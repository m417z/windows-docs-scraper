# SERCX_BUFFER_DESCRIPTOR_INIT function

## Description

The **SERCX_BUFFER_DESCRIPTOR_INIT** function initializes a [SERCX_BUFFER_DESCRIPTOR](https://learn.microsoft.com/windows-hardware/drivers/ddi/sercx/ns-sercx-sercx_buffer_descriptor) structure.

## Parameters

### `Descriptor` [out]

A pointer to the **SERCX_BUFFER_DESCRIPTOR** structure that is to be initialized.
