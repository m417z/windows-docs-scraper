# tagSMS_MESSAGE_TYPES enumeration

## Description

The **SMS_MESSAGE_TYPES** enumeration type describes the content type of a short message service (SMS) message.

## Constants

### `SMS_TEXT_MESSAGE`

A text message.

### `SMS_BINARY_MESSAGE`

A binary message.

## Remarks

This enumeration is used by the [WPD_COMMAND_SMS_SEND Command](https://learn.microsoft.com/windows/desktop/wpd_sdk/wpd-command-sms-send-command).
