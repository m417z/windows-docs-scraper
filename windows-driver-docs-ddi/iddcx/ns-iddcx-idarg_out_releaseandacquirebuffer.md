# IDARG_OUT_RELEASEANDACQUIREBUFFER structure

## Description

Gives information about the
acquired swap chain buffer.

## Members

### `MetaData`

[out] An [**IDDCX_METADATA**](https://learn.microsoft.com/windows-hardware/drivers/ddi/iddcx/ns-iddcx-iddcx_metadata) structure containing the metadata that accompanies the new buffer.
