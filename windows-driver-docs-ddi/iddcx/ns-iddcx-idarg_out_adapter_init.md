# IDARG_OUT_ADAPTER_INIT structure

## Description

**IDARG_OUT_ADAPTER_INIT** provides information about an initialized adapter.

## Members

### `AdapterObject`

An opaque [IDDCX_ADAPTER](https://learn.microsoft.com/windows-hardware/drivers/display/iddcx-objects) object that represents a single logical display adapter.

## Remarks

See [IDDCX objects](https://learn.microsoft.com/windows-hardware/drivers/display/iddcx-objects) for more information.
