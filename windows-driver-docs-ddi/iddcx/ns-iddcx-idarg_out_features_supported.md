# IDARG_OUT_FEATURES_SUPPORTED structure

## Description

Contains the IddCx 1.11 features that were requested.

## Members

### `Size`

[out] The total size of the structure.

### `Features_1_11`

[out] An [IDDCX_DEVICE_FEATURES_1_11](https://learn.microsoft.com/windows-hardware/drivers/ddi/iddcx/ne-iddcx-iddcx_device_features_1_11) enumeration indicating the IddCx 1.11 features supported by the OS.
