# IDARG_IN_REPORTFRAMESTATISTICS structure

## Description

Gives information about frame statistics.

## Members

### `FrameStatistics`

[in] Frame statistics being reported. See [**IDDCX_FRAME_STATISTICS**](https://learn.microsoft.com/windows-hardware/drivers/ddi/iddcx/ns-iddcx-iddcx_frame_statistics) for details on how a driver should fill this structure.
