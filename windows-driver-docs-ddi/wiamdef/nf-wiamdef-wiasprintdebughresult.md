# wiasPrintDebugHResult function

## Description

The **wiasPrintDebugHResult** function is obsolete for Windows XP and later, and is no longer supported. Use the [WIAS_LHRESULT](https://learn.microsoft.com/windows-hardware/drivers/ddi/wiamdef/nf-wiamdef-wias_lhresult) macro instead.

This function prints an HRESULT string on the Device Manager debug console.

## Parameters

### `hInstance`

The handle of the calling module.

### `hr`

Specifies the HRESULT to be printed.
