# DxgkUnInitialize function

## Description

Unloads the DirectX graphics kernel subsystem (Dxgkrnl.sys).

## Parameters

### `DriverObject`

A pointer to a [DRIVER_OBJECT](https://learn.microsoft.com/windows-hardware/drivers/ddi/wdm/ns-wdm-_driver_object) structure.

## Return value

This function returns NTSTATUS.
