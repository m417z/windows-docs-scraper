# DlFreePool function

## Description

Deallocates a block of pool memory previously allocated by the [DlAllocatePool](https://learn.microsoft.com/windows-hardware/drivers/ddi/dispmprt/nf-dispmprt-dlallocatepool) function.

## Parameters

### `DeviceHandle`

A handle that represents a display adapter.

### `Ptr`

Pointer to a memory pool.
