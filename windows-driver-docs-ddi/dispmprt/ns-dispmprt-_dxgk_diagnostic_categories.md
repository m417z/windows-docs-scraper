# _DXGK_DIAGNOSTIC_CATEGORIES structure

## Description

Structure with a bit-field for each defined category of diagnostic.

## Members

### `Notifications`

An OS defined category which has diagnostic types defined in the [DXGK_DIAGTYPE_NOTIFICATIONS](https://learn.microsoft.com/windows-hardware/drivers/ddi/dispmprt/ns-dispmprt-_dxgk_diagtype_notifications) structure.

### `Progressions`

A [DXGK_DIAGTYPE_PROGRESSIONS](https://learn.microsoft.com/windows-hardware/drivers/ddi/dispmprt/ns-dispmprt-_dxgk_diagtype_progressions) structure for indicating a Progressions diagnostic type.

### `Reserved`

Reserved.

### `Value`

Value of the categories.
