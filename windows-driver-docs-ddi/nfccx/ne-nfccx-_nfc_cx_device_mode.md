# _NFC_CX_DEVICE_MODE enumeration

## Description

Specifies device mode flags.

## Constants

### `NFC_CX_DEVICE_MODE_NCI`

Enable NCI mode.

### `NFC_CX_DEVICE_MODE_DTA`

Enable DTA mode.

### `NFC_CX_DEVICE_MODE_RAW`

Enable RAW mode.
