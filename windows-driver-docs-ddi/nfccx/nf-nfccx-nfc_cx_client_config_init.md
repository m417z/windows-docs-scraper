# NFC_CX_CLIENT_CONFIG_INIT function

## Description

The **NFC_CX_CLIENT_CONFIG_INIT** function initializes the [NFC_CX_CLIENT_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/nfccx/ns-nfccx-_nfc_cx_client_config) structure.

## Parameters

### `Config` [out]

A pointer to an [NFC_CX_CLIENT_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/nfccx/ns-nfccx-_nfc_cx_client_config) structure.

### `BusType` [in]

An [NFC_CX_TRANSPORT_TYPE](https://learn.microsoft.com/windows-hardware/drivers/ddi/nfccx/ne-nfccx-_nfc_cx_transport_type) enumeration.
