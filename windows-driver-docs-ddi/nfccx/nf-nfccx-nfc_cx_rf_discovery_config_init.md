# NFC_CX_RF_DISCOVERY_CONFIG_INIT function

## Description

The NFC_CX_RF_DISCOVERY_CONFIG_INIT function initializes the [NFC_CX_RF_DISCOVERY_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/nfccx/ns-nfccx-_nfc_cx_rf_discovery_config) structure.

## Parameters

### `Config` [out]

A pointer to the NFC_CX_RF_DISCOVERY_CONFIG structure that the function initializes.
