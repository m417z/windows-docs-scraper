# _NFC_CX_HARDWARE_EVENT structure

## Description

The NFC_CX_HARDWARE_EVENT structure is an input parameter to [NfcCxHardwareEvent.](https://learn.microsoft.com/windows-hardware/drivers/ddi/nfccx/nf-nfccx-nfccxhardwareevent)

## Members

### `HardwareStatus`

An [NTSTATUS](https://learn.microsoft.com/windows-hardware/drivers/kernel/ntstatus-values)  value.

### `HostAction`

An [NFC_CX_HOST_ACTION](https://learn.microsoft.com/windows-hardware/drivers/ddi/nfccx/ne-nfccx-_nfc_cx_host_action)-typed enumerator.
