# BTHHFP_AUDIO_DEVICE_CAPABILTIES_INIT function

## Description

The **BTHHFP_AUDIO_DEVICE_CAPABILTIES_INIT** method returns a pointer to an initialized [BTHHFP_AUDIO_DEVICE_CAPABILTIES](https://learn.microsoft.com/windows-hardware/drivers/ddi/bthhfpddi/ns-bthhfpddi-_bthhfp_audio_device_capabilties) data structure.

## Parameters

### `caps` [out]

A pointer to an initialized [BTHHFP_AUDIO_DEVICE_CAPABILTIES](https://learn.microsoft.com/windows-hardware/drivers/ddi/bthhfpddi/ns-bthhfpddi-_bthhfp_audio_device_capabilties) data structure.
