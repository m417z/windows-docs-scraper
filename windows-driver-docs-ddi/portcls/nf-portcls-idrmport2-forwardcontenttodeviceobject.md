# IDrmPort2::ForwardContentToDeviceObject

## Description

The `ForwardContentToDeviceObject` method accepts a device object representing a device to which the caller intends to forward protected content. Note that this method is identical in operation to the [DrmForwardContentToDeviceObject](https://learn.microsoft.com/windows-hardware/drivers/ddi/drmk/nf-drmk-drmforwardcontenttodeviceobject) function, and its parameter definitions and return value are also identical.
