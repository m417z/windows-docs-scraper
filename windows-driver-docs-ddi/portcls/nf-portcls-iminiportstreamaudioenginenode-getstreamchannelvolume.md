# IMiniportStreamAudioEngineNode::GetStreamChannelVolume

## Description

Gets the current volume level that is applied to the audio stream.

## Parameters

### `ulChannel` [in]

The audio stream channel.

### `plValue` [out]

The current volume level.

## Return value

**GetStreamChannelVolume** returns S_OK, if the call was successful. Otherwise, the method returns an appropriate error

code.
