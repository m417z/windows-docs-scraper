# NetAdapterGetNetLuid function

## Description

Retrieves the NET_LUID that is assigned to a network adapter.

## Parameters

### `Adapter` [_In_]

The network adapter object that the client created in a prior call to [NetAdapterCreate](https://learn.microsoft.com/windows-hardware/drivers/ddi/netadapter/nf-netadapter-netadaptercreate).

## Return value

Returns the NET_LUID for the specified NETADAPTER object.
