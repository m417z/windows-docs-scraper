# _BACKLIGHT_OPTIMIZATION_LEVEL enumeration

## Description

Specifies the type of backlight optimization.

## Constants

### `BacklightOptimizationDisable`

Optimization is disabled.

### `BacklightOptimizationDesktop`

Optimize for desktop.

### `BacklightOptimizationDynamic`

Dynamic optimization.

### `BacklightOptimizationDimmed`

Optimize for dimming.

### `BacklightOptimizationEDR`

Optimized for enhanced display response (EDR).
