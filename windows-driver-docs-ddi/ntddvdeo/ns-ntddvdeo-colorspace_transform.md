# COLORSPACE_TRANSFORM structure

## Description

Information about a colorspace transform.

## Members

### `Type`

The colorspace transform type.

### `Data`

The colorspace transform data.

### `Data.Rgb256x3x16`

RGB 256x3x16.

### `Data.Dxgi1`

Dxgi 1.

### `Data.T3x4`

Transform 3x4.

### `Data.MatrixV2`

Matrix v2.
