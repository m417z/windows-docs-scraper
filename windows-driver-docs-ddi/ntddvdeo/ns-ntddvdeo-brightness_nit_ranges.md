# BRIGHTNESS_NIT_RANGES structure

## Description

An array of [BRIGHTNESS_NIT_RANGE](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntddvdeo/ns-ntddvdeo-_brightness_nit_range) structures.

## Members

### `NormalRangeCount`

The normal range count.

### `RangeCount`

The range count.

### `PreferredMaximumBrightness`

The preferred maximum brightness.

### `SupportedRanges`

The array of supported ranges.
