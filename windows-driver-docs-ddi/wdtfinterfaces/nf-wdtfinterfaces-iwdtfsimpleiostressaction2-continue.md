# IWDTFSimpleIOStressAction2::Continue

## Description

Continues the I/O.

## Parameters

### `pResult` [out, retval]

True if the I/O operation succeeded; otherwise, false.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
