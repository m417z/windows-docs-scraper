# D3D11_1DDI_AUTHENTICATED_PROCESS_IDENTIFIER_TYPE enumeration

## Description

Specifies the type of process that is identified in the [D3D11_1DDI_AUTHENTICATED_QUERY_RESTRICTED_SHARED_RESOURCE_PROCESS_OUTPUT](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d10umddi/ns-d3d10umddi-d3d11_1ddi_authenticated_query_restricted_shared_resource_process_output) structure.

## Constants

### `D3D11_1DDI_PROCESSIDTYPE_UNKNOWN`

Unknown process type.

### `D3D11_1DDI_PROCESSIDTYPE_DWM`

DWM process.

### `D3D11_1DDI_PROCESSIDTYPE_HANDLE`

Handle to a process.
