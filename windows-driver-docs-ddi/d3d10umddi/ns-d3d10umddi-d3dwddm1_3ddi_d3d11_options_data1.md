# D3DWDDM1_3DDI_D3D11_OPTIONS_DATA1 structure

## Description

Specifies the level of support by the hardware and user-mode display driver for tiled resources.

## Members

### `TiledResourcesSupportFlags`

A value from the [D3DWDDM1_3DDI_TILED_RESOURCES_SUPPORT_FLAG](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d10umddi/ne-d3d10umddi-d3dwddm1_3ddi_tiled_resources_support_flag) enumeration that indicates the level of support for tiled resources.
