# D3D11_1DDIARG_CREATEVIDEODECODER structure

## Description

Specifies the attributes of a video decoder object.

## Members

### `Desc`

A [D3D11_1DDI_VIDEO_DECODER_DESC](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d10umddi/ns-d3d10umddi-d3d11_1ddi_video_decoder_desc) structure that describes the video stream and the decoder profile.

### `Config`

A [D3D11_1DDI_VIDEO_DECODER_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d10umddi/ns-d3d10umddi-d3d11_1ddi_video_decoder_config) structure that specifies the decoder configuration.
