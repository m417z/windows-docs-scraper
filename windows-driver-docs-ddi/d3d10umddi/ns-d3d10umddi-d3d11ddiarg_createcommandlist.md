# D3D11DDIARG_CREATECOMMANDLIST structure

## Description

The D3D11DDIARG_CREATECOMMANDLIST structure contains a handle to the deferred context that was created by the [CreateDeferredContext](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d10umddi/nc-d3d10umddi-pfnd3d11ddi_createdeferredcontext) function.

## Members

### `hDeferredContext` [in]

A handle to the deferred context.
