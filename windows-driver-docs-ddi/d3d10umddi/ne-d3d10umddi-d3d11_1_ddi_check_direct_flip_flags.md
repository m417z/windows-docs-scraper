# D3D11_1_DDI_CHECK_DIRECT_FLIP_FLAGS enumeration

## Description

Used by the *CheckDirectFlipFlags* parameter of the [CheckDirectFlipSupport(D3D11_1)](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d10umddi/nc-d3d10umddi-pfnd3d11_1ddi_checkdirectflipsupport) function to specify seamless flipping of video memory.

## Constants

### `D3D11_1DDI_CHECK_DIRECT_FLIP_IMMEDIATE`

Seamless flipping should occur immediately and does not have to be synchronized with a VSync interrupt.
