# PFND3DWDDM2_2DDI_SHADERCACHE_ADDREF_RELEASE_CB callback function

## Description

The *pfnShaderCacheAddRefCb* callback function supports the ability to extend the lifetime of a shader cache.

## Parameters

### `hCacheSession`

The handler of a cache session.

## Remarks

Access this callback function by using the [D3DWDDM2_2DDI_CORELAYER_DEVICECALLBACKS](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d10umddi/ns-d3d10umddi-d3dwddm2_2ddi_corelayer_devicecallbacks) structure.
