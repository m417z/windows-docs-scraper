# PFND3DWDDM2_2DDI_RELOCATEDEVICEFUNCS callback function

## Description

The *pfnRelocateDeviceFuncs* callback function specifies the device functions table.

## Parameters

### `unnamedParam1`

*hDevice*

A handle to the display device (graphics context).

### `unnamedParam2`

*DeviceFunctions* [in]

The device functions table, as a [D3DWDDM2_2DDI_DEVICEFUNCS](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d10umddi/ns-d3d10umddi-d3dwddm2_2ddi_devicefuncs) structure.
