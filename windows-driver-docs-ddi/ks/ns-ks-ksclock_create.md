# KSCLOCK_CREATE structure

## Description

The KSCLOCK_CREATE structure is used in clock create parameters for the [KsCreateClock](https://learn.microsoft.com/windows-hardware/drivers/ddi/ks/nf-ks-kscreateclock) function.

## Members

### `CreateFlags`

Currently consists of a flags item, which must be set to zero.
