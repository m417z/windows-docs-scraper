# IoSizeofWorkItem function

## Description

The **IoSizeofWorkItem** routine returns the size, in bytes, of an [IO_WORKITEM](https://learn.microsoft.com/windows-hardware/drivers/kernel/eprocess) structure.

## Return value

**IoSizeofWorkItem** returns the number of bytes necessary to hold an [IO_WORKITEM](https://learn.microsoft.com/windows-hardware/drivers/kernel/eprocess) structure.

## Remarks

For more information about work items, see [System Worker Threads](https://learn.microsoft.com/windows-hardware/drivers/kernel/system-worker-threads).
