## Description

The **PsFreeAffinityToken** function frees an affinity token that was allocated by [**PsAllocateAffinityToken**](https://learn.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-psallocateaffinitytoken).

## Parameters

### `AffinityToken`

Supplies a pointer to an affinity token.
