# _FILE_IS_REMOTE_DEVICE_INFORMATION structure

## Description

The **FILE_IS_REMOTE_DEVICE_INFORMATION** structure is used as an argument to the [ZwQueryInformationFile](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntifs/nf-ntifs-ntqueryinformationfile) routine.

## Members

### `IsRemote`

A value that indicates whether the file system that contains the file is a remote file system.
