# _FAULT_INFORMATION_ARCH enumeration

## Description

Specified in the [_FAULT_INFORMATION](https://learn.microsoft.com/windows-hardware/drivers/ddi/wdm/ns-wdm-_fault_information) structure.

## Constants

### `FaultInformationInvalid`

The architecture type is not valid.

### `FaultInformationArm64`

Specifies ARM64 architecture.

### `FaultInformationX64`

Specifies x64 architecture.
