# _CM_MCA_POS_DATA structure (wdm.h)

## Description

The **CM_MCA_POS_DATA** structure is obsolete. It defines IBM-compatible MCA POS configuration information for a slot.

## Members

### `AdapterId`

### `PosData1`

### `PosData2`

### `PosData3`

### `PosData4`

## Remarks

This structure is used by the obsolete [HalGetBusData](https://learn.microsoft.com/previous-versions/windows/hardware/drivers/ff546644(v=vs.85)) and [HalGetBusDataByOffset](https://learn.microsoft.com/previous-versions/windows/hardware/drivers/ff546644(v=vs.85)) routines.
