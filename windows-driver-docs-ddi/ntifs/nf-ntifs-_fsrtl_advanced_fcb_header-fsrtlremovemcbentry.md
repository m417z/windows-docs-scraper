# FsRtlRemoveMcbEntry function

## Description

The **FsRtlRemoveMcbEntry** routine is obsolete, but is exported to support existing driver binaries. Use [FsRtlRemoveLargeMcbEntry](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntifs/nf-ntifs-_fsrtl_advanced_fcb_header-fsrtlremovelargemcbentry) instead.

## Parameters

### `Mcb`

Reserved.

### `Vbn`

Reserved.

### `SectorCount`

Reserved.
