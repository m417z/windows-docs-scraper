# FsRtlReleaseFile function

The **FsRtlReleaseFile** routine is reserved for system use. See [FsRtlRegisterFileSystemFilterCallbacks](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntifs/nf-ntifs-fsrtlregisterfilesystemfiltercallbacks).
