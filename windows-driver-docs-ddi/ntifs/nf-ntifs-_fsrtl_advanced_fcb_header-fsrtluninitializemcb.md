# FsRtlUninitializeMcb function

## Description

The **FsRtlUninitializeMcb** routine is obsolete, but is exported to support existing driver binaries. Use [FsRtlUninitializeLargeMcb](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntifs/nf-ntifs-_fsrtl_advanced_fcb_header-fsrtluninitializelargemcb) instead.

## Parameters

### `Mcb`

Reserved.
