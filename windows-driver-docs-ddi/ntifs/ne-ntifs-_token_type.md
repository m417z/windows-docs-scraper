# TOKEN_TYPE enumeration

## Description

The **TOKEN_TYPE** enumeration type contains values that differentiate between a primary token and an impersonation token.

## Constants

### `TokenPrimary`

Indicates a primary token.

### `TokenImpersonation`

Indicates an impersonation token.
