# IoQueueThreadIrp function

## Description

The **IoQueueThreadIrp** routine is reserved for system use. See [IoBuildDeviceIoControlRequest](https://learn.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-iobuilddeviceiocontrolrequest) and [IoBuildSynchronousFsdRequest](https://learn.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-iobuildsynchronousfsdrequest).

## Parameters

### `Irp`

Reserved.
