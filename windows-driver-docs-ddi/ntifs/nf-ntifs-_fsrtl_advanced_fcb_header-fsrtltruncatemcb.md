# FsRtlTruncateMcb function

## Description

The **FsRtlTruncateMcb** routine is obsolete, but is exported to support existing driver binaries. Use [FsRtlTruncateLargeMcb](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntifs/nf-ntifs-_fsrtl_advanced_fcb_header-fsrtltruncatelargemcb) instead.

## Parameters

Reserved.

### `Vbn`

Reserved.
