## Description

The **LMR_QUERY_INFO_CLASS** enumeration specifies the type of information to query.

## Constants

### `LMRQuerySessionInfo:1`

Query the session ID information for the file or directory. The information is returned in an [**LMR_QUERY_SESSION_INFO**](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntifs/ns-ntifs-lmr_query_session_info) structure via **OutputBuffer**.
