# FsRtlAcquireFileExclusive function

## Description

The **FsRtlAcquireFileExclusive** routine is reserved for system use. Do not use this routine in your driver. See [FsRtlRegisterFileSystemFilterCallbacks](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntifs/nf-ntifs-fsrtlregisterfilesystemfiltercallbacks).

## Parameters

### `FileObject`

Reserved.
