## Description

The **CC_FILE_SIZES** structure contains file size information.

## Members

### `AllocationSize`

The section object size for the file, in bytes.

### `FileSize`

The size of the file, in bytes.

### `ValidDataLength`

The valid date length for the file, in bytes.
