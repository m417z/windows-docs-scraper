# __MIDL___MIDL_itf_keyworddetectoroemadapter_0000_0000_0001 structure (keyworddetectoroemadapter.h)

## Description

The **SOUNDDETECTOR_PATTERNHEADER** structure specifies the pattern header for the sound detector in the [KSPROPERTY_SOUNDDETECTOR_PATTERNS](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-sounddetector-patterns) property.

## Members

### `Size`

The size of the audio data.

### `PatternType`

The keyword pattern format, expressed as a GUID.
