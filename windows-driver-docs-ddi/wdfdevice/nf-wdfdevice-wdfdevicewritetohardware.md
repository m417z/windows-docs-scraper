# WdfDeviceWriteToHardware function

## Description

The **WdfDeviceWriteToHardware** method is used internally by the framework. Do not use.

Instead, use the [WDF Register/Port Access Functions](https://learn.microsoft.com/windows-hardware/drivers/ddi/wdfhwaccess/).
