# _UCMTCPCI_PORT_CONTROLLER_GET_CONTROL_IN_PARAMS structure

## Description

 This structure is used in the
[IOCTL_UCMTCPCI_PORT_CONTROLLER_GET_CONTROL](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmtcpciportcontrollerrequests/ni-ucmtcpciportcontrollerrequests-ioctl_ucmtcpci_port_controller_get_control) request.

## Members

### `PortControllerObject`

Handle to the port controller object that the client driver received in the previous call to [UcmTcpciPortControllerCreate](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmtcpciportcontroller/nf-ucmtcpciportcontroller-ucmtcpciportcontrollercreate).
