# UCM_PD_POWER_DATA_OBJECT_INIT_BATTERY function

## Description

Initializes a [UCM_PD_POWER_DATA_OBJECT](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmtypes/ns-ucmtypes-_ucm_pd_power_data_object) structure as a Battery Supply type Power Data Object.

## Parameters

### `Pdo` [out]

A pointer to a [UCM_PD_POWER_DATA_OBJECT](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmtypes/ns-ucmtypes-_ucm_pd_power_data_object) structure in which the **BatterySupplyPdo.Battery** member is set to **UcmPdPdoTypeBatterySupply**.
