## Description

This enumeration defines values used in the [PosDeviceBasicsType](https://learn.microsoft.com/windows-hardware/drivers/ddi/pointofservicedriverinterface/ns-pointofservicedriverinterface-_posdevicebasicstype) structure to indicate the type of device (for instance, barcode scanner or magnetic stripe reader).

## Constants

### `PosDeviceType_Unknown`

Indicates that the type of device is not known.

### `PosDeviceType_BarcodeScanner`

Indicates that the type of device is a barcode scanner.

### `PosDeviceType_MagneticStripeReader`

Indicates that the type of device is a magnetic stripe reader.

### `PosDeviceType_Max`

Unused.
