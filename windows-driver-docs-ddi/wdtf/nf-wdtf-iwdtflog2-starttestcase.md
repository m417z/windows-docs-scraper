# IWDTFLOG2::StartTestCase

## Description

Marks the start of a test case.

## Parameters

### `sTestCaseName` [in]

The test case name.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
