# IWDTFAction2::GetStatus

## Description

Returns the status code for the last operation.

## Parameters

### `pResult` [out, retval]

The status code for the last operation.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
