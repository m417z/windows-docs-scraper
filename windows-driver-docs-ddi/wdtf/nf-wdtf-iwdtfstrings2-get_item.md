# IWDTFStrings2::get_Item

## Description

Gets an individual string in the collection.

This property is read-only.

## Parameters

### `Index`

### `pString`

The string.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
