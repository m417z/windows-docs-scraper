# IWDTFTarget2::Eval

## Description

Evaluate whether this target matches an SDEL statement.

## Parameters

### `SDEL` [in]

The SDEL statement.

### `pResult` [out, retval]

True if the SDEL statement matches this target;
otherwise, false.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
