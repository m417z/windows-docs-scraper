# IWDTFLOG2::OutputError

## Description

Writes an error entry to the test case log.

## Parameters

### `sErrorString` [in]

The error string.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
