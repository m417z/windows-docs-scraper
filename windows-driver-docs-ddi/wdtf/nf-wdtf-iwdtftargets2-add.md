# IWDTFTargets2::Add

## Description

Add a single item to the collection.

## Parameters

### `pTarget` [in]

The item to add to this collection.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
