# IWDTFSystemDepot2::get_ThisSystem

## Description

Gets an [IWDTFTarget2](https://learn.microsoft.com/windows-hardware/drivers/ddi/wdtf/nn-wdtf-iwdtftarget2) value that represents the local computer.

This property is read-only.
