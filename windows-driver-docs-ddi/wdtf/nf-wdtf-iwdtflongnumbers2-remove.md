# IWDTFLongNumbers2::Remove

## Description

Removes a long number from the collection.

## Parameters

### `Index` [in]

The 0-based index value of the number to remove.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
