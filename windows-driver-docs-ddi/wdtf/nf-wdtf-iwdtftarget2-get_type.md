# IWDTFTarget2::get_Type

## Description

Gets a value that identifies the depot that the target comes from.

This property is read-only.

## Parameters

### `pVal`

### Return value

## Remarks

The value can be one of the following:

* **Device** if the target comes from the
  [IWDTFDeviceDepot2](https://learn.microsoft.com/windows-hardware/drivers/ddi/wdtf/nn-wdtf-iwdtfdevicedepot2) interface.
* **System** if the target comes from the
  [IWDTFSystemDepot2](https://learn.microsoft.com/windows-hardware/drivers/ddi/wdtf/nn-wdtf-iwdtfsystemdepot2) interface.
