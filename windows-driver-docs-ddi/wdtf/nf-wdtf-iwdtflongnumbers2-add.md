# IWDTFLongNumbers2::Add

## Description

Adds a single long number to the collection.

## Parameters

### `Number` [in]

The long number to add to this collection.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
