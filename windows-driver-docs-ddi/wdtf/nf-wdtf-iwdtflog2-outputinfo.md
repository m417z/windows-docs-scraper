# IWDTFLOG2::OutputInfo

## Description

Writes an informational entry to the test case log.

## Parameters

### `sInfoString` [in]

The information string.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
