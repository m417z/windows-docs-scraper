# IWDTFStrings2::Remove

## Description

Removes a string from the collection.

## Parameters

### `Index` [in]

The 0-based index value of the string to remove.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
