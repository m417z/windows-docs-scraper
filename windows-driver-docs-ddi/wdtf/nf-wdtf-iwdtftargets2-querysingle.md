# IWDTFTargets2::QuerySingle

## Description

Returns a single item from the collection.

## Parameters

### `SDEL` [in]

The SDEL query string.

### `ppTarget` [out, retval]

The requested item.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
