# IDebugBreakpoint3::GetGuid

## Description

Returns a GUID for the breakpoint.

## Parameters

### `Guid` [out]

A unique ID returned for the breakpoint.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
