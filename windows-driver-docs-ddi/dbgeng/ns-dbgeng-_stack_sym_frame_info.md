# _STACK_SYM_FRAME_INFO structure

## Description

Defines stack source information for an extended stack frame.

## Members

### `StackFrameEx`

A stack frame as a [DEBUG_STACK_FRAME_EX](https://learn.microsoft.com/windows-hardware/drivers/ddi/dbgeng/ns-dbgeng-_debug_stack_frame_ex) structure.

### `SrcInfo`

Stack source information as a [STACK_SRC_INFO](https://learn.microsoft.com/windows-hardware/drivers/ddi/dbgeng/ns-dbgeng-_stack_src_info) structure.
