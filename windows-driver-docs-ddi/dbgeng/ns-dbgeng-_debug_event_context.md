# _DEBUG_EVENT_CONTEXT structure

## Description

Defines context information about an event.

## Members

### `Size`

The size of the event.

### `ProcessEngineId`

The process engine ID of the event.

### `ThreadEngineId`

The process thread ID of the event.

### `FrameEngineId`

The frame engine ID of the event.
