# _DEBUG_CLIENT_CONTEXT structure

## Description

Contains a debug client constant to provide to the [IDebugClient7::SetClientContext](https://learn.microsoft.com/windows-hardware/drivers/ddi/dbgeng/nf-dbgeng-idebugclient7-setclientcontext) method.

## Members

### `cbSize`

A size value.

### `eClient`

A client value.
