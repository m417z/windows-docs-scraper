# IDebugOutputStream::Write

## Description

Writes to the debug output stream.

## Parameters

### `psz` [in]

A pointer to a Unicode character string of content to write.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
