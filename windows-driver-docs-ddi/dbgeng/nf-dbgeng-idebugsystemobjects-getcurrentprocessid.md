# IDebugSystemObjects::GetCurrentProcessId

## Description

The **GetCurrentProcessId** method returns the engine process ID for the current process.

## Parameters

### `Id` [out]

Receives the engine process ID for the current process.

## Return value

This method may also return error values. See [Return Values](https://learn.microsoft.com/windows-hardware/drivers/debugger/hresult-values) for more details.

| Return code | Description |
| --- | --- |
| **S_OK** | The method was successful. |

## Remarks

For more information about processes, see [Threads and Processes](https://learn.microsoft.com/windows-hardware/drivers/debugger/threads-and-processes).
