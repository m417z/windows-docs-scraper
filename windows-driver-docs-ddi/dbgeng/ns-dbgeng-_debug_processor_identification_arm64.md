# _DEBUG_PROCESSOR_IDENTIFICATION_ARM64 structure

## Description

Identifies an ARM64 processor.

## Members

### `Model`

The model of the processor.

### `Revision`

The revision of the processor.

### `VendorString`

A vendor specified string.
