## Description

Changes the selected option of the Print Schema Feature element to the specified [IPrintSchemaOption](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprintschemaoption) element.

This property is write-only.

## Parameters

### `pOption`

Defines the **IPrintSchemaOption** parameter *pOption*.

## Return value

Returns an **HRESULT** value. If the property call was not successful, it returns the appropriate **HRESULT** error code.
