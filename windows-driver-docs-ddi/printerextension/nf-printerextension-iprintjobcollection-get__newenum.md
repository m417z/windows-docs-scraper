## Description

Gets a pointer to the enumerants of [IPrintJobCollection](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprintjobcollection) objects.

This property is read-only.

## Parameters

### `ppUnk`

Defines the **IUnknown** parameter *ppUnk*.

## Return value

Returns an **HRESULT** value. If the property call was not successful, it returns the appropriate **HRESULT** error code.
