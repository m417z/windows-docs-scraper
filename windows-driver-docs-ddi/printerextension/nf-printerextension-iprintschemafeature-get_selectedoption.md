## Description

Gets an [IPrintSchemaOption](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprintschemaoption) representing the selected option.

This property is read-only.

## Parameters

### `ppOption`

Defines the **IPrintSchemaOption** parameter *ppOption*.

## Return value

Returns an **HRESULT** value. If the property call was not successful, it returns the appropriate **HRESULT** error code.
