## Description

The **DataType** property gets the [PrintSchemaParameterDataType](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/ne-printerextension-tagprintschemaparameterdatatype) enumerated value that indicates the expected data type for the Print Schema parameter.

This property is read-only.

## Parameters

### `pDataType`

Defines the **PrintSchemaParameterDataType** parameter *pDataType*.

## Return value

Returns an **HRESULT** value. If the property call was not successful, it returns the appropriate **HRESULT** error code.
