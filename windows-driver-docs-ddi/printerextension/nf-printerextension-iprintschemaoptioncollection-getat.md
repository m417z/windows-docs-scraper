## Description

Gets a pointer to an [IPrintSchemaOption](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprintschemaoption) object.

## Parameters

### `ulIndex` [in]

Index of the **IPrintSchemaOption** object within the collection.

### `ppOption` [out, retval, optional]

Pointer to an [IPrintSchemaOption](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprintschemaoption) object.

## Return value

Returns an **HRESULT** value. If the method call was not successful, it returns the appropriate **HRESULT** error code.
