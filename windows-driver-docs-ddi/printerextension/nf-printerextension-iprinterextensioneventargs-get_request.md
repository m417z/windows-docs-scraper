## Description

Gets the [IPrinterExtensionRequest](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprinterextensionrequest) object for the current event.

This interface is used to complete or cancel the event.

This property is read-only.

## Parameters

### `ppRequest`

Defines the **IPrinterExtensionRequest** parameter *ppRequest*.

## Return value

Returns an **HRESULT** value. If the property call was not successful, it returns the appropriate **HRESULT** error code.
