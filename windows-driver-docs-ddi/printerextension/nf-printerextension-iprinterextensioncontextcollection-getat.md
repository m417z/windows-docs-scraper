## Description

Gets a pointer to an [IPrinterExtensionContext](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprinterextensioncontext) object.

## Parameters

### `ulIndex` [in]

The index of the [IPrinterExtensionContext](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprinterextensioncontext) object within the collection.

### `ppContext` [out, retval]

Pointer to an [IPrinterExtensionContext](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprinterextensioncontext) interface.

## Return value

This method returns an **HRESULT** value.
