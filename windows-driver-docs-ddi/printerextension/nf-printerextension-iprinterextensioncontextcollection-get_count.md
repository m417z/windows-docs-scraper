## Description

Gets a count of the number of [IPrinterExtensionContext](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprinterextensioncontext) objects in the collection.

This property is read-only.

## Parameters

### `pulCount`

Defines the **ULONG** parameter *pulCount*.

## Return value

Returns an **HRESULT** value. If the property call was not successful, it returns the appropriate **HRESULT** error code.
