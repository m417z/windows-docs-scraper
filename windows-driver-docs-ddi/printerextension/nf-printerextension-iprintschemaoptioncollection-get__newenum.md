## Description

Gets a pointer to the enumerants of [**IPrintSchemaOptionCollection**](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprintschemaoptioncollection) objects.

This property is read-only.

## Parameters

### `ppUnk`

Pointer to object of type **IUnknown**.

## Return value

Returns an **HRESULT** value. If the property call was not successful, it returns the appropriate **HRESULT** error code.
