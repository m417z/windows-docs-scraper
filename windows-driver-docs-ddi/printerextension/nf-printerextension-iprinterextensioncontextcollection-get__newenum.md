## Description

Gets a pointer to the enumerants of [IPrinterExtensionContextCollection](https://learn.microsoft.com/windows-hardware/drivers/ddi/printerextension/nn-printerextension-iprinterextensioncontextcollection) objects.

## Parameters

### `ppUnk` [out, retval]

Defines the **IUnknown** parameter *ppUnk*.

## Return value

Returns an **HRESULT** value. If the property call was not successful, it returns the appropriate **HRESULT** error code.
