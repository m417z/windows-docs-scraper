## Description

The **PKTMON_NETEVT_PROVIDER_DISPATCH** structure contains a Packet Monitor NetEvt NMR provider dispatch table.

## Members

### `Header`

The [PKTMON_NETEVT_DISPATCH_HEADER](https://learn.microsoft.com/windows-hardware/drivers/ddi/pktmonnpik/ns-pktmonnpik-pktmon_netevt_dispatch_header) structure for this dispatch table.

### `HandlerCount`

The number of event handlers.

### `Handlers`

Pointer to the event handlers.
