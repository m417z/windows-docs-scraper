# _NDK_OBJECT_HEADER_RESERVED_BLOCK structure

## Description

The **NDK_OBJECT_HEADER_RESERVED_BLOCK** structure specifies reserved information in an NDK object.

## Members

### `rf`

Reserved.

## Remarks

The **NDK_OBJECT_HEADER_RESERVED_BLOCK** structure is used in the [NDK_OBJECT_HEADER](https://learn.microsoft.com/windows-hardware/drivers/ddi/ndkpi/ns-ndkpi-_ndk_object_header) structure.
