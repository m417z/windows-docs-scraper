# ExtRemoteTyped::GetSimpleValue method

## Description

The GetSimpleValue method is part of the **ExtRemoteTyped** class. The **ExtRemoteTyped** class provides the ability to manipulate typed data on the target. An instance of this class represents a small region of memory on the target. This region is interpreted as a specific type. This class provides methods for manipulating the memory according to the type and for accessing the object hierarchy on the target.
