# ExtRemoteTyped::OutSimpleValue

## Description

The **OutSimpleValue** method prints the value of the typed data represented by this object.

## Remarks

The **OutSimpleValue** method does not print as much detail as the [ExtRemoteTyped::OutFullValue](https://learn.microsoft.com/windows-hardware/drivers/ddi/engextcpp/nf-engextcpp-extremotetyped-outfullvalue) method.

The value is sent to the debugger engine's output callbacks.

### Returns

This method does not return a value.
