# IPartBase::SetPartCompression

## Description

The **SetPartCompression** method sets the compression of the part.

## Parameters

### `compression` [in]

A [ExpsCompressionOptions](https://learn.microsoft.com/windows-hardware/drivers/ddi/filterpipeline/ne-filterpipeline-__midl___midl_itf_filterpipeline_0000_0000_0001)-typed value that describes the compression option for the part.

## Return value

**SetPartCompression** returns an **HRESULT** value.
