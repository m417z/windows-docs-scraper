# IPartFont::SetFontOptions

## Description

The **SetFontOptions** method sets the options for the font.

## Parameters

### `options` [in]

An [ExpsFontOptions](https://learn.microsoft.com/windows-hardware/drivers/ddi/filterpipeline/ne-filterpipeline-__midl___midl_itf_filterpipeline_0000_0000_0002)-typed value that describes the options for the font.

## Return value

**SetFontOptions** returns an **HRESULT** value.
