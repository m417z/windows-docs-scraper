## Description

The **NDIS_KDNET_PF_USAGE_TAG** enumeration defines the PF usage. It is used to indicate the entity that owns the PF.
It is used by [NDIS_KDNET_QUERY_PF_INFORMATION](https://learn.microsoft.com/windows-hardware/drivers/ddi/kdnetpf/ns-kdnetpf-ndis_kdnet_query_pf_information).

## Constants

### `NdisKdnetPfUsageUnknown`

The usage is unknown.

### `NdisKdnetPfUsageKdModule`

The usage is a KdModule.
