## Description

The **NDIS_KDNET_BDF** structure describes the PCI location of the port to query.

## Members

### `SegmentNumber`

The PCI SegmentNumber.

### `BusNumber`

The PCI BusNumber.

### `DeviceNumber`

The PCI DeviceNumber.

### `FunctionNumber`

The PCI Function Number.

### `Reserved`

This field is reserved. Do not use.
