# UCM_MANAGER_CONFIG_INIT function

## Description

Initializes a [UCM_MANAGER_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmmanager/ns-ucmmanager-_ucm_manager_config) structure.

## Parameters

### `Config` [out]

Pointer to a caller-allocated [UCM_MANAGER_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmmanager/ns-ucmmanager-_ucm_manager_config) structure to initialize.
