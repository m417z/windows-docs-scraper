# _UCM_MANAGER_CONFIG structure

## Description

Describes the configuration options for the UCM Manager. An initialized **UCM_MANAGER_CONFIG** structure is an input parameter value to [UcmInitializeDevice](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmmanager/nf-ucmmanager-ucminitializedevice).

## Members

### `Size`

Size of the **UCM_MANAGER_CONFIG** structure. Initialize this structure by calling [UCM_MANAGER_CONFIG_INIT](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmmanager/nf-ucmmanager-ucm_manager_config_init).
