# _D3DKMT_QUERY_MIRACAST_DRIVER_TYPE structure

## Description

Used to query for a Miracast (wireless display) driver type.

## Members

### `MiracastDriverType`

Pointer to a [D3DKMT_MIRACAST_DRIVER_TYPE](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ne-d3dkmthk-_d3dkmt_miracast_driver_type) that specifies the driver type.
