# D3DKMTGetProcessDeviceRemovalSupport function

## Description

Used to get process device removal support.

## Parameters

### `unnamedParam1`

Pointer to a [D3DKMT_GETPROCESSDEVICEREMOVALSUPPORT](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-_d3dkmt_getprocessdeviceremovalsupport) structure.

## Return value

This function returns **NTSTATUS**.
