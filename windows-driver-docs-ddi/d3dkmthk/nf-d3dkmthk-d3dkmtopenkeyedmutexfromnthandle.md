# D3DKMTOpenKeyedMutexFromNtHandle function

## Description

Opens a keyed mutex from an NT handle to the process.

## Parameters

### `unnamedParam1`

*D3DKMT_OPENKEYEDMUTEXFROMNTHANDLE* [in, out]

Pointer to a [D3DKMT_OPENKEYEDMUTEXFROMNTHANDLE](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-_d3dkmt_openkeyedmutexfromnthandle) structure.

## Return value

This function returns NTSTATUS.
