# _D3DKMT_CLOSEADAPTER structure

## Description

The D3DKMT_CLOSEADAPTER structure specifies the adapter to close.

## Members

### `hAdapter`

*\_In\_* *D3DKMT\_HANDLE* *hAdapter*

A handle to the graphics adapter to close.
