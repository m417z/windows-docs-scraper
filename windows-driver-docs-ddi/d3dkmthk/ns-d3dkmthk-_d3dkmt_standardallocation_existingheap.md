# D3DKMT_STANDARDALLOCATION_EXISTINGHEAP structure

## Description

The **D3DKMT_STANDARDALLOCATION_EXISTINGHEAP** structure holds information about an existing heap.

## Members

### `Size`

Size in bytes of the heap.
