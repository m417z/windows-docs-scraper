# D3DKMTSetFSEBlock function

## Description

Sets FSE block.

## Parameters

### `unnamedParam1`

Pointer to a [D3DKMT_SETFSEBLOCK](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-_d3dkmt_setfseblock) structure.

## Return value

This function returns NTSTATUS.
