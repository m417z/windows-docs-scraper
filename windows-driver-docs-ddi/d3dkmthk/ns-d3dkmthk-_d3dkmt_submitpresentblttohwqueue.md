# _D3DKMT_SUBMITPRESENTBLTTOHWQUEUE structure

## Description

Contains arguments needed for the [D3DKMTSubmitPresentBltToHwQueue](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/nf-d3dkmthk-d3dkmtsubmitpresentblttohwqueue) function.

## Members

### `hHwQueue`

Handle to the created queue.

### `HwQueueProgressFenceId`

Handle to the hardware queue progress fence id.

### `PrivatePresentData`

The private present data.
