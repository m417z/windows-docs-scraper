# D3DKMTCheckMultiPlaneOverlaySupport3 function

## Description

The **D3DKMTCheckMultiPlaneOverlaySupport3** function checks details on support for multiplane overlays.

## Parameters

### `unnamedParam1`

Pointer to a [**D3DKMT_CHECKMULTIPLANEOVERLAYSUPPORT3**](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-_d3dkmt_checkmultiplaneoverlaysupport3) structure.

## Return value

This function returns NTSTATUS.

## Remarks

See [Multiplane overlay support](https://learn.microsoft.com/windows-hardware/drivers/display/multiplane-overlay-support) for more information.
