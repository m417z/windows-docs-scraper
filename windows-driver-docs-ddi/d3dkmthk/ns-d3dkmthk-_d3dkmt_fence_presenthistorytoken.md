# _D3DKMT_FENCE_PRESENTHISTORYTOKEN structure

## Description

The D3DKMT_FENCE_PRESENTHISTORYTOKEN structure identifies a fence present-history operation.

## Members

### `Key` [in]

A 64-bit value that specifies the key for the fence.
