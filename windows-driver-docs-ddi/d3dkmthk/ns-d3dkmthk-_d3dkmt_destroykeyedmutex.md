# _D3DKMT_DESTROYKEYEDMUTEX structure

## Description

The D3DKMT_DESTROYKEYEDMUTEX structure describes a keyed mutex that the [D3DKMTDestroyKeyedMutex](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/nf-d3dkmthk-d3dkmtdestroykeyedmutex) function destroys.

## Members

### `hKeyedMutex` [in]

A D3DKMT_HANDLE data type that represents a kernel-mode handle to the keyed mutex object to destroy.
