# _D3DKMT_GETRUNTIMEDATA structure

## Description

Used to get runtime data.

## Members

### `hAdapter` [in]

A handle to the graphics adapter.

### `hGlobalShare` [in]

A handle to a global share.

### `pRuntimeData` [in, out]

Pointer to runtime data.

### `RuntimeDataSize` [in]

The runtime data size.
