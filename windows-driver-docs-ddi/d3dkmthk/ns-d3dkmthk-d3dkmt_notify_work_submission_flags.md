## Description

The **D3DKMT_NOTIFY_WORK_SUBMISSION_FLAGS** structure specifies work submission notification flags.

## Members

### `Reserved`

Reserved for system use.

### `Value`

Value used to operate over this structure's members collectively.

## Remarks

For more information, see [User-mode work submission](https://learn.microsoft.com/windows-hardware/drivers/display/user-mode-work-submission).
