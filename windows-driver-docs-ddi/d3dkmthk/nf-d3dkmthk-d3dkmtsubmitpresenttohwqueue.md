# D3DKMTSubmitPresentToHwQueue function

## Description

> [NOTE!]
> This function has been deprecated, and replaced with the D3DKMTSubmitPresentBltToHwQueue function.

SubmitPresentToHwQueue submits a Blt Present operation to a hardware queue.

## Parameters

### `unnamedParam1` [in,out]

Pointer to a [D3DKMT_SUBMITPRESENTTOHWQUEUE](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-_d3dkmt_submitpresenttohwqueue) structure.

## Return value

This function returns NTSTATUS.
