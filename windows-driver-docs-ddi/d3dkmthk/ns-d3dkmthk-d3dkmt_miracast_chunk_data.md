# D3DKMT_MIRACAST_CHUNK_DATA structure

## Description

Contains wireless display (Miracast) chunk data.

## Members

### `ChunkInfo`

A [DXGK_MIRACAST_CHUNK_INFO](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dukmdt/ns-d3dukmdt-dxgk_miracast_chunk_info) structure.

### `PrivateDriverDataSize`

The size of the private data.

### `PrivateDriverData`
The private data buffer.
