# _D3DKMT_ACTIVATE_SPECIFIC_DIAG_TYPE enumeration

## Description

Indicates the specific diagonal type to activate.

## Constants

### `D3DKMT_ACTIVATE_SPECIFIC_DIAG_TYPE_EXTRA_CCD_DATABASE_INFO`

Extra CCD (Canonical Display Driver) database info.

### `D3DKMT_ACTIVATE_SPECIFIC_DIAG_TYPE_MODES_PRUNED`

Pruned modes. This value must be the same as *D3DKMT_ESCAPE_MODES_PRUNED_OUT(15)* in [D3DKMT_ESCAPE](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-_d3dkmt_escape), in order to maintain binary compatibility with Windows 8.
