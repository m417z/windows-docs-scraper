# D3DKMTQueryFSEBlock function

## Description

Query FSE block.

## Parameters

### `unnamedParam1`

Pointer to a [D3DKMT_QUERYFSEBLOCK](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-_d3dkmt_queryfseblock) structure.

## Return value

This function returns NTSTATUS.
