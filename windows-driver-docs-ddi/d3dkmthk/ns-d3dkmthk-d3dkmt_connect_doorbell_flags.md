## Description

The **D3DKMT_CONNECT_DOORBELL_FLAGS** structure specifies flags for [connecting a doorbell](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/nf-d3dkmthk-d3dkmtconnectdoorbell).

## Members

### `Reserved`

Reserved for system use.

### `Value`

An alternative way to access the structure members.

## Remarks

For more information, see [User-mode work submission](https://learn.microsoft.com/windows-hardware/drivers/display/user-mode-work-submission).
