# D3DKMT_MULTIPLANE_OVERLAY structure

## Description

The **D3DKMT_MULTIPLANE_OVERLAY** structure contains information that is used to define a multiplane overlay.

## Members

### `LayerIndex`

The layer index.

### `Enabled`

Boolean value that specifies whether the overlay is enabled for display.

### `hAllocation`

A handle to a device allocation.

### `PlaneAttributes`

A [**D3DKMT_MULTIPLANE_OVERLAY_ATTRIBUTES**](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-d3dkmt_multiplane_overlay_attributes) structure that contains the plane's attributes.
