# D3DKMTSubmitPresentBltToHwQueue function

## Description

Submits a present Blt to hardware queue.

## Parameters

### `unnamedParam1`

A [D3DKMT_SUBMITPRESENTBLTTOHWQUEUE](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-_d3dkmt_submitpresentblttohwqueue) structure.

## Return value

This function returns NTSTATUS.
