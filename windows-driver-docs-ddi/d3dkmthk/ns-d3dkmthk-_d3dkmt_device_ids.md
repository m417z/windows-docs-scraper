# D3DKMT_DEVICE_IDS structure

## Description

The **D3DKMT_DEVICE_IDS** structure contains ID information for a graphics device.

## Members

### `VendorID`

Vendor ID.

### `DeviceID`

Device ID.

### `SubVendorID`

Subvendor ID.

### `SubSystemID`

Subsystem ID.

### `RevisionID`

Revision ID.

### `BusType`

Bus type.
