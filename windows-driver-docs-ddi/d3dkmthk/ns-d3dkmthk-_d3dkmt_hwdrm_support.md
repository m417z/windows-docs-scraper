# D3DKMT_HWDRM_SUPPORT structure

## Description

The **D3DKMT_HWDRM_SUPPORT** structure indicates whether hardware digital rights management (DRM) is supported.

## Members

### `Supported`

Boolean value that indicates whether hardware DRM is supported.
