# D3DKMTPresentMultiPlaneOverlay3 function

## Description

Copies content from a source multiplane overlay allocation to a destination allocation.

## Parameters

### `unnamedParam1`

Pointer to a [D3DKMT_PRESENT_MULTIPLANE_OVERLAY3](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/ns-d3dkmthk-_d3dkmt_present_multiplane_overlay3) structure.

## Return value

This function returns NTSTATUS.
