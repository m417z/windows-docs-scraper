# D3DKMT_OUTDUPL_POINTER_SHAPE_TYPE enumeration

## Description

The **D3DKMT_OUTDUPL_POINTER_SHAPE_TYPE** enumeration specifies output duplicate pointer shape types.

## Constants

### `D3DKMT_OUTDUPL_POINTER_SHAPE_TYPE_MONOCHROME`

Monochrome.

### `D3DKMT_OUTDUPL_POINTER_SHAPE_TYPE_COLOR`

Color.

### `D3DKMT_OUTDUPL_POINTER_SHAPE_TYPE_MASKED_COLOR`

Masked color.
