# D3DKMT_CHANGESURFACEPOINTER structure

## Description

Used to change the surface pointer.

## Members

### `hDC`

The device context handle.

### `hBitmap`

The bitmap handle.

### `pSurfacePointer`

The new surface pointer.

### `Width`

Memory width.

### `Height`

Memory height.

### `Pitch`

Memory pitch.
