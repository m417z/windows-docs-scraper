## Description

The **WIFI_ADAPTER_TYPE** enumeration specifies the Wi-Fi adapter type.

## Constants

### `WIFI_ADAPTER_EXTENSIBLE_STATION`

Extensible station.

### `WIFI_ADAPTER_WIFI_DIRECT_ROLE`

Wi-Fi Direct Role.

### `WIFI_ADAPTER_SECONDARY_STATION`

Secondary station.
