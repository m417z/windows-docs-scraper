## Description

The **WIFI_ADAPTER_TX_DEMUX_TYPE** enumeration specifies the Tx DEMUX type.

## Constants

### `WifiAdapterTxDemuxTypePeerAddress:2`

The type is PEER_ADDRESS.

### `WifiAdapterTxDemuxTypeWmmInfo:3`

The type is WWMINFO.

## Remarks

For more information, see [Setting up multiple Tx queues](https://learn.microsoft.com/windows-hardware/drivers/netcx/writing-a-wificx-client-driver#setting-up-multiple-tx-queues).
