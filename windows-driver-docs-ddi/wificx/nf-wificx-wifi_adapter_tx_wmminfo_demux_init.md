## Description

The **WIFI_ADAPTER_TX_WMMINFO_DEMUX_INIT** function initializes a [**WIFI_ADAPTER_TX_DEMUX**](https://learn.microsoft.com/windows-hardware/drivers/ddi/wificx/ns-wificx-wifi_adapter_tx_demux) structure when the Tx DEMUX type is WMMINFO.

## Parameters

### `Demux`

[_Out_] A pointer to the [**WIFI_ADAPTER_TX_DEMUX**](https://learn.microsoft.com/windows-hardware/drivers/ddi/wificx/ns-wificx-wifi_adapter_tx_demux) structure.

## Remarks
