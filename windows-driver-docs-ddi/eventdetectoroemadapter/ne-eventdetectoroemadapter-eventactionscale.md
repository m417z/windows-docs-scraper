# EVENTACTIONSCALE enumeration

## Description

Enumeration defining the quality associated with a specific detection. It can be used for telemetry when no confidence is available or confidence is confusing to report.

## Constants

### `EVENTACTIONSCALE_High`

0 - High confidence

### `EVENTACTIONSCALE_Medium`

1 - Medium confidence

### `EVENTACTIONSCALE_Low`

2 - Low confidence

## Remarks

This enumeration is the part of the Event Detector OEM interface to be used for hardware keyword spotters. For more information, see [Voice Activation](https://learn.microsoft.com/windows-hardware/drivers/audio/voice-activation).
