# SENSOR_CONFIG_INIT function

## Description

This function initializes a [SENSOR_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/sensorscx/ns-sensorscx-_sensor_config) structure.

## Parameters

### `pConfig` [out]

A pointer to a new [SENSOR_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/sensorscx/ns-sensorscx-_sensor_config) structure.
