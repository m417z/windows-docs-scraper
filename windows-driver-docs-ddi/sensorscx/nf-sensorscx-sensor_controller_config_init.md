# SENSOR_CONTROLLER_CONFIG_INIT function

## Description

This function initializes a [SENSOR_CONTROLLER_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/sensorscx/ns-sensorscx-_sensor_controller_config) structure.

## Parameters

### `pConfig` [out]

A pointer to a new [SENSOR_CONTROLLER_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/sensorscx/ns-sensorscx-_sensor_controller_config) structure.
