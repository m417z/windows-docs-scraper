# _GET_LENGTH_INFORMATION structure

## Description

The GET_LENGTH_INFORMATION structure is used with the [IOCTL_DISK_GET_LENGTH_INFO](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntdddisk/ni-ntdddisk-ioctl_disk_get_length_info) to obtain the length, in bytes, of a disk, partition, or volume.

## Members

### `Length`

Contains the length, in bytes, of a disk, partition, or volume.
