# _AddPort_OUT structure

## Description

The AddPort_OUT structure is used by a WMI provider to report the output parameter data of the [AddPort](https://learn.microsoft.com/previous-versions/ff545022(v=vs.85)) WMI method to the WMI client.

## Members

### `HBAStatus`

Contains the status of the operation. For a list of allowed values and their descriptions, see [HBA_STATUS](https://learn.microsoft.com/windows-hardware/drivers/storage/hba-status).
