# _AddPort_IN structure

## Description

The AddPort_IN structure is used by a WMI client to deliver the input parameter data of the [AddPort](https://learn.microsoft.com/previous-versions/ff545022(v=vs.85)) WMI method to the HBA miniport driver.

## Members

### `PortWWN`

Contains a worldwide name that indicates the port whose events are to be reported.
