# _PEP_LOW_POWER_EPOCH structure (pep_x.h)

## Description

The **PEP_LOW_POWER_EPOCH** structure is used to provide data for a [PEP_DPM_LOW_POWER_EPOCH](https://learn.microsoft.com/windows-hardware/drivers/ddi/pepfx/ns-pepfx-_pep_low_power_epoch) notification (deprecated).

## Members

### `LowPowerEpoch`

When **TRUE**, indicates that the low power epoch is on.
