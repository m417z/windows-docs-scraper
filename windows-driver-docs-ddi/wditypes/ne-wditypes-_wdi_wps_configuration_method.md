# _WDI_WPS_CONFIGURATION_METHOD enumeration

## Description

> [!IMPORTANT]
> This topic is part of the [WDI driver model](https://learn.microsoft.com/windows-hardware/drivers/network/wdi-miniport-driver-design-guide) released in Windows 10. The WDI driver model is in maintenance mode and will only receive high priority fixes. [WiFiCx](https://learn.microsoft.com/windows-hardware/drivers/netcx/wifi-wdf-class-extension-wificx) is the Wi-Fi driver model released in Windows 11. We recommend that you use WiFiCx to take advantage of the latest features.

The WDI_WPS_CONFIGURATION_METHOD enumeration defines WPS configuration methods.

## Constants

### `WDI_WPS_CONFIGURATION_METHOD_NULL`

WFDS.

### `WDI_WPS_CONFIGURATION_METHOD_DISPLAY`

Pin display.

### `WDI_WPS_CONFIGURATION_METHOD_NFC_TAG`

NFC tag.

### `WDI_WPS_CONFIGURATION_METHOD_NFC_INTERFACE`

NFC interface.

### `WDI_WPS_CONFIGURATION_METHOD_PUSHBUTTON`

Push button.

### `WDI_WPS_CONFIGURATION_METHOD_KEYPAD`

Pin keypad.

### `WDI_WPS_CONFIGURATION_METHOD_WFDS_DEFAULT`

WFDS.

### `WDI_WPS_CONFIGURATION_METHOD_DUMMY_MAX_VALUE`

The maximum value for this type.
