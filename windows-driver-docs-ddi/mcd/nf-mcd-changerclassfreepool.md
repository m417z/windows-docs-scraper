# ChangerClassFreePool function

## Description

The **ChangerClassFreePool** routine frees pool memory previously allocated using [ChangerClassAllocatePool](https://learn.microsoft.com/windows-hardware/drivers/ddi/mcd/nf-mcd-changerclassallocatepool).

## Parameters

### `PoolToFree` [in]

Pointer to the block of memory to be freed.
