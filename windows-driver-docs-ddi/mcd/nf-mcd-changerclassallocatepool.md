# ChangerClassAllocatePool function

## Description

The **ChangerClassAllocatePool** function allocates pool memory.

## Parameters

### `PoolType` [in]

Indicates the type of pool memory to allocate. See [POOL_TYPE](https://learn.microsoft.com/windows-hardware/drivers/ddi/wdm/ne-wdm-_pool_type) for a list of types.

### `NumberOfBytes` [in]

Indicates number of bytes to allocate.

## Return value

None
