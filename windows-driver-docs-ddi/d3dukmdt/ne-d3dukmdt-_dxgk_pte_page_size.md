# _DXGK_PTE_PAGE_SIZE enumeration

## Description

The **DXGK_PTE_PAGE_SIZE** enumeration is used by [DXGK_PTE](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dukmdt/ns-d3dukmdt-_dxgk_pte) to indicate the size of lower page level pages.

## Constants

### `DXGK_PTE_PAGE_TABLE_PAGE_4KB`

Indicates the lower page table level uses 4 KB pages.

### `DXGK_PTE_PAGE_TABLE_PAGE_64KB`

Indicates the lower page table level uses 64 KB pages.
