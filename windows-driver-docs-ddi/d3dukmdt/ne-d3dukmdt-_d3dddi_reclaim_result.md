# _D3DDDI_RECLAIM_RESULT enumeration

## Description

Specifies result flags for the [ReclaimAllocations](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmthk/nc-d3dkmthk-pfnd3dkmt_reclaimallocations2) callback function.

## Constants

### `D3DDDI_RECLAIM_RESULT_OK`

The result is OK.

### `D3DDDI_RECLAIM_RESULT_DISCARDED`

The result is discarded.

### `D3DDDI_RECLAIM_RESULT_NOT_COMMITTED`

The result is not committed.
