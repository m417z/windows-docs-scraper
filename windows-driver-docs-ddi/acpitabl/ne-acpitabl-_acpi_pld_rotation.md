# _ACPI_PLD_ROTATION enumeration

## Description

Describes rotation bits 115:118.

## Constants

### `AcpiPldRotation0`

Value is 0.

### `AcpiPldRotation45`

Value is 1.

### `AcpiPldRotation90`

Value is 2.

### `AcpiPldRotation135`

Value is 3.

### `AcpiPldRotation180`

Value is 4.

### `AcpiPldRotation225`

Value is 5.

### `AcpiPldRotation270`

Value is 6.

### `AcpiPldRotation315`

Value is 7.
