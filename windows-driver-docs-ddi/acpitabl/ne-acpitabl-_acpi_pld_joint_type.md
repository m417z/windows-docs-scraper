# _ACPI_PLD_JOINT_TYPE enumeration

## Description

Describes joint type bits 6:9.

## Constants

### `AcpiPldJointTypeFixed`

Value is 0.

### `AcpiPldJointTypePlanar`

Value is 1.

### `AcpiPldJointTypeHinge`

Value is 2.

### `AcpiPldJointTypePivot`

Value is 3.

### `AcpiPldJointTypeSwivel`

Value is 4.
