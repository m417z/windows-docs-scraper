# _ACPI_PLD_PANEL enumeration

## Description

Describes panel surface bits 67:69.

## Constants

### `AcpiPldPanelTop`

Value is 0.

### `AcpiPldPanelBottom`

Value is 1.

### `AcpiPldPanelLeft`

Value is 2.

### `AcpiPldPanelRight`

Value is 3.

### `AcpiPldPanelFront`

Value is 4.

### `AcpiPldPanelBack`

Value is 5.

### `AcpiPldPanelUnknown`

Value is 6.
