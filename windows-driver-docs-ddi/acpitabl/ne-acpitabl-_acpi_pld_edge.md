# _ACPI_PLD_EDGE enumeration

## Description

Describes joint edge bits 10:12,24:26.

## Constants

### `AcpiPldEdgeTop`

Value is 0.

### `AcpiPldEdgeBottom`

Value is 1.

### `AcpiPldEdgeLeft`

Value is 2.

### `AcpiPldEdgeRight`

Value is 3.

### `AcpiPldEdgeUnknown`

Value is 4.
