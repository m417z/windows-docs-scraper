# _D3DDDIARG_FINISHSESSIONKEYREFRESH structure

## Description

The D3DDDIARG_FINISHSESSIONKEYREFRESH structure contains the handle to an encryption session to end in a call to the [FinishSessionKeyRefresh](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d10umddi/nc-d3d10umddi-pfnd3d11_1ddi_finishsessionkeyrefresh) function.

## Members

### `hCryptoSession` [in]

The handle to the encryption session to end.
