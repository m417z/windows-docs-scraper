# _D3DDDIRECTPATCH_INFO structure

## Description

Contains direct patch info.

## Members

### `StartVertexOffsetWidth`

The offset width of the start vertex.

### `StartVertexOffsetHeight`

The offset height of the start vertex.

### `Width`

The patch width.

### `Height`

The patch height.

### `Stride`

The patch stride.

### `Basis`

The patch basis.

### `Degree`

The patch degree.
