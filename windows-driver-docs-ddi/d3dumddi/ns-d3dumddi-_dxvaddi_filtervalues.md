# _DXVADDI_FILTERVALUES structure

## Description

The DXVADDI_FILTERVALUES structure describes values that are related to a filter.

## Members

### `Level` [in]

A [DXVADDI_FIXED32](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dumddi/ns-d3dumddi-_dxvaddi_fixed32) structure that specifies the level of the filter.

### `Threshold` [in]

A [DXVADDI_FIXED32](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dumddi/ns-d3dumddi-_dxvaddi_fixed32) structure that specifies the threshold of the filter.

### `Radius` [in]

A [DXVADDI_FIXED32](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dumddi/ns-d3dumddi-_dxvaddi_fixed32) structure that specifies the radius of the filter.
