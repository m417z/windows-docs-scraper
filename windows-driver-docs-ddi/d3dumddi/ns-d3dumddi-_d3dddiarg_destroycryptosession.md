# _D3DDDIARG_DESTROYCRYPTOSESSION structure

## Description

The D3DDDIARG_DESTROYCRYPTOSESSION structure contains the handle to an encryption session that is destroyed in a call to the [DestroyCryptoSession](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d10umddi/nc-d3d10umddi-pfnd3d11_1ddi_destroycryptosession) function.

## Members

### `hCryptoSession` [in]

The handle to the encryption session that the driver destroys.
