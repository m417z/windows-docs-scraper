# _D3DDDIARG_DESTROYAUTHENICATEDCHANNEL structure

## Description

The D3DDDIARG_DESTROYAUTHENTICATEDCHANNEL structure contains the handle to an authenticated channel that is destroyed in a call to the [DestroyAuthenticatedChannel](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d10umddi/nc-d3d10umddi-pfnd3d11_1ddi_destroyauthenticatedchannel) function.

## Members

### `hChannel` [in]

The handle to the authenticated channel that the driver destroys.
