# _D3DDDI_EXECUTIONSTATEESCAPE structure

## Description

Specifies the state of the device.

## Members

### `State` [Out]

A value of type [D3DDDI_DEVICEEXECUTION_STATE](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dumddi/ne-d3dumddi-_d3dddi_deviceexecution_state) that indicates the state of the device.
