# _D3DDDICB_DESTROYCONTEXT structure

## Description

The D3DDDICB_DESTROYCONTEXT structure contains the handle to a context to destroy.

## Members

### `hContext` [in]

A handle that the [pfnCreateContextCb](https://learn.microsoft.com/previous-versions/ff568895(v=vs.85)) function returns and that identifies the context to destroy.
