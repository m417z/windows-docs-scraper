## Description

The **AKM_FROM_TYPE** macro is used to compute the [**RSNA_AKM_SUITE**](https://learn.microsoft.com/windows-hardware/drivers/ddi/windot11/ne-windot11-rsna_akm_suite) and [**WPA_AKM_SUITE**](https://learn.microsoft.com/windows-hardware/drivers/ddi/windot11/ne-windot11-wpa_akm_suite) enum values.

## Parameters

### `_prefix`

The Robust Secure Network Association (RSNA) Authentication and Key Management (AKM) Organizationally Unique Identifier (OUI) prefix.

### `_akm`

The AKM type.
