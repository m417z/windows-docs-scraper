# _TUPLE_REQUEST structure

## Description

The TUPLE_REQUEST structure is used in conjunction with the [IOCTL_GET_TUPLE_DATA](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntddpcm/ni-ntddpcm-ioctl_get_tuple_data) request to retrieve tuple data from a PC Card's or CardBus card's attribute memory.

## Members

### `Socket`

Indicates the socket number.
