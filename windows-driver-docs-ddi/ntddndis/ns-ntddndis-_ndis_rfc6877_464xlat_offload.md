# _NDIS_RFC6877_464XLAT_OFFLOAD structure

## Description

The **NDIS_RFC6877_464XLAT_OFFLOAD** structure contains 464XLAT hardware offload information.

## Members

### `XlatOffload`

An [**NDIS_RFC6877_464XLAT_OFFLOAD_OPTIONS**](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntddndis/ne-ntddndis-_ndis_rfc6877_464xlat_offload_options) value that describes a miniport driver's 464XLAT hardware offload capabilities.

### `Flags`

Reserved for system use. This member is always zero (0).
