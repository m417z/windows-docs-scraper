# _USB_STOP_TRACKING_FOR_TIME_SYNC_INFORMATION structure

## Description

The input buffer for the [IOCTL_USB_STOP_TRACKING_FOR_TIME_SYNC](https://learn.microsoft.com/windows-hardware/drivers/ddi/usbioctl/ni-usbioctl-ioctl_usb_stop_tracking_for_time_sync) request.

## Members

### `TimeTrackingHandle`

The time racking handle received in the previous [IOCTL_USB_STOP_TRACKING_FOR_TIME_SYNC](https://learn.microsoft.com/windows-hardware/drivers/ddi/usbioctl/ni-usbioctl-ioctl_usb_stop_tracking_for_time_sync) request.
