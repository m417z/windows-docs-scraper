## Description

**D3D12DDI_MESH_SHADER_TIER** specifies the mesh shader tier.

## Constants

### `D3D12DDI_MESH_SHADER_TIER_NOT_SUPPORTED:0`

Mesh shading is not supported.

### `D3D12DDI_MESH_SHADER_TIER_1:10`

The device supports mesh shader tier 1.

## Remarks

See [**D3D12_MESH_SHADER_TIER**](https://learn.microsoft.com/windows/win32/api/d3d12/ne-d3d12-d3d12_mesh_shader_tier) for more information about the tiers.

See the [Mesh Shader Specification](https://microsoft.github.io/DirectX-Specs/d3d/MeshShader.html) for more information.
