# PFND3D12DDI_DESTROYCOMMANDALLOCATOR callback function

## Description

Destroys a command allocator.

## Parameters

### `unnamedParam1`

A handle to the display device (graphics context).

### `unnamedParam2`

A handle to a command allocator.
