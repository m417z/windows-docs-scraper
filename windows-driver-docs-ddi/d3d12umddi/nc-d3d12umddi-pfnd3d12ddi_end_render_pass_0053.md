# PFND3D12DDI_END_RENDER_PASS_0053 callback function

## Description

The user-mode display driver's (UMD) **pfnEndRenderPass** callback function marks the end of an [extended render pass](https://learn.microsoft.com/windows-hardware/drivers/display/d3d12-render-passes).

## Parameters

### `unnamedParam1`

A D3D12DDI_HCOMMANDLIST (command list) that contains the Render Pass.

For more information, see [Direct3D 12 Render Passes](https://learn.microsoft.com/windows-hardware/drivers/display/d3d12-render-passes).
