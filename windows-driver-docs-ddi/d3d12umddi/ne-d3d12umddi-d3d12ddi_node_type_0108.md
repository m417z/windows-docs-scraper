## Description

The **D3D12DDI_NODE_TYPE_0108** enumeration specifies the type of a node in a work graph.

## Constants

### `D3D12DDI_NODE_TYPE_SHADER_0108`

The node is a shader node, which is described by a [**D3D12DDI_SHADER_NODE_0108**](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d12umddi/ns-d3d12umddi-d3d12ddi_shader_node_0108) structure.

## Remarks

For more information, see [Work graphs](https://learn.microsoft.com/windows-hardware/drivers/display/work-graphs).
