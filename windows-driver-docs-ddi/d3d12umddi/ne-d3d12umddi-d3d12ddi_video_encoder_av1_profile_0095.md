## Description

The **D3D12DDI_VIDEO_ENCODER_AV1_PROFILE_0095** enumeration defines the list of distinct AV1 profiles that can be used in AV1 video encoding.

## Constants

### `D3D12DDI_VIDEO_ENCODER_AV1_PROFILE_0095_MAIN`

The AV1 Main profile.

### `D3D12DDI_VIDEO_ENCODER_AV1_PROFILE_0095_HIGH`

The AV1 High profile.

### `D3D12DDI_VIDEO_ENCODER_AV1_PROFILE_0095_PROFESSIONAL`

The AV1 Professional profile.

## Remarks

See [D3D12 AV1 video encoding](https://learn.microsoft.com/windows-hardware/drivers/display/video-encoding-d3d12-av1) for more information.
