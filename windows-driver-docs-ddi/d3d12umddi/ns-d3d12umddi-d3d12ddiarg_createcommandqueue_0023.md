# D3D12DDIARG_CREATECOMMANDQUEUE_0023 structure

## Description

Contains arguments used to create a command queue.

## Members

### `QueueFlags`

Command queue flags.

### `NodeMask`

A mask for a node.

### `QueueCreationFlags`

Command queue creation flag, as a [D3D12DDI_COMMAND_QUEUE_CREATION_FLAGS](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d12umddi/ne-d3d12umddi-d3d12ddi_command_queue_creation_flags) value.
