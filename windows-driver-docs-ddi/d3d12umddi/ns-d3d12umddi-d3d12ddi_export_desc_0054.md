# D3D12DDI_EXPORT_DESC_0054 structure

## Description

Describes information about exports.

## Members

### `Name`

The name of the export.

### `ExportToRename`

The name of use for export renaming.

### `Flags`

The [D3D12DDI_EXPORT_FLAGS](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d12umddi/ne-d3d12umddi-d3d12ddi_export_flags) flag option.
