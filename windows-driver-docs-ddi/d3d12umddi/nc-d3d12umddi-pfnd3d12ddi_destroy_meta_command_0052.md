# PFND3D12DDI_DESTROY_META_COMMAND_0052 callback function

## Description

**PFND3D12DDI_DESTROY_META_COMMAND_0052** destroys a meta-command.

## Parameters

### `unnamedParam1`

A handle to the display device (graphics context).

### `unnamedParam2`

Handle to a meta-command to destroy.
