# D3D12DDIARG_CREATESCHEDULINGGROUP_0050 structure

## Description

Arguments used in the call to [PFND3D12DDI_CREATESCHEDULINGGROUP_0050](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d12umddi/nc-d3d12umddi-pfnd3d12ddi_createschedulinggroup_0050) to create a scheduling group.

## Members

### `NodeMask`

The node mask of the command list.
