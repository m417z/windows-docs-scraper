# PFND3D12DDI_DESTROYHEAPANDRESOURCE callback function

## Description

Destroy heap and resource.

## Parameters

### `unnamedParam1`

A handle to the display device (graphics context).

### `unnamedParam2`

A heap handle.

### `unnamedParam3`

A resource handle.
