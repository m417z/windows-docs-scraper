# D3D12DDI_RESOURCE_DIMENSION enumeration

## Description

Identifies the type of resource dimension to use.

## Constants

### `D3D12DDI_RD_BUFFER`

Buffer.

### `D3D12DDI_RD_TEXTURE1D`

1D texture.

### `D3D12DDI_RD_TEXTURE2D`

2D texture.

### `D3D12DDI_RD_TEXTURE3D`

3D texture.

### `D3D12DDI_RD_TEXTURECUBE`

Cube texture.

### `D3D12DDI_RD_RAYTRACING_ACCELERATION_STRUCTURE_0042`

Ray tracing acceleration structure.
