## Description

**D3D12DDI_MIP_REGION_0075** describes the size of a MIP region.

## Members

### `Width`

Width of the MIP region, in texels.

### `Height`

Height of the MIP region, in texels.

### `Depth`

Depth of the MIP region, in texels.
