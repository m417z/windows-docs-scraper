# D3D12DDI_VIEW_INSTANCING_TIER enumeration

## Description

A **D3D12DDI_VIEW_INSTANCING_TIER** value indicates the tier level at which the driver supports view instancing.

## Constants

### `D3D12DDI_VIEW_INSTANCING_TIER_NOT_SUPPORTED`

View instancing is not supported.

### `D3D12DDI_VIEW_INSTANCING_TIER_1`

The driver provides tier 1 support.

### `D3D12DDI_VIEW_INSTANCING_TIER_2`

The driver provides tier 2 support.

### `D3D12DDI_VIEW_INSTANCING_TIER_3`

The driver provides tier 1 support.

## Remarks

See [**D3D12_VIEW_INSTANCING_TIER**](https://learn.microsoft.com/windows/win32/api/d3d12/ne-d3d12-d3d12_tiled_resources_tier) for a detailed description for each tier level.
