# D3D12DDI_RENDER_PASS_FUNCS_0053 structure

## Description

Callback functions used to begin and end a Render Pass.

## Members

### `pfnBeginRenderPass`

Pointer to the [PFND3D12DDI_BEGIN_RENDER_PASS_0053](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d12umddi/nc-d3d12umddi-pfnd3d12ddi_begin_render_pass_0053) callback to begin a Render Pass.

### `pfnEndRenderPass`

Pointer to the [PFND3D12DDI_END_RENDER_PASS_0053](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d12umddi/nc-d3d12umddi-pfnd3d12ddi_end_render_pass_0053) callback to end a Render Pass.
