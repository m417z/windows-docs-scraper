## Description

A **D3D12DDI_TILED_RESOURCES_TIER** value identifies the tier level at which the driver supports tiled resources.

## Constants

### `D3D12DDI_TILED_RESOURCES_TIER_NOT_SUPPORTED:0`

The driver does not support tiled resources.

### `D3D12DDI_TILED_RESOURCES_TIER_1:1`

The driver provides tier 1 support for tiled resources.

### `D3D12DDI_TILED_RESOURCES_TIER_2:2`

The driver provides tier 2 support for tiled resources.

### `D3D12DDI_TILED_RESOURCES_TIER_3:3`

The driver provides tier 3 support for tiled resources.

## Remarks

See [**D3D12_TILED_RESOURCES_TIER**](https://learn.microsoft.com/windows/win32/api/d3d12/ne-d3d12-d3d12_tiled_resources_tier) for a detailed description for each tier level.
