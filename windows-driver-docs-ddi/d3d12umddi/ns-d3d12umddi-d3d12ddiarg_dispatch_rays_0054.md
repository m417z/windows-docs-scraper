# D3D12DDIARG_DISPATCH_RAYS_0054 structure

## Description

Arguments used by the [DispatchRays](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d12umddi/nc-d3d12umddi-pfnd3d12ddi_dispatch_rays_0054) callback function.

## Members

### `RayGenerationShaderRecord`

Ray generation shader record.

### `MissShaderTable`

Miss shader table.

### `HitGroupTable`

Hit group table.

### `CallableShaderTable`

Callable shader table.

### `Width`

Width of the ray.

### `Height`

Height of the ray.

### `Depth`

Depth of the ray.
