## Description

The **D3D12DDI_VIDEO_ENCODER_LEVELS_HEVC_0080** enumeration lists the possible HEVC levels.

## Constants

### `D3D12DDI_VIDEO_ENCODER_LEVELS_HEVC_0080_1`
