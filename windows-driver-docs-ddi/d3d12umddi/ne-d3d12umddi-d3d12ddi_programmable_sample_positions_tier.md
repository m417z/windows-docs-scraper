## Description

A **D3D12DDI_PROGRAMMABLE_SAMPLE_POSITIONS_TIER** value specifies the level of support provided by the adapter for programmable sample positions.

## Constants

### `D3D12DDI_PROGRAMMABLE_SAMPLE_POSITIONS_TIER_NOT_SUPPORTED:0`

The adapter does not support programmable sample positions.

### `D3D12DDI_PROGRAMMABLE_SAMPLE_POSITIONS_TIER_1:1`

The driver provides tier 1 support.

### `D3D12DDI_PROGRAMMABLE_SAMPLE_POSITIONS_TIER_2:2`

The driver provides tier 2 support.

## Remarks

See [**D3D12_PROGRAMMABLE_SAMPLE_POSITIONS_TIER**](https://learn.microsoft.com/windows/win32/api/d3d12/ne-d3d12-d3d12_programmable_sample_positions_tier) for a detailed description for each tier level.
