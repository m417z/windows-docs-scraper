# D3D12DDI_RENDER_PASS_BEGINNING_ACCESS_CLEAR_PARAMETERS structure

## Description

Parameters needed to specify whether the application has a dependency on the resource being cleared, to a specific color.

## Members

### `ClearValue`

Pointer to a [D3D12DDI_CLEAR_VALUES](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3d12umddi/ns-d3d12umddi-d3d12ddi_clear_values) structure that holds the parameters need to begin a Render Pass.
