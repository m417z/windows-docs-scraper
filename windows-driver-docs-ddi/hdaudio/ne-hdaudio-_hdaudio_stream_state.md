# _HDAUDIO_STREAM_STATE enumeration

## Description

The **HDAUDIO_STREAM_STATE** enumeration defines constants that specify the different stream states supported by HDAudio.

## Constants

### `ResetState`

The reset state.

### `StopState`

The stop state.

### `PauseState`

The pause state.

### `RunState`

The run state.

## Remarks

This enumeration is used by the [PSET_DMA_ENGINE_STATE](https://learn.microsoft.com/windows-hardware/drivers/ddi/hdaudio/nc-hdaudio-pset_dma_engine_state).
