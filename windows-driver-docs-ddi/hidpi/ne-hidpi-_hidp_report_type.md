# _HIDP_REPORT_TYPE enumeration

## Description

The HIDP_REPORT_TYPE enumeration type is used to specify a HID report type.

## Constants

### `HidP_Input`

Indicates an input report.

### `HidP_Output`

Indicates an output report.

### `HidP_Feature`

Indicates a feature report.
