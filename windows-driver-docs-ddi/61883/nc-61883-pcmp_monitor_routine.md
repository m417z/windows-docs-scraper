# PCMP_MONITOR_ROUTINE callback function

## Description

This routine is called for plug monitoring.

## Parameters

### `MonitorInfo` [in]

Pointer to a [CMP_MONITOR_INFO](https://learn.microsoft.com/windows-hardware/drivers/ddi/61883/ns-61883-_cmp_monitor_info) structure containing the contents of the plug that was modified.
