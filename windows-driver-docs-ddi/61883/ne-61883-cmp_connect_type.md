# CMP_CONNECT_TYPE enumeration

## Description

This enumeration specifies a connection type.

## Constants

### `CMP_Broadcast`

Indicates that this is a broadcast connection.

### `CMP_PointToPoint`

Indicates that this is a point-to-point connection.
