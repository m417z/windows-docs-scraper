# _AV_PCR structure

## Description

The AV_PCR structure specifies settings for an input or output plug.

## Members

### `oPCR`

Contains an [OPCR](https://learn.microsoft.com/windows-hardware/drivers/ddi/61883/ns-61883-_opcr) structure that contains initialization values for an output plug.

### `iPCR`

Contains an [IPCR](https://learn.microsoft.com/windows-hardware/drivers/ddi/61883/ns-61883-_ipcr) structure that contains initialization values for an input plug.

### `ulongData`

Reserved for internal use.
