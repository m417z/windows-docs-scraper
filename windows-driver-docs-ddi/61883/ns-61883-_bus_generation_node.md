# _BUS_GENERATION_NODE structure

## Description

The BUS_GENERATION_NODE structure is used in conjunction with the [Av61883_GetUnitInfo](https://learn.microsoft.com/previous-versions/ff536983(v=vs.85)) request to retrieve bus characteristics.

## Members

### `GenerationCount`

The current bus generation count.

### `LocalNodeAddress`

The current node address of the local node.

### `DeviceNodeAddress`

The current node address of the device.
