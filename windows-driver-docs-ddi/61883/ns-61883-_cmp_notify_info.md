# _CMP_NOTIFY_INFO structure

## Description

This structure is used by the [PCMP_NOTIFY_ROUTINE](https://learn.microsoft.com/windows-hardware/drivers/ddi/61883/nc-61883-pcmp_notify_routine) callback.

## Members

### `hPlug`

The handle of a plug for the notification.

### `Pcr`

A plug control register.

### `Context`

A context.
