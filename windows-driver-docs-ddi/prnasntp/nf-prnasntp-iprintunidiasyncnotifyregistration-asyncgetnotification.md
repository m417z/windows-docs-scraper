## Description

Defines the **IPrintUnidiAsyncNotifyRegistration::AsyncGetNotification** method.

## Parameters

### `EQUALNULL`

Defines the **IAsyncGetSendNotificationCookie** parameter *EQUALNULL*.

## Return value
