# GNSS_SUPL_VERSION_2 structure

## Description

The **GNSS_SUPL_VERSION_2** structure contains Secure User Plane Location (SUPL) 2 version information.

## Members

### `MajorVersion`

Major version number.

### `MinorVersion`

Minor version number.

### `ServiceIndicator`

Service indicator version number.
