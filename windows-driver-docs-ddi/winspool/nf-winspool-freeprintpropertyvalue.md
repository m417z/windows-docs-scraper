# FreePrintPropertyValue function

## Description

Frees the value that is retrieved using the [GetJobNamedPropertyValue](https://learn.microsoft.com/windows-hardware/drivers/ddi/winspool/nf-winspool-getjobnamedpropertyvalue) function.

## Parameters

### `pValue` [in, out]

Pointer to **PrintPropertyValue** structure that is returned from [GetJobNamedPropertyValue](https://learn.microsoft.com/windows-hardware/drivers/ddi/winspool/nf-winspool-getjobnamedpropertyvalue).
