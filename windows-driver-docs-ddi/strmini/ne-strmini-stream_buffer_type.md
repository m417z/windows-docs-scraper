# STREAM_BUFFER_TYPE enumeration

## Description

This enumeration defines the buffer types for [StreamClassGetPhysicalAddress](https://learn.microsoft.com/dotnet/core/rid-catalog).

## Constants

### `PerRequestExtension`

Indicates the physical address of the SRB extension.

### `DmaBuffer`

Indicates the physical address of the DMA buffer.

### `SRBDataBuffer`

Indicates the physical address of a data buffer.
