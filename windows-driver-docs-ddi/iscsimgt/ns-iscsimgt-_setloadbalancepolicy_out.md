# _SetLoadBalancePolicy_OUT structure

## Description

The SetLoadBalancePolicy_OUT structure holds the output data for the SetLoadBalance method.

## Members

### `Status`

A value that provides the status of the SetLoadBalancePolicy operation.

## Remarks

You must implement this class. For a list of possible status qualifiers, see [ISCSI_STATUS_QUALIFIERS](https://learn.microsoft.com/windows-hardware/drivers/storage/iscsi-status-qualifiers).
