# USBPM_CONNECTOR_STATE_INIT function

## Description

Initializes a [**USBPM_CONNECTOR_STATE_INIT**](https://learn.microsoft.com/windows-hardware/drivers/ddi/usbpmapi/ns-usbpmapi-_usbpm_connector_state) structure. The client driver must call this function before calling [**UsbPm_RetrieveConnectorState**](https://learn.microsoft.com/windows-hardware/drivers/ddi/usbpmapi/nf-usbpmapi-usbpm_retrieveconnectorstate).

## Parameters

### `ConnectorState`

A pointer a [**USBPM_CONNECTOR_STATE_INIT**](https://learn.microsoft.com/windows-hardware/drivers/ddi/usbpmapi/ns-usbpmapi-_usbpm_connector_state) structure to initialize.
