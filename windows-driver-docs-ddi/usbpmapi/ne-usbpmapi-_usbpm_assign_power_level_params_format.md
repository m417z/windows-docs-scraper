# _USBPM_ASSIGN_POWER_LEVEL_PARAMS_FORMAT enumeration

## Description

Defines format values used in [**USBPM_ASSIGN_CONNECTOR_POWER_LEVEL_PARAMS**](https://learn.microsoft.com/windows-hardware/drivers/ddi/usbpmapi/ns-usbpmapi-_usbpm_assign_connector_power_level_params).

## Constants

### `UsbPmAssignPowerLevelParamsFormatInvalid`

Invalid format.

### `UsbPmAssignPowerLevelParamsFormatUcsi`

The format is UCSI.

### `UsbPmAssignPowerLevelParamsFormatRdo`

The format is in a PD Request Data Object.
