# USBPM_HUB_PROPERTIES_INIT function

## Description

Initializes a [**USBPM_HUB_PROPERTIES**](https://learn.microsoft.com/windows-hardware/drivers/ddi/usbpmapi/ns-usbpmapi-_usbpm_hub_properties) structure.

## Parameters

### `Properties` [Out]

A pointer to a [**USBPM_HUB_PROPERTIES**](https://learn.microsoft.com/windows-hardware/drivers/ddi/usbpmapi/ns-usbpmapi-_usbpm_hub_properties) structure to initialize.
