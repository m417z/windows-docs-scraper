# IWDTFSystemAction2::GetFirstSleepState

## Description

Returns the first supported sleep state.

## Parameters

### `pnSleepState` [out, retval]

The sleep state.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
