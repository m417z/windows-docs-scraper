# IWDTFSystemAction2::RebootRestartWithContext

## Description

Reboots the system and restarts the test script with context data.

## Parameters

### `psContextName` [in]

The context name.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
