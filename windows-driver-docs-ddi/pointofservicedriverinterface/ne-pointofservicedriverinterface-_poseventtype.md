## Description

This enumeration defines values used in the [PosEventDataHeader](https://learn.microsoft.com/previous-versions/windows/hardware/previsioning-framework/dn772232(v=vs.85)) structure to indicate the type of event that was raised.

## Constants

### `InvalidEvent`

The event code is not valid.

### `_MinEventType`

Reserved for internal use.

### `ReleaseDeviceRequested`

Represents the [ReleaseDeviceRequested](https://learn.microsoft.com/previous-versions/windows/hardware/previsioning-framework/dn790033(v=vs.85)) event.

### `StatusUpdated`

Represents the [StatusUpdated](https://learn.microsoft.com/previous-versions/windows/hardware/previsioning-framework/dn790040(v=vs.85)) event.

### `BarcodeScannerTriggerPressed`

Represents the [BarcodeScannerTriggerPressed](https://learn.microsoft.com/previous-versions/windows/hardware/previsioning-framework/dn757468(v=vs.85)) event.

### `BarcodeScannerTriggerReleased`

Represents the [BarcodeScannerTriggerReleased](https://learn.microsoft.com/previous-versions/windows/hardware/previsioning-framework/dn757469(v=vs.85)) event.

### `BarcodeScannerDataReceived`

Represents the [BarcodeScannerDataReceived](https://learn.microsoft.com/previous-versions/windows/hardware/previsioning-framework/dn757463(v=vs.85)) event.

### `BarcodeScannerErrorOccurred`

Represents the [BarcodeScannerErrorOccurred](https://learn.microsoft.com/previous-versions/windows/hardware/previsioning-framework/dn757464(v=vs.85)) event.

### `BarcodeScannerImagePreviewReceived`

Represents the [BarcodeScannerImagePreviewReceived](https://learn.microsoft.com/previous-versions/windows/hardware/previsioning-framework/dn757466(v=vs.85)) event.

### `MagneticStripeReaderDataReceived`

Represents the [MagneticStripeReaderDataReceived](https://learn.microsoft.com/previous-versions/windows/hardware/previsioning-framework/dn772149(v=vs.85)) event.

### `MagneticStripeReaderErrorOccurred`

Represents the [MagneticStripeReaderErrorOccurred](https://learn.microsoft.com/previous-versions/windows/hardware/previsioning-framework/dn772151(v=vs.85)) event.

### `_Max`

Reserved for internal use.
