# NdisFreeMdl function

## Description

The
**NdisFreeMdl** function frees an MDL that was allocated by calling the
[NdisAllocateMdl](https://learn.microsoft.com/windows-hardware/drivers/ddi/ndis/nf-ndis-ndisallocatemdl) function.

## Parameters

### `Mdl` [in]

A pointer to the MDL that NDIS should free.
