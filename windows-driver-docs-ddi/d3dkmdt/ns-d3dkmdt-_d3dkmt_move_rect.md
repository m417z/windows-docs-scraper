# _D3DKMT_MOVE_RECT structure

## Description

Provides information on a screen-to-screen move and a dirty rectangle copy operation.

## Members

### `SourcePoint`

A [POINT](https://learn.microsoft.com/windows/win32/api/windef/ns-windef-point) structure that describes the source (*x, y*) point where the source rectangle is copied from.

### `DestRect`

A [RECT](https://learn.microsoft.com/windows/win32/api/windef/ns-windef-rect) structure that contains the destination rectangle where the dirty rectangle is copied to.
