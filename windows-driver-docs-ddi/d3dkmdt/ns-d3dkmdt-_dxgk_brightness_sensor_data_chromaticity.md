# _DXGK_BRIGHTNESS_SENSOR_DATA_CHROMATICITY structure

## Description

The chromaticity coordinates from a color sensor.

## Members

### `ChromaticityX`

The X coordinate of the chromaticity.

### `ChromaticityY`

The Y coordinate of the chromaticity.
