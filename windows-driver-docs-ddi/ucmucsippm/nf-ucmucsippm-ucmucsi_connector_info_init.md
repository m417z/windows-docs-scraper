# UCMUCSI_CONNECTOR_INFO_INIT function

## Description

Initializes a [**UCMUCSI_CONNECTOR_INFO**](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmucsippm/ns-ucmucsippm-_ucmucsi_connector_info) structure.

## Parameters

### `ConnectorInfo` [out]

A pointer to a [**UCMUCSI_CONNECTOR_INFO**](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmucsippm/ns-ucmucsippm-_ucmucsi_connector_info) structure to initialize.
