# UCSI_CMD_SUCCEEDED function

## Description

On successful completion of a UCSI command the PPM firmware fills the CCI Data Structure provided by the client driver.

## Parameters

### `Cci`

A [**UCSI_CCI**](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmucsispec/ns-ucmucsispec-_ucsi_cci) structure.

## Return value

This function returns BOOLEAN.
