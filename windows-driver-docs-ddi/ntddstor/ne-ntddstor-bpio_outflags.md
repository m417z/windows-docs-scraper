## Description

**BPIO_OUTFLAGS** defines the BypassIO output flags for the [**IOCTL_STORAGE_MANAGE_BYPASS_IO**](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntddstor/ni-ntddstor-ioctl_storage_manage_bypass_io) control code.

## Constants

### `BPIO_OUTFL_NONE`

No output flag(s) are set.

## Remarks

See [BypassIO for storage drivers](https://learn.microsoft.com/windows-hardware/drivers/storage/bypassio) for more information.
