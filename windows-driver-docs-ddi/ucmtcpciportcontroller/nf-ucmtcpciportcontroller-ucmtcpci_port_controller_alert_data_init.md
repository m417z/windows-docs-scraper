# UCMTCPCI_PORT_CONTROLLER_ALERT_DATA_INIT function

## Description

Initializes the **UCMTCPCI_PORT_CONTROLLER_ALERT_DATA** structure.

Call this function before calling [UcmTcpciPortControllerAlert](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmtcpciportcontroller/nf-ucmtcpciportcontroller-ucmtcpciportcontrolleralert).

## Parameters

### `AlertData` [out]

A pointer to the driver-allocated **UCMTCPCI_PORT_CONTROLLER_ALERT_DATA** structure.
