## Description

The **GUIDREGINFO** structure contains GUID registration information.

## Members

### `Guid`

Specifies the GUID to register.

### `InstanceCount`

Specifies the count of instances of data block.

### `Flags`

Additional flags, for more information see the **[WMIREGINFOW](https://learn.microsoft.com/windows-hardware/drivers/ddi/wmistr/ns-wmistr-wmireginfow)** structure (wmistr.h).
