# _WWAN_UICC_RESET_INFO structure

## Description

The **WWAN_UICC_RESET_INFO** structure represents the passthrough status of a modem miniport adapter for a UICC smart card.

## Members

### `PassThroughStatus`

The passthrough status of the miniport adapter. For more info, see [WWAN_UICC_PASSTHROUGH_STATUS](https://learn.microsoft.com/windows-hardware/drivers/ddi/wwan/ne-wwan-_wwan_uicc_passthrough_status).
