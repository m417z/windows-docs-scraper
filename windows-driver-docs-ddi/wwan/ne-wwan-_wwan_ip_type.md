# _WWAN_IP_TYPE enumeration

## Description

The WWAN_IP_TYPE enumeration lists the different levels of supported IP.

## Constants

### `WwanIPTypeDefault`

Default IP.

### `WwanIPTypeIPv4`

IPv4.

### `WwanIPTypeIPv6`

IPv6.

### `WwanIPTypeIpv4v6`

IPv4 with IPv6.

### `WwanIPTypeXlat`

XLAT.

### `WwanSessionTypeEthernet`

Ethernet PDU session.

### `WwanSessionTypeUnstructured`

Unstructured PDU session.

### `WwanSessionTypeMax`
