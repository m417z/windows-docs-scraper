## Description

The **WWAN_LADN** structure represents a Local Area Data Network (LADN).

## Members

### `AccessString`

The selected APN/DNN for an established context/session.

### `TaiList`

A [**WWAN_TAI_LIST**](https://learn.microsoft.com/windows-hardware/drivers/ddi/wwan/ns-wwan-wwan_tai_list) structure that represents a tracking area identity (TAI) list.
