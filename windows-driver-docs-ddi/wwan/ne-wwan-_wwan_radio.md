# _WWAN_RADIO enumeration

## Description

The WWAN_RADIO enumeration lists the different types of radio power modes that are supported by the
MB device.

## Constants

### `WwanRadioOff`

The radio power is turned off.

### `WwanRadioOn`

The radio power is turned on.
