## Description

The **WWAN_LADN_LIST** structure represents a list of Local Area Data Networks (LADNs).

## Members

### `ElementCount`

The number of elements in the LADN list.

### `Ladns`

An array of [**WWAN_LADN**](https://learn.microsoft.com/windows-hardware/drivers/ddi/wwan/ns-wwan-wwan_ladn) structures that represent the LADN list.
