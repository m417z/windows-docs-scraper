# IKsAllocator::KsGetAllocatorStatus

## Description

Retrieves the status of an allocator.

## Parameters

### `AllocatorStatus`

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
