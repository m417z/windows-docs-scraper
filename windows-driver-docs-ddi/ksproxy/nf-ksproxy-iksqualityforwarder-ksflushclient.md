# IKsQualityForwarder::KsFlushClient

## Description

The **KsFlushClient** method flushes information from a pin.

## Parameters

### `Pin` [in]

Pointer to the [IKsPin](https://learn.microsoft.com/windows-hardware/drivers/ddi/ksproxy/nn-ksproxy-ikspin) interface for the pin from which to flush information.
