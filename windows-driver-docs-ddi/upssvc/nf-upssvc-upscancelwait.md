# UPSCancelWait function

## Description

The **UPSCancelWait** function cancels all waits initiated by calls to [UPSWaitForStateChange](https://learn.microsoft.com/windows-hardware/drivers/ddi/upssvc/nf-upssvc-upswaitforstatechange).

## Remarks

The call returns immediately.
