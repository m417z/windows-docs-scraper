# RECORDER_CONFIGURE_PARAMS_INIT function

## Description

The **RECORDER_CONFIGURE_PARAMS_INIT** function is used to initialize the [RECORDER_CONFIGURE_PARAMS](https://learn.microsoft.com/windows-hardware/drivers/ddi/wpprecorder/ns-wpprecorder-_recorder_configure_params) structure.

## Parameters

### `Params` [out]

A pointer to the [RECORDER_CONFIGURE_PARAMS](https://learn.microsoft.com/windows-hardware/drivers/ddi/wpprecorder/ns-wpprecorder-_recorder_configure_params) structure to initialize.
