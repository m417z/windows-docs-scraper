# _UCMUCSI_PPM_GET_UCSI_DATA_BLOCK_IN_PARAMS structure

## Description

Contains a USCI data block. This structure is the input buffer to the [IOCTL_UCMUCSI_PPM_GET_UCSI_DATA_BLOCK](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmucsippmrequests/ni-ucmucsippmrequests-ioctl_ucmucsi_ppm_get_ucsi_data_block) I/O request.

## Members

### `PpmObject`

A handle to a Platform Policy Manager (PPM) object that the client driver received in the previous call to [**UcmUcsiPpmCreate**](https://learn.microsoft.com/windows-hardware/drivers/ddi/ucmucsippm/nf-ucmucsippm-ucmucsippmcreate).
