# _SECURE_ELEMENT_ENDPOINT_INFO structure

## Description

SECURE_ELEMENT_ENDPOINT_INFO is a member of [SECURE_ELEMENT_ENDPOINT_LIST](https://learn.microsoft.com/windows-hardware/drivers/ddi/nfcsedev/ns-nfcsedev-_secure_element_endpoint_list).

## Members

### `guidSecureElementId`

This is a unique identifier for the secure element.

### `eSecureElementType`

Type of secure element endpoint (NFCEE).
