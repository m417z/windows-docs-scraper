## Description

The **ACX_PIN_CALLBACKS_INIT** function initializes an [ACX_PIN_CALLBACKS](https://learn.microsoft.com/windows-hardware/drivers/ddi/acxpin/ns-acxpin-acx_pin_callbacks) structure for use.

## Parameters

### `PinCallbacks` [out]

A pointer to the **ACX_PIN_CALLBACKS** structure to initialize.

## Remarks

### ACX requirements

**Minimum ACX version:** 1.0

For more information about ACX versions, see [ACX version overview](https://learn.microsoft.com/windows-hardware/drivers/audio/acx-version-overview).
