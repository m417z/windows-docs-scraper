# BranchOfficeJobDataContainer structure

## Description

This structure defines a container for one or more [BranchOfficeJobData](https://learn.microsoft.com/dotnet/core/rid-catalog) structures to sent to a server.

## Members

### `cJobDataEntries`

Describes the **DWORD** type member **cJobDataEntries**.

### `JobData`

Describes the **BranchOfficeJobData** type member **JobData**.
