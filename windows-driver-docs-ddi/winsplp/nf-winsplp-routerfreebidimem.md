# RouterFreeBidiMem function

## Description

`RouterFreeBidiMem` frees a block of memory that was previously allocated by [RouterAllocBidiMem](https://learn.microsoft.com/windows-hardware/drivers/ddi/winsplp/nf-winsplp-routerallocbidimem).

## Parameters

### `pMemPointer` [in]

Pointer to the memory to be freed.
