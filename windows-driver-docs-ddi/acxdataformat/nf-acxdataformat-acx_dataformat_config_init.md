## Description

The **ACX_DATAFORMAT_CONFIG_INIT** function initializes an [ACX_DATAFORMAT_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/acxdataformat/ns-acxdataformat-acx_dataformat_config) structure.

## Parameters

### `Config` [out]

A pointer to the [ACX_DATAFORMAT_CONFIG](https://learn.microsoft.com/windows-hardware/drivers/ddi/acxdataformat/ns-acxdataformat-acx_dataformat_config) structure to initialize.

## Remarks

### ACX requirements

**Minimum ACX version:** 1.0

For more information about ACX versions, see [ACX version overview](https://learn.microsoft.com/windows-hardware/drivers/audio/acx-version-overview).
