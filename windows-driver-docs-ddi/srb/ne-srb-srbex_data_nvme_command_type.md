## Description

**SRBEX_DATA_NVME_COMMAND_TYPE** enumerates the NVMe command types.

## Constants

### `SRBEX_DATA_NVME_COMMAND_TYPE_NVM`

The command is an NVM command.

### `SRBEX_DATA_NVME_COMMAND_TYPE_ADMIN`

The command is an Admin command.

### `SRBEX_DATA_NVME_COMMAND_TYPE_FABRICS`

The command is a Fabrics command.

## Remarks

**SRBEX_DATA_NVME_COMMAND_TYPE** is specified in a **[SRBEX_DATA_NVME_COMMAND](https://learn.microsoft.com/windows-hardware/drivers/ddi/srb/ns-srb-srbex_data_nvme_command)** structure.
