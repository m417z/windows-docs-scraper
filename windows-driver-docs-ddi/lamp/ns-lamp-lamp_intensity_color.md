# LAMP_INTENSITY_COLOR structure

## Description

This structure is the I/O parameter type of [IOCTL_LAMP_GET_INTENSITY_COLOR](https://learn.microsoft.com/windows-hardware/drivers/ddi/lamp/ni-lamp-ioctl_lamp_get_intensity_color) and [IOCTL_LAMP_SET_INTENSITY_COLOR](https://learn.microsoft.com/windows-hardware/drivers/ddi/lamp/ni-lamp-ioctl_lamp_set_intensity_color).

## Members

### `Red`

Red light intensity in percentage (0-100).

### `Green`

Green light intensity in percentage (0-100).

### `Blue`

Blue light intensity in percentage (0-100).
