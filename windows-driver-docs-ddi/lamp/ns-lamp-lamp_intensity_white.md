# LAMP_INTENSITY_WHITE structure

## Description

This structure is the I/O parameter type of [IOCTL_LAMP_GET_INTENSITY_WHITE](https://learn.microsoft.com/windows-hardware/drivers/ddi/lamp/ni-lamp-ioctl_lamp_get_intensity_white) and [IOCTL_LAMP_SET_INTENSITY_WHITE](https://learn.microsoft.com/windows-hardware/drivers/ddi/lamp/ni-lamp-ioctl_lamp_set_intensity_white).

## Members

### `Value`

White light intensity in percentage (0-100).
