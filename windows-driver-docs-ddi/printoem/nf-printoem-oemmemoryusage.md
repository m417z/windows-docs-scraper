## Description

This function is obsolete for Windows XP and later. It is supported only for earlier Unidrv plug-ins.

Use [IPrintOemUni::MemoryUsage](https://learn.microsoft.com/windows-hardware/drivers/ddi/prcomoem/nf-prcomoem-iprintoemuni-memoryusage) instead.

## Parameters

### `pdevobj`

Pointer to device object.

### `pMemoryUsage` [in, out]

Pointer to OEMMEMORYUSAGE structure.
