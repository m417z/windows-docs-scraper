## Description

The **OEMQueryFont** function is used by GDI to get the [IFIMETRICS](https://learn.microsoft.com/windows/win32/api/winddi/ns-winddi-ifimetrics) structure for a given font.

## Parameters

### `dhpdev`

Defines the **DHPDEV** parameter *dhpdev*.

### `iFile`

Defines the **ULONG_PTR** parameter *iFile*.

### `iFace`

Defines the **ULONG** parameter *iFace*.

### `pid`

Defines the **ULONG_PTR** parameter *pid*.

## Return value

Returns a **PIFIMETRICS** value.
