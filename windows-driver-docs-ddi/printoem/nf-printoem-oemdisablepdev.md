## Description

This function is obsolete for Windows XP and later. It is supported only for earlier plug-ins.

Use [IPrintOemPS::DisablePDEV (Pscript)](https://learn.microsoft.com/windows-hardware/drivers/ddi/prcomoem/nf-prcomoem-iprintoemps-disablepdev) or [IPrintOemUni::DisablePDEV (Unidrv)](https://learn.microsoft.com/windows-hardware/drivers/ddi/prcomoem/nf-prcomoem-iprintoemuni-disablepdev) instead.

## Parameters

### `pdevobj`

Pointer to the device object.
