# ISCSI_ADAPTER_EVENT_CODE enumeration

## Description

The ISCSI_ADAPTER_EVENT_CODE enumeration indicates the type of adapter event.

## Constants

### `ISCSI_ADAPTER_TARGETS_CHANGED`

An adapter discovered that its list of targets changed.
