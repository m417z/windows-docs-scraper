# _DeleteInitiatorNodeName_IN structure

## Description

The DeleteInitiatorNodeName_IN structure holds the input data for the [DeleteInitiatorNodeName](https://learn.microsoft.com/windows-hardware/drivers/storage/deleteinitiatornodename) method, which is used to delete an initiator node name.

## Members

### `DeletedInitiatorName`

The iSCSI initiator node name that is to be deleted.

## Remarks

It is optional that you implement this method.
