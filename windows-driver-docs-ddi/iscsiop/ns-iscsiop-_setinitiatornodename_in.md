# _SetInitiatorNodeName_IN structure

## Description

The SetInitiatorNodeName_IN structure holds the input data for the [SetInitiatorNodeName](https://learn.microsoft.com/windows-hardware/drivers/storage/setinitiatornodename) method.

## Members

### `CreatedInitiatorName`

The new initiator name.

## Remarks

It is optional that you implement this class.
