# _HARDWARE_COUNTER_TYPE enumeration

## Description

The **HARDWARE_COUNTER_TYPE** enumeration specifies the type of a hardware counter.

## Constants

### `PMCCounter`

Performance monitor counter. This type of counter is used by thread-profiling applications.

### `MaxHardwareCounterType`

The maximum value in this enumeration type.

## Remarks

The **Type** member of the [HARDWARE_COUNTER](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntddk/ns-ntddk-_hardware_counter) structure contains a **HARDWARE_COUNTER_TYPE** enumeration value.
