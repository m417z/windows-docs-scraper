# _IO_FOEXT_SILO_PARAMETERS structure

## Description

This structure describes the Container context that's identified by the [IoGetSiloParameters](https://learn.microsoft.com/windows-hardware/drivers/ddi/ntddk/nf-ntddk-iogetsiloparameters) routine.

## Members

### `Length`

The length of the structure. This also serves as the structure version.

### `SpareFlags`

### `Flags`

### `SiloContext`
