# _SET_LOCAL_HOST_PROPS2 structure

## Description

SET_LOCAL_HOST_PROPS2 sets a lower bound on the value the bus will use for its gap count.

## Members

### `GapCountLowerBound`

Specifies lower bound of the gap count.
