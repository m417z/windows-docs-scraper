# _IRB_REQ_ISOCH_QUERY_CURRENT_CYCLE_TIME structure

## Description

 This structure contains the fields necessary for the Bus driver to carry out an
**IsochQueryCurrentCycleTime** request.

## Members

### `CycleTime`

On success, specifies the current isochronous cycle time. See the [CYCLE_TIME](https://learn.microsoft.com/windows-hardware/drivers/ddi/1394/ns-1394-_cycle_time) entry for details.
