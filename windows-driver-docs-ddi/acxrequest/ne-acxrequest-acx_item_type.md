## Description

The **ACX_ITEM_TYPE** enumeration describes the possible types of items.

## Constants

### `AcxItemTypeDefault`

The default type.

### `AcxItemTypeCircuit`

A Circuit type.

### `AcxItemTypeStream`

A Stream type.

### `AcxItemTypeFactoryCircuit`

A FactoryCircuit type.

### `AcxItemTypePin`

A Pin type.

### `AcxItemTypeElement`

An Element type.

### `AcxItemTypeMax`

For internal validation, do not use.
