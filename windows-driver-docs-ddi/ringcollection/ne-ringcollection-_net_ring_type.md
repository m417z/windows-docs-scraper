# _NET_RING_TYPE enumeration

## Description

The **NET_RING_TYPE** enumeration defines types of net rings for a packet queue.

## Constants

### `NetRingTypePacket:0`

The [**NET_RING**](https://learn.microsoft.com/windows-hardware/drivers/ddi/ring/ns-ring-_net_ring) is a packet ring.

### `NetRingTypeFragment:1`

The [**NET_RING**](https://learn.microsoft.com/windows-hardware/drivers/ddi/ring/ns-ring-_net_ring) is a fragment ring.

### `NetRingTypeDataBuffer`
Reserved for system use.
