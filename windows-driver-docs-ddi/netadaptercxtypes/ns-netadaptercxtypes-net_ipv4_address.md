# NET_IPV4_ADDRESS union

## Description

The **NET_IPV4_ADDRESS** union specifies an IPv4 address.

## Members

### `Address`

A UINT32 value that represents the IPv4 address.

### `Value`

A UINT8 array that represents the IPv4 address.
