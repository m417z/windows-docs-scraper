# TAPE_INIT_DATA structure

## Description

**TAPE_INIT_DATA** is used only by legacy tape miniclass drivers. Use [**TAPE_INIT_DATA_EX**](https://learn.microsoft.com/windows-hardware/drivers/ddi/minitape/ns-minitape-_tape_init_data_ex) instead.
