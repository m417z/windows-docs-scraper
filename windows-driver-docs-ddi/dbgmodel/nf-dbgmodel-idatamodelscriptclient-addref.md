# IDataModelScriptClient::AddRef

## Description

Increments the reference count for an interface on an object. This method should be called for every new copy of a pointer to an interface on an object.

For more information, see [IUnknown::AddRef](https://learn.microsoft.com/windows/win32/api/unknwn/nf-unknwn-iunknown-addref) and [Introduction to COM](https://learn.microsoft.com/cpp/atl/introduction-to-com).

## Return value

This method returns ULONG.
