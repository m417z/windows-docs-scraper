## Description

Defines the kind of address space being used.

## Constants

### `Disjoint`

The address space is Disjoint.

### `Equal`

The address space is Equal.

### `Overlapping`

The address space is overlapping.

### `Subset`

The address space is a subset.

### `Superset`

The address space is a superset.
