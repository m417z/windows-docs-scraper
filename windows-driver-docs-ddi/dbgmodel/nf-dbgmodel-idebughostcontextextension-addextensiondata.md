## Description

The AddExtensionData method adds a set of extension data to a host context.

## Parameters

### `blobId`

The identifier for the blob.

### `dataSize`

The size of the data to be added.

### `data`

A pointer to the data to be added.

## Return value

This method returns ULONG.
