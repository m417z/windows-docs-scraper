# _DsmSetLoadBalancePolicy_OUT structure

## Description

The **DsmSetLoadBalancePolicy_OUT** structure reports the output parameter of the [DsmSetLoadBalancePolicy](https://learn.microsoft.com/windows-hardware/drivers/storage/dsmsetloadbalancepolicy) method.

## Members

### `Status`

The status of the [DsmSetLoadBalancePolicy](https://learn.microsoft.com/windows-hardware/drivers/storage/dsmsetloadbalancepolicy) operation.
