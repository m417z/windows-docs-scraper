# _DsmSetLoadBalancePolicyALUA_OUT structure

## Description

The **DsmSetLoadBalancePolicyALUA_OUT** structure reports the output of the [DsmSetLoadBalancePolicyALUA](https://learn.microsoft.com/windows-hardware/drivers/storage/dsmsetloadbalancepolicyalua) method.

## Members

### `Status`

The status of the [DsmSetLoadBalancePolicyALUA](https://learn.microsoft.com/windows-hardware/drivers/storage/dsmsetloadbalancepolicyalua) operation.
