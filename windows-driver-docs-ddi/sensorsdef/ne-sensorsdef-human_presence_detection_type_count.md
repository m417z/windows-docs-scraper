## Description

The **HUMAN_PRESENCE_DETECTION_TYPE_COUNT** enumeration represents the number of human presence detection types in the [HUMAN_PRESENCE_DETECTION_TYPE](https://learn.microsoft.com/windows-hardware/drivers/ddi/sensorsdef/ne-sensorsdef-human_presence_detection_type) enum.

## Constants

### `HumanPresenceDetectionTypeCount`

Indicates the number of types in the **HUMAN_PRESENCE_DETECTION_TYPE** enum.
