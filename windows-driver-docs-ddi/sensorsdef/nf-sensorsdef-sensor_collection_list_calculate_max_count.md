# SENSOR_COLLECTION_LIST_CALCULATE_MAX_COUNT function

## Description

This function calculates the number of [SENSOR_VALUE_PAIR](https://learn.microsoft.com/windows-hardware/drivers/ddi/sensorsdef/ns-sensorsdef-sensor_value_pair) elements in a [SENSOR_COLLECTION_LIST](https://learn.microsoft.com/windows-hardware/drivers/ddi/sensorsdef/ns-sensorsdef-sensor_collection_list) structure.

## Parameters

### `pCollectionList` [in]

A pointer to a [SENSOR_COLLECTION_LIST](https://learn.microsoft.com/windows-hardware/drivers/ddi/sensorsdef/ns-sensorsdef-sensor_collection_list) structure.

## Return value

The **SENSOR_COLLECTION_LIST_CALCULATE_MAX_COUNT** function returns a ULONG value.
