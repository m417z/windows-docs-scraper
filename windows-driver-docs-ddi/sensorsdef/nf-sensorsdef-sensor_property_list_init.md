# SENSOR_PROPERTY_LIST_INIT function

## Description

This function initializes a [SENSOR_PROPERTY_LIST](https://learn.microsoft.com/windows-hardware/drivers/ddi/sensorsdef/ns-sensorsdef-sensor_property_list) structure.

## Parameters

### `pPropertyList` [out]

The list of PROPERTYKEY elements.

### `PropertyListSize` [in]

The size of pPropertyList.
