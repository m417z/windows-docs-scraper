# _UCX_USBDEVICE_CHARACTERISTIC_TYPE enumeration

## Description

Defines values that indicates the type of device characteristic.

## Constants

### `UCX_USBDEVICE_CHARACTERISTIC_TYPE_PATH_DELAY`

The type of characteristic of the device.
