## Description

Defines values for FLDR and PLDR trigger resets.

## Constants

### `UcxUsbDeviceRecoverActionFunctionLevelDeviceReset`

### `UcxUsbDeviceRecoverActionPlatformLevelDeviceReset`

PLDR trigger reset.

### `UcxUsbDeviceRecoverActionNone`

None.
