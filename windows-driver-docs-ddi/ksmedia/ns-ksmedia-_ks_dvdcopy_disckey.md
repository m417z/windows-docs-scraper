# _KS_DVDCOPY_DISCKEY structure

## Description

The KS_DVDCOPY_DISCKEY structure is used to describe the disc key information for the DVD copyright protection authentication process.

## Members

### `DiscKey`

Specifies the DVD's disc key.

## Remarks

The KS_DVDCOPY_DISCKEY structure is used by the [KSPROPERTY_DVDCOPY_DISC_KEY](https://learn.microsoft.com/windows-hardware/drivers/stream/ksproperty-dvdcopy-disc-key) property.

For more information, see [DVD Copyright Protection](https://learn.microsoft.com/windows-hardware/drivers/stream/dvd-copyright-protection).
