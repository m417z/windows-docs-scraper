# _KSPROPERTY_SPPAL structure

## Description

The KSPROPERTY_SPPAL structure is used to describe the palette of a subpicture display.

## Members

### `sppal`

Specifies an array of 16 YUV color values.

## Remarks

The KSPROPERTY_SPPAL structure is used by the [KSPROPERTY_DVDSUBPIC_PALETTE](https://learn.microsoft.com/windows-hardware/drivers/stream/ksproperty-dvdsubpic-palette) property.
