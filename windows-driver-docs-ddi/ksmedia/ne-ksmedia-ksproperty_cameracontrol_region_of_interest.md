# KSPROPERTY_CAMERACONTROL_REGION_OF_INTEREST enumeration

## Description

Used to specify a camera region of interest.

## Constants

### `KSPROPERTY_CAMERACONTROL_REGION_OF_INTEREST_PROPERTY_ID`

The [KSPROPERTY_CAMERACONTROL_REGION_OF_INTEREST_PROPERTY](https://learn.microsoft.com/windows-hardware/drivers/stream/ksproperty-cameracontrol-region-of-interest-property) property is used to specify a region of interest.
