# _tagKSTOPOLOGY_ENDPOINTID structure

## Description

The **KSTOPOLOGY_ENDPOINTID** structure specifies the name and the pin ID of a topology endpoint.

## Members

### `TopologyName`

The name of the topology endpoint.

### `PinId`

The pin ID of the topology endpoint.
