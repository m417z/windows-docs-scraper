# tagKS_DATAFORMAT_VIDEOINFOHEADER2 structure

## Description

The KS_DATAFORMAT_VIDEOINFOHEADER2 structure describes a video stream that includes settings for bob or weave.

## Members

### `DataFormat`

Specifies the data format.

### `VideoInfoHeader2`

Specifies the details of the video stream, including options for specifying bob or weave settings.

## Remarks

Minidrivers that do not need to specify bob or weave settings may use the [KS_DATAFORMAT_VIDEOINFOHEADER](https://learn.microsoft.com/windows-hardware/drivers/ddi/ksmedia/ns-ksmedia-tagks_dataformat_videoinfoheader) structure.
