## Description

**KSCAMERA_EXTENDEDPROP_DIGITALWINDOW_CONFIGCAPSHEADER** contains size and count information for the **KSCAMERA_EXTENDEDPROP_DIGITALWINDOW_CONFIGCAPSHEADER** and [KSCAMERA_EXTENDEDPROP_DIGITALWINDOW_CONFIGCAPS](https://learn.microsoft.com/windows-hardware/drivers/ddi/ksmedia/ns-ksmedia-kscamera_extendedprop_digitalwindow_configcaps) structures.

## Members

### `Size`

The size in bytes of this structure plus all other structures that follow.

### `Count`

The number of **KSCAMERA_EXTENDEDPROP_DIGITAL_WINDOW_CONFIGCAPS** structures that follow.
