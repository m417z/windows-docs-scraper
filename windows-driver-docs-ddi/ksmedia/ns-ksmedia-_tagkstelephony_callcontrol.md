# _tagKSTELEPHONY_CALLCONTROL structure

## Description

The **KSTELEPHONY_CALLCONTROL** structure specifies the phone call type and control operation to use for the [KSPROPERTY_TELEPHONY_CALLCONTROL](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-telephony-callcontrol) property.

## Members

### `CallType`

Specifies the type of phone call (circuit-switched, LTE packet-switched, or WLAN packet-switched).

### `CallControlOp`

Specifies the call control operation (enable or disable).
