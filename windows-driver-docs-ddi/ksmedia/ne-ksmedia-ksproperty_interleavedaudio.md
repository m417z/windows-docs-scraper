# KSPROPERTY_INTERLEAVEDAUDIO enumeration

## Description

The **KSPROPERTY_INTERLEAVEDAUDIO** enumeration defines a constant which can be used by the Hardware Keyword Spotter pin to include loopback audio interleaved with the microphone audio.

## Constants

### `KSPROPERTY_INTERLEAVEDAUDIO_FORMATINFORMATION`

Specifies the ID of the [KSPROPERTY_INTERLEAVEDAUDIO_FORMATINFORMATION](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-interleavedaudio-formatinformation) property.
