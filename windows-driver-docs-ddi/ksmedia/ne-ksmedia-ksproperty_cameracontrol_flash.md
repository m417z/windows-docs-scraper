# KSPROPERTY_CAMERACONTROL_FLASH enumeration

## Description

Used to specify camera flash control.

## Constants

### `KSPROPERTY_CAMERACONTROL_FLASH_PROPERTY_ID`

The [KSPROPERTY_CAMERACONTROL_FLASH_PROPERTY](https://learn.microsoft.com/windows-hardware/drivers/stream/ksproperty-cameracontrol-flash-property) property is used to specify flash control.
