# _tagKSTOPOLOGY_ENDPOINTIDPAIR structure

## Description

The **KSTOPOLOGY_ENDPOINTIDPAIR** structure specifies the render and capture endpoint IDs to use for the [KSPROPERTY_TELEPHONY_ENDPOINTIDPAIR](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-telephony-endpointidpair) property.

## Members

### `RenderEndpoint`

Specifies the render endpoint ID.

### `CaptureEndpoint`

Specifies the capture endpoint ID.
