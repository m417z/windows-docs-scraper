# tagKSCAMERA_EXTENDEDPROP_ROI_CONFIGCAPSHEADER structure

## Description

This structure contains the header information for ROI capabilities.

## Members

### `Size`

The size of this header and all [KSCAMERA_EXTENDEDPROP_ROI_CONFIGCAPS](https://learn.microsoft.com/windows-hardware/drivers/ddi/ksmedia/ns-ksmedia-tagkscamera_extendedprop_roi_configcaps) structures that follow.

### `ConfigCapCount`

Contains the number of **KSCAMERA_EXTENDEDPROP_ROI_CONFIGCAPS** structures that follow.

### `Reserved`

Reserved for future use.
