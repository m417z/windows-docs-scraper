# KSWAVE_VOLUME structure

## Description

The KSWAVE_VOLUME structure is used to describe sample volume.

## Members

### `LeftAttenuation`

Specifies the amount of left attenuation.

### `RightAttenuation`

Specifies the amount of right attenuation.

## Remarks

This structure is used by the [KSPROPERTY_WAVE_VOLUME](https://learn.microsoft.com/windows-hardware/drivers/stream/ksproperty-wave-volume) property.
