# _KS_DVDCOPY_CHLGKEY structure

## Description

The KS_DVDCOPY_CHLGKEY structure is used to describe the challenge key information for the DVD copyright protection authentication process.

## Members

### `ChlgKey`

Specifies the DVD decoder minidriver's challenge key.

### `Reserved`

Reserved. Do not use.

## Remarks

The KS_DVDCOPY_CHLGKEY structure is used by the [KSPROPERTY_DVDCOPY_CHLG_KEY](https://learn.microsoft.com/windows-hardware/drivers/stream/ksproperty-dvdcopy-chlg-key) property.

For more information, see [DVD Copyright Protection](https://learn.microsoft.com/windows-hardware/drivers/stream/dvd-copyright-protection).
