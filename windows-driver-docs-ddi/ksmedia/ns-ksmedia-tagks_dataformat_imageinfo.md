# tagKS_DATAFORMAT_IMAGEINFO structure

## Description

Specifies an image data format that is used for an independent image pin (or stream).

## Members

### `DataFormat`

A [KSDATAFORMAT](https://learn.microsoft.com/windows-hardware/drivers/ddi/ks/ns-ks-ksdataformat) structure that specifies the data format of the image stream.

### `ImageInfoHeader`

A [KS_BITMAPINFOHEADER](https://learn.microsoft.com/windows-hardware/drivers/ddi/ksmedia/ns-ksmedia-tagks_bitmapinfoheader) structure that specifies image color and dimension information that the still image capture stream would provide.
