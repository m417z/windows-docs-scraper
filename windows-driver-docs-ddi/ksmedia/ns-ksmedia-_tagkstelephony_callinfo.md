# _tagKSTELEPHONY_CALLINFO structure

## Description

The **KSTELEPHONY_CALLINFO** structure specifies the type and state of a phone call for the [KSPROPERTY_TELEPHONY_CALLINFO](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-telephony-callinfo) property.

## Members

### `CallType`

Specifies the type of phone call (circuit-switched, LTE packet-switched, or WLAN packet-switched).

### `CallState`

Specifies the state of the phone call.
