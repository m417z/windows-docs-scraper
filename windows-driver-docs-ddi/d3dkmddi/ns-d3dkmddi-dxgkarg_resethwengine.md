## Description

The **DXGKARG_RESETHWENGINE** structure contains arguments used to [reset the hardware engine](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmddi/nc-d3dkmddi-dxgkddi_resethwengine).

## Members

### `NodeOrdinal`

[in] GPU node ordinal for the GPU engine that is being requested to reset.

### `EngineOrdinal`

[in] GPU engine ordinal being requested to reset.
