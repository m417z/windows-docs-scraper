## Description

The **DXGKARG_NOTIFYWORKSUBMISSION_FLAGS** structure specifies work submission flags for [**DxgkDdiNotifyWorkSubmission**](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmddi/nc-d3dkmddi-dxgkddi_notifyworksubmission).

## Members

### `Reserved`

Reserved; set to zero.

### `Value`

An alternative way to access the structure members.

## Remarks

For more information, see [User-mode work submission](https://learn.microsoft.com/windows-hardware/drivers/display/user-mode-work-submission).
