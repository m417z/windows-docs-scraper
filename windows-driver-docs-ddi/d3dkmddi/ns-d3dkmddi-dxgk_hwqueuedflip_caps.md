## Description

The **DXGK_HWQUEUEDFLIP_CAPS** structure describes hardware flip queue capabilities.

## Members

### `Reserved`

Reserved for system use.

### `Value`

An alternative way to access the union's bits.

## Remarks

See [Hardware flip queue](https://learn.microsoft.com/windows-hardware/drivers/display/hardware-flip-queue) for more information.
