## Description

The **DXGKDDI_KERNELMODETESTINGINTERFACE** structure contains pointers to functions that are used for the kernel-mode testing interface.

## Members

### `pfnBuildTestCommandBuffer`

Pointer to the [**DXGKDDI_BUILDTESTCOMMANDBUFFER**](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmddi/nc-d3dkmddi-dxgkddi_buildtestcommandbuffer) function.

## Remarks

For more information, see [Kernel-mode testing](https://learn.microsoft.com/windows-hardware/drivers/display/kernel-mode-testing-of-wddm-features).
