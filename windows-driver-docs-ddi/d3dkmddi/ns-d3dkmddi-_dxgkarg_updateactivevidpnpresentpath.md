# _DXGKARG_UPDATEACTIVEVIDPNPRESENTPATH structure

## Description

The DXGKARG_UPDATEACTIVEVIDPNPRESENTPATH structure contains a [D3DKMDT_VIDPN_PRESENT_PATH](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmdt/ns-d3dkmdt-_d3dkmdt_vidpn_present_path) structure, which contains arguments for the [DxgkDdiUpdateActiveVidPnPresentPath](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmddi/nc-d3dkmddi-dxgkddi_updateactivevidpnpresentpath) function.

## Members

### `VidPnPresentPathInfo`

A [D3DKMDT_VIDPN_PRESENT_PATH](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmdt/ns-d3dkmdt-_d3dkmdt_vidpn_present_path) structure that contains the arguments required by [DxgkDdiUpdateActiveVidPnPresentPath](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmddi/nc-d3dkmddi-dxgkddi_updateactivevidpnpresentpath).
