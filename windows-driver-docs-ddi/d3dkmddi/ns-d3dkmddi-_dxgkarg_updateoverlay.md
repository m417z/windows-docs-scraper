# _DXGKARG_UPDATEOVERLAY structure

## Description

The DXGKARG_UPDATEOVERLAY structure describes parameters for modifying an overlay.

## Members

### `OverlayInfo` [in]

A [DXGK_OVERLAYINFO](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmddi/ns-d3dkmddi-_dxgk_overlayinfo) structure that contains parameters that are required to modify the overlay.
