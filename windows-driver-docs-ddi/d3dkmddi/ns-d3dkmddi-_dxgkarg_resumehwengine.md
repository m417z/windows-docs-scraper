# DXGKARG_RESUMEHWENGINE structure

## Description

The **DXGKARG_RESUMEHWENGINE** structure contains arguments used to [resume the hardware engine](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmddi/nc-d3dkmddi-dxgkddi_resumehwengine).

## Members

### `NodeOrdinal`

[in] GPU node ordinal for the GPU engine that is being requested to resume.

### `EngineOrdinal`

[in] GPU engine ordinal being requested to resume.
