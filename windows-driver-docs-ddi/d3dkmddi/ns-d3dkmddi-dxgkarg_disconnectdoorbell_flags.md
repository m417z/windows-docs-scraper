## Description

The **DXGKARG_DISCONNECTDOORBELL_FLAGS** structure specifies doorbell disconnection flags for [**DxgkDdiDisconnectDoorbell**](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmddi/nc-d3dkmddi-dxgkddi_disconnectdoorbell).

## Members

### `Reserved`

Reserved; set to zero.

### `Value`

An alternative way to access the structure members.

For more information, see [User-mode work submission](https://learn.microsoft.com/windows-hardware/drivers/display/user-mode-work-submission).
