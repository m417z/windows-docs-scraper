## Description

The **DXGKARGCB_ISFEATUREENABLED2_FLAGS** structure contains optional flags for querying feature enabled state.

## Members

### `Reserved`

Reserved for system use.

### `Value`

An alternative way to access members of this structure.

## Remarks

For more information, see [Querying WDDM feature support and enablement](https://learn.microsoft.com/windows-hardware/drivers/display/querying-wddm-feature-support-and-enablement).
