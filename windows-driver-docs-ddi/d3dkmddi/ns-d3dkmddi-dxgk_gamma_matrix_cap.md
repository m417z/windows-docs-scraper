# DXGK_GAMMA_MATRIX_CAP structure

## Description

The stage capability for a matrix lookup table.

## Members

### `MatrixSizeX`

Size of x-dimension of the matrix.

### `MatrixSizeY`

Size of y-dimension of the matrix.

### `Value`

### `GammaDataCap`

Pointer to a [DKGK_GAMMA_DATA_CAP](https://learn.microsoft.com/windows-hardware/drivers/ddi/d3dkmddi/ns-d3dkmddi-dkgk_gamma_data_cap) structure.
