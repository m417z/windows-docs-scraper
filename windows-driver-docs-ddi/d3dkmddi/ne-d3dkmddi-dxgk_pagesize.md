## Description

The **DXGK_PAGESIZE** enumeration defines a compressed 16-bit representation of a page size. It represents powers of two, ranging from 4 KB to 128 MB.

## Constants

### `DXGK_PAGESIZE_4KB`

Represents a page size of 4 KB.
