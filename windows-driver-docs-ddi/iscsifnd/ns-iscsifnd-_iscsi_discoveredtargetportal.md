# _ISCSI_DiscoveredTargetPortal structure

## Description

The ISCSI_DiscoveredTargetPortal structure provides information that is associated with a discovered target portal.

## Members

### `Socket`

The socket number of the portal.

### `Address`

The network address of the portal.

### `SymbolicName`

A wide character string that indicates the portal's symbolic name.

## Remarks

The WMI tool suite automatically generates a declaration of the ISCSI_DiscoveredTargetPortal structure when it compiles the [ISCSI_DiscoveredTargetPortal WMI Class](https://learn.microsoft.com/windows-hardware/drivers/storage/iscsi-discoveredtargetportal-wmi-class) in *Discover.mof*.
