# DsmSetLoadBalancePolicy function

The **DsmSetLoadBalancePolicy** method is used to set the DSM load balance policy.

## Parameters

*LoadBalancePolicy*
A [**DsmSetLoadBalancePolicy\_OUT**](https://learn.microsoft.com/windows-hardware/drivers/ddi/mpiodisk/ns-mpiodisk-_dsmsetloadbalancepolicy_out) structure.

*Status*
The status of the operation.

## Return value

None

## Remarks

This WMI method belongs to the [DSM\_LB\_Operations](https://learn.microsoft.com/windows-hardware/drivers/storage/dsm-lb-operations-wmi-class) WMI class.

## Requirements

|  |  |
| --- | --- |
