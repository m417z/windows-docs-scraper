# RefreshInformation function

The **RefreshInformation** WMI method updates all tables for the HBA that corresponds to the calling instance object.

## Parameters

This function has no parameters.

## Return value

Not applicable.

## Remarks

The **RefreshInformation** WMI method refreshes port attribute data that is retrieved by the following WMI methods:

[**GetDiscoveredPortAttributes**](https://learn.microsoft.com/windows-hardware/drivers/storage/getdiscoveredportattributes)

[**GetFC3MgmtInfo**](https://learn.microsoft.com/windows-hardware/drivers/storage/getfc3mgmtinfo)

[**GetFC4Statistics**](https://learn.microsoft.com/windows-hardware/drivers/storage/getfc4statistics)

[**GetFCPStatistics**](https://learn.microsoft.com/windows-hardware/drivers/storage/getfcpstatistics)

[**GetPortAttributesByWWN**](https://learn.microsoft.com/windows-hardware/drivers/storage/getportattributesbywwn)

This WMI method belongs to the [MSFC\_HBAAdapterMethods WMI Class](https://learn.microsoft.com/windows-hardware/drivers/storage/msfc-hbaadaptermethods-wmi-class).
