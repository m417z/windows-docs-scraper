# KSPROPERTY\_SOUNDDETECTOR enumeration

The **KSPROPERTY\_SOUNDDETECTOR** enumeration defines constants that are used to register a filter for an audio capture device that also supports a detector.

## Constants

**KSPROPERTY\_SOUNDDETECTOR\_SUPPORTEDPATTERNS**
Specifies the ID for the [**KSPROPERTY\_SOUNDDETECTOR\_SUPPORTEDPATTERNS**](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-sounddetector-supportedpatterns) property.

**KSPROPERTY\_SOUNDDETECTOR\_PATTERNS**
Specifies the ID for the [**KSPROPERTY\_SOUNDDETECTOR\_PATTERNS**](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-sounddetector-patterns) property.

**KSPROPERTY\_SOUNDDETECTOR\_ARMED**
Specifies the ID for the [**KSPROPERTY\_SOUNDDETECTOR\_ARMED**](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-sounddetector-armed) property.

**KSPROPERTY\_SOUNDDETECTOR\_MATCHRESULT**
Specifies the ID for the [**KSPROPERTY\_SOUNDDETECTOR\_MATCHRESULT**](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-sounddetector-matchresult) property.
