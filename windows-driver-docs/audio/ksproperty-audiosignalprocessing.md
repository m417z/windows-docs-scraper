# KSPROPERTY\_AUDIOSIGNALPROCESSING enumeration

The KSPROPERTY\_AUDIOSIGNALPROCESSING enumeration defines a constant that is used by audio drivers in connection with audio processing modes on pins.

## Constants

**KSPROPERTY\_AUDIOSIGNALPROCESSING\_MODES**
Specifies the ID for the [**KSPROPERTY\_AUDIOSIGNALPROCESSING\_MODES**](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-audiosignalprocessing-modes) property.
