# KSPROPERTY\_JACK enumeration

The `KSPROPERTY_JACK` enumeration defines new property IDs that are used by audio jack structures.

## Constants

**KSPROPERTY\_JACK\_DESCRIPTION**
Specifies the ID for the [**KSPROPERTY\_JACK\_DESCRIPTION**](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-jack-description) property.

**KSPROPERTY\_JACK\_DESCRIPTION2**
Specifies the ID for the [**KSPROPERTY\_JACK\_DESCRIPTION2**](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-jack-description2) property.

**KSPROPERTY\_JACK\_SINK\_INFO**
Specifies the ID for the [**KSPROPERTY\_JACK\_SINK\_INFO**](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-jack-sink-info) property.

**KSPROPERTY\_JACK\_CONTAINERID**
Specifies the ID for the [**KSPROPERTY\_JACK\_CONTAINERID**](https://learn.microsoft.com/windows-hardware/drivers/audio/ksproperty-jack-containerid) property.
