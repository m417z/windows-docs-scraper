# DlpNotifyPreOpenDocument function

Provides the system with information about a document before an open operation is initiated.

## Parameters

*DocumentInfo* \[in\]

A pointer to a [PDLP_DOCUMENT_INFO](https://learn.microsoft.com/windows/win32/lwef/endpointdlp-dlp_document_info) structure containing information about the document to be opened.

## Return value

Return void.
