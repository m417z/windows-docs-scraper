# MPVERSION\_INFO structure

Version information for the malware protection manager's components.

## Members

**Product**

Type: **[**MPCOMPONENT\_VERSION**](https://learn.microsoft.com/windows/win32/lwef/mpcomponent-version)**

Product version.

**Service**

Type: **[**MPCOMPONENT\_VERSION**](https://learn.microsoft.com/windows/win32/lwef/mpcomponent-version)**

Service component version.

**FileSystemFilter**

Type: **[**MPCOMPONENT\_VERSION**](https://learn.microsoft.com/windows/win32/lwef/mpcomponent-version)**

File system filter component version.

**Engine**

Type: **[**MPCOMPONENT\_VERSION**](https://learn.microsoft.com/windows/win32/lwef/mpcomponent-version)**

Engine component version.

**ASSignature**

Type: **[**MPCOMPONENT\_VERSION**](https://learn.microsoft.com/windows/win32/lwef/mpcomponent-version)**

Antispyware signature component version.

**AVSignature**

Type: **[**MPCOMPONENT\_VERSION**](https://learn.microsoft.com/windows/win32/lwef/mpcomponent-version)**

Antivirus signature component version.

**NISEngine**

Type: **[**MPCOMPONENT\_VERSION**](https://learn.microsoft.com/windows/win32/lwef/mpcomponent-version)**

NIS engine version.

**NISSignature**

Type: **[**MPCOMPONENT\_VERSION**](https://learn.microsoft.com/windows/win32/lwef/mpcomponent-version)**

NIS Signature component version.

**Reserved**

Type: **[**MPCOMPONENT\_VERSION**](https://learn.microsoft.com/windows/win32/lwef/mpcomponent-version)\[4\]**

Reserved fields.
