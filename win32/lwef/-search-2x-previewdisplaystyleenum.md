# PreviewDisplayStyle enumeration

> [!NOTE]
> Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for Windows XP and Windows Server 2003. On later releases, use the [Windows Search API](https://learn.microsoft.com/windows/win32/search/-search-reference-entry-page) instead.

Used by [**IResultsViewer::PreviewStyle**](https://learn.microsoft.com/windows/win32/lwef/-search-2x-iresultsviewer-previewstyle) to set or determine the display style currently being used.

## Constants

**AutoPreview**

Indicates AutoPreview.

**RightPreview**

Indicates RightPreview.

**BottomPreview**

Indicates BottomPreview.

**NoPreview**

Indicates NoPreview.
