# ViewContents enumeration

> [!NOTE]
> Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for Windows XP and Windows Server 2003. On later releases, use the [Windows Search API](https://learn.microsoft.com/windows/win32/search/-search-reference-entry-page) instead.

Used by [**IResultsViewer::Contents**](https://learn.microsoft.com/windows/win32/lwef/-search-2x-iresultsviewer-contents) to indicate or set how the current query return set is displayed.

## Constants

**ResultsDisplayed**

Indicates the type of contents being displayed in the results view.

**ShellViewDisplayed**

Indicates the type of contents being displayed in the results view is currently set to the Shell view.

**WebBrowserDisplayed**

Indicates the type of contents being displayed in the results view is currently set to the browser view.
