# MPSCAN\_RESOURCES structure

Resource information passed during a scan operation.

## Members

**dwResourceCount**

Type: **DWORD**

Count of resources.

**pResourceList**

Type: **PMPRESOURCE\_INFO**

Array of resources. See [**MPRESOURCE\_INFO**](https://learn.microsoft.com/windows/win32/lwef/mpresource-info).
