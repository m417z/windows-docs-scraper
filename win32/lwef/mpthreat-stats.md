# MPTHREAT\_STATS structure

Threat-related statistics.

## Members

**ThreatCount**

Type: **UINT**

Number of threats detected.

**SuspiciousThreatCount**

Type: **UINT**

Number of threats detected with behavior monitoring.

**Reserved**

Type: **UINT\[4\]**

Fields reserved for future use.
