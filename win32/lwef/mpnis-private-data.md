# MPNIS\_PRIVATE\_DATA structure

Private NIS notifications.

## Members

**dwNotificationType**

Type: **DWORD**

Type of notification.

**cbDataSize**

Type: **DWORD**

Size of reserved data, in bytes.

**pbData**

Type: **BYTE\***

Pointer to reserved data.
