# ResultsDisplayStyle enumeration

> [!NOTE]
> Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for Windows XP and Windows Server 2003. On later releases, use the [Windows Search API](https://learn.microsoft.com/windows/win32/search/-search-reference-entry-page) instead.

Used by [**IResultsViewer::ResultsStyle**](https://learn.microsoft.com/windows/win32/lwef/-search-2x-iresultsviewer-resultsstyle) to set or determine how results are displayed.

## Constants

**SmallIconResults**

Indicates Results are displayed as small icons.

**LargeIconResults**

Indicates Results are displayed as large icons.
