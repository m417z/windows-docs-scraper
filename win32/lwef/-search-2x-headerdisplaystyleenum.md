# HeaderDisplayStyle enumeration

> [!NOTE]
> Windows Desktop Search 2.x is an obsolete technology that was originally available as an add-in for Windows XP and Windows Server 2003. On later releases, use the [Windows Search API](https://learn.microsoft.com/windows/win32/search/-search-reference-entry-page) instead.

Used by [**IResultsViewer::HeaderStyle**](https://learn.microsoft.com/windows/win32/lwef/-search-2x-iresultsviewer-headerstyle) to set or return the current header style.

## Constants

**FullHeader**

Indicates or sets the use of full headers.

**CompactHeader**

Indicates or sets the use of compact headers.

**NoHeader**

Indicates or sets the use of no headers.
