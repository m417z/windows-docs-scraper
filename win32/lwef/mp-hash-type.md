# MP\_HASH\_TYPE enumeration

Possible hash types.

## Constants

**MP\_HASH\_TYPE\_NONE**

No hash.

**MP\_HASH\_TYPE\_CRC32**

CRC32

**MP\_HASH\_TYPE\_MD5**

MD5

**MP\_HASH\_TYPE\_SHA1**

SHA1

**MP\_HASH\_TYPE\_SHA256**

SHA 256
