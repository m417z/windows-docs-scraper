# CreateInkDivider function

Instantiates an implementation of the [**InkDivider**](https://learn.microsoft.com/windows/win32/tablet/inkdivider-class) interface and returns its handle.

This function is not intended to be used by application code.

## Parameters

This function has no parameters.

## Return value

Returns a handle to the created object.

## Remarks

This class inherits from the [**InkDivider**](https://learn.microsoft.com/windows/win32/tablet/inkdivider-class) base class and implements virtual functions.

| Requirement | Value |
|-------------------------------------|---------------------------------------------------------------------------------------|
| Minimum supported client<br> | Windows XP Tablet PC Edition \[desktop apps only\]<br> |
| Minimum supported server<br> | None supported<br> |
| Library<br> | InkDiv.dll |
