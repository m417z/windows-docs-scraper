# SMS\_MESSAGE\_TYPES enumeration

The **SMS\_MESSAGE\_TYPES** enumeration type describes the content type of a short message service (SMS) message.

## Constants

**SMS\_TEXT\_MESSAGE**

A text message.

**SMS\_BINARY\_MESSAGE**

A binary message.

## Remarks

This enumeration is used by the [**WPD\_COMMAND\_SMS\_SEND Command**](https://learn.microsoft.com/windows/win32/wpd_sdk/wpd-command-sms-send-command).
