# D3DX\_R8G8B8A8\_SINT\_to\_INT4 function

Unpacks DXGI\_FORMAT\_R8G8B8A8\_SINT shader data to an XMINT4.

## Parameters

*packedInput*

The packed shader data.

## Return value

The unpacked shader data.

## Requirements

| Requirement | Value |
|-------------------|--------------------------------------------------------------------------------------------------------|
| Header<br> | D3DX\_DXGIFormatConvert.inl |
