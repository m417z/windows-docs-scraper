# D3DX\_INT2\_to\_R16G16\_SINT function

Packs the given XMINT2 back into a DXGI\_FORMAT\_R16G16\_SINT.

## Parameters

*unpackedInput*

The shader data to pack.

## Return value

The packed shader data.

## Requirements

| Requirement | Value |
|-------------------|--------------------------------------------------------------------------------------------------------|
| Header<br> | D3DX\_DXGIFormatConvert.inl |
