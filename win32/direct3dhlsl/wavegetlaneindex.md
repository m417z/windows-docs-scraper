# WaveGetLaneIndex function

Returns the index of the current lane within the current wave.

## Parameters

This function has no parameters.

## Return value

The current lane index. The result will be between 0 and the result returned from [**WaveGetLaneCount**](https://learn.microsoft.com/windows/win32/direct3dhlsl/wavegetlanecount).

## Remarks

This function is supported from shader model 6.0 in all shader stages.
