# dst function

Calculates a distance vector.

## Parameters

*src0* \[in\]

Type: **[fVector](https://learn.microsoft.com/windows/win32/direct3dhlsl/dx-graphics-hlsl-vector)**

The first vector.

*src1* \[in\]

Type: **[fVector](https://learn.microsoft.com/windows/win32/direct3dhlsl/dx-graphics-hlsl-vector)**

The second vector.

## Return value

Type: **[fVector](https://learn.microsoft.com/windows/win32/direct3dhlsl/dx-graphics-hlsl-vector)**

The computed distance vector.

## Remarks

This intrinsic function provides the same functionality as the Vertex Shader instruction [dst](https://learn.microsoft.com/windows/win32/direct3dhlsl/dst---vs).
