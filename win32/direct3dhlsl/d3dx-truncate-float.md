# D3DX\_Truncate\_FLOAT function

Truncates a FLOAT value.

## Parameters

*\_V*

The value to truncate.

## Return value

The truncated value.

## Requirements

| Requirement | Value |
|-------------------|--------------------------------------------------------------------------------------------------------|
| Header<br> | D3DX\_DXGIFormatConvert.inl |
