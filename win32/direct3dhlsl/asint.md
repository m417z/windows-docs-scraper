# asint function

Interprets the bit pattern of an input value as an integer. For more information about the **asint** intrinsic function, see [**asint (DirectX HLSL)**](https://learn.microsoft.com/windows/win32/direct3dhlsl/dx-graphics-hlsl-asint).

## Parameters

*value* \[in\]

The input value.

## Return value

Type: **int**

The input interpreted as an integer.
