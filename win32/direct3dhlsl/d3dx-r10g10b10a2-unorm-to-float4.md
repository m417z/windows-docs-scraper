# D3DX\_R10G10B10A2\_UNORM\_to\_FLOAT4 function

Unpacks DXGI\_FORMAT\_R10G10B10A2\_UNORM shader data to an XMFLOAT4.

## Parameters

*packedInput*

The packed shader data.

## Return value

The unpacked shader data.

## Requirements

| Requirement | Value |
|-------------------|--------------------------------------------------------------------------------------------------------|
| Header<br> | D3DX\_DXGIFormatConvert.inl |
