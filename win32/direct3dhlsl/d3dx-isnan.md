# D3DX\_IsNan function

Determines if the value is a NaN (Not a Number).

## Parameters

*\_V*

The specified value.

## Return value

true if a NaN; otherwise false.

## Requirements

| Requirement | Value |
|-------------------|--------------------------------------------------------------------------------------------------------|
| Header<br> | D3DX\_DXGIFormatConvert.inl |
