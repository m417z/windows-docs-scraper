# D3DX\_Saturate\_FLOAT function

Retrieves the saturated value of the given FLOAT.

## Parameters

*\_V*

The value to saturate.

## Return value

The saturated value.

## Requirements

| Requirement | Value |
|-------------------|--------------------------------------------------------------------------------------------------------|
| Header<br> | D3DX\_DXGIFormatConvert.inl |
