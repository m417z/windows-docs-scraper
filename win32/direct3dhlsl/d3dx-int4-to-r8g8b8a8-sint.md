# D3DX\_INT4\_to\_R8G8B8A8\_SINT function

Packs the given XMINT4 back into a DXGI\_FORMAT\_R8G8B8A8\_SINT.

## Parameters

*unpackedInput*

The shader data to pack.

## Return value

The packed shader data.

## Requirements

| Requirement | Value |
|-------------------|--------------------------------------------------------------------------------------------------------|
| Header<br> | D3DX\_DXGIFormatConvert.inl |
