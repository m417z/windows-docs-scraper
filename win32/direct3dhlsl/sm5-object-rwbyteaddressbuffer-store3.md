# Store3 function

Sets three values.

## Parameters

*address* \[in\]

Type: **uint**

The input address in bytes, which must be a multiple of 4.

*values* \[in\]

Type: **uint3**

Three input values.

## Return value

This function does not return a value.

## Remarks

This function is supported for the following types of shaders:

| Vertex | Hull | Domain | Geometry | Pixel | Compute |
|--------|------|--------|----------|-------|---------|
| | | | | x | x |
