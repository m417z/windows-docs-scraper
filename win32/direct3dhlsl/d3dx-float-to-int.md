# D3DX\_FLOAT\_to\_INT function

Converts a FLOAT value to INT.

## Parameters

*\_V*

The v value.

*\_Scale*

The scale value.

## Return value

The converted FLOAT value

## Requirements

| Requirement | Value |
|-------------------|--------------------------------------------------------------------------------------------------------|
| Header<br> | D3DX\_DXGIFormatConvert.inl |
