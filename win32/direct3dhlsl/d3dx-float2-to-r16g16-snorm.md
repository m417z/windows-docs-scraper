# D3DX\_FLOAT2\_to\_R16G16\_SNORM function

Packs the given XMFLOAT2 back into a DXGI\_FORMAT\_R16G16\_SNORM.

## Parameters

*unpackedInput*

The unpacked shader data.

## Return value

The packed shader data.

## Requirements

| Requirement | Value |
|-------------------|--------------------------------------------------------------------------------------------------------|
| Header<br> | D3DX\_DXGIFormatConvert.inl |
