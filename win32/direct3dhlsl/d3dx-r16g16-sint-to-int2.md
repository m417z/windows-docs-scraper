# D3DX\_R16G16\_SINT\_to\_INT2 function

Unpacks DXGI\_FORMAT\_R16G16\_SINT shader data to an XMINT2.

## Parameters

*packedInput*

The packed shader data.

## Return value

The unpacked shader data.

## Requirements

| Requirement | Value |
|-------------------|--------------------------------------------------------------------------------------------------------|
| Header<br> | D3DX\_DXGIFormatConvert.inl |
