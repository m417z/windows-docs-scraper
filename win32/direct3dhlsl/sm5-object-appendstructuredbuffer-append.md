# Append function

Appends a value to the end of the buffer.

## Parameters

*value* \[in\]

Type: **T**

The input value.

## Return value

None

## Remarks

T can be any data type including structures.

This function is supported for the following types of shaders:

| Vertex | Hull | Domain | Geometry | Pixel | Compute |
|--------|------|--------|----------|-------|---------|
| x | x | x | x | x | x |
