Proxy function for the [**Commit**](https://learn.microsoft.com/windows/desktop/api/Wincodec/nf-wincodec-iwicfastmetadataencoder-commit) method.

## Parameters

*THIS\_PTR* \[in\]

Type: **[**IWICFastMetadataEncoder**](https://learn.microsoft.com/windows/desktop/api/Wincodec/nn-wincodec-iwicfastmetadataencoder)\***

Pointer to this [**IWICFastMetadataEncoder**](https://learn.microsoft.com/windows/desktop/api/Wincodec/nn-wincodec-iwicfastmetadataencoder) object.

## Return value

Type: **HRESULT**

If this function succeeds, it returns **S\_OK**. Otherwise, it returns an **HRESULT** error code.
