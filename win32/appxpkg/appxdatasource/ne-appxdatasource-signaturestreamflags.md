# SignatureStreamFlags enumeration

Defines constants that specify options for the [GetSignatureFileStream](https://learn.microsoft.com/windows/win32/appxpkg/appxdatasource/nf-appxdatasource-iappxstreamingdatasource4-getsignaturefilestream) method.

## Constants

|   |
| -- |
| `SignatureStreamFlagsNone`<br> Specifies no options. |
| `SignatureStreamFlagsAddP7xHeader`<br> Add a p7x header (defined in the **P7xEditor** class) to the stream. |

## Requirements

|   |   |
| ---- |:---- |
| **Header** | appxdatasource.h |
