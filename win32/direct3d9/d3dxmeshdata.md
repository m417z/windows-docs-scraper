# D3DXMESHDATA structure

Mesh data structure.

## Members

**Type**

Type: **[**D3DXMESHDATATYPE**](https://learn.microsoft.com/windows/win32/direct3d9/d3dxmeshdatatype)**

Defines the mesh data type. See [**D3DXMESHDATATYPE**](https://learn.microsoft.com/windows/win32/direct3d9/d3dxmeshdatatype).

**pMesh**

Type: **[**LPD3DXMESH**](https://learn.microsoft.com/windows/win32/direct3d9/id3dxmesh)**

Pointer to a mesh. See [**ID3DXMesh**](https://learn.microsoft.com/windows/win32/direct3d9/id3dxmesh).

**pPatchMesh**

Type: **LPD3DXPATCHMESH**

Pointer to a patch mesh. See [**ID3DXPatchMesh**](https://learn.microsoft.com/windows/win32/direct3d9/id3dxpatchmesh).
