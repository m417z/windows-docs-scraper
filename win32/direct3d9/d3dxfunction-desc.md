# D3DXFUNCTION\_DESC structure

Describes a function used by an effect.

## Members

**Name**

Type: **[**LPCSTR**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

Function name.

**Annotations**

Type: **[**UINT**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

Unused. This member will always be set to zero by [**GetFunctionDesc**](https://learn.microsoft.com/windows/win32/direct3d9/id3dxbaseeffect--getfunctiondesc).
