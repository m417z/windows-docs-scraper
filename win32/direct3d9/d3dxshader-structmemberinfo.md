# D3DXSHADER\_STRUCTMEMBERINFO structure

A helper structure containing member structure information.

## Members

**Name**

Type: **[**DWORD**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

Offset from the beginning of this structure, in bytes, to the string that contains the structure member name.

**TypeInfo**

Type: **[**DWORD**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

Offset from the beginning of this structure, in bytes, to the string that contains the type information. See [**D3DXSHADER\_TYPEINFO**](https://learn.microsoft.com/windows/win32/direct3d9/d3dxshader-typeinfo).
