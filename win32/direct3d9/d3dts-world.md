# D3DTS\_WORLD macro

Identifies the transformation matrix being set as the world transformation matrix.

## Parameters

This macro has no parameters.

## Return value

A [**D3DTRANSFORMSTATETYPE**](https://learn.microsoft.com/windows/win32/direct3d9/d3dtransformstatetype) equivalent to [**D3DTS\_WORLDMATRIX(0)**](https://learn.microsoft.com/windows/win32/direct3d9/d3dts-worldmatrix).

## Remarks

This macro is provided to facilitate porting existing applications to Direct3D 9.
