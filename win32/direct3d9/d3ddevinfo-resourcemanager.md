# D3DDEVINFO\_RESOURCEMANAGER structure

Resource usage statistics.

## Members

**stats**

Type: **[**D3DRESOURCESTATS**](https://learn.microsoft.com/windows/win32/direct3d9/d3dresourcestats)**

Array of resource statistics elements. See [**D3DRESOURCESTATS**](https://learn.microsoft.com/windows/win32/direct3d9/d3dresourcestats).

## Remarks

D3DRTYPECOUNT refers to the number of enumerations in [**D3DRESOURCETYPE**](https://learn.microsoft.com/windows/win32/direct3d9/d3dresourcetype).
