# D3DXCOLOR structure (D3dx9math.h)

> [!Note]
> The D3DX utility library is deprecated. We recommend that you use [DirectXMath](https://learn.microsoft.com/windows/win32/dxmath/pg-xnamath-migration-d3dx) instead.

Describes color values.

## Members

**r**

Type: **[**FLOAT**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

Red component of the color.

**g**

Type: **[**FLOAT**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

Green component of the color.

**b**

Type: **[**FLOAT**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

Blue component of the color.

**a**

Type: **[**FLOAT**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

Alpha component of the color.

## Remarks

C++ programmers can take advantage of operator overloading and type casting with the [**D3DXCOLOR Extensions**](https://learn.microsoft.com/windows/win32/direct3d9/d3dxcolor-extensions), which implement overloaded constructors, as well as assignment, unary, and binary (including equality) operators.
