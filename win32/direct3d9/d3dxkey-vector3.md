# D3DXKEY\_VECTOR3 structure

Describes a vector key for use in key frame animation. It specifies a vector at a given time. This is used for scale and translation keys.

## Members

**Time**

Type: **[**FLOAT**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

Key frame time stamp.

**Value**

Type: **[**D3DXVECTOR3**](https://learn.microsoft.com/windows/win32/direct3d9/d3dxvector3)**

[**D3DXVECTOR3**](https://learn.microsoft.com/windows/win32/direct3d9/d3dxvector3) 3D vector that supplies scale and/or translation values.
