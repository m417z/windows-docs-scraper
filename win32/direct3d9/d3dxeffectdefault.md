# D3DXEFFECTDEFAULT structure

Effect default parameters.

## Members

**pParamName**

Type: **LPSTR**

Parameter name.

**Type**

Type: **[**D3DXEFFECTDEFAULTTYPE**](https://learn.microsoft.com/windows/win32/direct3d9/d3dxeffectdefaulttype)**

Data type in pValue. For more information, see [**D3DXEFFECTDEFAULTTYPE**](https://learn.microsoft.com/windows/win32/direct3d9/d3dxeffectdefaulttype)

**NumBytes**

Type: **[**DWORD**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

Size, in bytes, of the data pointed to by pValue.

**pValue**

Type: **[**LPVOID**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

Pointer to the memory location that contains the data.
