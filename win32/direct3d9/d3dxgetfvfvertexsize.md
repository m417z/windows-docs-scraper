# D3DXGetFVFVertexSize function

Returns the size of a vertex for a flexible vertex format (FVF).

## Parameters

*FVF* \[in\]

Type: **[**DWORD**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

FVF to be queried. A combination of [D3DFVF](https://learn.microsoft.com/windows/win32/direct3d9/d3dfvf).

## Return value

Type: **[**UINT**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

The FVF vertex size, in bytes.
