# DSTORAGE_SOURCE_MEMORY structure (dstorage.h)

Describes a source for a request with *SourceType* [DSTORAGE_REQUEST_SOURCE_MEMORY](https://learn.microsoft.com/windows/win32/dstorage/dstorage/ne-dstorage-dstorage_request_source_type).

## Members

`Source`

Address of the source buffer to be read from.

`Size`

Number of bytes to read from the source buffer.
