# CAPICOM\_STORE\_SAVE\_AS\_TYPE enumeration

The **CAPICOM\_STORE\_SAVE\_AS\_TYPE** enumeration type indicates the encoding of a [*certificate store*](https://learn.microsoft.com/windows/win32/secgloss/c-gly).

## Members

| Member | Description | Value |
|------------------------------------------|-----------------------------------------------------|-------|
| **CAPICOM\_STORE\_SAVE\_AS\_SERIALIZED** | The store is saved in serialized format.<br> | 0 |
| **CAPICOM\_STORE\_SAVE\_AS\_PKCS7** | The store is saved in PKCS \#7 format.<br> | 1 |

## Requirements

| Requirement | Value |
|----------------------------|--------------------------------------------------------------------------------------|
| Redistributable<br> | CAPICOM 2.0 or later on Windows Server 2003 and Windows XP<br> |
| Header<br> | Capicom.h |
