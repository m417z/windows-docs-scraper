# ENCRYPTION_PROTECTOR_LIST structure

Under WIP policy, includes all the identities a file is protected to.

## Members

### NumberOfProtectors

Number of protectors in the *Protectors* array.

### Protectors

A pointer to an [ENCRYPTION_PROTECTOR](https://learn.microsoft.com/windows/win32/devnotes/encryption_protector-structure) structure.

## Requirements

|-------------------------------------|-----------------------------------------|

| Minimum supported server | Windows 10 |
