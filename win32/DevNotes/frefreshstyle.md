# FRefreshStyle function

Reloads color settings from registry.

## Parameters

This function has no parameters.

## Return value

Returns TRUE on success; otherwise, FALSE.

## Remarks

This function is not associated with an import library or header file; it must be called using the [**LoadLibrary**](https://learn.microsoft.com/windows/win32/devnotes/-loadlibrary) and [**GetProcAddress**](https://learn.microsoft.com/windows/win32/devnotes/-getprocaddress-) functions.

## Requirements

| Requirement | Value |
|----------------|-----------------------------------------------------------------------------------------|
| DLL<br> | Imeshare.dll |
