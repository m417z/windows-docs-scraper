# SdbReadApphelpDetailsData function

Retrieves data from the Apphelp details database.

## Parameters

*pdb* \[in\]

A handle to the Apphelp details database, Apphelp.sdb.

*pData* \[out\]

An [**APPHELP\_DATA**](https://learn.microsoft.com/windows/win32/devnotes/apphelp-data) structure.

## Return value

The function returns **TRUE** on success or **FALSE** on failure.

## Requirements

| Requirement | Value |
|-------------------------------------|----------------------------------------------------------------------------------------|
| Minimum supported client<br> | Windows Vista \[desktop apps only\]<br> |
| Minimum supported server<br> | Windows Server 2008 \[desktop apps only\]<br> |
| DLL<br> | Apphelp.dll |
