# \_PeekMessage function

\[This function is a wrapper over the **PeekMessage** function. This function may be altered or unavailable in the future. Applications should call **PeekMessage** directly.\]

Dispatches incoming sent messages, checks the thread message queue for a posted message, and retrieves the message (if any exist). See [**PeekMessage**](https://learn.microsoft.com/windows/win32/api/winuser/nf-winuser-peekmessagea).
