# TF\_RENDERINGMARKUP structure

The [**TF\_RENDERINGMARKUP**](https://learn.microsoft.com/windows/desktop/api/Msctf/ns-msctf-tf_da_color) structure structure contains a range and the display attribute information.

## Members

**pRange**

A pointer to an [ITfRange](https://learn.microsoft.com/windows/desktop/api/Msctf/nn-msctf-itfrange) interface.

**tfDisplayAttr**

Display attribute information.

## Remarks

This structure is not currently in the public header files. To use this API, you must MIDL-compile the [prototype](https://learn.microsoft.com/windows/win32/tsf/prototypes).
