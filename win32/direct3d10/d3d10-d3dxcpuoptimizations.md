# D3DXCpuOptimizations function

Enables or disables CPU optimizations.

## Parameters

*Enable* \[in\]

Type: **[**BOOL**](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

**TRUE** to enable CPU optimizations; otherwise **FALSE**.

## Return value

Type: **[**D3DX\_CPU\_OPTIMIZATION**](https://learn.microsoft.com/windows/win32/direct3d10/d3dx-cpu-optimization)**

Returns the type of CPU detected, and for which optimizations exist (see [**D3DX\_CPU\_OPTIMIZATION**](https://learn.microsoft.com/windows/win32/direct3d10/d3dx-cpu-optimization)).
