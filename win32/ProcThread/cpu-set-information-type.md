# CPU\_SET\_INFORMATION\_TYPE enumeration

Represents the type of information in the [**SYSTEM\_CPU\_SET\_INFORMATION**](https://learn.microsoft.com/windows/desktop/api/winnt/ns-winnt-system_cpu_set_information) structure.

## Constants

**CpuSetInformation**

The structure contains CPU Set information.
