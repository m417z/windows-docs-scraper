# VMDVDDriveAttachmentType enumeration

\[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the [Hyper-V WMI provider (V2)](https://learn.microsoft.com/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Specifies what is attached to a DVD drive.

## Constants

**vmDVDDrive\_None**

There is nothing attached.

**vmDVDDrive\_Image**

There is an ISO image file attached.

**vmDVDDrive\_HostDrive**

There is a host CD or DVD drive attached.
