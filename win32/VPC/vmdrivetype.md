# VMDriveType enumeration

\[Windows Virtual PC is no longer available for use as of Windows 8. Instead, use the [Hyper-V WMI provider (V2)](https://learn.microsoft.com/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Specifies the type of drive attached to a bus location.

## Constants

**vmDriveType\_Null**

There is no drive attached.

**vmDriveType\_HardDisk**

There is a hard disk attached.

**vmDriveType\_DVD**

There is a CD or DVD drive attached.
