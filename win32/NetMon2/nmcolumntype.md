# NMCOLUMNTYPE enumeration

The **NMCOLUMNTYPE** enumeration specifies the members of the [**NMCOLUMNVARIANT**](https://learn.microsoft.com/windows/win32/netmon2/nmcolumnvariant) structure.

## Constants

**NMCOLUMNTYPE\_UINT8**

8-bit unsigned integer.

**NMCOLUMNTYPE\_SINT8**

8-bit signed integer.

**NMCOLUMNTYPE\_UINT16**

16-bit unsigned integer.

**NMCOLUMNTYPE\_SINT16**

16-bit signed integer.

**NMCOLUMNTYPE\_UINT32**

32-bit unsigned integer.

**NMCOLUMNTYPE\_SINT32**

32-bit signed integer.

**NMCOLUMNTYPE\_FLOAT64**

64-bit float.

**NMCOLUMNTYPE\_FRAME**

Frame number.

**NMCOLUMNTYPE\_YESNO**

"Yes" or "No".

**NMCOLUMNTYPE\_ONOFF**

"On" or "Off"

**NMCOLUMNTYPE\_TRUEFALSE**

"True" or "False".

**NMCOLUMNTYPE\_MACADDR**

MAC address.

**NMCOLUMNTYPE\_IPXADDR**

IPX address.

**NMCOLUMNTYPE\_IPADDR**

IP address.

**NMCOLUMNTYPE\_VARTIME**

Variant time.

**NMCOLUMNTYPE\_STRING**

Pointer to a string.
