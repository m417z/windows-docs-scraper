# FRAME structure

The **FRAME** structure is an actual frame from the driver.

## Members

**TimeStamp**

Elapsed time, in microseconds, between the start of the capture and the time when the frame was captured.

**FrameLength**

MAC frame length.

**nBytesAvail**

Actual frame length copied.

**MacFrame**

Frame data.
