# PF\_FOLLOWENTRY structure

The **PF\_FOLLOWENTRY** structure defines a protocol that Network Monitor adds to the follow set of a parser.

## Members

**szProtocol**

The name of the protocol.

## Remarks

The [PF\_FOLLOWSET](https://learn.microsoft.com/windows/win32/netmon2/pf-followset) structure uses an array of **PF\_FOLLOWENTRY** structures.
