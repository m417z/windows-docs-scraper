# CONFIGUREDEXPERT structure

The **CONFIGUREDEXPERT** structure associates an expert with its configuration data.

## Members

**hExpert**

Handle to an expert.

**StartupFlags**

Startup flag values of the expert.

**pConfig**

Pointer to an [**EXPERTCONFIG**](https://learn.microsoft.com/windows/win32/netmon2/expertconfig) structure.
