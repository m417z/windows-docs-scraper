# CB\_RESOURCE\_TYPE enumeration

Specifies the type of resource that the incoming connection is connecting to. This enumeration is used with the [**CB\_CONNECTION\_INFO**](https://learn.microsoft.com/windows/win32/termserv/cb-connection-info) structure.

## Constants

**CB\_RESOURCE\_UNDEFINED**

The resource type is not defined.

**CB\_RESOURCE\_SESSION**

The resource is a remote session.

**CB\_RESOURCE\_VM**

The resource is a virtual machine.
