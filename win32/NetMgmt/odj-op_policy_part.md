# OP_POLICY_PART structure

Contains an array of OP_POLICY_ELEMENT_LIST structures.

## Members

### cElementLists

Contains the number of elements in pElementLists.

### pElementLists

Contains an array of OP_POLICY_ELEMENT_LIST structures.

### Extension

Reserved for future use and must contain all zeros.
