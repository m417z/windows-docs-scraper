# WINHTTP_WEB_SOCKET_OPERATION enumeration

## Description

The **WINHTTP_WEB_SOCKET_OPERATION** enumeration includes the WebSocket operation type.

## Constants

### `WINHTTP_WEB_SOCKET_SEND_OPERATION:0`

A [WinHttpWebSocketSend](https://learn.microsoft.com/windows/desktop/api/winhttp/nf-winhttp-winhttpwebsocketsend) operation.

### `WINHTTP_WEB_SOCKET_RECEIVE_OPERATION:1`

A [WinHttpWebSocketReceive](https://learn.microsoft.com/windows/desktop/api/winhttp/nf-winhttp-winhttpwebsocketreceive) operation.

### `WINHTTP_WEB_SOCKET_CLOSE_OPERATION:2`

A [WinHttpWebSocketClose](https://learn.microsoft.com/windows/desktop/api/winhttp/nf-winhttp-winhttpwebsocketclose) operation.

### `WINHTTP_WEB_SOCKET_SHUTDOWN_OPERATION:3`

A [WinHttpWebSocketShutdown](https://learn.microsoft.com/windows/desktop/api/winhttp/nf-winhttp-winhttpwebsocketshutdown) operation.
