## Description

Represents a connection group.

## Members

### `cConnections`

Type: **[ULONG](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

The number of connections marked with *guidGroup*.

### `guidGroup`

Type: **[GUID](https://learn.microsoft.com/windows/win32/api/guiddef/ns-guiddef-guid)**

A http connection **GUID**.
