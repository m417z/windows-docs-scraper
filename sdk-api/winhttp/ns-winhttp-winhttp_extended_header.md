## Description

Represents an HTTP request header as a name/value string pair.

## Members

### `pwszName`

Type: IN **[PCWSTR](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

A wide string containing a name.

### `pszName`

Type: IN **[PCSTR](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

A narrow string containing a name.

### `pwszValue`

Type: IN **[PCWSTR](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

A wide string containing a value.

### `pszValue`

Type: IN **[PCSTR](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)**

A narrow string containing a value.
