# DEVICE_INFO structure

## Description

Contains information about a media sharing device.

## Members

### `pFriendlyDeviceName`

The friendly name of the device.

### `pUniqueDeviceName`

A string that uniquely identifies the device.

### `pManufacturerName`

The manufacturer name.

### `pModelName`

The model name.

### `pIconURL`

The URL of an icon for the device.
