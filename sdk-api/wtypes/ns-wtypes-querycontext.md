# QUERYCONTEXT structure

## Description

Contains a list of attributes used to look up a class implementation.

## Members

### `dwContext`

The execution context.

### `Platform`

The operating system platform and processor architecture. For more information, see [CSPLATFORM](https://learn.microsoft.com/windows/desktop/api/wtypes/ns-wtypes-csplatform).

### `Locale`

The locale identifier. For more information, see [Language Identifier Constants and Strings](https://learn.microsoft.com/windows/desktop/Intl/language-identifier-constants-and-strings).

### `dwVersionHi`

The high version number.

### `dwVersionLo`

The low version number.
