# PCCARD_RESOURCE structure

## Description

The PCCARD_RESOURCE structure is used for specifying either a resource list or a resource requirements list that describes resource usage by a PC Card instance. For more information about resource lists and resource requirements lists, see [Hardware Resources](https://learn.microsoft.com/windows-hardware/drivers/kernel/hardware-resources).

## Members

### `PcCard_Header`

A [PCCARD_DES](https://learn.microsoft.com/windows/desktop/api/cfgmgr32/ns-cfgmgr32-pccard_des) structure.
