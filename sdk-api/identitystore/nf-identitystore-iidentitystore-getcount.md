# IIdentityStore::GetCount

## Description

The **GetCount** method gets the number of identity providers registered on the system.

## Parameters

### `pdwProviders` [out]

The number of identity providers registered on the system.

## Return value

 If the method succeeds, it returns **S_OK**.

If the method fails, it returns an error code. For a list of common error codes, see [Common HRESULT Values](https://learn.microsoft.com/windows/desktop/SecCrypto/common-hresult-values).
