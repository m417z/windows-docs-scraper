# IInkHostWorkItem::Invoke

## Description

Executes the ink operation on an [IInkDesktopHost](https://learn.microsoft.com/previous-versions/windows/desktop/api/inkpresenterdesktop/nn-inkpresenterdesktop-iinkdesktophost) object thread.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
