# D2D1_FEATURE enumeration

## Description

Defines capabilities of the underlying Direct3D device which may be queried using [ID2D1EffectContext::CheckFeatureSupport](https://learn.microsoft.com/windows/desktop/api/d2d1effectauthor/nf-d2d1effectauthor-id2d1effectcontext-checkfeaturesupport).

## Constants

### `D2D1_FEATURE_DOUBLES:0`

A D2D1_FEATURE_DATA_DOUBLES structure should be filled.

### `D2D1_FEATURE_D3D10_X_HARDWARE_OPTIONS:1`

A D2D1_FEATURE_DATA_D3D10_X_HARDWARE_OPTIONS structure should be filled.

### `D2D1_FEATURE_FORCE_DWORD:0xffffffff`
