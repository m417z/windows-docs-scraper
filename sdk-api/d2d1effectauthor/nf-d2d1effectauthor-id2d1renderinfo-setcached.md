# ID2D1RenderInfo::SetCached

## Description

Specifies that the output of the transform in which the render information is encapsulated is or is not cached.

## Parameters

### `isCached`

Type: **BOOL**

**TRUE** if the output of the transform is cached; otherwise, **FALSE**.
