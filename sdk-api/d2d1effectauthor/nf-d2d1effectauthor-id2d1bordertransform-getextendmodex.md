# ID2D1BorderTransform::GetExtendModeX

## Description

Gets the extend mode in the x direction.

## Return value

Type: **[D2D1_EXTEND_MODE](https://learn.microsoft.com/windows/desktop/api/d2d1/ne-d2d1-d2d1_extend_mode)**

This method returns the extend mode in the x direction.
