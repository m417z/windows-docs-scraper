# MI_SubscriptionDeliveryOptions_GetOptionCount function

## Description

Gets the number of previously set options.

## Parameters

### `self` [in]

[MI_SubscriptionDeliveryOptions](https://learn.microsoft.com/windows/desktop/api/mi/ns-mi-mi_subscriptiondeliveryoptions) structure

### `count` [out, optional]

Returned number of previously set options.

## Return value

A value of the [MI_Result](https://learn.microsoft.com/windows/desktop/api/mi/ne-mi-mi_result) enumeration that specifies the function return code. This can be one of the following codes.
