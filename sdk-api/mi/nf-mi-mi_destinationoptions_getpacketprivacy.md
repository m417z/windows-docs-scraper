# MI_DestinationOptions_GetPacketPrivacy function

## Description

Gets the packet privacy (encryption) setting.

## Parameters

### `options` [in]

[MI_DestinationOptions](https://learn.microsoft.com/windows/desktop/api/mi/ns-mi-mi_destinationoptions) object returned from [MI_Application_NewDestinationOptions](https://learn.microsoft.com/previous-versions/windows/desktop/api/mi/nf-mi-mi_application_newdestinationoptions).

### `privacy` [out]

The returned privacy (encryption) setting, a Boolean value. If this setting is **MI_TRUE**, then the session contains protection against eavesdropping.

## Return value

A value of the [MI_Result](https://learn.microsoft.com/windows/desktop/api/mi/ne-mi-mi_result) enumeration that specifies the function return code. This can be one of the following codes.
