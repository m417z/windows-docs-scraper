# MI_OperationOptions_GetResourceUriPrefix function

## Description

Gets the resource URI prefix being used for an operation.

## Parameters

### `options` [in]

[MI_OperationOptions](https://learn.microsoft.com/windows/desktop/api/mi/ns-mi-mi_operationoptions) structure.

### `ruriPrefix`

Returned resource URI being used for the operation.

## Return value

A value of the [MI_Result](https://learn.microsoft.com/windows/desktop/api/mi/ne-mi-mi_result) enumeration that specifies the function return code. This can be one of the following codes.
