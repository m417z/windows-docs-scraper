# MI_Instance_ClearElementAt function

## Description

Clears the value of the element (CIM property) at the specified index and sets it to **NULL**.

## Parameters

### `self` [in, out]

Instance whose property will be set.

### `index`

Zero-based index of the element that will be set.

## Return value

A value of the [MI_Result](https://learn.microsoft.com/windows/desktop/api/mi/ne-mi-mi_result) enumeration that specifies the function return code. This can be one of the following codes.
