# IValueMap::CreateValueMapItem

## Description

Creates a value map item.

## Parameters

### `Item` [out]

An [IValueMapItem](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-ivaluemapitem) interface that you use to define a name/value pair.

## Return value

Returns S_OK if successful.
