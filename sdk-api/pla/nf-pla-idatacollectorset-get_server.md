# IDataCollectorSet::get_Server

## Description

Retrieves the name of the server where the data collector set is run.

This property is read-only.

## Parameters

## Remarks

The name is set when you call the [IDataCollectorSet::Commit](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nf-pla-idatacollectorset-commit) method.
