# ITraceDataProviderCollection::AddRange

## Description

Adds one or more trace providers to the collection.

## Parameters

### `providers` [in]

An [ITraceDataProviderCollection](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-itracedataprovidercollection) interface to a collection of one or more trace providers to add to this collection.

## Return value

Returns S_OK if successful.
