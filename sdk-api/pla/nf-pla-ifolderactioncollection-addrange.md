# IFolderActionCollection::AddRange

## Description

Adds one or more folder actions to the collection.

## Parameters

### `Actions` [in]

An [IFolderActionCollection](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-ifolderactioncollection) interface to a collection of one or more folder actions to add to this collection.

## Return value

Returns S_OK if successful.
