# IFolderActionCollection::Add

## Description

Adds a folder action to the collection.

## Parameters

### `Action` [in]

An [IFolderAction](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-ifolderaction) interface of the action to add to the collection.

## Return value

Returns S_OK if successful.
