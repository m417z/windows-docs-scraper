# IValueMap::AddRange

## Description

Adds one or more items to the collection.

## Parameters

### `map` [in]

An [IValueMap](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-ivaluemap) interface that contains a collection of items to add to this collection.

## Return value

Returns S_OK if successful.
