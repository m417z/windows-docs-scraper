# IDataManager::get_ReportFileName

## Description

Retrieves or sets the name of the HTML file that results from converting the file in the [IDataManager::RuleTargetFileName](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nf-pla-idatamanager-get_ruletargetfilename) property from XML to HTML.

This property is read/write.

## Remarks

For details, see the **plaCreateHtml** value of the [DataManagerSteps](https://learn.microsoft.com/windows/win32/api/pla/ne-pla-datamanagersteps) enumeration.
