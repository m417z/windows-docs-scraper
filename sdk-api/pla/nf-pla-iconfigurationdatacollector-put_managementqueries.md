# IConfigurationDataCollector::put_ManagementQueries

## Description

Retrieves or sets [Windows Management Instrumentation](https://learn.microsoft.com/windows/desktop/WmiSdk/wmi-start-page) (WMI) queries to run.

This property is read/write.
