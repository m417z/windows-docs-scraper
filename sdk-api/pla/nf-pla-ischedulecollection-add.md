# IScheduleCollection::Add

## Description

Adds a schedule to the collection.

## Parameters

### `pSchedule` [in]

An [ISchedule](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-ischedule) interface of the schedule to add to the collection.

## Return value

Returns S_OK if successful.
