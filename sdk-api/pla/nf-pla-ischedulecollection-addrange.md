# IScheduleCollection::AddRange

## Description

Adds one or more schedules to the collection.

## Parameters

### `pSchedules` [in]

An [IScheduleCollection](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-ischedulecollection) interface to a collection of one or more schedules to add to this collection.

## Return value

Returns S_OK if successful.
