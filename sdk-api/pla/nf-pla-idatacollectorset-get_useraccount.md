# IDataCollectorSet::get_UserAccount

## Description

Retrieves the user account under which the data collector set will run.

This property is read-only.

## Parameters

## Remarks

The user name is set using the [IDataCollectorSet::SetCredentials](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nf-pla-idatacollectorset-setcredentials) method.
