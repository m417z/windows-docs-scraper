# IPerformanceCounterDataCollector::put_DataSourceName

## Description

Retrieves or sets the data source name if the log file is an SQL log file.

This property is read/write.

## Parameters

## Remarks

Specify the data source name only if the [IPerformanceCounterDataCollector::LogFileFormat](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nf-pla-iperformancecounterdatacollector-get_logfileformat) property is **plaSql**.
