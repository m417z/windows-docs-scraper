# IDataCollectorSetCollection::Add

## Description

Adds a data collector set to the collection.

## Parameters

### `set` [in]

An [IDataCollectorSet](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-idatacollectorset) interface of the data collector set to add to this collection.

## Return value

Returns S_OK if successful.
