# IValueMapItem::put_Key

## Description

Retrieves or sets the name of the item.

This property is read/write.

## Parameters

## Remarks

Note that if you set the [IValueMap::Value](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nf-pla-ivaluemap-get_value) property and PLA generates the items, the items are not named.
