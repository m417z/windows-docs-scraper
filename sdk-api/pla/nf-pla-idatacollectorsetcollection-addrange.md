# IDataCollectorSetCollection::AddRange

## Description

Adds one or more data collector sets to the collection.

## Parameters

### `sets` [in]

An [IDataCollectorSetCollection](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-idatacollectorsetcollection) interface to a collection of one or more data collector sets to add to this collection.

## Return value

Returns S_OK if successful.
