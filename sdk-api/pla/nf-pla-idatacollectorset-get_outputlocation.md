# IDataCollectorSet::get_OutputLocation

## Description

Retrieves the decorated folder name if PLA were to create it now.

This property is read-only.

## Parameters

## Remarks

The [IDataCollectorSet::LatestOutputLocation](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nf-pla-idatacollectorset-get_latestoutputlocation) property contains the decorated file name used the last time the collector ran.
