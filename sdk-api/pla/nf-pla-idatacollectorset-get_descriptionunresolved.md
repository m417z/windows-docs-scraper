# IDataCollectorSet::get_DescriptionUnresolved

## Description

Retrieves the description of the data collector set in its original form.

This property is read-only.

## Parameters

## Remarks

This property returns the description as you originally set it in the [IDataCollectorSet::Description](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nf-pla-idatacollectorset-get_description) property. Typically, you would use this property if you set the description using the form @*binary*,#*id*.
