# IDataManager::Extract

## Description

Extracts the specified CAB file.

## Parameters

### `CabFilename` [in]

The name of the CAB file to extract.

### `DestinationPath` [in]

The path where you want to place the CAB file.

## Return value

Returns S_OK if successful.
