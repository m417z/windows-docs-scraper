# IDataCollectorSet::get_Xml

## Description

Retrieves an XML string that describes the values of the data collector set properties, including those of the data collectors contained in the set.

This property is read-only.

## Parameters

## Remarks

For details on the property elements contained in the XML string, see the Remarks section of [IDataCollectorSet](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-idatacollectorset) and the individual data collectors.
