# ITraceDataProviderCollection::CreateTraceDataProvider

## Description

Creates a trace data provider object.

## Parameters

### `Provider` [out]

An [ITraceDataProvider](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-itracedataprovider) interface that you use to specify a trace provider to enable in the trace session.

## Return value

Returns S_OK if successful.
