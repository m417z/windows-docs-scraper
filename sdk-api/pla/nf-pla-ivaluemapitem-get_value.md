# IValueMapItem::get_Value

## Description

Retrieves or sets the value of the item.

This property is read/write.

## Parameters

## Remarks

The variant type is VT_UI8 if the [ValueMapType](https://learn.microsoft.com/windows/win32/api/pla/ne-pla-valuemaptype) enumeration is plaIndex, plaFlag or plaFlagArray.

The variant type is VT_UI4 if [ValueMapType](https://learn.microsoft.com/windows/win32/api/pla/ne-pla-valuemaptype) is plaValidation.
