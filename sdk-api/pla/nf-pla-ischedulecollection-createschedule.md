# IScheduleCollection::CreateSchedule

## Description

Creates a schedule object.

## Parameters

### `Schedule` [out]

An [ISchedule](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-ischedule) interface that you use to specify when the data collector set runs.

## Return value

Returns S_OK if successful.
