# ITraceDataProviderCollection::Add

## Description

Adds a trace provider to the collection.

## Parameters

### `pProvider` [in]

An [ITraceDataProvider](https://learn.microsoft.com/previous-versions/windows/desktop/api/pla/nn-pla-itracedataprovider) interface of the trace provider to add to this collection.

## Return value

Returns S_OK if successful.
