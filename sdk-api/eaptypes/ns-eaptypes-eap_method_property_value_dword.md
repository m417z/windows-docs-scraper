# EAP_METHOD_PROPERTY_VALUE_DWORD structure

## Description

The **EAP_METHOD_PROPERTY_VALUE_DWORD** structure contains the DWORD value of an EAP method property.

## Members

### `length`

The size, in bytes, of **value**.

### `value`

A DWORD value of the method property.
