# ITocEntry::SetDescriptor

## Description

The **SetDescriptor** method associates a descriptor with the entry.

## Parameters

### `pDescriptor` [in]

Pointer to a [TOC_ENTRY_DESCRIPTOR](https://learn.microsoft.com/windows/desktop/api/wmcodecdsp/ns-wmcodecdsp-toc_entry_descriptor) structure that contains the descriptor.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
