# OnDemandUnRegisterNotification function

## Description

The **OnDemandUnregisterNotification** function allows an application to unregister for notifications and clean up resources.

## Parameters

### `registrationHandle` [in]

A HANDLE obtained from a successful [OnDemandRegisterNotification](https://learn.microsoft.com/windows/desktop/api/ondemandconnroutehelper/nf-ondemandconnroutehelper-ondemandregisternotification) call.

## Return value

Returns S_OK on success.
