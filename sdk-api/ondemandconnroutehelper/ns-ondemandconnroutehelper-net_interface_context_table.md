# NET_INTERFACE_CONTEXT_TABLE structure

## Description

The table of [NET_INTERFACE_CONTEXT](https://learn.microsoft.com/windows/win32/api/ondemandconnroutehelper/ns-ondemandconnroutehelper-net_interface_context) structures.

## Members

### `InterfaceContextHandle`

A handle to the interface context.

### `NumberOfEntries`

The number of entries in the table.

### `InterfaceContextArray`

An array of [NET_INTERFACE_CONTEXT](https://learn.microsoft.com/windows/win32/api/ondemandconnroutehelper/ns-ondemandconnroutehelper-net_interface_context) structures.
