# NET_INTERFACE_CONTEXT structure

## Description

The interface context that is part of the [NET_INTERFACE_CONTEXT_TABLE](https://learn.microsoft.com/windows/win32/api/ondemandconnroutehelper/ns-ondemandconnroutehelper-net_interface_context_table) structure.

## Members

### `InterfaceIndex`

The interface index.

### `ConfigurationName`

The configuration name.
