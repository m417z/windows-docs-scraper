# FreeInterfaceContextTable function

## Description

This function frees the interface context table retrieved using the [GetInterfaceContextTableForHostName](https://learn.microsoft.com/windows/desktop/api/ondemandconnroutehelper/nf-ondemandconnroutehelper-getinterfacecontexttableforhostname) function.

## Parameters

### `InterfaceContextTable` [in]

The interface context table retrieved using the [GetInterfaceContextTableForHostName](https://learn.microsoft.com/windows/desktop/api/ondemandconnroutehelper/nf-ondemandconnroutehelper-getinterfacecontexttableforhostname) function.
