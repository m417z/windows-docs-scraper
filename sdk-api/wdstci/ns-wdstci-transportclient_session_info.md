# TRANSPORTCLIENT_SESSION_INFO structure

## Description

This structure is used by the [PFN_WdsTransportClientSessionStartEx](https://learn.microsoft.com/windows/desktop/api/wdstci/nc-wdstci-pfn_wdstransportclientsessionstartex) callback function.

## Members

### `ulStructureLength`

The length of this structure in bytes.

### `ullFileSize`

The size of the file, in bytes.

### `ulBlockSize`

The size of a receive block, in bytes.
