## Description

Specifies the data field identifiers for the [FWPS_LAYER_RPC_EP_ADD](https://learn.microsoft.com/windows/win32/api/fwpsu/ne-fwpsu-fwps_builtin_layers) run-time filtering layer.

## Constants

### `FWPS_FIELD_RPC_EP_ADD_PROCESS_WITH_RPC_IF_UUID`

The UUID of the process with the RPC interface.

### `FWPS_FIELD_RPC_EP_ADD_PROTOCOL`

The possible condition values are **RPC_PROTSEQ_TCP**, **RPC_PROTSEQ_HTTP**, and **RPC_PROTSEQ_NMP**.

### `FWPS_FIELD_RPC_EP_ADD_EP_VALUE`

Reserved for internal use.

### `FWPS_FIELD_RPC_EP_ADD_EP_FLAGS`

Reserved for internal use.

### `FWPS_FIELD_RPC_EP_ADD_MAX`

The maximum value for this enumeration. This value might change in future versions of the NDIS
header files and binaries.
