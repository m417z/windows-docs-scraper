## Description

Frees the memory allocated to a [DNS_QUERY_RAW_RESULT](https://learn.microsoft.com/windows/win32/api/windns/ns-windns-dns_query_raw_result) structure object. Also see [DNS_QUERY_RAW_COMPLETION_ROUTINE](https://learn.microsoft.com/windows/win32/api/windns/nc-windns-dns_query_raw_completion_routine).

## Parameters

### `queryResults`

Type: \_In\_ **[DNS_QUERY_RAW_RESULT](https://learn.microsoft.com/windows/win32/api/windns/ns-windns-dns_query_raw_result)\***

The object whose memory should be freed.
