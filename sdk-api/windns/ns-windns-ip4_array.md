# IP4_ARRAY structure

## Description

The **IP4_ARRAY** structure stores an array of IPv4 addresses.

## Members

### `AddrCount`

The number of IPv4 addresses in **AddrArray**.

### `AddrArray.size_is`

### `AddrArray.size_is.AddrCount`

### `AddrArray`

An array of [IP4_ADDRESS](https://learn.microsoft.com/windows/desktop/DNS/dns-data-types) data types that contains a list of IPv4 address.
