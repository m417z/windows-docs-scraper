# ResUtilInitializeResourceStatus macro

## Description

Initializes a [RESOURCE_STATUS](https://learn.microsoft.com/previous-versions/windows/desktop/api/resapi/ns-resapi-resource_status) structure.

## Parameters

### `_resource_status_` [in]

Pointer to the [RESOURCE_STATUS](https://learn.microsoft.com/previous-versions/windows/desktop/api/resapi/ns-resapi-resource_status) structure to initialize.
