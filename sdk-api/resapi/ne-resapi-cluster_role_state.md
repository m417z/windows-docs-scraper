# CLUSTER_ROLE_STATE enumeration

## Description

Defines the potential return values for the [ResUtilGetClusterRoleState](https://learn.microsoft.com/previous-versions/windows/desktop/api/resapi/nf-resapi-resutilgetclusterrolestate) function.

## Constants

### `ClusterRoleUnknown:-1`

It is unknown whether or not the role is clustered.

### `ClusterRoleClustered`

The role is clustered.

### `ClusterRoleUnclustered`

The role is not clustered.
