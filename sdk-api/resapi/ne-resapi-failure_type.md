# FAILURE_TYPE enumeration

## Description

Defines the failure types for cluster resources.

## Constants

### `FAILURE_TYPE_GENERAL:0`

A general failure.

### `FAILURE_TYPE_EMBEDDED:1`

An embedded failure.

### `FAILURE_TYPE_NETWORK_LOSS:2`

A network failure.
