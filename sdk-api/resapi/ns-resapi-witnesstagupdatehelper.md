# WitnessTagUpdateHelper structure

## Description

Contains information used to update and validate a [PaxosTagCStruct](https://learn.microsoft.com/windows/desktop/api/resapi/ns-resapi-paxostagcstruct) structure.

## Members

### `Version`

The cluster configuration version to validate.

### `paxosToSet`

The Paxos tag to update.

### `paxosToValidate`

The Paxos tag to validate.
