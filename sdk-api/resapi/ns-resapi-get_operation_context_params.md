# GET_OPERATION_CONTEXT_PARAMS structure

## Description

Represents context parameters that are used as input for the [CLUSCTL_RESOURCE_GET_OPERATION_CONTEXT](https://learn.microsoft.com/previous-versions/windows/desktop/mscs/clusctl-resource-get-operation-context) control code.

## Members

### `Size`

The size of this structure, in bytes.

### `Version`

The version of this structure.

### `Type`

A [RESDLL_CONTEXT_OPERATION_TYPE](https://learn.microsoft.com/previous-versions/windows/desktop/api/resapi/ne-resapi-resdll_context_operation_type) enumeration value that specifies the context operation type.
