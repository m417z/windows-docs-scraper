# PSET_INTERNAL_STATE callback function

## Description

Sets the internal state of a resource

## Parameters

### `unnamedParam1`

### `stateType` [in]

A [CLUSTER_RESOURCE_APPLICATION_STATE](https://learn.microsoft.com/previous-versions/windows/desktop/api/resapi/ne-resapi-cluster_resource_application_state) value

### `active` [in]

Whether the resource is active

#### - [in]

A resource handle
