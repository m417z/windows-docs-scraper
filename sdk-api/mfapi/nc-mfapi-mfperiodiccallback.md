# MFPERIODICCALLBACK callback function

## Description

Callback function for the [MFAddPeriodicCallback](https://learn.microsoft.com/windows/desktop/api/mfapi/nf-mfapi-mfaddperiodiccallback) function.

## Parameters

### `pContext` [in]

Pointer to the **IUnknown** interface, or **NULL**. This pointer is specified by the caller in the [MFAddPeriodicCallback](https://learn.microsoft.com/windows/desktop/api/mfapi/nf-mfapi-mfaddperiodiccallback) function.
