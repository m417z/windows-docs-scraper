# MFHeapFree function

## Description

Frees a block of memory that was allocated by calling the [MFHeapAlloc](https://learn.microsoft.com/windows/desktop/api/mfapi/nf-mfapi-mfheapalloc) function.

## Parameters

### `pv` [in]

Pointer to the memory to free.
