# TAKE_SNAPSHOT_VHDSET_FLAG enumeration

## Description

Contains flags affecting the behavior of the [TakeSnapshotVhdSet](https://learn.microsoft.com/windows/win32/api/virtdisk/nf-virtdisk-takesnapshotvhdset) function.

## Constants

### `TAKE_SNAPSHOT_VHDSET_FLAG_NONE:0x00000000`

No flag specified.
