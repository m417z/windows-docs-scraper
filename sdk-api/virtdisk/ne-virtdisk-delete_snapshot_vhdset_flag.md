# DELETE_SNAPSHOT_VHDSET_FLAG enumeration

## Description

Contains flags affecting the behavior of the [DeleteSnapshotVhdSet](https://learn.microsoft.com/windows/win32/api/virtdisk/nf-virtdisk-deletesnapshotvhdset) function.

## Constants

### `DELETE_SNAPSHOT_VHDSET_FLAG_NONE:0x00000000`

No flag specified.

### `DELETE_SNAPSHOT_VHDSET_FLAG_PERSIST_RCT:0x00000001`

A reference point should be persisted in the VHD Set after the snapshot is deleted.
