# STORAGE_DEPENDENCY_INFO_VERSION enumeration

## Description

Contains the version of the virtual hard disk (VHD) [STORAGE_DEPENDENCY_INFO](https://learn.microsoft.com/windows/win32/api/virtdisk/ns-virtdisk-storage_dependency_info) structure to use in calls to VHD functions.

## Constants

### `STORAGE_DEPENDENCY_INFO_VERSION_UNSPECIFIED:0`

The version is not specified.

### `STORAGE_DEPENDENCY_INFO_VERSION_1:1`

Specifies [STORAGE_DEPENDENCY_INFO_TYPE_1](https://learn.microsoft.com/windows/win32/api/virtdisk/ns-virtdisk-storage_dependency_info_type_1).

### `STORAGE_DEPENDENCY_INFO_VERSION_2:2`

Specifies [STORAGE_DEPENDENCY_INFO_TYPE_2](https://learn.microsoft.com/windows/win32/api/virtdisk/ns-virtdisk-storage_dependency_info_type_2).
