# COMPACT_VIRTUAL_DISK_VERSION enumeration

## Description

Contains the version of the virtual hard disk (VHD) [COMPACT_VIRTUAL_DISK_PARAMETERS](https://learn.microsoft.com/windows/win32/api/virtdisk/ns-virtdisk-compact_virtual_disk_parameters) structure to use in calls to VHD functions.

## Constants

### `COMPACT_VIRTUAL_DISK_VERSION_UNSPECIFIED:0`

### `COMPACT_VIRTUAL_DISK_VERSION_1:1`
