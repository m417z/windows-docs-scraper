# EXPAND_VIRTUAL_DISK_VERSION enumeration

## Description

Contains the version of the virtual disk
[EXPAND_VIRTUAL_DISK_PARAMETERS](https://learn.microsoft.com/windows/win32/api/virtdisk/ns-virtdisk-expand_virtual_disk_parameters) structure
to use in calls to virtual disk functions.

## Constants

### `EXPAND_VIRTUAL_DISK_VERSION_UNSPECIFIED:0`

### `EXPAND_VIRTUAL_DISK_VERSION_1:1`
