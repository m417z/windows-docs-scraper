# APPLY_SNAPSHOT_VHDSET_FLAG enumeration

## Description

Contains flags affecting the behavior of the [ApplySnapshotVhdSet](https://learn.microsoft.com/windows/win32/api/virtdisk/nf-virtdisk-applysnapshotvhdset) function.

## Constants

### `APPLY_SNAPSHOT_VHDSET_FLAG_NONE:0x00000000`

No flag specified.

### `APPLY_SNAPSHOT_VHDSET_FLAG_WRITEABLE:0x00000001`

Indicates that the snapshot to be applied was created as a writable snapshot type.
