# RAW_SCSI_VIRTUAL_DISK_VERSION enumeration

## Description

Contains the version of the [RAW_SCSI_VIRTUAL_DISK_PARAMETERS](https://learn.microsoft.com/windows/win32/api/virtdisk/ns-virtdisk-raw_scsi_virtual_disk_parameters) structure to use in calls to virtual disk functions.

## Constants

### `RAW_SCSI_VIRTUAL_DISK_VERSION_UNSPECIFIED:0`

### `RAW_SCSI_VIRTUAL_DISK_VERSION_1:1`
