# APPLY_SNAPSHOT_VHDSET_VERSION enumeration

## Description

Enumerates the possible versions for parameters for the
[ApplySnapshotVhdSet](https://learn.microsoft.com/windows/win32/api/virtdisk/nf-virtdisk-applysnapshotvhdset) function.

## Constants

### `APPLY_SNAPSHOT_VHDSET_VERSION_UNSPECIFIED:0`

Not Supported.

### `APPLY_SNAPSHOT_VHDSET_VERSION_1:1`

The Version1 member structure will be used.
