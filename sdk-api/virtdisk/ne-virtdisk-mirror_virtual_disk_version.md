# MIRROR_VIRTUAL_DISK_VERSION enumeration

## Description

Contains the version of the virtual disk
[MIRROR_VIRTUAL_DISK_PARAMETERS](https://learn.microsoft.com/windows/win32/api/virtdisk/ns-virtdisk-mirror_virtual_disk_parameters) structure
used by the [MirrorVirtualDisk](https://learn.microsoft.com/windows/win32/api/virtdisk/nf-virtdisk-breakmirrorvirtualdisk)
function.

## Constants

### `MIRROR_VIRTUAL_DISK_VERSION_UNSPECIFIED:0`

Unsupported.

### `MIRROR_VIRTUAL_DISK_VERSION_1:1`

Use the **Version1** member.
