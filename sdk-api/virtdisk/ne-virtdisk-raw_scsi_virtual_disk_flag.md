# RAW_SCSI_VIRTUAL_DISK_FLAG enumeration

## Description

Contains flags affecting the behavior of the [RawSCSIVirtualDisk](https://learn.microsoft.com/windows/win32/api/virtdisk/nf-virtdisk-rawscsivirtualdisk) function.

## Constants

### `RAW_SCSI_VIRTUAL_DISK_FLAG_NONE:0X00000000`

No flag specified.
