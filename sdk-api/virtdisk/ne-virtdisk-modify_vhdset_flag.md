# MODIFY_VHDSET_FLAG enumeration

## Description

Contains flags affecting the behavior of the [ModifyVhdSet](https://learn.microsoft.com/windows/win32/api/virtdisk/nf-virtdisk-modifyvhdset) function.

## Constants

### `MODIFY_VHDSET_FLAG_NONE:0x00000000`

No flag specified.

### `MODIFY_VHDSET_FLAG_WRITEABLE_SNAPSHOT:0x00000001`
