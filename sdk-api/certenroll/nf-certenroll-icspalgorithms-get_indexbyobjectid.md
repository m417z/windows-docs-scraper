# ICspAlgorithms::get_IndexByObjectId

## Description

The **IndexByObjectId** property retrieves the index of an [ICspAlgorithm](https://learn.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-icspalgorithm) object by [object identifier](https://learn.microsoft.com/windows/desktop/SecGloss/o-gly) (OID).

This property is read-only.
