# Pkcs10AllowedSignatureTypes enumeration

## Description

The **Pkcs10AllowedSignatureTypes** enumeration type specifies the type of signature permitted when signing a [certificate request](https://learn.microsoft.com/windows/desktop/SecGloss/c-gly). This enumeration is used by the [CheckSignature](https://learn.microsoft.com/windows/desktop/api/certenroll/nf-certenroll-ix509certificaterequestpkcs10-checksignature) method on the [IX509CertificateRequestPkcs10](https://learn.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-ix509certificaterequestpkcs10) interface to determine whether a signature exists and is valid.

## Constants

### `AllowedKeySignature:0x1`

Signatures generated by using [cryptographic keys](https://learn.microsoft.com/windows/desktop/SecGloss/c-gly) are allowed.

### `AllowedNullSignature:0x2`

Null signatures are allowed. A null signature is a SHA-1 [hash](https://learn.microsoft.com/windows/desktop/SecGloss/h-gly).
