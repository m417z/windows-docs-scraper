# ICryptAttribute::get_Values

## Description

The **Values** property retrieves an [IX509Attributes](https://learn.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-ix509attributes) object that contains a collection of attributes.

This property is read-only.
