# IX509CertificateRequest::get_EnrollmentContext

## Description

The **EnrollmentContext** property retrieves a value that specifies whether the certificate is intended for a computer or a user.

This property is read-only.

## Parameters

## Remarks

For a PKCS #7 or CMC request, the property value is retrieved from the inner request object.
