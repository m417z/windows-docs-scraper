# ICspAlgorithms::Add

## Description

The **Add** method adds an [ICspAlgorithm](https://learn.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-ix509namevaluepair) object to the collection. This method is web enabled.

## Parameters

### `pVal` [in]

Pointer to an [ICspAlgorithm](https://learn.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-ix509namevaluepair) interface to add.

## Return value

If the function succeeds, the function returns **S_OK**.

If the function fails, it returns an **HRESULT** value that indicates the error. For a list of common error codes, see [Common HRESULT Values](https://learn.microsoft.com/windows/desktop/SecCrypto/common-hresult-values).
