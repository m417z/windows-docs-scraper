# ICspStatuses::get_ItemByName

## Description

The **ItemByName** property retrieves an [ICspStatus](https://learn.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-icspstatus) object from the collection by provider and algorithm name. This property is web enabled.

This property is read-only.
