# ICertPropertyAutoEnroll::get_TemplateName

## Description

The **TemplateName** property retrieves a string that contains the name of the template that the certificate can use for autoenrollment.

This property is read-only.

## Remarks

Call the [Initialize](https://learn.microsoft.com/windows/desktop/api/certenroll/nf-certenroll-icertpropertyautoenroll-initialize) method to specify the property value.
