# IX509Extensions::get_Count

## Description

The **Count** property retrieves the number of [IX509Extension](https://learn.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-ix509extension) objects in the collection. This property is web enabled.

This property is read-only.
