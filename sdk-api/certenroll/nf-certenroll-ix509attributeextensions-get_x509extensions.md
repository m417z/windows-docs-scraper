# IX509AttributeExtensions::get_X509Extensions

## Description

The **X509Extensions** property retrieves the certificate extensions.

This property is read-only.

## Parameters

## Remarks

Call the [InitializeEncode](https://learn.microsoft.com/windows/desktop/api/certenroll/nf-certenroll-ix509attributeextensions-initializeencode) method or the [InitializeDecode](https://learn.microsoft.com/windows/desktop/api/certenroll/nf-certenroll-ix509attributeextensions-initializedecode) method to initialize the **X509Extensions** property.
