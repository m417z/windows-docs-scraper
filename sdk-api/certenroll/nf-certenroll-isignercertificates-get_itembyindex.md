# ISignerCertificates::get_ItemByIndex

## Description

The **ItemByIndex** property retrieves an [ISignerCertificate](https://learn.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-isignercertificate) object from the collection by index number.

This property is read-only.
