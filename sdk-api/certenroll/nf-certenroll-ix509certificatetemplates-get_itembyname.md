# IX509CertificateTemplates::get_ItemByName

## Description

The **ItemByName** property retrieves an [IX509CertificateTemplate](https://learn.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-ix509certificatetemplate) object from the collection by name. This property is web enabled.

This property is read-only.
