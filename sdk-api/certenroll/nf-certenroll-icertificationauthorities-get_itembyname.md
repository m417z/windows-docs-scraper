# ICertificationAuthorities::get_ItemByName

## Description

The **ItemByName** property retrieves an [ICertificationAuthority](https://learn.microsoft.com/windows/desktop/api/certenroll/nn-certenroll-icertificationauthority) object from the collection by certification authority name. This property is web enabled.

This property is read-only.
