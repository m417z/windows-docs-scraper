# ICertPropertyDescription::get_Description

## Description

The **Description** property retrieves a description of the certificate. The description identifies the intended use of the certificate.

This property is read-only.

## Parameters

## Remarks

 Call the [Initialize](https://learn.microsoft.com/windows/desktop/api/certenroll/nf-certenroll-icertpropertydescription-initialize) method to create a description.
