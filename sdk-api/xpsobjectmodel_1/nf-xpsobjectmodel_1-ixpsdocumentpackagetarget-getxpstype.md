# IXpsDocumentPackageTarget::GetXpsType

## Description

Gets the [XPS_DOCUMENT_TYPE](https://learn.microsoft.com/windows/win32/api/xpsobjectmodel_1/ne-xpsobjectmodel_1-xps_document_type) enumerated value for the document package.

## Parameters

### `documentType` [out, retval]

The [XPS_DOCUMENT_TYPE](https://learn.microsoft.com/windows/win32/api/xpsobjectmodel_1/ne-xpsobjectmodel_1-xps_document_type) enumerated value.

## Return value

This method returns an HRESULT value. If the method call fails, it returns the appropriate HRESULT error code.
