## Description

Targets a single window for the creation of a graphics capture item.

## Parameters

### `window`

Type: **HWND**

The window handle that represents the window to capture.

### `riid`

Type: **REFIID**

GUID for the type returned. Supported value: [GraphicsCaptureItem](https://learn.microsoft.com/uwp/api/windows.graphics.capture.graphicscaptureitem).

### `result`

Type: **void\*\***

Out pointer for the object to receive.

## Return value

Type: **HRESULT**

The return error code.
