# IDWriteFontList::GetFontCollection

## Description

 Gets the font collection that contains the fonts in
the font list.

## Parameters

### `fontCollection` [out]

Type: **[IDWriteFontCollection](https://learn.microsoft.com/windows/win32/api/dwrite/nn-dwrite-idwritefontcollection)****

When this method returns, contains the address of a pointer to the current [IDWriteFontCollection](https://learn.microsoft.com/windows/win32/api/dwrite/nn-dwrite-idwritefontcollection) object.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
