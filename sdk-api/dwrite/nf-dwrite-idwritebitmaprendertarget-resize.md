# IDWriteBitmapRenderTarget::Resize

## Description

 Resizes the bitmap.

## Parameters

### `width`

Type: **UINT32**

The new bitmap width, in pixels.

### `height`

Type: **UINT32**

The new bitmap height, in pixels.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
