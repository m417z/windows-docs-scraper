# IDWriteTextLayout::SetMaxWidth

## Description

 Sets the layout maximum width.

## Parameters

### `maxWidth`

Type: **FLOAT**

A value that indicates the maximum width of the layout box.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
