# IDWriteFont::GetFontFamily

## Description

 Gets the font family to which the specified font belongs.

## Parameters

### `fontFamily` [out]

Type: **[IDWriteFontFamily](https://learn.microsoft.com/windows/win32/api/dwrite/nn-dwrite-idwritefontfamily)****

When this method returns, contains an address of a pointer to the font family object to which the specified font belongs.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
