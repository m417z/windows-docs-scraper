# IDWriteTextFormat::SetWordWrapping

## Description

 Sets the word wrapping option.

## Parameters

### `wordWrapping`

Type: **[DWRITE_WORD_WRAPPING](https://learn.microsoft.com/windows/win32/api/dwrite/ne-dwrite-dwrite_word_wrapping)**

The word wrapping option being set for a paragraph; see [DWRITE_WORD_WRAPPING](https://learn.microsoft.com/windows/win32/api/dwrite/ne-dwrite-dwrite_word_wrapping) for more information.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
