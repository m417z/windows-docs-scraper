# IDWriteTextLayout::SetUnderline

## Description

Sets underlining for text within a specified text range.

## Parameters

### `hasUnderline`

Type: **BOOL**

A Boolean flag that indicates whether underline takes place within a specified text range.

### `textRange`

Type: **[DWRITE_TEXT_RANGE](https://learn.microsoft.com/windows/win32/api/dwrite/ns-dwrite-dwrite_text_range)**

Text range to which this change applies.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
