# IDWriteFont::CreateFontFace

## Description

 Creates a font face object for the font.

## Parameters

### `fontFace` [out]

Type: **[IDWriteFontFace](https://learn.microsoft.com/windows/win32/api/dwrite/nn-dwrite-idwritefontface)****

When this method returns, contains an address of a pointer to the newly created font face object.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
