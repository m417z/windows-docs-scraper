# IDWriteFontFileEnumerator::GetCurrentFontFile

## Description

 Gets a reference to the current font file.

## Parameters

### `fontFile` [out]

Type: **[IDWriteFontFile](https://learn.microsoft.com/windows/win32/api/dwrite/nn-dwrite-idwritefontfile)****

When this method returns, the address of a pointer to the newly created [IDWriteFontFile](https://learn.microsoft.com/windows/win32/api/dwrite/nn-dwrite-idwritefontfile) object.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
