# ID2D1GradientStopCollection1::GetColorInterpolationMode

## Description

Retrieves the color interpolation mode that the gradient stop collection uses.

## Return value

Type: **[D2D1_COLOR_INTERPOLATION_MODE](https://learn.microsoft.com/windows/desktop/api/d2d1_1/ne-d2d1_1-d2d1_color_interpolation_mode)**

The color interpolation mode.
