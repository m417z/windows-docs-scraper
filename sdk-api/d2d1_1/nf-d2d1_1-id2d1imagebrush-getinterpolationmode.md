# ID2D1ImageBrush::GetInterpolationMode

## Description

Gets the interpolation mode of the image brush.

## Return value

Type: **[D2D1_INTERPOLATION_MODE](https://learn.microsoft.com/windows/desktop/api/d2d1_1/ne-d2d1_1-d2d1_interpolation_mode)**

This method returns the interpolation mode.
