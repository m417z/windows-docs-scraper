# ID2D1Bitmap1::GetOptions

## Description

Gets the options used in creating the bitmap.

## Return value

Type: **[D2D1_BITMAP_OPTIONS](https://learn.microsoft.com/windows/desktop/api/d2d1_1/ne-d2d1_1-d2d1_bitmap_options)**

This method returns the options used.
