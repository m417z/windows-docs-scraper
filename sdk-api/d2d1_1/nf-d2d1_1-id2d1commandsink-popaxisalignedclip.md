# ID2D1CommandSink::PopAxisAlignedClip

## Description

Removes an axis-aligned clip from the layer and clip stack.

## Return value

Type: **HRESULT**

If the method succeeds, it returns **S_OK**. If it fails, it returns an **HRESULT** error code.
