# ID2D1CommandSink::SetTextRenderingParams

## Description

Indicates more detailed text rendering parameters.

## Parameters

### `textRenderingParams` [in]

Type: **[IDWriteRenderingParams](https://learn.microsoft.com/windows/desktop/api/dwrite/nn-dwrite-idwriterenderingparams)***

The parameters to use for text rendering.

## Return value

Type: **HRESULT**

If the method succeeds, it returns **S_OK**. If it fails, it returns an **HRESULT** error code.
