# ID2D1CommandSink::SetTextAntialiasMode

## Description

Indicates the new default antialiasing mode for text.

## Parameters

### `textAntialiasMode`

Type: **[D2D1_TEXT_ANTIALIAS_MODE](https://learn.microsoft.com/windows/desktop/api/d2d1/ne-d2d1-d2d1_text_antialias_mode)**

The antialiasing mode for the text.

## Return value

Type: **HRESULT**

If the method succeeds, it returns **S_OK**. If it fails, it returns an **HRESULT** error code.
