# ID2D1DeviceContext::IsBufferPrecisionSupported

## Description

Indicates whether the buffer precision is supported by the underlying Direct3D [device.](https://learn.microsoft.com/windows/desktop/api/d2d1_1/nn-d2d1_1-id2d1device)

## Parameters

### `bufferPrecision`

Type: **[D2D1_BUFFER_PRECISION](https://learn.microsoft.com/windows/desktop/api/d2d1_1/ne-d2d1_1-d2d1_buffer_precision)**

The buffer precision to check.

## Return value

Type: **BOOL**

Returns TRUE if the buffer precision is supported. Returns FALSE if the buffer precision is not supported.
