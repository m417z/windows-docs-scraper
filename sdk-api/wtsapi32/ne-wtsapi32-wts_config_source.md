# WTS_CONFIG_SOURCE enumeration

## Description

Specifies the source of configuration information returned by the [WTSQueryUserConfig](https://learn.microsoft.com/windows/desktop/api/wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) function. This enumeration type is used in the [WTSUSERCONFIG](https://learn.microsoft.com/windows/desktop/api/wtsapi32/ns-wtsapi32-wtsuserconfiga) structure.

## Constants

### `WTSUserConfigSourceSAM`

The configuration information came from the Security Accounts Manager (SAM) database.
