# WTSINFOEX_LEVEL_A structure

## Description

Contains a [WTSINFOEX_LEVEL1](https://learn.microsoft.com/windows/desktop/api/wtsapi32/ns-wtsapi32-wtsinfoex_level1_a) structure that contains extended information about a Remote Desktop Services session.

## Members

### `WTSInfoExLevel1`

A [WTSINFOEX_LEVEL1](https://learn.microsoft.com/windows/desktop/api/wtsapi32/ns-wtsapi32-wtsinfoex_level1_a) structure that contains extended session information.
