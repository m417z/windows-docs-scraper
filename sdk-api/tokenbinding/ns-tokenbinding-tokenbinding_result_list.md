# TOKENBINDING_RESULT_LIST structure

## Description

Contains the results for each of the token bindings that [TokenBindingVerifyMessage](https://learn.microsoft.com/windows/desktop/api/tokenbinding/nf-tokenbinding-tokenbindingverifymessage) verified.

## Members

### `resultCount`

The number of elements in the array that the **resultData** member contains.

### `resultData`

An array of results, one for each of the token bindings that [TokenBindingVerifyMessage](https://learn.microsoft.com/windows/desktop/api/tokenbinding/nf-tokenbinding-tokenbindingverifymessage) verified.
