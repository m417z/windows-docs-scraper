# TIMECAPS structure

## Description

The **TIMECAPS** structure contains information about the resolution of the timer.

## Members

### `wPeriodMin`

The minimum supported resolution, in milliseconds.

### `wPeriodMax`

The maximum supported resolution, in milliseconds.
