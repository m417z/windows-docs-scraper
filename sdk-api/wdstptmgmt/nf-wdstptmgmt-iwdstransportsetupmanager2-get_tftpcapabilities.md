# IWdsTransportSetupManager2::get_TftpCapabilities

## Description

Receives a mask of [WDSTRANSPORT_TFTP_CAPABILITY](https://learn.microsoft.com/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_tftp_capability) values that indicates which WDS TFTP features are supported by the WDS TFTP server.

This property is read-only.
