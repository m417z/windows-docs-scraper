# IWdsTransportServer::get_SetupManager

## Description

Returns a pointer to the object of an [IWdsTransportSetupManager](https://learn.microsoft.com/windows/desktop/api/wdstptmgmt/nn-wdstptmgmt-iwdstransportsetupmanager) interface used to manage the setup functionality on this server.

This property is read-only.
