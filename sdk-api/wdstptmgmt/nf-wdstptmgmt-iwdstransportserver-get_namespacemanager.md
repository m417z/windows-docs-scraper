# IWdsTransportServer::get_NamespaceManager

## Description

Returns a pointer to the object of an [IWdsTransportNamespaceManager](https://learn.microsoft.com/windows/desktop/api/wdstptmgmt/nn-wdstptmgmt-iwdstransportnamespacemanager) interface used to manage namespaces on this server.

This property is read-only.
