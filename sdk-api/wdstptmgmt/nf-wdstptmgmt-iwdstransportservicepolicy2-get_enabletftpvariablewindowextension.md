# IWdsTransportServicePolicy2::get_EnableTftpVariableWindowExtension

## Description

Receives a [WDSTRANSPORT_TFTP_CAPABILITY](https://learn.microsoft.com/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_tftp_capability) value that specifies whether variable-window extension is enabled on the TFTP server. The value enables a WDS client computer to configure the WDS TFTP server to use variable-window extensions.

This property is read/write.
