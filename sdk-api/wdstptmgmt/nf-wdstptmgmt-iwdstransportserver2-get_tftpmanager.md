# IWdsTransportServer2::get_TftpManager

## Description

Receives a pointer to the object of the [IWdsTransportTftpManager](https://learn.microsoft.com/windows/desktop/api/wdstptmgmt/nn-wdstptmgmt-iwdstransporttftpmanager) interface used to manage the WDS TFTP server. This property is available beginning with Windows Server 2012.

This property is read-only.
