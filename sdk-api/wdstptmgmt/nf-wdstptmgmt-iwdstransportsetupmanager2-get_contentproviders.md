# IWdsTransportSetupManager2::get_ContentProviders

## Description

Receives a pointer to an instance of the [IWdsTransportCollection](https://learn.microsoft.com/windows/desktop/api/wdstptmgmt/nn-wdstptmgmt-iwdstransportcollection) interface. The collection contains objects of the [IWdsTransportContentProvider](https://learn.microsoft.com/windows/desktop/api/wdstptmgmt/nn-wdstptmgmt-iwdstransportcontentprovider) interface for the content providers registered on the server. This property is available beginning with Windows Server 2012.

This property is read-only.
