# IWdsTransportServer::get_ConfigurationManager

## Description

Returns a pointer to the object of an [IWdsTransportConfigurationManager](https://learn.microsoft.com/windows/desktop/api/wdstptmgmt/nn-wdstptmgmt-iwdstransportconfigurationmanager) interface used to manage the configuration of this server.

This property is read-only.
