# IWdsTransportContent::get_Namespace

## Description

Receives a pointer to an object of an [IWdsTransportNamespace](https://learn.microsoft.com/windows/desktop/api/wdstptmgmt/nn-wdstptmgmt-iwdstransportnamespace) interface that represents the namespace associated with this content.

This property is read-only.
