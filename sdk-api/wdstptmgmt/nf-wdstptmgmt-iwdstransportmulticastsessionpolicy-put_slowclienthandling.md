# IWdsTransportMulticastSessionPolicy::put_SlowClientHandling

## Description

Sets or retrieves a [WDSTRANSPORT_SLOW_CLIENT_HANDLING_TYPE](https://learn.microsoft.com/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_slow_client_handling_type) value that indicates the method used by the server to handle clients that are slowing down a multicast transmission.

This property is read/write.
