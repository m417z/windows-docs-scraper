# THEMESIZE enumeration

## Description

Identifies the type of size value to retrieve for a visual style part.

## Constants

### `TS_MIN`

Receives the minimum size of a visual style part.

### `TS_TRUE`

Receives the size of the visual style part that will best fit the available space.

### `TS_DRAW`

Receives the size that the theme manager uses to draw a part.

## Remarks

A value from the **THEMESIZE** enumeration is used with the [GetThemePartSize](https://learn.microsoft.com/windows/desktop/api/uxtheme/nf-uxtheme-getthemepartsize) function to specify the type of size value to retrieve for a particular visual style part.
