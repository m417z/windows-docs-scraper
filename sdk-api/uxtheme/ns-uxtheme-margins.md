# MARGINS structure

## Description

Returned by the [GetThemeMargins](https://learn.microsoft.com/windows/desktop/api/uxtheme/nf-uxtheme-getthememargins) function to define the margins of windows that have visual styles applied.

## Members

### `cxLeftWidth`

Type: **int**

Width of the left border that retains its size.

### `cxRightWidth`

Type: **int**

Width of the right border that retains its size.

### `cyTopHeight`

Type: **int**

Height of the top border that retains its size.

### `cyBottomHeight`

Type: **int**

Height of the bottom border that retains its size.
