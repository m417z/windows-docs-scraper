# BP_ANIMATIONSTYLE enumeration

## Description

Used in the [BP_ANIMATIONPARAMS](https://learn.microsoft.com/windows/desktop/api/uxtheme/ns-uxtheme-bp_animationparams) structure to declare animation options.

## Constants

### `BPAS_NONE`

No animation. Not implemented in Windows Vista.

### `BPAS_LINEAR`

Linear fade animation.

### `BPAS_CUBIC`

Cubic fade animation. Not implemented in Windows Vista.

### `BPAS_SINE`

Sinusoid fade animation. Not implemented in Windows Vista.
