# CLUSTER_GROUP_PRIORITY enumeration

## Description

Specifies the priority level of a group.

## Constants

### `PriorityDisabled:0`

Disabled priority. A group that has a disabled priority does not start automatically.

### `PriorityLow:1000`

Low priority.

### `PriorityMedium:2000`

Medium priority.

### `PriorityHigh:3000`

High priority.
