# DOT11_NETWORK structure

## Description

The **DOT11_NETWORK** structure contains information about an available wireless network.

## Members

### `dot11Ssid`

A [DOT11_SSID](https://learn.microsoft.com/windows/desktop/NativeWiFi/dot11-ssid) structure that contains the SSID of a visible wireless network.

### `dot11BssType`

A [DOT11_BSS_TYPE](https://learn.microsoft.com/windows/desktop/NativeWiFi/dot11-bss-type) value that indicates the BSS type of the network.
