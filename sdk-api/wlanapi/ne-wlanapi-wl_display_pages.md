# WL_DISPLAY_PAGES enumeration

## Description

Specifies the active tab when the wireless profile user interface dialog box appears.

## Constants

### `WLConnectionPage`

Displays the **Connection** tab.

### `WLSecurityPage`

Displays the **Security** tab.
