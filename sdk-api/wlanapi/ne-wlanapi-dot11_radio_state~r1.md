# DOT11_RADIO_STATE enumeration

## Description

The **DOT11_RADIO_STATE** enumeration specifies an 802.11 radio state.

## Constants

### `dot11_radio_state_unknown:0`

The radio state is unknown.

### `dot11_radio_state_on`

The radio is on.

### `dot11_radio_state_off`

The radio is off.
