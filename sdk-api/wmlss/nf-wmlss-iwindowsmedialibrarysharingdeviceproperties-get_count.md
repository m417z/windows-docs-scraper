# IWindowsMediaLibrarySharingDeviceProperties::get_Count

## Description

The **get_Count** method retrieves the number of properties associated with an individual media device.

## Parameters

### `count` [out]

A pointer to a **LONG** that receives the number of properties.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
