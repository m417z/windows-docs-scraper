# Matrix4x4F::SkewY

## Description

Skews the matrix in the Y direction.

## Parameters

### `degreeY`

Type: **FLOAT**

The skew amount.

## Return value

Type: **[Matrix4x4F](https://learn.microsoft.com/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)**

The result matrix.
