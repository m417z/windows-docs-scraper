# Point2L function

## Description

Returns a point struct of long integers.

## Parameters

### `x`

Type: **INT32**

The x-coordinate of the point.

### `y`

Type: **INT32**

The y-coordinate of the point.

## Return value

Type: **[D2D1_POINT_2L](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/hh847948(v=vs.85))**

A POINT structure that defines the x- and y- coordinates of a point.
