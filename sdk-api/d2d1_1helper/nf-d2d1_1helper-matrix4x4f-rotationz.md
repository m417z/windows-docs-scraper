# Matrix4x4F::RotationZ

## Description

Rotates the transform matrix around the Z axis.

## Parameters

### `degreeZ`

Type: **FLOAT**

The amount of rotation.

## Return value

Type: **[Matrix4x4F](https://learn.microsoft.com/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)**

The result matrix.
