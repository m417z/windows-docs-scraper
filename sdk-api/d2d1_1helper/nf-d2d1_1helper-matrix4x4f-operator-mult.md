# Matrix4x4F::operator-mult

## Description

Multiplies this matrix with the specified matrix and returns the result.

## Parameters

### `matrix` [ref]

Type: **const [Matrix4x4F](https://learn.microsoft.com/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)**

The matrix to multiply by.

## Return value

Type: **[Matrix4x4F](https://learn.microsoft.com/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)**

The result of the matrix multiplication.
