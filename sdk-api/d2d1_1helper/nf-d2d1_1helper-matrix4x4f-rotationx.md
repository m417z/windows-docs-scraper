# Matrix4x4F::RotationX

## Description

Rotates the transform matrix around the X axis.

## Parameters

### `degreeX`

Type: **FLOAT**

The amount of rotation.

## Return value

Type: **[Matrix4x4F](https://learn.microsoft.com/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)**

The result matrix.
