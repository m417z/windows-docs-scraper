# Matrix4x4F::ReinterpretBaseType(D2D1_MATRIX_4X4_F)

## Description

Converts the specified [D2D1_MATRIX_4X4_F](https://learn.microsoft.com/windows/desktop/Direct2D/d2d1-matrix-3x2-f) matrix to a [Matrix4x4F](https://learn.microsoft.com/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f) without making a copy.

## Parameters

### `pMatrix`

Type: **[D2D1_MATRIX_4X4_F](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/hh848012(v=vs.85))***

The matrix to convert.

## Return value

Type: **[Matrix4x4F](https://learn.microsoft.com/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)***

The converted matrix.
