# RoGetActivatableClassRegistration function

## Description

Enables retrieving class registration information.

## Parameters

### `activatableClassId` [in]

Type: **[HSTRING](https://learn.microsoft.com/windows/desktop/WinRT/hstring)**

The ID of the activatable class.

### `activatableClassRegistration` [out]

Type: **PActivatableClassRegistration***

The registration information about the activatable class.

## Return value

Type: **HRESULT**

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
