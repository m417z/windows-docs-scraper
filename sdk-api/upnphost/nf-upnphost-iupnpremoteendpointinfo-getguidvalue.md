# IUPnPRemoteEndpointInfo::GetGuidValue

## Description

The **GetGuidValue** method currently is not supported.

## Parameters

### `bstrValueName` [in]

Not supported.

### `pguidValue` [out]

Not supported.

## Return value

This method returns E_INVALIDARG, a COM error code defined in WinError.h.
