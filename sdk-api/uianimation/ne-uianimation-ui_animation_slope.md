# UI_ANIMATION_SLOPE enumeration

## Description

Defines animation slope characteristics.

## Constants

### `UI_ANIMATION_SLOPE_INCREASING:0`

An increasing slope.

### `UI_ANIMATION_SLOPE_DECREASING:1`

A decreasing slope.
