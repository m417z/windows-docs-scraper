# IUIAnimationVariable2::GetFinalIntegerValue

## Description

Gets the final integer value of the animation variable. This is the value after all currently scheduled animations have completed.

## Parameters

### `finalValue` [out]

The final value of the animation variable as an integer.

## Return value

If this method succeeds, it returns S_OK. Otherwise, it returns an **HRESULT** error code. See [Windows Animation Error Codes](https://learn.microsoft.com/windows/desktop/UIAnimation/uianimation-error-codes) for a list of error codes.
