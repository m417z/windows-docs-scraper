# IUIAnimationTimer::IsEnabled

## Description

Determines whether the timer is currently enabled.

## Return value

Returns S_OK if the animation timer is enabled, S_FALSE if the animation timer is disabled, or an **HRESULT** error code. See [Windows Animation Error Codes](https://learn.microsoft.com/windows/desktop/UIAnimation/uianimation-error-codes) for a list of error codes.
