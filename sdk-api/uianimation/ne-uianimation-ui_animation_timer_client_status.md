# UI_ANIMATION_TIMER_CLIENT_STATUS enumeration

## Description

Defines activity status for a timer's client.

## Constants

### `UI_ANIMATION_TIMER_CLIENT_IDLE:0`

The client is idle.

### `UI_ANIMATION_TIMER_CLIENT_BUSY:1`

The client is busy.
