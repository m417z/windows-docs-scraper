# IUIAnimationVariable2::SetVariableCurveChangeHandler

## Description

Specifies a handler for changes to the animation curve of the animation variable.

## Parameters

### `handler` [in, optional]

A pointer to the handler for changes to the animation curve of the animation variable. This parameter can be **NULL**.

## Return value

Returns S_OK if successful; otherwise an **HRESULT** error code. See [Windows Animation Error Codes](https://learn.microsoft.com/windows/desktop/UIAnimation/uianimation-error-codes) for a list of error codes.
