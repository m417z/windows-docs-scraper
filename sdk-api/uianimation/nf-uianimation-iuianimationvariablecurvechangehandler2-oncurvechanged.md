# IUIAnimationVariableCurveChangeHandler2::OnCurveChanged

## Description

Handles events that occur when the animation curve of an animation variable changes.

## Parameters

### `variable` [in]

The animation variable for which the animation curve has been updated.

## Return value

If this method succeeds, it returns S_OK. Otherwise, it returns an **HRESULT** error code. See [Windows Animation Error Codes](https://learn.microsoft.com/windows/desktop/UIAnimation/uianimation-error-codes) for a list of error codes.
