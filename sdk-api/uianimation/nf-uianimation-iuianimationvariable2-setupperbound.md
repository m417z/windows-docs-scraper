# IUIAnimationVariable2::SetUpperBound

## Description

Sets the upper bound (ceiling) for the value of the animation variable. The value of the animation variable should not rise above the specified value.

## Parameters

### `bound` [in]

The upper bound for the value of the animation variable.

## Return value

Returns S_OK if successful; otherwise an **HRESULT** error code. See [Windows Animation Error Codes](https://learn.microsoft.com/windows/desktop/UIAnimation/uianimation-error-codes) for a list of error codes.
