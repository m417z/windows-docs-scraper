# IUIAnimationVariable2::SetLowerBound

## Description

Sets the lower bound (floor) for the value of the animation variable. The value of the animation variable should not fall below the specified value.

## Parameters

### `bound` [in]

The lower bound for the value of the animation variable.

## Return value

Returns S_OK if successful; otherwise an **HRESULT** error code. See [Windows Animation Error Codes](https://learn.microsoft.com/windows/desktop/UIAnimation/uianimation-error-codes) for a list of error codes.
