# IFECommon::InvokeDictToolDialog

## Description

Invokes the Microsoft IME's Dictionary Tool from the app.

## Parameters

### `pimedlg` [in]

Pointer to an [IMEDLG](https://learn.microsoft.com/windows/desktop/api/msime/ns-msime-imedlg) structure.

## Return value

**S_OK** if successful,
otherwise **E_FAIL**.
