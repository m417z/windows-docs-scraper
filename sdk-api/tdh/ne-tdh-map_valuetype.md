# MAP_VALUETYPE enumeration

## Description

Defines if the value map value is in a ULONG data type or a string.

## Constants

### `EVENTMAP_ENTRY_VALUETYPE_ULONG`

Use the **Value** member of [EVENT_MAP_ENTRY](https://learn.microsoft.com/windows/desktop/api/tdh/ns-tdh-event_map_entry) to access the map value.

### `EVENTMAP_ENTRY_VALUETYPE_STRING`

Use the **InputOffset** member of [EVENT_MAP_ENTRY](https://learn.microsoft.com/windows/desktop/api/tdh/ns-tdh-event_map_entry) to access the map value.
