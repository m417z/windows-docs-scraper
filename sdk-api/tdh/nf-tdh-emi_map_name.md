# EMI_MAP_NAME function (tdh.h)

## Description

Macro that retrieves the event map name.

## Parameters

### `MapInfo`

The metadata about the event map ([EVENT_MAP_INFO structure](https://learn.microsoft.com/windows/win32/api/tdh/ns-tdh-event_map_info)).

## Return value

The event map name, or NULL.
