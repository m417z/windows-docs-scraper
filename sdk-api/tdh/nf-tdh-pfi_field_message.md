# PFI_FIELD_MESSAGE function (tdh.h)

## Description

Macro that retrieves the Provider Field Information (PFI) field message.

## Parameters

### `FieldInfoArray`

The [PROVIDER_FIELD_INFOARRAY structure](https://learn.microsoft.com/windows/win32/api/tdh/ns-tdh-provider_field_infoarray).

### `FieldInfo`

Provider field info ([PROVIDER_FIELD_INFO structure](https://learn.microsoft.com/windows/win32/api/tdh/ns-tdh-provider_field_info)).

## Return value

The provider field message, or NULL.
