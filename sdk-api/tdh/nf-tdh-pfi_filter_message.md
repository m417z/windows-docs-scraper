# PFI_FILTER_MESSAGE function (tdh.h)

## Description

Macro that filters the Provider Field Information (PFI) field message.

## Parameters

### `FilterInfo` [in]

Provider filter info ([PROVIDER_FILTER_INFO structure](https://learn.microsoft.com/windows/win32/api/tdh/ns-tdh-provider_filter_info)).

## Return value

The Provider Field Information (PFI) field message, or NULL.
