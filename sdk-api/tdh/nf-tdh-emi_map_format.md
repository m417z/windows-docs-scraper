# EMI_MAP_FORMAT function (tdh.h)

Macro that retrieves the event map format.

## Description

## Parameters

### `MapInfo` [in]

The metadata about the event map ([EVENT_MAP_INFO structure](https://learn.microsoft.com/windows/win32/api/tdh/ns-tdh-event_map_info)).

## Return value

The event map format, or NULL.
