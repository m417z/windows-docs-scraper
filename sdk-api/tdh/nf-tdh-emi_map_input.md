# EMI_MAP_INPUT function (tdh.h)

## Description

Macro that retrieves the event map input.

## Parameters

### `MapInfo`

The metadata about the event map ([EVENT_MAP_INFO structure](https://learn.microsoft.com/windows/win32/api/tdh/ns-tdh-event_map_info)).

### `Map`

A single value map entry ([EVENT_MAP_ENTRY structure](https://learn.microsoft.com/windows/win32/api/tdh/ns-tdh-event_map_entry)).

## Return value

The event map input, or NULL.
