# IStrokeBuilder::get_Ink

## Description

Gets or sets the ink object that is associated with the [IStrokeBuilder](https://learn.microsoft.com/windows/desktop/api/rtscom/nn-rtscom-istrokebuilder) object.

This property is read/write.
