# IDynamicRenderer::put_HWND

## Description

Gets or sets the window handle, HWND, associated with the [DynamicRenderer Class](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/ms701168(v=vs.85)) object.

This property is read/write.
