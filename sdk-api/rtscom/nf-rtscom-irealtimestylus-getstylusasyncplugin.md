# IRealTimeStylus::GetStylusAsyncPlugin

## Description

Retrieves the plug-in at the specified index in the asynchronous plug-in collection.

## Parameters

### `iIndex` [in]

The index for the plug-in that is in the asynchronous plug-in collection.

### `ppiPlugin` [out]

 A pointer to the plug-in.

## Return value

For a description of the return values see [RealTimeStylus Classes and Interfaces](https://learn.microsoft.com/windows/desktop/tablet/realtimestylus-classes-and-interfaces).
