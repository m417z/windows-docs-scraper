# IRealTimeStylus::GetStylusSyncPluginCount

## Description

Retrieves the number of plug-ins in the synchronous plug-in collection.

## Parameters

### `pcPlugins` [out]

 The number of plug-ins in the synchronous plug-in collection.

## Return value

For a description of the return values, see [RealTimeStylus Classes and Interfaces](https://learn.microsoft.com/windows/desktop/tablet/realtimestylus-classes-and-interfaces).
