# UpdateOBStatusInWindowsServerBackup function

## Description

The **UpdateOBStatusInWindowsServerBackup** function updates the cloud backup provider status in the Windows Server Backup MMC snap-in.

## Parameters

### `pOBRegistrationInfo` [in]

Pointer to a [WSB_OB_STATUS_INFO](https://learn.microsoft.com/windows/desktop/api/wsbonline/ns-wsbonline-wsb_ob_status_info) structure.

## Return value

The return values listed here are in addition to the normal **HRESULT**s that may be returned at any time
from the function.
