# WSB_OB_STATUS_ENTRY_VALUE_TYPE_PAIR structure

## Description

 The **WSB_OB_STATUS_ENTRY_VALUE_TYPE_PAIR** structure contains the value and value type for a parameter used to expand the value resource string in the [WSB_OB_STATUS_ENTRY](https://learn.microsoft.com/windows/desktop/api/wsbonline/ns-wsbonline-wsb_ob_status_entry) structure.

## Members

### `m_wszObStatusEntryPairValue`

Specifies the value for the parameter.

### `m_ObStatusEntryPairType`

Specifies the type of the value for the parameter.
