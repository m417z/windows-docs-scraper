# IWICFormatConverterInfo::CreateInstance

## Description

Creates a new [IWICFormatConverter](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicformatconverter) instance.

## Parameters

### `ppIConverter` [out]

Type: **[IWICFormatConverter](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicformatconverter)****

A pointer that receives a new [IWICFormatConverter](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicformatconverter) instance.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
