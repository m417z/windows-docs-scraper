# IWICDevelopRaw::SetRenderMode

## Description

Sets the current [WICRawRenderMode](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wicrawrendermode).

## Parameters

### `RenderMode` [in]

Type: **[WICRawRenderMode](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wicrawrendermode)**

The render mode to use.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
