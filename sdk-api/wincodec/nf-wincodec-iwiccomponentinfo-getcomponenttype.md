# IWICComponentInfo::GetComponentType

## Description

Retrieves the component's [WICComponentType](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wiccomponenttype).

## Parameters

### `pType` [out]

Type: **[WICComponentType](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wiccomponenttype)***

A pointer that receives the [WICComponentType](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wiccomponenttype).

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
