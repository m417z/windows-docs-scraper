# IWICDevelopRaw::SetNotificationCallback

## Description

Sets the notification callback method.

## Parameters

### `pCallback` [in]

Type: **[IWICDevelopRawNotificationCallback](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicdeveloprawnotificationcallback)***

Pointer to the notification callback method.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
