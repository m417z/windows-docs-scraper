# IWICDevelopRaw::SetDestinationColorContext

## Description

Sets the destination color context.

## Parameters

### `pColorContext` [in]

Type: **const [IWICColorContext](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwiccolorcontext)***

The destination color context.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
