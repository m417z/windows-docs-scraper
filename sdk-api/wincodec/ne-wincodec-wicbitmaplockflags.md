# WICBitmapLockFlags enumeration

## Description

Specifies access to an [IWICBitmap](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicbitmap).

## Constants

### `WICBitmapLockRead:0x1`

A read access lock.

### `WICBitmapLockWrite:0x2`

A write access lock.

### `WICBITMAPLOCKFLAGS_FORCE_DWORD:0x7fffffff`
