# IWICJpegFrameDecode::ClearIndexing

## Description

Removes the indexing from a JPEG that has been indexed using [IWICJpegFrameDecode::SetIndexing](https://learn.microsoft.com/windows/desktop/api/wincodec/nf-wincodec-iwicjpegframedecode-setindexing).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns S_OK upon successful completion.
