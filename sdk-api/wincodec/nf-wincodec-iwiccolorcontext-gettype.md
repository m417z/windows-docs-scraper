# IWICColorContext::GetType

## Description

Retrieves the color context type.

## Parameters

### `pType` [out]

Type: **[WICColorContextType](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wiccolorcontexttype)***

A pointer that receives the [WICColorContextType](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wiccolorcontexttype) of the color context.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
