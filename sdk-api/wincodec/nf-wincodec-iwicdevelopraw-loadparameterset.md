# IWICDevelopRaw::LoadParameterSet

## Description

Sets the desired [WICRawParameterSet](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wicrawparameterset) option.

## Parameters

### `ParameterSet` [in]

Type: **[WICRawParameterSet](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wicrawparameterset)**

The desired [WICRawParameterSet](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wicrawparameterset) option.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
