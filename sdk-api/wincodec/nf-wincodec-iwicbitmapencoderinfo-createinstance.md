# IWICBitmapEncoderInfo::CreateInstance

## Description

Creates a new [IWICBitmapEncoder](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicbitmapencoder) instance.

## Parameters

### `ppIBitmapEncoder` [out]

Type: **[IWICBitmapEncoder](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicbitmapencoder)****

A pointer that receives a pointer to a new [IWICBitmapEncoder](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicbitmapencoder) instance.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
