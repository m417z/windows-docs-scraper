# IWICEnumMetadataItem::Clone

## Description

Creates a copy of the current [IWICEnumMetadataItem](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicenummetadataitem).

## Parameters

### `ppIEnumMetadataItem` [out]

Type: **[IWICEnumMetadataItem](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicenummetadataitem)****

A pointer that receives a pointer to the [IWICEnumMetadataItem](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicenummetadataitem) copy.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
