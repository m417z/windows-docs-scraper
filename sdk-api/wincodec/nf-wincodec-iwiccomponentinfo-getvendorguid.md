# IWICComponentInfo::GetVendorGUID

## Description

Retrieves the vendor GUID.

## Parameters

### `pguidVendor` [out]

Type: **GUID***

A pointer that receives the component's vendor GUID.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
