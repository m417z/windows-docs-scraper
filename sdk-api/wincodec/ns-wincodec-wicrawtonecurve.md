# WICRawToneCurve structure

## Description

Represents a raw image tone curve.

## Members

Type: **UINT**

The number of tone curve points.

### `aPoints`

Type: **[WICRawToneCurvePoint](https://learn.microsoft.com/windows/desktop/api/wincodec/ns-wincodec-wicrawtonecurvepoint)[1]**
