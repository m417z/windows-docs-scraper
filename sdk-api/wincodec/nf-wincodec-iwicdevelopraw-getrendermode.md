# IWICDevelopRaw::GetRenderMode

## Description

Gets the current [WICRawRenderMode](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wicrawrendermode).

## Parameters

### `pRenderMode` [out]

Type: **[WICRawRenderMode](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wicrawrendermode)***

A pointer that receives the current [WICRawRenderMode](https://learn.microsoft.com/windows/desktop/api/wincodec/ne-wincodec-wicrawrendermode).

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
