# IWICDevelopRaw::GetCurrentParameterSet

## Description

Gets the current set of parameters.

## Parameters

### `ppCurrentParameterSet` [out]

Type: **[IPropertyBag2](https://learn.microsoft.com/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa768192(v=vs.85))****

A pointer that receives a pointer to the current set of parameters.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
