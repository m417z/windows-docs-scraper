# IWebApplicationNavigationEvents::DownloadComplete

## Description

Download of a page has completed.

## Return value

Type: **HRESULT**

Ignored by the host. If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
