# IWebApplicationUpdateEvents::OnCssChanged

## Description

Notifies the authoring app that the Cascading Style Sheets (CSS) has changed.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
