# LATTICE_METRICS structure

## Description

Describes the baseline and the midline height.

## Members

### `lsBaseline`

The [LINE_SEGMENT](https://learn.microsoft.com/windows/desktop/api/rectypes/ns-rectypes-line_segment) structure containing the start and end point of the baseline, in ink coordinates.

### `iMidlineOffset`

Offset of the midline, relative to the baseline, in HIMETRIC units.
