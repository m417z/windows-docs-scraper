# BTH_QUERY_DEVICE structure

## Description

The
**BTH_QUERY_DEVICE** structure is used when querying for the presence of a Bluetooth device.

## Members

### `LAP`

Reserved. Must be set to zero.

### `length`

Requested length of the inquiry, in seconds.

## Remarks

See the Bluetooth specification at
[www.bluetooth.com](https://www.bluetooth.com/) for additional information about LAP.
