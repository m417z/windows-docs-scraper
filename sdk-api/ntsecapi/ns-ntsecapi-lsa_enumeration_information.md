# LSA_ENUMERATION_INFORMATION structure

## Description

The **LSA_ENUMERATION_INFORMATION** structure is used with the
[LsaEnumerateAccountsWithUserRight](https://learn.microsoft.com/windows/desktop/api/ntsecapi/nf-ntsecapi-lsaenumerateaccountswithuserright) function to return a pointer to a SID.

## Members

### `Sid`

Pointer to a SID.
