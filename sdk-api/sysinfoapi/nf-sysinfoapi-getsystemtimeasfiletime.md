# GetSystemTimeAsFileTime function

## Description

Retrieves the current system date and time. The information is in Coordinated Universal Time (UTC) format.

## Parameters

### `lpSystemTimeAsFileTime` [out]

A pointer to a
[FILETIME](https://learn.microsoft.com/windows/desktop/api/minwinbase/ns-minwinbase-filetime) structure to receive the current system date and time in UTC format.
