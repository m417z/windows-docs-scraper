# NMTOOLTIPSCREATED structure

## Description

Contains information used with [NM_TOOLTIPSCREATED](https://learn.microsoft.com/windows/desktop/Controls/nm-tooltipscreated) notification codes.

## Members

### `hdr`

Type: **[NMHDR](https://learn.microsoft.com/windows/desktop/api/richedit/ns-richedit-nmhdr)**

An [NMHDR](https://learn.microsoft.com/windows/desktop/api/richedit/ns-richedit-nmhdr) structure that contains additional information about this notification.

### `hwndToolTips`

Type: **[HWND](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

The window handle to the tooltip control created.
