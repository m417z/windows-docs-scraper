# OPM_OMAC structure

## Description

Contains a Message Authentication Code (MAC) for an [Output Protection Manager](https://learn.microsoft.com/windows/desktop/medfound/output-protection-manager) (OPM) message.

## Members

### `abOMAC`

A buffer that contains the cryptographic MAC value of the message.
