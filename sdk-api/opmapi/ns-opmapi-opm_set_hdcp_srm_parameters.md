# OPM_SET_HDCP_SRM_PARAMETERS structure

## Description

Contains parameters for the [OPM_SET_HDCP_SRM](https://learn.microsoft.com/windows/desktop/medfound/opm-set-hdcp-srm) command. This command updates the system renewability message (SRM) for High-Bandwidth Digital Content Protection (HDCP).

## Members

### `ulSRMVersion`

Contains the SRM version number in little-endian format. This number is contained in the **SRM Version** field of the SRM. For more information, see the HDCP specification.
