# OPM_DPCP_PROTECTION_LEVEL enumeration

## Description

Specifies the protection level for DisplayPort Content Protection (DPCP).

## Constants

### `OPM_DPCP_OFF:0`

DPCP is disabled.

### `OPM_DPCP_ON:1`

DPCP is enabled.

### `OPM_DPCP_FORCE_ULONG:0x7fffffff`

Reserved.
