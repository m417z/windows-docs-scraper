## Description

Specifies how ink is inserted onto the [InkEdit](https://learn.microsoft.com/windows/win32/tablet/inkedit-control-reference) control.

## Constants

### `IEM_InsertText:0`

Default. Drawn ink is inserted as text.

### `IEM_InsertInk:1`

Drawn ink is inserted as ink.
