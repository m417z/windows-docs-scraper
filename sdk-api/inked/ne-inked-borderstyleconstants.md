# BorderStyleConstants enumeration

## Description

Specifies how the borders of an [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control-reference) control appear on the screen.

## Constants

### `rtfNoBorder:0`

No border.

### `rtfFixedSingle:1`

Single line border.
