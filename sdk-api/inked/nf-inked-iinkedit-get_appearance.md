# IInkEdit::get_Appearance

## Description

Gets or sets a value that determines the appearance of the [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control-reference) control - whether it is flat (painted with no visual effects) or 3D (painted with three-dimensional effects).

This property is read/write.
