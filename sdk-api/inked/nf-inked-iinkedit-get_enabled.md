# IInkEdit::get_Enabled

## Description

Gets or sets a value that determines whether the [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control-reference) control can respond to user-generated events.

This property is read/write.

## Parameters

## Remarks

The **Enabled** property allows objects to be enabled or disabled at run time.
