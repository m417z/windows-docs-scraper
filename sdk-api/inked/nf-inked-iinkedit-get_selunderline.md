# IInkEdit::get_SelUnderline

## Description

Gets or sets a value that specifies whether the font style of the currently selected text in the [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control) control is underlined (run time only).

This property is read/write.
