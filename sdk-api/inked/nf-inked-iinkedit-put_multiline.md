# IInkEdit::put_MultiLine

## Description

 Gets or sets a value indicating whether an [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control-reference) control can accept and display multiple lines of text.

This property is read/write.

## Parameters

## Remarks

A multiple-line [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control-reference) control wraps text as the user types text extending beyond the text box.

You can also add scroll bars to a larger [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control-reference) control using the [ScrollBars](https://learn.microsoft.com/windows/desktop/api/inked/nf-inked-iinkedit-get_scrollbars) property. If no HScrollBar control (horizontal scroll bar) is specified, the text in a multiple-line InkEdit control automatically wraps.
