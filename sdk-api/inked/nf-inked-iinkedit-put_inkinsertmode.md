# IInkEdit::put_InkInsertMode

## Description

Gets or sets a value that specifies how ink is inserted onto the [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control-reference) control, either as text or as ink.

This property is read/write.
