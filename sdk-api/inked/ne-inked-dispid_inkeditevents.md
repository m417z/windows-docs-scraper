# DISPID_InkEditEvents enumeration

## Description

Specifies various ink editor events.

## Constants

### `DISPID_IeeChange`

Content changed.

### `DISPID_IeeSelChange`

Selection changed.

### `DISPID_IeeKeyDown`

Key down.

### `DISPID_IeeKeyUp`

Key up.

### `DISPID_IeeMouseUp`

Mouse button up.

### `DISPID_IeeMouseDown`

Mouse button down.

### `DISPID_IeeKeyPress`

Key press and release.

### `DISPID_IeeDblClick`

Double click.

### `DISPID_IeeClick`

Single click.

### `DISPID_IeeMouseMove`

Mouse move.

### `DISPID_IeeCursorDown`

Cursor down.

### `DISPID_IeeStroke`

Ink stroke.

### `DISPID_IeeGesture`

Ink gesture.

### `DISPID_IeeRecognitionResult`

Recognition result.
