# IInkEdit::put_SelRTF

## Description

Gets or sets the currently selected Rich Text Format (RTF) formatted text in the [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control) control (run time only).

This property is read/write.

## Parameters

## Remarks

Setting the SelRTF property replaces any selected text in the [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control) control with the new string. This property returns a zero-length string ("") if no text is selected in the control.
