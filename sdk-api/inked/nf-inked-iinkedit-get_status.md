# IInkEdit::get_Status

## Description

Gets a value that specifies whether the [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control-reference) control is idle, collecting ink, or recognizing ink.

This property is read-only.

## Parameters

## Remarks

This property is available at run time only.
