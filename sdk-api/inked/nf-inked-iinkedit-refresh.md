# IInkEdit::Refresh

## Description

Causes the [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control-reference) control to redraw.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
