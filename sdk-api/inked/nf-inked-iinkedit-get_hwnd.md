# IInkEdit::get_Hwnd

## Description

Gets a handle to the [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control-reference) control.

This property is read-only.

## Parameters

## Remarks

**Note** Because the value of this property can change while a program is running, never store the Hwnd value in a variable.
