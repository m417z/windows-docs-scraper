# IInkEdit::get_Font

## Description

Gets or sets a Font object.

This property is read/write.

## Parameters

## Remarks

Use the **Font** property of an [InkEdit](https://learn.microsoft.com/windows/desktop/tablet/inkedit-control-reference) control to identify a specific Font object whose properties you want to use. For example, the following code changes the **Bold** property setting of a Font object identified by the **Font** property of an InkEdit control:

`myInkEditControl.Font.Bold = true;`
