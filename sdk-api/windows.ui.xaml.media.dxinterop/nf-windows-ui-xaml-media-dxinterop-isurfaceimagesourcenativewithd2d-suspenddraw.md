# ISurfaceImageSourceNativeWithD2D::SuspendDraw (windows.ui.xaml.media.dxinterop.h)

## Description

Suspends the drawing operation.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
