# IVirtualSurfaceUpdatesCallbackNative::xaml

## Description

Performs the drawing behaviors when an update to [VirtualSurfaceImageSource](https://learn.microsoft.com/windows/desktop/api/windows.ui.xaml.media.dxinterop/nn-windows-ui-xaml-media-dxinterop-ivirtualsurfaceimagesourcenative) is requested.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.

## Remarks

This method is implemented by the developer.
