# ISwapChainBackgroundPanelNative::xaml

## Description

Sets the DirectX swap chain for [SwapChainBackgroundPanel](https://learn.microsoft.com/uwp/api/windows.ui.xaml.controls.swapchainbackgroundpanel).

## Parameters

### `swapChain` [in]

A configured [IDXGISwapChain](https://learn.microsoft.com/windows/desktop/api/dxgi/nn-dxgi-idxgiswapchain).

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
