# ISurfaceImageSourceNative::EndDraw (windows.ui.xaml.media.dxinterop.h)

## Description

Closes the surface draw operation.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
