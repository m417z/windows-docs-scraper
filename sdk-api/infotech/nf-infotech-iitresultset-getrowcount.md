# IITResultSet::GetRowCount

## Description

Gets the number of rows in a result set.

## Parameters

### `lNumberOfRows` [out, ref]

Number of rows.

## Return value

This method can return one of these values.

| Return code | Description |
| --- | --- |
| **S_OK** | The number of rows was successfully retrieved. |
