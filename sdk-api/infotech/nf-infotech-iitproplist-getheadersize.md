# IITPropList::GetHeaderSize

## Description

Returns the number of bytes needed to save the header.

## Parameters

### `dwHdrSize` [out, ref]

Size, in bytes.

## Return value

This method can return one of these values.

| Return code | Description |
| --- | --- |
| **S_OK** | The size was successfully returned. |
