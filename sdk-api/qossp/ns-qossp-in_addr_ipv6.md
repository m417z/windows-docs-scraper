# IN_ADDR_IPV6 structure

## Description

The **IN_ADDR_IPV6** structure stores an IPv6 address for use with RSVP FILTERSPECs.

## Members

### `Addr`

IPv6 address.

## Remarks

When working with IPv4 addresses, use [IN_ADDR_IPV4](https://learn.microsoft.com/windows/desktop/api/qossp/ns-qossp-in_addr_ipv4).
