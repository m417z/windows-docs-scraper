# RSVP_FILTERSPEC_V6_FLOW structure

## Description

The **RSVP_FILTERSPEC_V6_FLOW** structure provides flow label information for an IPv6 FILTERSPEC.

## Members

### `Address`

IPv4 address for which the FILTERSPEC flow label applies, expressed as an [IN_ADDR_IPV6](https://learn.microsoft.com/windows/desktop/api/qossp/ns-qossp-in_addr_ipv6) structure.

### `UnUsed`

### `FlowLabel`

Label for the flow.

### `Unused`

Reserved. Do not use.
