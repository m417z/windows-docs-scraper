# IN_ADDR_IPV4 structure

## Description

The **IN_ADDR_IPV4** union stores an IPv4 address for use with RSVP FILTERSPECs.

## Members

### `Addr`

IPv4 address, expressed as a ULONG.

### `AddrBytes`

IPv4 address, expressed as four UCHARs.

## Remarks

When working with IPv6 addresses, use [IN_ADDR_IPV6](https://learn.microsoft.com/windows/desktop/api/qossp/ns-qossp-in_addr_ipv6).
