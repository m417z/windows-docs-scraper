# ICEnroll::freeRequestInfo

## Description

[This method is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **freeRequestInfo** method releases session identifiers when they are no longer needed. This method was first defined in the [ICEnroll](https://learn.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-icenroll) interface.

## Parameters

### `PKCS7OrPKCS10` [in]

Specifies the session identifier that represents the data.
