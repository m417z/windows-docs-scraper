# IEnroll2::SetHStoreMy

## Description

[This method is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **SetHStoreMy** method specifies the handle to use for the MY store. This method was first defined in the [IEnroll2](https://learn.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-ienroll2) interface.

## Parameters

### `hStore` [in]

Certificate store handle to use for the MY store.

## Return value

The return value is an **HRESULT**. A value of S_OK indicates success.
