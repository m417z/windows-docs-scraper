# IEnroll4::resetAttributes

## Description

[This method is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **resetAttributes** method removes all attributes from the request. This method was first defined in the [IEnroll4](https://learn.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-ienroll4) interface.
