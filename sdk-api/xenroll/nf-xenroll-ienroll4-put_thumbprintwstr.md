# IEnroll4::put_ThumbPrintWStr

## Description

[This property is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **ThumbPrintWStr** property sets or retrieves a [hash](https://learn.microsoft.com/windows/desktop/SecGloss/h-gly) of the certificate data.
The thumbprint is used to point to the pending certificate.

This property was first defined in the [IEnroll4](https://learn.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-ienroll4) interface.

This property is read/write.
