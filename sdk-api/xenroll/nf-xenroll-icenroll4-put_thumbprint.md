# ICEnroll4::put_ThumbPrint

## Description

[This property is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **ThumbPrint** property sets or retrieves a [hash](https://learn.microsoft.com/windows/desktop/SecGloss/h-gly) of the certificate data.

This property was first defined in the [ICEnroll4](https://learn.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-icenroll4) interface.

This property is read/write.

## Parameters

## Remarks

The thumbprint is used to point to a pending certificate.
