# IEnroll4::removePendingRequestWStr

## Description

[This method is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **removePendingRequestWStr** method removes a pending request from the client's request store. This method was first defined in the [IEnroll4](https://learn.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-ienroll4) interface.

## Parameters

### `thumbPrintBlob` [in]

[CRYPT_DATA_BLOB](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/aa381414(v=vs.85)) structure that represents the thumbprint, or [hash](https://learn.microsoft.com/windows/desktop/SecGloss/h-gly), of the certificate data.
