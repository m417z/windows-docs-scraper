# IEnroll4::get_IncludeSubjectKeyID

## Description

[This property is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **IncludeSubjectKeyID** property determines whether the subject key ID extension is added to the certificate request that is generated. The [object identifier](https://learn.microsoft.com/windows/desktop/SecGloss/o-gly) (OID) for the subject key ID extension is "2.5.29.14", which is the defined value of the szOID_SUBJECT_KEY_IDENTIFIER constant in Wincrypt.h. This property was first defined in the [IEnroll4](https://learn.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-ienroll4) interface.

This property is read/write.
