# IEnroll::getROOTHStore

## Description

[This method is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **getROOTHStore** method is not implemented.

## Return value

This method always returns **NULL**. [GetLastError](https://learn.microsoft.com/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror) will return **ERROR_CALL_NOT_IMPLEMENTED**.
