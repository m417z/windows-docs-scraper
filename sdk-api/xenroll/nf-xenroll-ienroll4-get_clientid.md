# IEnroll4::get_ClientId

## Description

[This property is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **ClientId** property sets or retrieves a client ID request attribute. The client ID request attribute indicates the source of the certificate request. This property was first defined in the [IEnroll4](https://learn.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-ienroll4) interface.

This property is read/write.
