# ICEnroll4::resetBlobProperties

## Description

[This method is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **resetBlobProperties** method resets the properties of a [BLOB](https://learn.microsoft.com/windows/desktop/SecGloss/b-gly).

This property was first defined in the [ICEnroll4](https://learn.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-icenroll4) interface.
