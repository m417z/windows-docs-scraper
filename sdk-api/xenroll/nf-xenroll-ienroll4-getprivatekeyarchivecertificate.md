# IEnroll4::GetPrivateKeyArchiveCertificate

## Description

[This method is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **GetPrivateKeyArchiveCertificate** method retrieves the certificate used to archive the private key. This method was first defined in the [IEnroll4](https://learn.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-ienroll4) interface.

## Return value

Pointer to a [CERT_CONTEXT](https://learn.microsoft.com/windows/desktop/api/wincrypt/ns-wincrypt-cert_context) structure that represents the certificate used to archive the private key, or **NULL** if an error is encountered.
