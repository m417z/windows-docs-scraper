# IEnroll2::Reset

## Description

[This method is no longer available for use as of Windows Server 2008 and Windows Vista.]

The **Reset** method returns the
certificate enrollment control object to its initial [state](https://learn.microsoft.com/windows/desktop/SecGloss/s-gly) and thereby allows reuse of the control. This method was first defined in the [IEnroll2](https://learn.microsoft.com/windows/desktop/api/xenroll/nn-xenroll-ienroll2) interface.

## Return value

The return value is an **HRESULT**. A value of S_OK indicates success.
