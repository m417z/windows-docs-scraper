# RTM_ENTITY_EXPORT_METHODS structure

## Description

The
**RTM_ENTITY_EXPORT_METHODS** structure contains the set of methods exported by a client.

## Members

### `NumMethods`

Specifies the number of methods exported by the client in the **Methods** member.

### `Methods`

Specifies which methods the client is exporting.
