# ITAllocatorProperties::SetBufferSize

## Description

The
**SetBufferSize** method sets the size of the allocator buffer.

## Parameters

### `BufferSize` [in]

Size of the allocator buffer.

## Return value

If the method succeeds, it returns S_OK. Otherwise, it returns an error value.
