# ITAllocatorProperties::GetBufferSize

## Description

The
**GetBufferSize** method gets the size of the allocator buffer.

## Parameters

### `pBufferSize` [out]

Size of current allocator buffer.

## Return value

If the method succeeds, it returns S_OK. Otherwise, it returns an error value.
