# PathGradientBrush::PathGradientBrush(IN const GraphicsPath)

## Description

Creates a **PathGradientBrush::PathGradientBrush** object based on a [GraphicsPath](https://learn.microsoft.com/windows/desktop/api/gdipluspath/nl-gdipluspath-graphicspath) object.

## Parameters

### `path` [in]

Type: **const [GraphicsPath](https://learn.microsoft.com/windows/desktop/api/gdipluspath/nl-gdipluspath-graphicspath)***

Pointer to a [GraphicsPath](https://learn.microsoft.com/windows/desktop/api/gdipluspath/nl-gdipluspath-graphicspath) object that specifies the boundary path of the path gradient brush.
