# GraphicsPath::Clone

## Description

The **GraphicsPath::Clone** method creates a new [GraphicsPath](https://learn.microsoft.com/windows/desktop/api/gdipluspath/nl-gdipluspath-graphicspath) object, and initializes it with the contents of this **GraphicsPath** object.

## Return value

Type: **[GraphicsPath](https://learn.microsoft.com/windows/desktop/api/gdipluspath/nl-gdipluspath-graphicspath)***

This method returns a pointer to the new [GraphicsPath](https://learn.microsoft.com/windows/desktop/api/gdipluspath/nl-gdipluspath-graphicspath) object.
