# GraphicsPath::GetFillMode

## Description

The **GraphicsPath::GetFillMode** method gets the fill mode of this path.

## Return value

Type: **[FillMode](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-fillmode)**

This method returns an element of the [FillMode](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-fillmode) enumeration.
