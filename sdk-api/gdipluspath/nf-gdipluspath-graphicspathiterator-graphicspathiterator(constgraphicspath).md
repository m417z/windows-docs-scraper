## Description

Creates a new **GraphicsPathIterator::GraphicsPathIterator** object and associates it with a [GraphicsPath](https://learn.microsoft.com/windows/desktop/api/gdipluspath/nl-gdipluspath-graphicspath) object.

## Parameters

### `path` [in]

Type: **[GraphicsPath](https://learn.microsoft.com/windows/desktop/api/gdipluspath/nl-gdipluspath-graphicspath)***

Pointer to a [GraphicsPath](https://learn.microsoft.com/windows/desktop/api/gdipluspath/nl-gdipluspath-graphicspath) object that will be associated with this **GraphicsPathIterator::GraphicsPathIterator** object.
