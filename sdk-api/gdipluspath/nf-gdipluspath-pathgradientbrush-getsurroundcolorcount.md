# PathGradientBrush::GetSurroundColorCount

## Description

The **PathGradientBrush::GetSurroundColorCount** method gets the number of colors that have been specified for the boundary path of this path gradient brush.

## Return value

Type: **INT**

This method returns the number of colors that have been specified for the boundary path of this path gradient brush.
