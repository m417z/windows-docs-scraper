# WINBIO_ADAPTER_INTERFACE_VERSION structure

## Description

The **WINBIO_ADAPTER_INTERFACE_VERSION** structure contains a major and minor version number used in the engine, sensor, and storage adapter interface tables.

## Members

### `MajorVersion`

Contains the major version number.

### `MinorVersion`

Contains the minor version number.

## Remarks

This structure is used by the following structures:

* [WINBIO_ENGINE_INTERFACE](https://learn.microsoft.com/windows/win32/api/winbio_adapter/ns-winbio_adapter-winbio_engine_interface)
* [WINBIO_SENSOR_INTERFACE](https://learn.microsoft.com/windows/win32/api/winbio_adapter/ns-winbio_adapter-winbio_sensor_interface)
* [WINBIO_STORAGE_INTERFACE](https://learn.microsoft.com/windows/win32/api/winbio_adapter/ns-winbio_adapter-winbio_storage_interface)
