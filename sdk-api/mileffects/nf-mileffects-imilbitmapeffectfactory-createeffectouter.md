# IMILBitmapEffectFactory::CreateEffectOuter

## Description

Creates an outer [IMILBitmapEffect](https://learn.microsoft.com/previous-versions/windows/desktop/api/mileffects/nn-mileffects-imilbitmapeffect) object.

## Parameters

### `ppEffect` [out]

Type: **[IMILBitmapEffect](https://learn.microsoft.com/previous-versions/windows/desktop/api/mileffects/nn-mileffects-imilbitmapeffect)****

A pointer that receives a pointer to the new [IMILBitmapEffect](https://learn.microsoft.com/previous-versions/windows/desktop/api/mileffects/nn-mileffects-imilbitmapeffect) object.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
