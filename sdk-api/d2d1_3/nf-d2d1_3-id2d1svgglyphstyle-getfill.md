# ID2D1SvgGlyphStyle::GetFill

## Description

Returns the requested fill parameters.

## Parameters

### `brush` [out]

Type: **[ID2D1Brush](https://learn.microsoft.com/windows/desktop/api/d2d1/nn-d2d1-id2d1brush)****

Describes how the area is painted.
