## Description

Creates an [ID2D1Device4](https://learn.microsoft.com/windows/desktop/api/d2d1_3/nn-d2d1_3-id2d1device4) object.

## Parameters

### `dxgiDevice` [in]

Type: **[IDXGIDevice](https://learn.microsoft.com/windows/desktop/api/dxgi/nn-dxgi-idxgidevice)***

The IDXGIDevice object used when creating the [ID2D1Device4](https://learn.microsoft.com/windows/desktop/api/d2d1_3/nn-d2d1_3-id2d1device4).

### `d2dDevice4` [out]

Type: **[ID2D1Device4](https://learn.microsoft.com/windows/desktop/api/d2d1_3/nn-d2d1_3-id2d1device4)****

The requested [ID2D1Device4](https://learn.microsoft.com/windows/desktop/api/d2d1_3/nn-d2d1_3-id2d1device4) object.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

This method returns an HRESULT success or error code.
