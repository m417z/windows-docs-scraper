# IDiskQuotaUserBatch::RemoveAll

## Description

Removes all
[IDiskQuotaUser](https://learn.microsoft.com/windows/desktop/api/dskquota/nn-dskquota-idiskquotauser) pointers from the batch list.

## Return value

This method returns one of the following values.

| Return code | Description |
| --- | --- |
| **S_OK** | Success. |
| **E_UNEXPECTED** | An unexpected exception occurred. |
