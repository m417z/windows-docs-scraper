# IMFMediaEngine::GetNetworkState

## Description

Gets the current network state of the media engine.

## Return value

Returns an [MF_MEDIA_ENGINE_NETWORK](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/ne-mfmediaengine-mf_media_engine_network) enumeration value.

## Remarks

This method corresponds to the **networkState** attribute of the **HTMLMediaElement** interface in HTML5.
