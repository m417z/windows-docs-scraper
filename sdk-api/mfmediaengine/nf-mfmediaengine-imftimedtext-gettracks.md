# IMFTimedText::GetTracks

## Description

Retrieves a list of all timed-text tracks registered with the [IMFTimedText](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imftimedtext).

## Parameters

### `tracks` [out]

Type: **IMFTimedTextTrackList****

Receives a pointer to track list.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
