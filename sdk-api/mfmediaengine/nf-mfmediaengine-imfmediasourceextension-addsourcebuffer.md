# IMFMediaSourceExtension::AddSourceBuffer

## Description

Adds a [IMFSourceBuffer](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfsourcebuffer) to the collection of buffers associated with the [IMFMediaSourceExtension](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfmediasourceextension).

## Parameters

### `type` [in]

### `pNotify` [in]

### `ppSourceBuffer` [out]

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
