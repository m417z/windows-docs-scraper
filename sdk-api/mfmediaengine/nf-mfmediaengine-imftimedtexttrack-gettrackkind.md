# IMFTimedTextTrack::GetTrackKind

## Description

Gets the kind of timed-text track.

## Return value

Type: **[MF_TIMED_TEXT_TRACK_KIND](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/ne-mfmediaengine-mf_timed_text_track_kind)**

Returns a [MF_TIMED_TEXT_TRACK_KIND](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/ne-mfmediaengine-mf_timed_text_track_kind)-typed value that specifies the kind of timed-text track.
