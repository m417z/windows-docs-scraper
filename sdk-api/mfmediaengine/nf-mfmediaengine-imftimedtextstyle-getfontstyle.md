# IMFTimedTextStyle::GetFontStyle

## Description

Gets the font style of the timed-text style.

## Parameters

### `fontStyle` [out]

Type: **[MF_TIMED_TEXT_FONT_STYLE](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/ne-mfmediaengine-mf_timed_text_font_style)***

A pointer to a variable that receives a [MF_TIMED_TEXT_FONT_STYLE](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/ne-mfmediaengine-mf_timed_text_font_style)-typed value that specifies the font style.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
