# IMFTimedTextRegion::GetBackgroundColor

## Description

Gets the background color of the region.

## Parameters

### `bgColor` [out]

Type: **[MFARGB](https://learn.microsoft.com/windows/desktop/api/mfobjects/ns-mfobjects-mfargb)***

A pointer to a variable that receives a [MFARGB](https://learn.microsoft.com/windows/desktop/api/mfobjects/ns-mfobjects-mfargb) structure that describes the background color.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
