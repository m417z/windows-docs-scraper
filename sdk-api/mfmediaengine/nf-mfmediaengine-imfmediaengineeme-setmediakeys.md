# IMFMediaEngineEME::SetMediaKeys

## Description

Sets the media keys object to use with the media engine.

## Parameters

### `keys`

The media keys.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
