# IMFTimedTextStyle::GetBold

## Description

Determines whether the style of timed text is bold.

## Parameters

### `bold` [out]

Type: **BOOL***

A pointer to a variable that receives a value that specifies whether the style of timed text is bold. The variable specifies **TRUE** if the style is bold; otherwise, **FALSE**.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
