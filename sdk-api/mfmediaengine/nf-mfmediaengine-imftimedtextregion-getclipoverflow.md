# IMFTimedTextRegion::GetClipOverflow

## Description

Determines whether a clip of text overflowed the region.

## Parameters

### `clipOverflow` [out]

Type: **BOOL***

A pointer to a variable that receives a value that specifies whether a clip of text overflowed the region. The variable specifies **TRUE** if the clip overflowed; otherwise, **FALSE**.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
