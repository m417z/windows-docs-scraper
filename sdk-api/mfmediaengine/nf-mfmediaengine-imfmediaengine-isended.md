# IMFMediaEngine::IsEnded

## Description

Queries whether playback has ended.

## Return value

Returns **TRUE** if the direction of playback is forward and playback has reached the end of the media resource. Returns **FALSE** otherwise.

## Remarks

This method corresponds to the **ended** attribute of the **HTMLMediaElement** interface in HTML5.
