# IMFMediaEngine::SetMuted

## Description

Mutes or unmutes the audio.

## Parameters

### `Muted` [in]

Specify **TRUE** to mute the audio, or **FALSE** to unmute the audio.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
