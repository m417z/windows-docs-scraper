# IMFTimedText::SetInBandEnabled

## Description

Enables or disables inband mode.

## Parameters

### `enabled` [in]

Type: **BOOL**

Specifies whether inband mode is enabled. If **TRUE**, inband mode is enabled. If **FALSE**, inband mode is disabled.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
