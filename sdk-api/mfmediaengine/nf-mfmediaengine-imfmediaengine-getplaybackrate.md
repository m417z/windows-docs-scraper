# IMFMediaEngine::GetPlaybackRate

## Description

Gets the current playback rate.

## Return value

Returns the playback rate, as a multiple of normal (1Ã—) playback. A negative value indicates reverse playback.

## Remarks

This method corresponds to getting the **playbackRate** attribute of the **HTMLMediaElement** interface in HTML5.
