# IMFMediaEngineEx::ApplyStreamSelections

## Description

Applies the stream selections from previous calls to [SetStreamSelection](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nf-mfmediaengine-imfmediaengineex-setstreamselection).

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
