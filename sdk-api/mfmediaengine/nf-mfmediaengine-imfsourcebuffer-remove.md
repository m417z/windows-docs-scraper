# IMFSourceBuffer::Remove

## Description

Removes the media segments defined by the specified time range from the [IMFSourceBuffer](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfsourcebuffer).

## Parameters

### `start` [in]

The start of the time range.

### `end` [in]

The end of the time range.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
