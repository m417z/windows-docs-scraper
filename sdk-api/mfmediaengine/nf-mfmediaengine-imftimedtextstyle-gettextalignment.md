# IMFTimedTextStyle::GetTextAlignment

## Description

Gets the text alignment of the timed-text style.

## Parameters

### `textAlign` [out]

Type: **[MF_TIMED_TEXT_ALIGNMENT](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/ne-mfmediaengine-mf_timed_text_alignment)***

A pointer to a variable that receives a [MF_TIMED_TEXT_ALIGNMENT](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/ne-mfmediaengine-mf_timed_text_alignment)-typed value that specifies the text alignment.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
