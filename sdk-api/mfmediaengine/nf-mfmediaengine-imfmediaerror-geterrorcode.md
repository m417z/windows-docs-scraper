# IMFMediaError::GetErrorCode

## Description

Gets the error code.

## Return value

Returns a value from the [MF_MEDIA_ENGINE_ERR](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/ne-mfmediaengine-mf_media_engine_err) enumeration.
