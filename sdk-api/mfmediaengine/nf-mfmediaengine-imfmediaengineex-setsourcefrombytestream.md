# IMFMediaEngineEx::SetSourceFromByteStream

## Description

Opens a media resource from a byte stream.

## Parameters

### `pByteStream` [in]

A pointer to the [IMFByteStream](https://learn.microsoft.com/windows/desktop/api/mfobjects/nn-mfobjects-imfbytestream) interface of the byte stream.

### `pURL` [in]

The URL of the byte stream.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
