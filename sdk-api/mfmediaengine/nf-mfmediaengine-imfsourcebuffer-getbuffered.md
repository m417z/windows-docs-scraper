# IMFSourceBuffer::GetBuffered

## Description

Gets the buffered time range.

## Parameters

### `ppBuffered` [out]

The buffered time range.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
