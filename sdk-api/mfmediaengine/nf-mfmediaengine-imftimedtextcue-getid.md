# IMFTimedTextCue::GetId

## Description

Gets the identifier of a timed-text cue.

## Return value

Type: **DWORD**

The identifier of a timed-text cue.

## Remarks

The identifier is retrieved by this method is dynamically generated by the system and is guaranteed to uniquely identify a cue within a single timed-text track. It is not guaranteed to be unique across tracks. If a cue already has an identifier that is provided in the text-track data format, this ID can be retrieved by calling [GetOriginalId](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nf-mfmediaengine-imftimedtextcue-getoriginalid).
