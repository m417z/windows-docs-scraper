# IMFBufferListNotify::OnRemoveSourceBuffer

## Description

Indicates that a [IMFSourceBuffer](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfsourcebuffer) has been removed.
