# IMFMediaKeySession::get_SessionId

## Description

Gets a unique session id created for this session.

## Parameters

### `sessionId`

The media key session id.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
