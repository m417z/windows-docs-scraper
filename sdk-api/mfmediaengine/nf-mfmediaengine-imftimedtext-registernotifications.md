# IMFTimedText::RegisterNotifications

## Description

Registers a timed-text notify object.

## Parameters

### `notify` [in, optional]

Type: **[IMFTimedTextNotify](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imftimedtextnotify)***

A pointer to the [IMFTimedTextNotify](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imftimedtextnotify) interface for the timed-text notify object to register.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
