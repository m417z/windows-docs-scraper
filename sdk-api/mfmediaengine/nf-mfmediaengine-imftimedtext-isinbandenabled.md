# IMFTimedText::IsInBandEnabled

## Description

Determines whether inband mode is enabled.

## Return value

Type: **BOOL**

Returns whether inband mode is enabled. If **TRUE**, inband mode is enabled. If **FALSE**, inband mode is disabled.
