# IMFMediaEngineWebSupport::ConnectWebAudio

## Description

Connects web audio to Media Engine using the specified sample rate.

## Parameters

### `dwSampleRate`

The sample rate of the web audio.

### `ppSourceProvider` [out]

The sample rate of the web audio.

## Return value

Returns S_OK on successful completion.
