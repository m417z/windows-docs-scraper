# IMFMediaEngine::GetReadyState

## Description

Gets the ready state, which indicates whether the current media resource can be rendered.

## Return value

Returns an [MF_MEDIA_ENGINE_READY](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/ne-mfmediaengine-mf_media_engine_ready) enumeration value.

## Remarks

This method corresponds to the **readyState** attribute of the **HTMLMediaElement** interface in HTML5.
