# IMFMediaEngineEx::GetNumberOfStreams

## Description

Gets the number of streams in the media resource.

## Parameters

### `pdwStreamCount` [out]

Receives the number of streams.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
