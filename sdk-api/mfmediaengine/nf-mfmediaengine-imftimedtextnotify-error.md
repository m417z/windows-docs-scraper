# IMFTimedTextNotify::Error

## Description

Called when an error occurs in a text track.

## Parameters

### `errorCode` [in]

Type: **[MF_TIMED_TEXT_ERROR_CODE](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/ne-mfmediaengine-mf_timed_text_error_code)**

An MF_TIMED_TEXT_ERROR_CODE representing the last error.

### `extendedErrorCode` [in]

Type: **extendedErrorCode**

The extended error code for the last error.

### `sourceTrackId` [in]

Type: **extendedErrorCode**

The identifier of the track on which the error occurred.
