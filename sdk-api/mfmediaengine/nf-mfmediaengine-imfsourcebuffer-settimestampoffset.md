# IMFSourceBuffer::SetTimeStampOffset

## Description

Sets the timestamp offset for media segments appended to the [IMFSourceBuffer](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfsourcebuffer).

## Parameters

### `offset` [in]

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
