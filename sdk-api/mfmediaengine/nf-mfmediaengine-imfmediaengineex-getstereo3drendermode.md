# IMFMediaEngineEx::GetStereo3DRenderMode

## Description

For stereoscopic 3D video, queries how the Media Engine renders the 3D video content.

## Parameters

### `outputType` [out]

Receives a member of the [MF3DVideoOutputType](https://learn.microsoft.com/windows/desktop/api/mftransform/ne-mftransform-mf3dvideooutputtype) enumeration.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
