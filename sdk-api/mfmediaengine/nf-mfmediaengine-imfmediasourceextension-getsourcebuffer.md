# IMFMediaSourceExtension::GetSourceBuffer

## Description

Gets the [IMFSourceBuffer](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfsourcebuffer) at the specified index in the collection of buffers.

## Parameters

### `dwStreamIndex` [in]

The location of the buffer in the collection.

## Return value

The source buffer.
