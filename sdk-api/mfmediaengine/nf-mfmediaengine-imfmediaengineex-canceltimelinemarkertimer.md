# IMFMediaEngineEx::CancelTimelineMarkerTimer

## Description

Cancels the next pending timeline marker.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.

## Remarks

Call this method to cancel the [IMFMediaEngineEx::SetTimelineMarkerTimer](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/nf-mfmediaengine-imfmediaengineex-settimelinemarkertimer) method.
