# IMFMediaEngineEx::GetStereo3DFramePackingMode

## Description

For stereoscopic 3D video, gets the layout of the two views within a video frame.

## Parameters

### `packMode` [out]

Receives a member of the [MF_MEDIA_ENGINE_S3D_PACKING_MODE](https://learn.microsoft.com/windows/desktop/api/mfmediaengine/ne-mfmediaengine-mf_media_engine_s3d_packing_mode) enumeration.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
