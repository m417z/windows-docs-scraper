# SL_ACTIVATION_TYPE enumeration

## Description

Represents the type of offline activation for a license. This enumeration is used by the **type** member of the [SL_ACTIVATION_INFO_HEADER](https://learn.microsoft.com/windows/desktop/api/slpublic/ns-slpublic-sl_activation_info_header) structure.

## Constants

### `SL_ACTIVATION_TYPE_DEFAULT:0`

Retail phone activation.

### `SL_ACTIVATION_TYPE_ACTIVE_DIRECTORY:1`

The product activation is through Active Directory.
