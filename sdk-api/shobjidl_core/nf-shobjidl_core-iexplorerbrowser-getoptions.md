# IExplorerBrowser::GetOptions

## Description

Gets the current browser options.

## Parameters

### `pdwFlag` [out]

Type: **[EXPLORER_BROWSER_OPTIONS](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/ne-shobjidl_core-explorer_browser_options)***

When this method returns, contains the current [EXPLORER_BROWSER_OPTIONS](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/ne-shobjidl_core-explorer_browser_options) for the browser.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
