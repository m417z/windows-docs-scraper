# IOperationsProgressDialog::SetOperation

## Description

Sets which progress dialog operation is occurring, and whether we are in pre-flight or undo mode.

## Parameters

### `action` [in]

Type: **[SPACTION](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/ne-shobjidl_core-spaction)**

Specifies operation. See [SPACTION](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/ne-shobjidl_core-spaction).

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
