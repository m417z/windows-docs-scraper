# IBandSite::RemoveBand

## Description

Removes a band from the band site.

## Parameters

### `dwBandID` [in]

Type: **DWORD**

The ID of the band to remove.

## Return value

Type: **HRESULT**

Returns S_OK if successful, or a COM-defined error code otherwise.
