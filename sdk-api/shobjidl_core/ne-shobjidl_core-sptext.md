# SPTEXT enumeration

## Description

Specifies the type of descriptive text being provided to an [IActionProgress](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-iactionprogress) interface.

## Constants

### `SPTEXT_ACTIONDESCRIPTION:1`

The text is a high level, short description.

### `SPTEXT_ACTIONDETAIL`

The text is a detailed description.
