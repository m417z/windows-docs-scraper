# IOperationsProgressDialog::GetOperationStatus

## Description

Gets operation status for progress dialog.

## Parameters

### `popstatus` [out]

Type: **[PDOPSTATUS](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/ne-shobjidl_core-pdopstatus)***

Contains pointer to the operation status. See [PDOPSTATUS](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/ne-shobjidl_core-pdopstatus).

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
