# INamespaceWalkCB2::WalkComplete

## Description

Removes data collected during a namespace walk.

## Parameters

### `hr` [in]

Type: **HRESULT**

The results of [Walk](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nf-shobjidl_core-inamespacewalk-walk).

## Return value

Type: **HRESULT**

Always returns S_OK.
