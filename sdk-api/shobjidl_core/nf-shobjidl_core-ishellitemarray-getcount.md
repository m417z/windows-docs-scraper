# IShellItemArray::GetCount

## Description

Gets the number of items in the given [IShellItem](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ishellitem) array.

## Parameters

### `pdwNumItems` [out]

Type: **DWORD***

When this method returns, contains the number of items in the [IShellItemArray](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ishellitemarray).

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
