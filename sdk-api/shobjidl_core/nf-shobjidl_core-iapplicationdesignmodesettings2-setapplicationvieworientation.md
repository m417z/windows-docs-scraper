# IApplicationDesignModeSettings2::SetApplicationViewOrientation

## Description

Sets the window orientation used for the design mode window.

## Parameters

### `viewOrientation` [in]

Type: **[APPLICATION_VIEW_ORIENTATION](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/ne-shobjidl_core-application_view_orientation)**

The orientation of the design mode window to use. Either **AVO_LANDSCAPE** or **AVO_PORTRAIT**.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
