# IShellLinkA::SetPath

## Description

Sets the path and file name for the target of a Shell link object.

## Parameters

### `pszFile`

Type: **LPCTSTR**

The address of a buffer that contains the new path.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
