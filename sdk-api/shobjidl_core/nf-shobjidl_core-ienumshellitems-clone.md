# IEnumShellItems::Clone

## Description

Gets a copy of the current enumeration.

## Parameters

### `ppenum` [out]

Type: **[IEnumShellItems](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ienumshellitems)****

The address of a pointer that receives a copy of this enumeration.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
