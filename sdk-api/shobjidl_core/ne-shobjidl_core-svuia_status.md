# SVUIA_STATUS enumeration

## Description

Used with the [IBrowserService2::_UIActivateView](https://learn.microsoft.com/windows/win32/api/shdeprecated/nf-shdeprecated-ibrowserservice2-_uiactivateview) method to set the state of a browser view.

## Constants

### `SVUIA_DEACTIVATE:0`

The browser view has been deactivated.

### `SVUIA_ACTIVATE_NOFOCUS:1`

The browser view is activated and does not have focus.

### `SVUIA_ACTIVATE_FOCUS:2`

The browser view is activated and has focus.

### `SVUIA_INPLACEACTIVATE:3`

The browser view is activated in place.
