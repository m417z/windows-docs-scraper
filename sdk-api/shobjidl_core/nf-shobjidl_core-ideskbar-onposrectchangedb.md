# IDeskBar::OnPosRectChangeDB

## Description

Notifies the object that the rectangle has changed.

## Parameters

### `prc` [in]

Type: **LPRECT**

A pointer to a [RECT](https://learn.microsoft.com/windows/desktop/api/windef/ns-windef-rect) structure that specifies the child bar's desired size.

## Return value

Type: **HRESULT**

Returns S_OK if successful, or an error value otherwise.
