# ISearchFolderItemFactory::SetGroupColumn

## Description

Sets a group column, as specified. If no group column is specified, no grouping occurs.

## Parameters

### `keyGroup` [in]

Type: **REFPROPERTYKEY**

A reference to a group column [PROPERTYKEY](https://learn.microsoft.com/windows/desktop/api/wtypes/ns-wtypes-propertykey).

## Return value

Type: **HRESULT**

Returns a success value if successful, or an error value otherwise.
