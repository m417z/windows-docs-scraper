# IFolderViewSettings::GetColumnPropertyList

## Description

Gets an ordered list of columns that corresponds to the column enumerated.

## Parameters

### `riid`

A reference to the interface identifier (IID) of the IPropertyDescriptionList.

### `ppv`

Type: **[IPropertyDescriptionList](https://learn.microsoft.com/windows/desktop/api/propsys/nn-propsys-ipropertydescriptionlist)****

The address of an [IPropertyDescriptionList](https://learn.microsoft.com/windows/desktop/api/propsys/nn-propsys-ipropertydescriptionlist) interface pointer.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
