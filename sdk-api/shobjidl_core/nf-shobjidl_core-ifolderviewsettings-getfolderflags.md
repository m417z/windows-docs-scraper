# IFolderViewSettings::GetFolderFlags

## Description

Gets folder view options flags.

## Parameters

### `pfolderMask` [out]

Type: **[FOLDERFLAGS](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/ne-shobjidl_core-folderflags)***

A pointer to a mask for folder view options.

### `pfolderFlags` [out]

Type: **[FOLDERFLAGS](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/ne-shobjidl_core-folderflags)***

A pointer to a flag for folder view options.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
