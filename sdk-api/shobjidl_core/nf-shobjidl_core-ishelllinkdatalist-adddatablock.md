# IShellLinkDataList::AddDataBlock

## Description

Adds a data block to a link.

## Parameters

### `pDataBlock` [in]

Type: **VOID***

The data block structure. For a list of supported structures, see [IShellLinkDataList](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ishelllinkdatalist).

## Return value

Type: **HRESULT**

Returns S_OK if successful, or a COM error code otherwise.
