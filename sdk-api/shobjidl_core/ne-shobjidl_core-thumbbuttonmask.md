# THUMBBUTTONMASK enumeration

## Description

Used by the [THUMBBUTTON](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/ns-shobjidl_core-thumbbutton) structure to specify which members of that structure contain valid data.

## Constants

### `THB_BITMAP:0x1`

The **iBitmap** member contains valid information.

### `THB_ICON:0x2`

The **hIcon** member contains valid information.

### `THB_TOOLTIP:0x4`

The **szTip** member contains valid information.

### `THB_FLAGS:0x8`

The **dwFlags** member contains valid information.
