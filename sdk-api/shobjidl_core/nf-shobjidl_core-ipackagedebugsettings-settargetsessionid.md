# IPackageDebugSettings::SetTargetSessionId

## Description

Sets the session identifier.

## Parameters

### `sessionId` [in]

The session identifier.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
