# IPackageDebugSettings::ActivateBackgroundTask

## Description

Activates the specified background task.

## Parameters

### `taskId` [in]

The identifier of the background task to activate.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.

## Remarks

Use the **ActivateBackgroundTask** method to test the code that handles your background tasks.
