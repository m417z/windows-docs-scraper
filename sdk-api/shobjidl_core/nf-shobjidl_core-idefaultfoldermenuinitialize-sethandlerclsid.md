# IDefaultFolderMenuInitialize::SetHandlerClsid

## Description

Sets the shortcut menu handler for the [IDefaultFolderMenuInitialize](https://learn.microsoft.com/windows/win32/api/shobjidl_core/nn-shobjidl_core-idefaultfoldermenuinitialize) object.

## Parameters

### `rclsid` [in]

Type: **REFCLSID**

The CLSID for the object defines the shortcut menu handler.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
