# IShellItemResources::EnumResources

## Description

Gets a resource enumerator object.

## Parameters

### `ppenumr` [out]

Type: **[IEnumResources](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ienumresources)****

The address of an [IEnumResources](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ienumresources) interface pointer.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
