# LIBRARYFOLDERFILTER enumeration

## Description

Defines options for filtering folder items.

## Constants

### `LFF_FORCEFILESYSTEM:1`

Return only file system items.

### `LFF_STORAGEITEMS:2`

Return items that can be bound to an IStorage object.

### `LFF_ALLITEMS:3`

Return all items.
