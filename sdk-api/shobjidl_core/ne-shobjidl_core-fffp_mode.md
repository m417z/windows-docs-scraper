# FFFP_MODE enumeration

## Description

Describes match criteria. Used by methods of the [IKnownFolderManager](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-iknownfoldermanager) interface.

## Constants

### `FFFP_EXACTMATCH:0`

Exact match.

### `FFFP_NEARESTPARENTMATCH`

Nearest parent match.
