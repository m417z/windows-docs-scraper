# LIBRARYMANAGEDIALOGOPTIONS enumeration

## Description

Used by [SHShowManageLibraryUI](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nf-shobjidl_core-shshowmanagelibraryui) to define options for handling a name collision when saving a library.

## Constants

### `LMD_DEFAULT:0`

Show default warning UI to the user.

### `LMD_ALLOWUNINDEXABLENETWORKLOCATIONS:0x1`

Do not display a warning dialog to the user in collisions that concern network locations that cannot be indexed.
