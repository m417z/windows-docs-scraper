# IEnumFullIDList::Clone

## Description

Creates a new item enumeration object with the same contents and state as the current one.

## Parameters

### `ppenum` [out]

Type: **[IEnumFullIDList](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ienumfullidlist)****

On success, contains the address of an [IEnumFullIDList](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ienumfullidlist) interface pointer.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
