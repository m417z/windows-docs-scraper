# ATTACHMENT_ACTION enumeration

## Description

Provides a set of flags to be used with [IAttachmentExecute::Prompt](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nf-shobjidl_core-iattachmentexecute-prompt) to indicate the action to be performed upon user confirmation.

## Constants

### `ATTACHMENT_ACTION_CANCEL:0`

Cancel

### `ATTACHMENT_ACTION_SAVE:0x1`

Save

### `ATTACHMENT_ACTION_EXEC:0x2`

Execute
