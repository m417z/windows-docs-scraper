# IPersistIDList::GetIDList

## Description

Gets an item identifier list.

## Parameters

### `ppidl` [out]

Type: **LPITEMIDLIST***

The address of a pointer to the item identifier list to get.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
