# IBandSite::AddBand

## Description

Adds a band to a band site object.

## Parameters

### `punk` [in]

Type: **IUnknown***

The interface of a band site object.

## Return value

Type: **HRESULT**

Returns the band ID in ShortFromResult(hresult).
