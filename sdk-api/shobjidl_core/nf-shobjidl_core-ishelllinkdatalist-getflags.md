# IShellLinkDataList::GetFlags

## Description

Gets the current option settings.

## Parameters

### `pdwFlags` [out]

Type: **DWORD***

Pointer to one or more of the [SHELL_LINK_DATA_FLAGS](https://learn.microsoft.com/windows/desktop/api/shlobj_core/ne-shlobj_core-shell_link_data_flags) that indicate the current option settings.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
