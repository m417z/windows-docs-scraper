# PREVIEWHANDLERFRAMEINFO structure

## Description

Accelerator table structure. Used by [IPreviewHandlerFrame::GetWindowContext](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nf-shobjidl_core-ipreviewhandlerframe-getwindowcontext).

## Members

### `haccel`

Type: **HACCEL**

A handle to the accelerator table.

### `cAccelEntries`

Type: **UINT**

The number of entries in the accelerator table.
