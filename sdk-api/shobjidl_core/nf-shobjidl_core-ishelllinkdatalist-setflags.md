# IShellLinkDataList::SetFlags

## Description

Sets the current option settings.

## Parameters

### `dwFlags` [in]

Type: **DWORD**

One or more of the [SHELL_LINK_DATA_FLAGS](https://learn.microsoft.com/windows/desktop/api/shlobj_core/ne-shlobj_core-shell_link_data_flags) that indicate the option settings.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
