# IExecuteCommand::SetShowWindow

## Description

Sets the specified window's visual state.

## Parameters

### `nShow` [in]

Type: **int**

It can be any of the values that can be specified in the *nCmdShow* parameter for the [ShowWindow](https://learn.microsoft.com/windows/desktop/api/winuser/nf-winuser-showwindow) function.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
