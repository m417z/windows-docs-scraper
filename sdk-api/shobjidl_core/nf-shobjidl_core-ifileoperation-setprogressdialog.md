# IFileOperation::SetProgressDialog

## Description

Specifies a dialog box used to display the progress of the operation.

## Parameters

### `popd` [in]

Type: **[IOperationsProgressDialog](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ioperationsprogressdialog)***

Pointer to an [IOperationsProgressDialog](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ioperationsprogressdialog) object that represents the dialog box.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
