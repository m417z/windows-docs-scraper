# BLUETOOTH_FIND_RADIO_PARAMS structure

## Description

The **BLUETOOTH_FIND_RADIO_PARAMS** structure facilitates enumerating installed Bluetooth radios.

## Members

### `dwSize`

Size of the **BLUETOOTH_FIND_RADIO_PARAMS** structure, in bytes.
