# BLUETOOTH_NUMERIC_COMPARISON_INFO structure

## Description

The **BLUETOOTH_NUMERIC_COMPARISON_INFO** structure contains the numeric value used for authentication via numeric comparison.

## Members

### `NumericValue`

The numeric value.
