# ICoreWindowInterop::get_WindowHandle

## Description

Obtains the handle (HWND) to the [CoreWindow](https://msdn.microsoft.com/60b1c8c6-c136-4c4c-8e46-69a792d58ed0) for an app.

This property is read-only.
