# ICoreWindowInterop::put_MessageHandled

## Description

Sets whether or not the message to the [CoreWindow](https://msdn.microsoft.com/60b1c8c6-c136-4c4c-8e46-69a792d58ed0) has been handled.

This property is write-only.
