# ID3D11FunctionReflection::GetFunctionParameter

## Description

Gets the function parameter reflector.

## Parameters

### `ParameterIndex` [in]

Type: **INT**

The zero-based index of the function parameter reflector to retrieve.

## Return value

Type: **[ID3D11FunctionParameterReflection](https://learn.microsoft.com/windows/desktop/api/d3d11shader/nn-d3d11shader-id3d11functionparameterreflection)***

A pointer to a [ID3D11FunctionParameterReflection](https://learn.microsoft.com/windows/desktop/api/d3d11shader/nn-d3d11shader-id3d11functionparameterreflection) interface that represents the function parameter reflector.
