# MCAST_CLIENT_UID structure

## Description

The
**MCAST_CLIENT_UID** structure describes the unique client identifier for each multicast request.

## Members

### `ClientUID`

Buffer containing the unique client identifier.

### `ClientUIDLength`

Size of the **ClientUID** member, in bytes.
