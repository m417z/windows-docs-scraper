# XMSHORT2::XMSHORT2

## Description

Default constructor for `XMSHORT2`.

Default constructor for [XMSHORT2](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmshort2).

**Note** This constructor is only available under C++.

## Remarks

`XMSHORT2()` constructs a new instance of the `XMSHORT2` structure without
defining the value of any of its members.
