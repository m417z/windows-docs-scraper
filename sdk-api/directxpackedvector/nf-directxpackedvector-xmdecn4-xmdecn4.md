# XMDECN4::XMDECN4

## Description

Default constructor for `XMDECN4`.

Default constructor for [XMDECN4](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmdecn4).

**Note** This constructor is only available under C++.

## Remarks

`XMDECN4()` constructs a new instance of the `XMDECN4` structure without
defining the value of any of its members.
