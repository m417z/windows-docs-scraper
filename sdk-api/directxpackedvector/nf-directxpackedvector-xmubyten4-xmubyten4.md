# XMUBYTEN4::XMUBYTEN4

Default constructor for `XMUBYTEN4`

**Note** This constructor is only available under C++.

## Remarks

`XMUBYTEN4()` constructs a new instance of the `XMUBYTEN4` structure without
defining the value of any of its members.
