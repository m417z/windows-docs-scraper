# XMBYTEN4::XMBYTEN4

## Description

Default constructor for `XMBYTEN4`

Default constructor for [XMBYTEN4](https://msdn.microsoft.com/62d61a35-8674-4855-b09c-f351363cd50b)

**Note** This constructor is only available under C++.

## Remarks

`XMBYTEN4()` constructs a new instance of the `XMBYTEN4` structure without
defining the value of any of its members.
