# XMUSHORT4::XMUSHORT4

## Description

Default constructor for `XMUSHORT4`.

Default constructor for [XMUSHORT4](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmushort4).

**Note** This constructor is only available under C++.

## Remarks

`XMUSHORT4()` constructs a new instance of the `XMUSHORT4` structure without
defining the value of any of its members.
