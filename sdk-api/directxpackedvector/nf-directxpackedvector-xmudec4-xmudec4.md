# XMUDEC4::XMUDEC4

## Description

Default constructor for `XMUDEC4`.

Default constructor for [XMUDEC4](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmudec4) .

**Note** This constructor is only available under C++.

## Remarks

`XMUDEC4()` constructs a new instance of the `XMUDEC4` structure without
defining the value of any of its members.
