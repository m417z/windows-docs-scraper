# XMBYTE4::XMBYTE4

## Description

Default constructor for `XMBYTE4`.

Default constructor for [XMBYTE4](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmbyte4).

**Note** This constructor is only available under C++.

## Remarks

`XMBYTE4()` constructs a new instance of the `XMBYTE4` structure without
defining the value of any of its members.
