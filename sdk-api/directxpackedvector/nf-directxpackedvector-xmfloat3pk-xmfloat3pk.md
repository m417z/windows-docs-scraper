# XMFLOAT3PK::XMFLOAT3PK

## Description

Default constructor for `XMFLOAT3PK`.

Default constructor for [XMFLOAT3PK](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmfloat3pk).

**Note** This constructor is only available under C++.

## Remarks

`XMFLOAT3PK()` constructs a new instance of the `XMFLOAT3PK` structure without
defining the value of any of its members.
