# XMXDEC4::XMXDEC4

## Description

Default constructor for `XMXDEC4`.

Default constructor for [XMXDEC4](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmxdec4) .

**Note** This constructor is only available under C++.

## Remarks

`XMXDEC4()` constructs a new instance of the `XMXDEC4` structure without
defining the value of any of its members.
