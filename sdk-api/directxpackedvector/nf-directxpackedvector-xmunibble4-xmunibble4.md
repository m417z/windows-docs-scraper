# XMUNIBBLE4::XMUNIBBLE4

## Description

Default constructor for `XMUNIBBLE4`

Default constructor for [XMUNIBBLE4](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmunibble4)

**Note** This is only available for C++ based development.

## Remarks

`XMUNIBBLE4()` constructs a new instance of the `XMUNIBBLE4` structure without
defining the value of any of its members.
