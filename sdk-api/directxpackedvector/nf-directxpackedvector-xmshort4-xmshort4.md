# XMSHORT4::XMSHORT4

## Description

Default constructor for `XMSHORT4`.

Default constructor for [XMSHORT4](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmshort4).

**Note** This constructor is only available under C++.

## Remarks

`XMSHORT4()` constructs a new instance of the `XMSHORT4` structure without
defining the value of any of its members.
