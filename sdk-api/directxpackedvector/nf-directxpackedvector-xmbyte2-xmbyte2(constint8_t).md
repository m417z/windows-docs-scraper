# XMBYTE2::XMBYTE2(const int8_t)

## Description

Initializes a new instance of [XMBYTE2](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmbyte2) from a two-element `int8_t` array argument.

This constructor initializes a new instance of [XMBYTE2](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmbyte2) from a two-element `int8_t` array argument.

**Note** This constructor is only available with C++.

## Parameters

### `pArray`

Two-element `int8_t` array containing the values used to initialize the two components of a new instance of **XMBYTE2**.

## Remarks

The following pseudocode demonstrates the operation of this constructor:

```cpp
XMBYTE2 instance;
instance.x = pArray[0];
instance.y = pArray[1];
```
