# XMHALF2::XMHALF2

## Description

Default constructor for `XMHALF2`.

Default constructor for [XMHALF2](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmhalf2).

**Note** This constructor is only available under C++.

## Remarks

`XMHALF2()` constructs a new instance of the `XMHALF2` structure without
defining the value of any of its members.
