# XMU565::XMU565

## Description

Default constructor for `XMU565`

Default constructor for [XMU565](https://learn.microsoft.com/windows/desktop/api/directxpackedvector/ns-directxpackedvector-xmu565)

**Note** This is only available for C++ based development.

## Remarks

`XMU565()` constructs a new instance of the `XMU565` structure without
defining the value of any of its members.
