# MFCreateASFMultiplexer function

## Description

Creates the ASF Multiplexer.

## Parameters

### `ppIMultiplexer`

Receives a pointer to the [IMFASFMultiplexer](https://learn.microsoft.com/windows/desktop/api/wmcontainer/nn-wmcontainer-imfasfmultiplexer) interface. The caller must release the interface.

## Return value

The function returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
| --- | --- |
| **S_OK** | The function succeeded. |
