# ASF_STATUSFLAGS enumeration

## Description

Defines status conditions for the [IMFASFSplitter::GetNextSample](https://learn.microsoft.com/windows/desktop/api/wmcontainer/nf-wmcontainer-imfasfsplitter-getnextsample) method.

## Constants

### `ASF_STATUSFLAGS_INCOMPLETE:0x1`

The operation is incomplete.

### `ASF_STATUSFLAGS_NONFATAL_ERROR:0x2`

 One or more non-critical errors
occurred while parsing the ASF data.
