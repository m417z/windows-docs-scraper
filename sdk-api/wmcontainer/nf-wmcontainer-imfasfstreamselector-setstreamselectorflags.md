# IMFASFStreamSelector::SetStreamSelectorFlags

## Description

Sets options for the stream selector.

## Parameters

### `dwStreamSelectorFlags` [in]

Bitwise [MFASF_STREAMSELECTOR_FLAGS](https://learn.microsoft.com/windows/win32/api/wmcontainer/ne-wmcontainer-mfasf_streamselector_flags) enumeration specifying the options to use.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
