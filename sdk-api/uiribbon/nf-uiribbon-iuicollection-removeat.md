# IUICollection::RemoveAt

## Description

Removes an item from the [IUICollection](https://learn.microsoft.com/windows/desktop/api/uiribbon/nn-uiribbon-iuicollection) at the specified index.

## Parameters

### `index` [in]

Type: **UINT32**

The zero-based index of the item to remove from the [IUICollection](https://learn.microsoft.com/windows/desktop/api/uiribbon/nn-uiribbon-iuicollection).

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
