# UI_FONTPROPERTIES enumeration

## Description

Specifies values that identify the font property state of a [FontControl](https://learn.microsoft.com/windows/desktop/windowsribbon/windowsribbon-element-fontcontrol), such as **Strikethrough**.

## Constants

### `UI_FONTPROPERTIES_NOTAVAILABLE:0`

The property is not available.

### `UI_FONTPROPERTIES_NOTSET:1`

The property is not set.

### `UI_FONTPROPERTIES_SET:2`

The property is set.
