# IUICollection::Clear

## Description

Deletes all items from the [IUICollection](https://learn.microsoft.com/windows/desktop/api/uiribbon/nn-uiribbon-iuicollection).

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
