# IUICollection::GetCount

## Description

Retrieves the number of items contained in the [IUICollection](https://learn.microsoft.com/windows/desktop/api/uiribbon/nn-uiribbon-iuicollection).

## Parameters

### `count` [out]

Type: **UINT32***

When this method returns, contains the item count.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
