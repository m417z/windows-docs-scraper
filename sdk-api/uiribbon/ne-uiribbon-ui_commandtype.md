# UI_COMMANDTYPE enumeration

## Description

Specifies values that identify the type of Command associated with a Ribbon control.

## Constants

### `UI_COMMANDTYPE_UNKNOWN:0`

The type of command is not known.

### `UI_COMMANDTYPE_GROUP:1`

[Group](https://learn.microsoft.com/windows/desktop/windowsribbon/windowsribbon-controls-group)

### `UI_COMMANDTYPE_ACTION:2`

[Help Button](https://learn.microsoft.com/windows/desktop/windowsribbon/windowsribbon-controls-helpbutton)

### `UI_COMMANDTYPE_ANCHOR:3`

[Split Button](https://learn.microsoft.com/windows/desktop/windowsribbon/windowsribbon-controls-splitbutton)

[Tab](https://learn.microsoft.com/windows/desktop/windowsribbon/windowsribbon-controls-tab)

[Tab Group](https://learn.microsoft.com/windows/desktop/windowsribbon/windowsribbon-controls-tabgroup)
