# UI_EVENTLOCATION enumeration

## Description

Identifies the locations where events associated with a Ribbon control can originate.

## Constants

### `UI_EVENTLOCATION_Ribbon:0`

The [Ribbon](https://learn.microsoft.com/windows/desktop/windowsribbon/windowsribbon-element-ribbon).

### `UI_EVENTLOCATION_QAT:1`

The [QuickAccessToolbar](https://learn.microsoft.com/windows/desktop/windowsribbon/windowsribbon-element-quickaccesstoolbar).

### `UI_EVENTLOCATION_ApplicationMenu:2`

The [ApplicationMenu](https://learn.microsoft.com/windows/desktop/windowsribbon/windowsribbon-element-applicationmenu).

### `UI_EVENTLOCATION_ContextPopup:3`

The [ContextPopup](https://learn.microsoft.com/windows/desktop/windowsribbon/windowsribbon-element-contextpopup).
