# UI_VIEWVERB enumeration

## Description

Specifies values that identify the type of action to complete on a Ribbon framework View.

## Constants

### `UI_VIEWVERB_CREATE:0`

Create a View.

### `UI_VIEWVERB_DESTROY:1`

Destroy a View.

### `UI_VIEWVERB_SIZE:2`

Resize a View.

### `UI_VIEWVERB_ERROR:3`

Unable to complete the action.
