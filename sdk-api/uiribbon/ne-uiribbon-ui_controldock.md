# UI_CONTROLDOCK enumeration

## Description

Specifies values that identify the dock state of the Quick Access Toolbar (QAT).

## Constants

### `UI_CONTROLDOCK_TOP:1`

The QAT is docked in the nonclient area of the Ribbon host application, as shown in the following screen shot.

![Screen shot showing the Quick Access Toolbar docked above the Ribbon in the nonclient area.](https://learn.microsoft.com/windows/win32/api/uiribbon/images/QAT_DockTop.png)

### `UI_CONTROLDOCK_BOTTOM:3`

The QAT is docked as a visually integral band below the Ribbon, as shown in the following screen shot.

![Screen shot showing the Quick Access Toolbar docked below the Ribbon.](https://learn.microsoft.com/windows/win32/api/uiribbon/images/QAT_DockBottom.png)

## Remarks

The QAT dock position is based on the **UI_CONTROLDOCK** value in [UI_PKEY_QuickAccessToolbarDock](https://learn.microsoft.com/windows/desktop/windowsribbon/windowsribbon-reference-properties-uipkey-quickaccesstoolbardock).
