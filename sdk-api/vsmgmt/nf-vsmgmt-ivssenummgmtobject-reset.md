# IVssEnumMgmtObject::Reset

## Description

The **Reset** method resets the enumerator
so that [IVssEnumMgmtObject](https://learn.microsoft.com/windows/desktop/api/vsmgmt/nn-vsmgmt-ivssenummgmtobject) starts at the first
enumerated object.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
