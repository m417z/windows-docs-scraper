# IBufferByteAccess::Buffer

## Description

Gets the array of bytes in the buffer.

## Parameters

### `value` [out]

The byte array.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
