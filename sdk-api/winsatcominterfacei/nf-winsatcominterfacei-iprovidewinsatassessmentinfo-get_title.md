# IProvideWinSATAssessmentInfo::get_Title

## Description

[IProvideWinSATAssessmentInfo::Title may be altered or unavailable for releases after Windows 8.1.]

Retrieves the title of the subcomponent.

This property is read-only.

## Parameters

## Remarks

Note that the title includes a trailing colon character. For example, the title for the CPU subcomponent is "Processor:".

#### Examples

For an example, see the [IProvideWinSATResultsInfo::GetAssessmentInfo](https://learn.microsoft.com/windows/desktop/api/winsatcominterfacei/nf-winsatcominterfacei-iprovidewinsatresultsinfo-getassessmentinfo) method.
