# ITPluggableTerminalClassRegistration::put_TerminalClass

## Description

The
**put_TerminalClass** method sets the terminal's terminal class.

## Parameters

### `bstrTerminalClass` [in]

The **BSTR** representation of the
[terminal class](https://learn.microsoft.com/windows/desktop/Tapi/terminal-class).

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
