# ITTerminalControl::get_AddressHandle

## Description

The
**get_AddressHandle** method gets the MSP address handle.

## Parameters

### `phtAddress` [out]

Pointer to MSP address handle.

## Return value

This method can return one of these values.

| Value | Meaning |
| --- | --- |
| **S_OK** | Method succeeded. |
| **E_POINTER** | The *phtAddress* parameter is not a valid pointer. |
