# ITTerminalControl::StopRenderFilter

## Description

The
**StopRenderFilter** method stops the rightmost render filter in the terminal. Needed for dynamic filter graphs.

## Return value

This method can return one of these values.

| Value | Meaning |
| --- | --- |
| **S_OK** | Method succeeded. |
| **E_NOTIMPL** | Method not implemented. |
