# ITPluggableTerminalClassRegistration::Delete

## Description

The
**Delete** method deletes the terminal class from the registry.

## Parameters

### `bstrSuperclass` [in]

The **BSTR** representation of the CLSID for the terminal superclass.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
