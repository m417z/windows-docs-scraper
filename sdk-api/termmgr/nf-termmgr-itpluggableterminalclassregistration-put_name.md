# ITPluggableTerminalClassRegistration::put_Name

## Description

The
**put_Name** method sets the name of the terminal class being registered.

## Parameters

### `bstrName` [in]

The **BSTR** representation of the name.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
