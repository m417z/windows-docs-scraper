# ITPluggableTerminalSuperclassRegistration::GetTerminalSuperclassInfo

## Description

The
**GetTerminalSuperclassInfo** method gets class information from the registry for the current terminal superclass.

## Return value

This method can return one of these values.

| Return code | Description |
| --- | --- |
| **S_OK** | Method succeeded. |
| **E_FAIL** | Method failed. |
