# ITPluggableTerminalClassRegistration::GetTerminalClassInfo

## Description

The
**GetTerminalClassInfo** method gets all the information from the registry for a specific terminal.

## Parameters

### `bstrSuperclass` [in]

The **BSTR** representation of the CLSID for the terminal superclass.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
