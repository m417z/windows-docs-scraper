# ITPluggableTerminalClassRegistration::put_Version

## Description

The
**put_Version** method sets the terminal version.

## Parameters

### `bstrVersion` [in]

The **BSTR** representation of the terminal version.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
