# ITPluggableTerminalClassRegistration::put_Direction

## Description

The
**put_Direction** method sets the direction supported by the terminal.

## Parameters

### `nDirection` [in]

The
[TMGR_DIRECTION](https://learn.microsoft.com/windows/win32/api/termmgr/ne-termmgr-tmgr_direction) descriptor of the terminal direction.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
