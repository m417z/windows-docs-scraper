# SpatialAudioHrtfDirectivityCone structure

## Description

Represents a cone-shaped directivity model for an [ISpatialAudioObjectForHrtf](https://learn.microsoft.com/windows/desktop/api/spatialaudiohrtf/nn-spatialaudiohrtf-ispatialaudioobjectforhrtf).

## Members

### `directivity`

A structure that expresses the direction in which sound is emitted by an [ISpatialAudioObjectForHrtf](https://learn.microsoft.com/windows/desktop/api/spatialaudiohrtf/nn-spatialaudiohrtf-ispatialaudioobjectforhrtf).

### `InnerAngle`

The inner angle of the cone.

### `OuterAngle`

The outer angle of the cone.
