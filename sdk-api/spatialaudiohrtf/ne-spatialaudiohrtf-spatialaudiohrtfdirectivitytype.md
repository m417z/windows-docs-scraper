# SpatialAudioHrtfDirectivityType enumeration

## Description

Specifies the shape in which sound is emitted by an [ISpatialAudioObjectForHrtf](https://learn.microsoft.com/windows/desktop/api/spatialaudiohrtf/nn-spatialaudiohrtf-ispatialaudioobjectforhrtf).

## Constants

### `SpatialAudioHrtfDirectivity_OmniDirectional:0`

The sound is emitted in all directions.

### `SpatialAudioHrtfDirectivity_Cardioid`

The sound is emitted in a cardioid shape.

### `SpatialAudioHrtfDirectivity_Cone`

The sound is emitted in a cone shape.
