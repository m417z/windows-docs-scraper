# SpatialAudioHrtfDirectivityUnion structure

## Description

Defines a spatial audio directivity model for an [ISpatialAudioObjectForHrtf](https://learn.microsoft.com/windows/desktop/api/spatialaudiohrtf/nn-spatialaudiohrtf-ispatialaudioobjectforhrtf).

## Members

### `Cone`

 A cone-shaped directivity model

### `Cardioid`

### `Omni`

And omni-direction directivity model that can be interpolated linearly with one of the other directivity models.

#### - Cardioid

 A cardioid-shaped directivity model
