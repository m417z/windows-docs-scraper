# SpatialAudioHrtfDirectivityCardioid structure

## Description

Represents a cardioid-shaped directivity model for an [ISpatialAudioObjectForHrtf](https://learn.microsoft.com/windows/desktop/api/spatialaudiohrtf/nn-spatialaudiohrtf-ispatialaudioobjectforhrtf).

## Members

### `directivity`

A structure that expresses the direction in which sound is emitted by an [ISpatialAudioObjectForHrtf](https://learn.microsoft.com/windows/desktop/api/spatialaudiohrtf/nn-spatialaudiohrtf-ispatialaudioobjectforhrtf).

### `Order`

The order of the cardioid.
