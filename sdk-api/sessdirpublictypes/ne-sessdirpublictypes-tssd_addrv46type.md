# TSSD_AddrV46Type enumeration

## Description

Indicates the IP address type.

## Constants

### `TSSD_ADDR_UNDEFINED:0`

The IP address is not valid.

### `TSSD_ADDR_IPv4:4`

The address is in IPv4 format.

### `TSSD_ADDR_IPv6:6`

The address is in IPv6 format.
