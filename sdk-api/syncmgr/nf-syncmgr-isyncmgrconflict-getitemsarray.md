# ISyncMgrConflict::GetItemsArray

## Description

Retrieves a conflict items array.

## Parameters

### `ppArray` [out]

Type: **[ISyncMgrConflictItems](https://learn.microsoft.com/windows/desktop/api/syncmgr/nn-syncmgr-isyncmgrconflictitems)****

The address of a pointer to an [ISyncMgrConflictItems](https://learn.microsoft.com/windows/desktop/api/syncmgr/nn-syncmgr-isyncmgrconflictitems) array.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
