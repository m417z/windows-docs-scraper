# IEnumSyncMgrSyncItems::Clone

## Description

Not used. Clones an [IEnumSyncMgrSyncItems](https://learn.microsoft.com/windows/desktop/api/syncmgr/nn-syncmgr-ienumsyncmgrsyncitems) object.

## Parameters

### `ppenum` [out]

Type: **[IEnumSyncMgrSyncItems](https://learn.microsoft.com/windows/desktop/api/syncmgr/nn-syncmgr-ienumsyncmgrsyncitems)****

The address of the cloned [IEnumSyncMgrSyncItems](https://learn.microsoft.com/windows/desktop/api/syncmgr/nn-syncmgr-ienumsyncmgrsyncitems) interface pointer.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
