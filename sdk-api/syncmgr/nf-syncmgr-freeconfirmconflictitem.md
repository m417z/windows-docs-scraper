# FreeConfirmConflictItem function

## Description

Frees the resources that have been allocated for a [CONFIRM_CONFLICT_ITEM](https://learn.microsoft.com/windows/desktop/api/syncmgr/ns-syncmgr-confirm_conflict_item) structure.

## Parameters

### `pcci` [in, out]

Type: **[CONFIRM_CONFLICT_ITEM](https://learn.microsoft.com/windows/desktop/api/syncmgr/ns-syncmgr-confirm_conflict_item)***

A pointer to a [CONFIRM_CONFLICT_ITEM](https://learn.microsoft.com/windows/desktop/api/syncmgr/ns-syncmgr-confirm_conflict_item) structure that stores pointers to the items for which memory is to be freed.
