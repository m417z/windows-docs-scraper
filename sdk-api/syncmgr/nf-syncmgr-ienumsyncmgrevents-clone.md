# IEnumSyncMgrEvents::Clone

## Description

Clones an [IEnumSyncMgrEvents](https://learn.microsoft.com/windows/desktop/api/syncmgr/nn-syncmgr-ienumsyncmgrevents) object.

## Parameters

### `ppenum` [out]

Type: **[IEnumSyncMgrEvents](https://learn.microsoft.com/windows/desktop/api/syncmgr/nn-syncmgr-ienumsyncmgrevents)****

The address of the cloned [IEnumSyncMgrEvents](https://learn.microsoft.com/windows/desktop/api/syncmgr/nn-syncmgr-ienumsyncmgrevents) interface pointer.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
