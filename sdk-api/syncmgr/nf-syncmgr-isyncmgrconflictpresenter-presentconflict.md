# ISyncMgrConflictPresenter::PresentConflict

## Description

Presents the conflict to the user.

## Parameters

### `pConflict` [in]

Type: **[ISyncMgrConflict](https://learn.microsoft.com/windows/desktop/api/syncmgr/nn-syncmgr-isyncmgrconflict)***

Specifies the conflict. See [ISyncMgrConflict](https://learn.microsoft.com/windows/desktop/api/syncmgr/nn-syncmgr-isyncmgrconflict).

### `pResolveInfo` [in]

Type: **[ISyncMgrConflictResolveInfo](https://learn.microsoft.com/windows/desktop/api/syncmgr/nn-syncmgr-isyncmgrconflictresolveinfo)***

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
