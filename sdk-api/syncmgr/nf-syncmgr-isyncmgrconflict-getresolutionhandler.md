# ISyncMgrConflict::GetResolutionHandler

## Description

Gets the resolution handler for the conflict.

## Parameters

### `riid` [in]

Type: **REFIID**

A reference to the ID for the resolution handler.

### `ppvResolutionHandler` [out]

Type: **void****

When this method returns, contains the interface pointer requested in *riid*.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
