# IEnumSyncMgrSyncItems::Skip

## Description

Skips forward in the enumeration the specified number of items.

## Parameters

### `celt` [in]

Type: **ULONG**

The number of items to skip.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
