# WSAVERSION structure

## Description

The
**WSAVERSION** structure provides version comparison in Windows Sockets.

## Members

### `dwVersion`

Version of Windows Sockets.

### `ecHow`

[WSAECOMPARATOR](https://learn.microsoft.com/windows/desktop/api/winsock2/ne-winsock2-wsaecomparator) enumeration, used in the comparison.
