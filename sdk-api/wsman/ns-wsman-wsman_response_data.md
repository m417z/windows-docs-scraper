# WSMAN_RESPONSE_DATA structure

## Description

Represents the output data received from a [WSMan](https://learn.microsoft.com/windows/desktop/WinRM/wsman) operation.

## Members

### `receiveData`

Represents the output data received from a [WSManReceiveShellOutput](https://learn.microsoft.com/windows/desktop/api/wsman/nf-wsman-wsmanreceiveshelloutput) method.

### `connectData`

Represents the output data received from a [WSManConnectShell](https://learn.microsoft.com/windows/desktop/api/wsman/nf-wsman-wsmanconnectshell) or [WSManConnectShellCommand](https://learn.microsoft.com/windows/desktop/api/wsman/nf-wsman-wsmanconnectshellcommand) method.
