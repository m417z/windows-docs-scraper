# WSManDataType enumeration

## Description

Specifies the current data type of the union in the [WSMAN_DATA](https://learn.microsoft.com/windows/desktop/api/wsman/ns-wsman-wsman_data) structure.

## Constants

### `WSMAN_DATA_NONE:0`

The structure is not valid yet.

### `WSMAN_DATA_TYPE_TEXT:1`

The structure contains text.

### `WSMAN_DATA_TYPE_BINARY:2`

The structure contains binary data.

### `WSMAN_DATA_TYPE_DWORD:4`

The structure contains a DWORD integer.
