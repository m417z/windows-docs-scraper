# WSMAN_ENVIRONMENT_VARIABLE_SET structure

## Description

Defines an array of environment variables.

## Members

### `varsCount`

Specifies the number of environment variables contained within the **vars** array.

### `vars`

Defines an array of environment variables. Each element of the array is of type [WSMAN_ENVIRONMENT_VARIABLE](https://learn.microsoft.com/windows/desktop/api/wsman/ns-wsman-wsman_environment_variable).
