# WSMAN_SELECTOR_SET structure

## Description

Defines a set of keys that represent the identity of a resource.

## Members

### `numberKeys`

Specifies the number of keys (selectors).

### `keys`

An array of [WSMAN_KEY](https://learn.microsoft.com/windows/desktop/api/wsman/ns-wsman-wsman_key) structures that specify key names and values.
