# WSMAN_DATA_BINARY structure

## Description

A [WSMAN_DATA](https://learn.microsoft.com/windows/desktop/api/wsman/ns-wsman-wsman_data) structure component that holds binary data for use with various Windows Remote Management functions.

## Members

### `dataLength`

Represents the number of BYTEs stored in the data field.

### `data`

Specifies the storage location for the binary data.
