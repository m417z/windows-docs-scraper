# WSMAN_DATA_TEXT structure

## Description

A [WSMAN_DATA](https://learn.microsoft.com/windows/desktop/api/wsman/ns-wsman-wsman_data) structure component that holds textual data for use with various Windows Remote Management functions.

## Members

### `bufferLength`

Specifies the number of UNICODE characters stored in the buffer.

### `buffer`

Specifies the storage location for the textual data.
