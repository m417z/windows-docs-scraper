# IWRdsProtocolConnection::SetErrorInfo

## Description

Sets error information in the protocol.

## Parameters

### `ulError` [in]

The error code.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
