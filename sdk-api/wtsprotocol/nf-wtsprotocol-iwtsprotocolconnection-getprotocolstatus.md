# IWTSProtocolConnection::GetProtocolStatus

## Description

[**IWTSProtocolConnection::GetProtocolStatus** is no longer available for use as of Windows Server 2012. Instead, use [IWRdsProtocolConnection::GetProtocolStatus](https://learn.microsoft.com/windows/desktop/api/wtsprotocol/nf-wtsprotocol-iwrdsprotocolconnection-getprotocolstatus).]

Retrieves information about the protocol status.

## Parameters

### `pProtocolStatus` [out]

A pointer to a [WTS_PROTOCOL_STATUS](https://learn.microsoft.com/windows/desktop/api/wtsdefs/ns-wtsdefs-wts_protocol_status) structure that contains counter, signal, and cache information for the protocol.
