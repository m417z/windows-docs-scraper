# IWTSProtocolConnection::GetClientData

## Description

[**IWTSProtocolConnection::GetClientData**
is no longer available for use as of Windows Server 2012. Instead, use
[IWRdsProtocolConnection::GetClientData](https://learn.microsoft.com/windows/desktop/api/wtsprotocol/nf-wtsprotocol-iwrdsprotocolconnection-getclientdata).]

Requests client settings from the protocol.

## Parameters

### `pClientData` [out]

A pointer to a [WTS_CLIENT_DATA](https://learn.microsoft.com/windows/desktop/api/wtsdefs/ns-wtsdefs-wts_client_data) structure that contains the client settings.
