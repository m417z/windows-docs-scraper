# IWTSProtocolConnection::GetLastInputTime

## Description

[**IWTSProtocolConnection::GetLastInputTime** is no longer available for use as of Windows Server 2012. Instead, use [IWRdsProtocolConnection::GetLastInputTime](https://learn.microsoft.com/windows/desktop/api/wtsprotocol/nf-wtsprotocol-iwrdsprotocolconnection-getlastinputtime).]

Returns the last time the protocol received input data.

## Parameters

### `pLastInputTime` [out]

An integer that contains the elapsed time, in milliseconds.
