# IWTSProtocolConnection::SetErrorInfo

## Description

[**IWTSProtocolConnection::SetErrorInfo** is no longer available for use as of Windows Server 2012. Instead, use [IWRdsProtocolConnection::SetErrorInfo](https://learn.microsoft.com/windows/desktop/api/wtsprotocol/nf-wtsprotocol-iwrdsprotocolconnection-seterrorinfo).]

Sends an error code to the client.

## Parameters

### `ulError` [in]

An integer that contains the error code.
