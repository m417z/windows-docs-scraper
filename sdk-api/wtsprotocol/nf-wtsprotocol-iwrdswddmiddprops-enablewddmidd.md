# IWRdsWddmIddProps::EnableWddmIdd

## Description

Termsrv uses this method to tell protocol stack which mode it is
operating.

## Parameters

### `Enabled` [in]

Boolean flag that instructs protocol stack that termsrv supports WDDM IDD mode.

## Return value

S_OK or error code.
