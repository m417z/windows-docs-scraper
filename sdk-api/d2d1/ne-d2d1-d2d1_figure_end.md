# D2D1_FIGURE_END enumeration

## Description

Indicates whether a specific [ID2D1SimplifiedGeometrySink](https://learn.microsoft.com/windows/win32/api/d2d1/nn-d2d1-id2d1simplifiedgeometrysink) figure is open or closed.

## Constants

### `D2D1_FIGURE_END_OPEN:0`

The figure is open.

### `D2D1_FIGURE_END_CLOSED:1`

The figure is closed.

### `D2D1_FIGURE_END_FORCE_DWORD:0xffffffff`
