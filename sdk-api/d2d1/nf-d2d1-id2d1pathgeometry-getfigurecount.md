# ID2D1PathGeometry::GetFigureCount

## Description

Retrieves the number of figures in the path geometry.

## Parameters

### `count` [out]

Type: **UINT32***

A pointer that receives the number of figures in the path geometry when this method returns. You must allocate storage for this parameter.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an [**HRESULT**](https://learn.microsoft.com/windows/desktop/com/structure-of-com-error-codes) error code.
