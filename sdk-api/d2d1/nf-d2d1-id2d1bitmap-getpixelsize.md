# ID2D1Bitmap::GetPixelSize

## Description

Returns the size, in device-dependent units (pixels), of the bitmap.

## Return value

Type: **[D2D1_SIZE_U](https://learn.microsoft.com/windows/win32/Direct2D/d2d1-size-u)**

The size, in pixels, of the bitmap.
