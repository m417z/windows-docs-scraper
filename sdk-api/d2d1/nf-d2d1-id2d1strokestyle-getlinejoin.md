# ID2D1StrokeStyle::GetLineJoin

## Description

Retrieves the type of joint used at the vertices of a shape's outline.

## Return value

Type: **[D2D1_LINE_JOIN](https://learn.microsoft.com/windows/win32/api/d2d1/ne-d2d1-d2d1_line_join)**

A value that specifies the type of joint used at the vertices of a shape's outline.
