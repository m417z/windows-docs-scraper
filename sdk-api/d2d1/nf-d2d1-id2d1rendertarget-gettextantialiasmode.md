# ID2D1RenderTarget::GetTextAntialiasMode

## Description

Gets the current antialiasing mode for text and glyph drawing operations.

## Return value

Type: **[D2D1_TEXT_ANTIALIAS_MODE](https://learn.microsoft.com/windows/win32/api/d2d1/ne-d2d1-d2d1_text_antialias_mode)**

The current antialiasing mode for text and glyph drawing operations.
