# ID2D1RadialGradientBrush::GetCenter

## Description

Retrieves the center of the gradient ellipse.

## Return value

Type: **[D2D1_POINT_2F](https://learn.microsoft.com/windows/win32/Direct2D/d2d1-point-2f)**

The center of the gradient ellipse. This value is expressed in the brush's coordinate space.
