# ID2D1Layer::GetSize

## Description

Gets the size of the layer in device-independent pixels.

## Return value

Type: **[D2D1_SIZE_F](https://learn.microsoft.com/windows/win32/Direct2D/d2d1-size-f)**

The size of the layer in device-independent pixels.
