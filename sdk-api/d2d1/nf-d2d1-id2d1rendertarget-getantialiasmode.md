# ID2D1RenderTarget::GetAntialiasMode

## Description

Retrieves the current antialiasing mode for nontext drawing operations.

## Return value

Type: **[D2D1_ANTIALIAS_MODE](https://learn.microsoft.com/windows/win32/api/d2d1/ne-d2d1-d2d1_antialias_mode)**

The current antialiasing mode for nontext drawing operations.
