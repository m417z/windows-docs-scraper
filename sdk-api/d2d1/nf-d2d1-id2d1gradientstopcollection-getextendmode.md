# ID2D1GradientStopCollection::GetExtendMode

## Description

Indicates the behavior of the gradient outside the normalized gradient range.

## Return value

Type: **[D2D1_EXTEND_MODE](https://learn.microsoft.com/windows/win32/api/d2d1/ne-d2d1-d2d1_extend_mode)**

The behavior of the gradient outside the [0,1] normalized gradient range.
