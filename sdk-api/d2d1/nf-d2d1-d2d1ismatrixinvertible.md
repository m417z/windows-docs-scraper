# D2D1IsMatrixInvertible function

## Description

Indicates whether the specified matrix is invertible.

## Parameters

### `matrix` [in]

Type: **const [D2D1_MATRIX_3X2_F](https://learn.microsoft.com/windows/win32/Direct2D/d2d1-matrix-3x2-f)***

The matrix to test.

## Return value

Type: **BOOL**

**true** if the matrix was inverted; otherwise, **false**.
