## Description

Adds a single arc to the path geometry.

## Parameters

### `arc`

Type: [in] **const [D2D1_ARC_SEGMENT](https://learn.microsoft.com/windows/win32/api/d2d1/ns-d2d1-d2d1_arc_segment) &**

The arc segment to add to the figure.
