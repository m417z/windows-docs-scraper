# ID2D1SolidColorBrush::GetColor

## Description

Retrieves the color of the solid color brush.

## Return value

Type: **[D2D1_COLOR_F](https://learn.microsoft.com/windows/win32/Direct2D/d2d1-color-f)**

The color of this solid color brush.
