# Rect::Rect

## Description

Creates a [Rect](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) object whose x-coordinate, y-coordinate, width, and height are all zero. This is the default constructor.
