# RectF::RectF

## Description

Creates a [RectF](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) object and initializes the
**X**,
**Y**,
**Width**, and
**Height** data members to zero. This is the default constructor.
