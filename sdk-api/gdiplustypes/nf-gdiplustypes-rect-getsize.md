# Rect::GetSize

## Description

The **Rect::GetSize** method gets the width and height of the rectangle.

## Parameters

### `size` [out]

Type: **Size***

Pointer to a [Size](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) object that receives the width and height of the rectangle.
