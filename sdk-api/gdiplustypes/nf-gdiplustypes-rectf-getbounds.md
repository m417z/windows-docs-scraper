# RectF::GetBounds

## Description

The **RectF::GetBounds** method makes a copy of this rectangle.

## Parameters

### `rect` [out]

Type: **RectF***

Pointer to a
[RectF](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) object that receives the copy of this rectangle.
