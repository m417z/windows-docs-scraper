# RectF::GetLocation

## Description

The **RectF::GetLocation** method gets the coordinates of the upper-left corner of this rectangle.

## Parameters

### `point` [out]

Type: **PointF***

Pointer to a [PointF](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) object that receives the coordinates of the upper-left corner of this rectangle.
