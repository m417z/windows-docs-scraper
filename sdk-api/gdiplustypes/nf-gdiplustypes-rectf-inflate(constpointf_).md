# RectF::Inflate(IN const PointF &)

## Description

The **RectF::Inflate** method expands the rectangle by the value of
*point*.**X** on the left and right edges, and by the value of
*point*.**Y** on the top and bottom edges.

## Parameters

### `point` [in]

Type: **const PointF&**

Reference to a [PointF](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) object whose
**X** data member specifies the amount to expand the rectangle on the left and right edges, and whose
**Y** data member specifies the amount to expand the rectangle on the top and bottom edges.
