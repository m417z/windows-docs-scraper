# Rect::GetLocation

## Description

The **Rect::GetLocation** method gets the coordinates of the upper-left corner of the rectangle.

## Parameters

### `point` [out]

Type: **Point***

Pointer to a [Point](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) object that receives the coordinates of the upper-left corner.
