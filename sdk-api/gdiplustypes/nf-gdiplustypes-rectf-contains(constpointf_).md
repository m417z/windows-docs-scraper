# RectF::Contains(IN const PointF &)

## Description

The **RectF::Contains** method determines whether a point is inside this rectangle.

## Parameters

### `pt` [in]

Type: **const PointF&**

Reference to a point to be tested.

## Return value

Type: **BOOL**

If the point is inside the rectangle, this method returns **TRUE**; otherwise, it returns **FALSE**.
