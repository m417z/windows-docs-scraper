# Rect::GetBounds

## Description

The **Rect::GetBounds** method makes a copy of this rectangle.

## Parameters

### `rect` [out]

Type: **Rect***

Pointer to a
[Rect](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) object that receives the copy of this rectangle.
