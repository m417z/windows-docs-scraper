# PointF::PointF(IN REAL,IN REAL)

## Description

Creates a [PointF](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) object using two real numbers to specify the
**X** and
**Y** data members.

## Parameters

### `x` [in]

Type: **REAL**

Real number that specifies the x-coordinate of the [PointF](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) object.

### `y` [in]

Type: **REAL**

Real number that specifies the y-coordinate of the [PointF](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) object.
