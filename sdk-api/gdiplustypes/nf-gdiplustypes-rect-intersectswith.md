# Rect::IntersectsWith

## Description

The **Rect::IntersectsWith** method determines whether this rectangle intersects another rectangle.

## Parameters

### `rect` [in]

Type: **const Rect&**

Reference to a rectangle to test.

## Return value

Type: **BOOL**

If the intersection of the two rectangles is not empty, this method returns **TRUE**; otherwise, it returns **FALSE**.
