# Size::Size(IN INT,IN INT)

## Description

Creates a [Size](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) object and initializes its
**Width** and
**Height** data members.

## Parameters

### `width` [in]

Type: **INT**

Integer that specifies the initial value of the
**Width** data member.

### `height` [in]

Type: **INT**

Integer that specifies the initial value of the
**Height** data member.
