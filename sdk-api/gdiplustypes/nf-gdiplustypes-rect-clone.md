# Rect::Clone

## Description

The **Rect::Clone** method creates a new
[Rect](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) object and initializes it with the contents of this **Rect** object.

## Return value

Type: **[Rect](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect)***

This method returns a pointer to the new
[Rect](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) object.
