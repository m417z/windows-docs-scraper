# eAvrfResourceTypes enumeration

## Description

Specifies the types of resources that can be enumerated using the [VerifierEnumerateResource](https://learn.microsoft.com/windows/desktop/api/avrfsdk/nf-avrfsdk-verifierenumerateresource) function.

## Constants

### `AvrfResourceHeapAllocation`

Indicates heap-allocation information is being obtained.

### `AvrfResourceHandleTrace`

Indicates handle trace information is being obtained.

### `AvrfResourceMax`

Indicates the upper boundary of the current implementation's resource type.
