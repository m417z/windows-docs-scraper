# IWMDRMTranscryptionManager::CreateTranscryptor

## Description

Creates a DRM transcryptor object.

## Parameters

### `ppTranscryptor` [out]

Address of a pointer to the [IWMDRMTranscryptor](https://learn.microsoft.com/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmdrmtranscryptor) interface of the newly created DRM transcryptor object.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
