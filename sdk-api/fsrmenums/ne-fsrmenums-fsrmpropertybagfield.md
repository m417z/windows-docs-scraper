# FsrmPropertyBagField enumeration

## Description

Describes the type of property bag.

## Constants

### `FsrmPropertyBagField_AccessVolume:0`

Indicates if the property bag should include the name of the volume being accessed, which may be a
snapshot.

### `FsrmPropertyBagField_VolumeGuidName:1`

Indicates if the property bag should include the volume **GUID** name of the
original volume.
