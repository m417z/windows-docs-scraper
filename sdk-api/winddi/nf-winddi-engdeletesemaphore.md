# EngDeleteSemaphore function

## Description

The **EngDeleteSemaphore** function deletes a semaphore object from the system's resource list.

## Parameters

### `hsem` [in]

Handle to the semaphore to be deleted. The semaphore was created in [EngCreateSemaphore](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-engcreatesemaphore).

## Return value

None
