# GAMMARAMP structure

## Description

The GAMMARAMP structure is used by [DrvIcmSetDeviceGammaRamp](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-drvicmsetdevicegammaramp) to set the hardware [gamma ramp](https://learn.microsoft.com/windows-hardware/drivers/) of a particular display device.

## Members

### `Red`

Is the 256-entry ramp for the red color channel.

### `Green`

Is the 256-entry ramp for the green color channel.

### `Blue`

Is the 256-entry ramp for the blue color channel.
