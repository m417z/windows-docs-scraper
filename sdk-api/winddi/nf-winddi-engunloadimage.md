# EngUnloadImage macro

## Description

The **EngUnloadImage** function unloads an image loaded by [EngLoadImage](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-engloadimage).

## Parameters

### `h` [in]

Handle to the image to be unloaded from system memory.
