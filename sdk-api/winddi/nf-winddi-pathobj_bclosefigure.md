# PATHOBJ_bCloseFigure function

## Description

The **PATHOBJ_bCloseFigure** function closes an open figure in a path by drawing a line from the current position to the first point of the figure.

## Parameters

### `ppo`

Pointer to a [PATHOBJ](https://learn.microsoft.com/windows/desktop/api/winddi/ns-winddi-pathobj) structure that identifies the path to be closed.

## Return value

The return value is **TRUE** if the function is successful.
