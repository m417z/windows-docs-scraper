# PALOBJ structure

## Description

[The PALOBJ structure has no public members.]

The **PALOBJ**
structure is a user object that represents an indexed color palette.

## Members

## Remarks

This structure is used in conjunction with the [PALOBJ_cGetColors](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-palobj_cgetcolors) service routine.
