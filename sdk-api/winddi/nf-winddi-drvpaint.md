# DrvPaint function

## Description

The **DrvPaint** function is obsolete, and is no longer called by GDI in Windows 2000 and later. New drivers should implement one or more of [DrvFillPath](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-drvfillpath), [DrvStrokePath](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-drvstrokepath), or [DrvStrokeAndFillPath](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-drvstrokeandfillpath).
