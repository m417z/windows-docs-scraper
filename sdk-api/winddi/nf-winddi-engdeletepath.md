# EngDeletePath function

## Description

The **EngDeletePath** function deletes a path previously allocated by [EngCreatePath](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-engcreatepath).

## Parameters

### `ppo`

Pointer to the [PATHOBJ](https://learn.microsoft.com/windows/desktop/api/winddi/ns-winddi-pathobj) structure to be deleted.

## Return value

None
