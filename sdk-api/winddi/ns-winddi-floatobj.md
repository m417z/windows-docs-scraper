# FLOATOBJ structure

## Description

The FLOATOBJ structure is used to emulate a floating-point number.

## Members

### `ul1`

Reserved for system use.

### `ul2`

Reserved for system use.

## Remarks

This structure, in conjunction with the **FLOATOBJ_***Xxx* service routines, allows graphics drivers to emulate floating-point arithmetic in the NT kernel. Floating-point arithmetic is not otherwise supported in the NT kernel code.
