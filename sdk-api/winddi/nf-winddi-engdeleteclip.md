# EngDeleteClip function

## Description

The **EngDeleteClip** function deletes a [CLIPOBJ](https://learn.microsoft.com/windows/desktop/api/winddi/ns-winddi-clipobj) structure allocated by [EngCreateClip](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-engcreateclip).

## Parameters

### `pco`

Pointer to the CLIPOBJ structure to delete.

## Return value

None
