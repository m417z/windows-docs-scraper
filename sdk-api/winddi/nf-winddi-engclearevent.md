# EngClearEvent function

## Description

The **EngClearEvent** function sets a specified event object to the nonsignaled state.

## Parameters

### `pEvent` [in]

Pointer to the event object returned by a previous call to either [EngCreateEvent](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-engcreateevent) or [EngMapEvent](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-engmapevent).

## Return value

None
