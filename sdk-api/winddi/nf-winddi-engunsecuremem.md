# EngUnsecureMem function

## Description

The **EngUnsecureMem** function unlocks an address range that is locked down in memory.

## Parameters

### `hSecure` [in]

Handle to the address range that is locked down. This handle is returned by [EngSecureMem](https://learn.microsoft.com/windows/desktop/api/winddi/nf-winddi-engsecuremem).

## Return value

None
