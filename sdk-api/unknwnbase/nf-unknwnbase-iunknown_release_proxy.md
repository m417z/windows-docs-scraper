# IUnknown_Release_Proxy function

## Description

[IUnknown_Release_Proxy is not supported and may be altered or unavailable in the future.]

The [IUnknown_Release_Proxy](https://learn.microsoft.com/windows/desktop/api/unknwnbase/nf-unknwnbase-iunknown_queryinterface_proxy) function implements the [Release](https://learn.microsoft.com/windows/desktop/api/unknwn/nf-unknwn-iunknown-release) method for all interface proxies.

## Parameters

### `This`

Pointer to the proxy object.

## Return value

Returns an integer from 1 to *n*, indicating the value of the new reference count.

## Remarks

 The return value is for diagnostic and testing purposes only; the return value may be unstable.

This function is used internally by proxies that are generated by MIDL.
