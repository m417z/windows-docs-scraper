# SBinaryArray structure

## Description

Do not use. An array of entry identifiers representing MAPI objects. Uses the same implementation as [SBinaryArray](https://learn.microsoft.com/previous-versions/office/developer/office-2007/cc815398(v=office.12)).

## Members

### `cValues`

Type: **ULONG**

Variable of type **ULONG** that specifies the number of entry identifiers.

### `lpbin`

Type: **[SBinary](https://learn.microsoft.com/previous-versions/office/developer/office-2007/cc815817(v=office.12))***

Array of variables of type [SBinary](https://learn.microsoft.com/previous-versions/office/developer/office-2007/cc815817(v=office.12)) that specify the entry identifiers.
