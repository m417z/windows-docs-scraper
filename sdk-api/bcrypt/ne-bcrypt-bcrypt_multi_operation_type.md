# BCRYPT_MULTI_OPERATION_TYPE enumeration

## Description

The **BCRYPT_MULTI_OPERATION_TYPE** enumeration specifies type of multi-operation that is passed to the [BCryptProcessMultiOperations](https://learn.microsoft.com/windows/desktop/api/bcrypt/nf-bcrypt-bcryptprocessmultioperations) function.

## Constants

### `BCRYPT_OPERATION_TYPE_HASH:1`

A hash operation.
