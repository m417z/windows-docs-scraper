# BCRYPT_PROVIDER_NAME structure

## Description

The **BCRYPT_PROVIDER_NAME** structure contains the name of a CNG provider.

## Members

### `pszProviderName`

A pointer to a null-terminated Unicode string that contains the name of the provider.
