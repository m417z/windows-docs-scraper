# CXAPOBase::GetRegistrationPropertiesInternal

## Description

Returns a pointer to the [XAPO_REGISTRATION_PROPERTIES](https://learn.microsoft.com/windows/desktop/api/xapo/ns-xapo-xapo_registration_properties) structure containing the registration properties the XAPO was created with.

## Return value

Pointer to the [XAPO_REGISTRATION_PROPERTIES](https://learn.microsoft.com/windows/desktop/api/xapo/ns-xapo-xapo_registration_properties) structure containing the registration properties the XAPO was created with.

CXAPOBase

## Remarks

### Platform Requirements

Windows 10 (XAudio2.9); Windows 8, Windows Phone 8 (XAudio 2.8); DirectX SDK (XAudio 2.7)
