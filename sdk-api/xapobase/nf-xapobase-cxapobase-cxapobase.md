# CXAPOBase::CXAPOBase

## Description

Creates an instance of the [CXAPOBase](https://learn.microsoft.com/windows/desktop/api/xapobase/nl-xapobase-cxapobase) class.

## Parameters

### `pRegistrationProperties`

Pointer to an [XAPO_REGISTRATION_PROPERTIES](https://learn.microsoft.com/windows/desktop/api/xapo/ns-xapo-xapo_registration_properties) structure containing the registration properties for the XAPO.

## Remarks

The object created by this [CXAPOBase](https://learn.microsoft.com/windows/desktop/api/xapobase/nl-xapobase-cxapobase) will have a reference count of 1.

### Platform Requirements

Windows 10 (XAudio2.9); Windows 8, Windows Phone 8 (XAudio 2.8); DirectX SDK (XAudio 2.7)
