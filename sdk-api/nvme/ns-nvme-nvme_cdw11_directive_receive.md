# NVME_CDW11_DIRECTIVE_RECEIVE structure

## Description

Contains parameters for the Directive Receive command.

This structure is used as the value of the **CDW11** parameter in the **DIRECTIVERECEIVE** field of the [Command](https://learn.microsoft.com/windows/win32/api/nvme/ns-nvme-nvme_command) structure.

## Members

### `DUMMYSTRUCTNAME`

### `DUMMYSTRUCTNAME.DOPER`

The directive operation.

### `DUMMYSTRUCTNAME.DTYPE`

The directive type.

### `DUMMYSTRUCTNAME.DSPEC`

A directive specific value.

### `AsUlong`
