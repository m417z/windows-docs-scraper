# NVME_TEMPERATURE_THRESHOLD_TYPES enumeration

## Description

Contains values that indicate the type of threshold for the temperature of the overall device (controller and NVM included).

## Constants

### `NVME_TEMPERATURE_OVER_THRESHOLD`

Over Temperature Threshold

### `NVME_TEMPERATURE_UNDER_THRESHOLD`

Under Temperature Threshold

## Remarks

Values from this enumeration are used in the **THSEL** field of the [NVME_CDW11_FEATURE_TEMPERATURE_THRESHOLD](https://learn.microsoft.com/windows/win32/api/nvme/ns-nvme-nvme_cdw11_feature_temperature_threshold) structure.
