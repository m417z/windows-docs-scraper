# NVME_CDW10_DIRECTIVE_SEND structure

## Description

Contains parameters for the Directive Send command.

This structure is used as the value of the **CDW10** parameter in the **DIRECTIVESEND** field of the [Command](https://learn.microsoft.com/windows/win32/api/nvme/ns-nvme-nvme_command) structure.

## Members

### `NUMD`

Indicates the number of Dwords.
