# IDWriteAsyncResult::GetWaitHandle

## Description

Returns a handle that can be used to wait for the asynchronous operation to complete. The handle remains valid until the interface is released.

## Return value

Type: **HANDLE**

Returns a handle that can be used to wait for the asynchronous operation to complete. The handle remains valid until the interface is released.
