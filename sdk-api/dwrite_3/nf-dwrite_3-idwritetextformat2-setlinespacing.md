# IDWriteTextFormat2::SetLineSpacing

## Description

Set line spacing.

## Parameters

### `lineSpacingOptions` [in]

Type: **const [DWRITE_LINE_SPACING](https://learn.microsoft.com/windows/win32/api/dwrite_3/ns-dwrite_3-dwrite_line_spacing)***

How to manage space between lines.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
