# IDWriteFontDownloadQueue::CancelDownload

## Description

 Removes all download requests from the queue and cancels any active download
operations.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
