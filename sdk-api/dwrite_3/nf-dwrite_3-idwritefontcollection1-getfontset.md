# IDWriteFontCollection1::GetFontSet

## Description

Gets the underlying font set used by this collection.

## Parameters

### `fontSet` [out]

Type: **[IDWriteFontSet](https://learn.microsoft.com/windows/win32/api/dwrite_3/nn-dwrite_3-idwritefontset)****

Returns the font set used by the collection.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
