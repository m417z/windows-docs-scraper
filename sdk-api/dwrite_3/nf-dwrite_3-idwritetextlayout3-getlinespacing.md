# IDWriteTextLayout3::GetLineSpacing

## Description

Gets line spacing information.

## Parameters

### `lineSpacingOptions` [out]

How to manage space between lines.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
