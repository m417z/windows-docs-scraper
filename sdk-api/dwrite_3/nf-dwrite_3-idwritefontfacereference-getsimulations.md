# IDWriteFontFaceReference::GetSimulations

## Description

Obtains the algorithmic style simulation flags of a font face.

## Return value

Type: **[DWRITE_FONT_SIMULATIONS](https://learn.microsoft.com/windows/win32/api/dwrite/ne-dwrite-dwrite_font_simulations)**

Returns the algorithmic style simulation flags of a font face.
