# ATTRIBUTEINFO enumeration

## Description

The values of the
**ATTRIBUTEINFO** type enumerate characteristics of a specified attribute.

## Constants

### `NAME:1`

The name of the attribute.

### `SYNTAX`

The
[syntax](https://learn.microsoft.com/windows/desktop/api/sdoias/ne-sdoias-attributesyntax) of the attribute.

### `RESTRICTIONS`

[Restrictions](https://learn.microsoft.com/windows/desktop/api/sdoias/ne-sdoias-attributerestrictions) on how the attribute can be used.

### `DESCRIPTION`

Description of the attribute.

### `VENDORID`

The vendor ID for Vendor Specific Attributes (VSA).

### `LDAPNAME`

The Lightweight Directory Access Protocol (LDAP) name for the attribute.

### `VENDORTYPE`

The attribute type for Vendor Specific Attributes (VSA).
