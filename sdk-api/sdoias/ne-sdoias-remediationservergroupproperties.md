# REMEDIATIONSERVERGROUPPROPERTIES enumeration

## Description

The values of the
**REMEDIATIONSERVERGROUPPROPERTIES** enumeration type enumerate the properties of a remediation server group.

## Constants

### `PROPERTY_REMEDIATIONSERVERGROUP_SERVERS_COLLECTION`

The collection of servers in the remediation server group.
