# RADIUSPROXYPROPERTIES enumeration

## Description

The values in the
**RADIUSPROXYPROPERTIES** enumeration type enumerate properties related to the RADIUS proxy service.

## Constants

### `PROPERTY_RADIUSPROXY_SERVERGROUPS`

The collection of RADIUS proxy server groups.
