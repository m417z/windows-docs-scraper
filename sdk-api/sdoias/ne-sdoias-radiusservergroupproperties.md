# RADIUSSERVERGROUPPROPERTIES enumeration

## Description

The values in the
**RADIUSSERVERGROUPPROPERTIES** enumeration type enumerate properties of a RADIUS server group.

## Constants

### `PROPERTY_RADIUSSERVERGROUP_SERVERS_COLLECTION`

The collection of servers in the RADIUS server group.

## Remarks

The
[IASPROPERTIES](https://learn.microsoft.com/windows/desktop/api/sdoias/ne-sdoias-iasproperties) enumeration type contains the value PROPERTY_IAS_RADIUSSERVERGROUPS_COLLECTION, which corresponds to the collection of RADIUS server groups.
