# REMEDIATIONSERVERPROPERTIES enumeration

## Description

The values of the
**REMEDIATIONSERVERPROPERTIES** enumeration type enumerate the properties of a remediation server.

## Constants

### `PROPERTY_REMEDIATIONSERVER_ADDRESS`

IP address value for a Remediation Server. Used in Network Policy Server (NPS) user interface.

### `PROPERTY_REMEDIATIONSERVER_FRIENDLY_NAME`

Host-name for a Remediation Server. Used in NPS user interface.
