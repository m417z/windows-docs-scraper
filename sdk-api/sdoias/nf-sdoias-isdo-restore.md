# ISdo::Restore

## Description

The
**Restore** method reloads the values of the Server Data Objects (SDO) properties from persistent storage.

## Return value

If the method succeeds the return value is **S_OK**.

If the method fails, the return value is one of the following error codes.

## Remarks

Use the
**Restore** method to cancel changes made by calls to the
[ISdo::PutProperty](https://learn.microsoft.com/windows/desktop/api/sdoias/nf-sdoias-isdo-putproperty) method.
