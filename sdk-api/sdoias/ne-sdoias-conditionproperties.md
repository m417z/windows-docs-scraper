# CONDITIONPROPERTIES enumeration

## Description

The values of the
**CONDITIONPROPERTIES** enumeration type specify the properties of a Network Access Policy (NAP) condition.

## Constants

### `PROPERTY_CONDITION_TEXT`

The text of the NAP condition.

## Remarks

The
[POLICYPROPERTIES](https://learn.microsoft.com/windows/desktop/api/sdoias/ne-sdoias-policyproperties) enumeration type contains the value PROPERTY_POLICY_CONDITIONS_COLLECTION, which is the property for the conditions collection.
