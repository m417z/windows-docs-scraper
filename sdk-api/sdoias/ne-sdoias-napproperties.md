# NAPPROPERTIES enumeration

## Description

The values of the
**NAPPROPERTIES** enumeration type specify properties for Network Access Policies (NAP).

## Constants

### `PROPERTY_NAP_POLICIES_COLLECTION`

The network access policies collection.

### `PROPERTY_SHV_TEMPLATES_COLLECTION`

Collection of System Health Validator (SHV) templates. See [NAP documentation](https://learn.microsoft.com/windows/desktop/NAP/network-access-protection-start-page) for more information on SHV.
