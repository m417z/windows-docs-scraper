# IRemoteDesktopClientActions::ExecuteRemoteAction

## Description

Causes an action to be performed in the remote session.

## Parameters

### `remoteAction` [in]

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
