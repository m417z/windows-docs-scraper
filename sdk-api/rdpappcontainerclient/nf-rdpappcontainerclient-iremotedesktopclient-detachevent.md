# IRemoteDesktopClient::detachEvent

## Description

Detaches an event handler from an event.

## Parameters

### `eventName` [in]

### `callback` [in]

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
