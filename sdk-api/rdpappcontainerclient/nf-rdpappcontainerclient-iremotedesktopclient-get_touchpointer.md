Contains the [RemoteDesktopClientTouchPointer](https://learn.microsoft.com/windows/desktop/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) object for the Remote Desktop Protocol (RDP) app container client.

This property is read-only.
