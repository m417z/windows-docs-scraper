# IRemoteDesktopClientActions::ResumeScreenUpdates

## Description

Resumes screen updates being sent to the client.

Use the [SuspendScreenUpdates](https://learn.microsoft.com/windows/desktop/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-suspendscreenupdates) method to suspend screen updates.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
