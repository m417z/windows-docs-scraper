# IRemoteDesktopClientTouchPointer::put_EventsEnabled

## Description

Whether touch pointer event notifications are enabled for the RDP app container client control. If this property is enabled, the [OnTouchPointerCursorMoved](https://learn.microsoft.com/windows/desktop/TermServ/iremotedesktopclientevents-ontouchpointercursormoved) method will handle events when the touch pointer cursor is moved.

This property is read/write.
