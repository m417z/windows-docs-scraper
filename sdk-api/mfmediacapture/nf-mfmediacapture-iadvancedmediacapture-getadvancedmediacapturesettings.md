# IAdvancedMediaCapture::GetAdvancedMediaCaptureSettings

## Description

Gets the advanced media capture settings.

## Parameters

### `value` [out]

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
