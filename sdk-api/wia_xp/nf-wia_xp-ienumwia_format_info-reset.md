# IEnumWIA_FORMAT_INFO::Reset

## Description

The **IEnumWIA_FORMAT_INFO::Reset** method sets the enumeration back to the first [WIA_FORMAT_INFO](https://learn.microsoft.com/windows/desktop/api/wia_xp/ns-wia_xp-wia_format_info) structure.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
