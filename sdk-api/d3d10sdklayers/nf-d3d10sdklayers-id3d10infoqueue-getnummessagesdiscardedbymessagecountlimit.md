# ID3D10InfoQueue::GetNumMessagesDiscardedByMessageCountLimit

## Description

Get the number of messages that were discarded due to the message count limit.

## Return value

Type: **[UINT64](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

Number of messages discarded.

## Remarks

Get and set the message count limit with [ID3D10InfoQueue::GetMessageCountLimit](https://learn.microsoft.com/windows/desktop/api/d3d10sdklayers/nf-d3d10sdklayers-id3d10infoqueue-getmessagecountlimit) and [ID3D10InfoQueue::SetMessageCountLimit](https://learn.microsoft.com/windows/desktop/api/d3d10sdklayers/nf-d3d10sdklayers-id3d10infoqueue-setmessagecountlimit), respectively.
