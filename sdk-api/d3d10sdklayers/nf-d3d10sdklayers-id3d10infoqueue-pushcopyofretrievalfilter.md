# ID3D10InfoQueue::PushCopyOfRetrievalFilter

## Description

Push a copy of retrieval filter currently on the top of the retrieval-filter stack onto the retrieval-filter stack.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

This method returns one of the following [Direct3D 10 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues).
