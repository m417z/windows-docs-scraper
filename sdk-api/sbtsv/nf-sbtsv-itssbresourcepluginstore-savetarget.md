# ITsSbResourcePluginStore::SaveTarget

## Description

Saves a target.

## Parameters

### `pTarget` [in]

Pointer to the [ITsSbTarget](https://learn.microsoft.com/windows/desktop/api/sbtsv/nn-sbtsv-itssbtarget) object to save.

### `bForceWrite` [in]

Set to TRUE to force writing the saved object.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
