# ITsSbProvisioning::CancelJob

## Description

Cancels a provisioning job.

## Parameters

### `JobGuid` [in]

A **GUID** that identifies the job.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
