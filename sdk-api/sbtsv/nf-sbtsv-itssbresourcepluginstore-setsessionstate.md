# ITsSbResourcePluginStore::SetSessionState

## Description

Sets the session state.

## Parameters

### `sbSession` [in]

A pointer to the session to modify.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
