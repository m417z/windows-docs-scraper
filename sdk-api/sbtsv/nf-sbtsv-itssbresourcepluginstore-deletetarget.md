# ITsSbResourcePluginStore::DeleteTarget

## Description

 Deletes a target.

## Parameters

### `targetName` [in]

The name of the target.

### `hostName` [in]

The name of the computer that hosts the target.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
