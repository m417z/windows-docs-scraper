# ITsSbResourcePluginStore::SaveSession

## Description

Saves a session.

## Parameters

### `pSession` [in]

A Pointer to the [ITsSbSession](https://learn.microsoft.com/windows/desktop/api/sbtsv/nn-sbtsv-itssbsession) object to save.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
