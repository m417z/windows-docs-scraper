# ITsSbTaskInfo::get_Status

## Description

Retrieves an [RDV_TASK_STATUS](https://learn.microsoft.com/windows/win32/api/sessdirpublictypes/ne-sessdirpublictypes-rdv_task_status) enumeration value that represents the state of the task.

This property is read-only.
