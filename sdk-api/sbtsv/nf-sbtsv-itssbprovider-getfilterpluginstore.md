# ITsSbProvider::GetFilterPluginStore

## Description

Retrieves a [FilterPluginStore](https://learn.microsoft.com/windows/desktop/api/sbtsv/nn-sbtsv-itssbfilterpluginstore) instance of the filter plugin store.

## Parameters

### `ppStore` [out]

The address of an [ITsSbFilterPluginStore](https://learn.microsoft.com/windows/desktop/api/sbtsv/nn-sbtsv-itssbfilterpluginstore) interface pointer.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
