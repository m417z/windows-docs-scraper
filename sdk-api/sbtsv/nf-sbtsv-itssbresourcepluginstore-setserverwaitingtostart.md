# ITsSbResourcePluginStore::SetServerWaitingToStart

## Description

Indicates to the session host that the server is waiting to start.

## Parameters

### `PoolName` [in]

Name of the pool.

### `serverName` [in]

Name of the server.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
