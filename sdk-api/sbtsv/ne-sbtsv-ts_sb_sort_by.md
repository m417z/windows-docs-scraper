# TS_SB_SORT_BY enumeration

## Description

Specifies sort order. It is used as a parameter in the [EnumerateTargets](https://learn.microsoft.com/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcepluginstore-enumeratetargets) method.

## Constants

### `TS_SB_SORT_BY_NONE:0`

Do not sort.

### `TS_SB_SORT_BY_NAME:0x1`

Sort by target name.

### `TS_SB_SORT_BY_PROP:0x2`

Sort by a specified property.
