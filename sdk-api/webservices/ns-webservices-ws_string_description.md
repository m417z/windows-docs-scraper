# WS_STRING_DESCRIPTION structure

## Description

This type description is used with [WS_STRING_TYPE](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_type) and is optional.
It is used to specify constraints on the set of values
which can be deserialized.

## Members

### `minCharCount`

Specifies the minimum number of characters.

### `maxCharCount`

Specifies the maximum number of characters.
