# WS_DOUBLE_DESCRIPTION structure

## Description

An optional type description used with [WS_DOUBLE_TYPE](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_type).
It is used to specify constraints on the set of values
which can be deserialized.

## Members

### `minValue`

The minimum value.

### `maxValue`

The maximum value.
