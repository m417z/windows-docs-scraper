# WS_XML_QNAME_TEXT structure

## Description

Represents a qname formatted as the text "prefix:localName"

## Members

### `text`

The base type for all types that derive from [WS_XML_TEXT](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_text).

### `prefix`

The prefix.

### `localName`

The local name.
