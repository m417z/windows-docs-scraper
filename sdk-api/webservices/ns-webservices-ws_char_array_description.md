# WS_CHAR_ARRAY_DESCRIPTION structure

## Description

Specifies constraints on the set of values
which can be deserialized.This type description is used with [WS_CHAR_ARRAY_TYPE](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_type) and is optional.

## Members

### `minCharCount`

The minimum number of characters.

### `maxCharCount`

The maximum number of characters.
