# WS_XML_GUID_TEXT structure

## Description

Represents a GUID formatted as the text "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx".

## Members

### `text`

The base type for all types that derive from [WS_XML_TEXT](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_text).

### `value`

The GUID value
