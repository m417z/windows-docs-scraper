# WS_XML_UTF16_TEXT structure

## Description

Represents text encoded as UTF-16 bytes.

## Members

### `text`

The base type for all types that derive from [WS_XML_TEXT](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_text).

### `bytes`

The bytes that point to UTF-16 encoded data.

### `byteCount`

The length in bytes of the UTF-16 encoded data.
