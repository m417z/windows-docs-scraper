# WS_XML_READER_PROPERTIES structure

## Description

A structure that is used to specify a set of [WS_XML_READER_PROPERTY](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_reader_property)s.

## Members

### `properties`

An array of properties. The number of elements in the array is specified
using the propertyCount parameter. This field may be **NULL** if the propertyCount
is 0.

### `propertyCount`

The number of elements in the properties array.
