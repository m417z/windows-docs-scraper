# WS_SSL_TRANSPORT_SECURITY_BINDING_POLICY_DESCRIPTION structure

## Description

This type description is used with template APIs to describe
the templates generated accordingly to input policy setting.

See also,
[WsCreateServiceProxyFromTemplate](https://learn.microsoft.com/windows/desktop/api/webservices/nf-webservices-wscreateserviceproxyfromtemplate),
[WsCreateServiceEndpointFromTemplate](https://learn.microsoft.com/windows/desktop/api/webservices/nf-webservices-wscreateserviceendpointfromtemplate)

## Members

### `securityBindingProperties`

Specifies the security binding properties according to the specific policy.
