# WS_HTTP_HEADER_MAPPING structure

## Description

Specifies an individual header that is mapped as part of [WS_HTTP_MESSAGE_MAPPING](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_http_message_mapping).

## Members

### `headerName`

The name of the HTTP header.

### `headerMappingOptions`

A set of flags that control how headers are mapped.
See [WS_HTTP_HEADER_MAPPING_OPTIONS](https://learn.microsoft.com/windows/win32/api/webservices/ne-webservices-ws_xml_canonicalization_algorithm) for more information.
