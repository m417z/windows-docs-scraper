# WS_USERNAME_CREDENTIAL_TYPE enumeration

## Description

The type of the username/password credential, used as a selector for
subtypes of [WS_USERNAME_CREDENTIAL](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_username_credential).

## Constants

### `WS_STRING_USERNAME_CREDENTIAL_TYPE:1`

Type ID for the username/password credential [WS_STRING_USERNAME_CREDENTIAL](https://learn.microsoft.com/windows/win32/api/webservices/ns-webservices-ws_string_username_credential).
