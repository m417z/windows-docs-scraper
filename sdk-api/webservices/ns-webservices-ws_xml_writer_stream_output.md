# WS_XML_WRITER_STREAM_OUTPUT structure

## Description

Specifies that the generated bytes should be sent to callback.

## Members

### `output`

The base type for all types that derive from [WS_XML_WRITER_OUTPUT](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_writer_output).

### `writeCallback`

A callback that will get invoked when [WsFlushWriter](https://learn.microsoft.com/windows/desktop/api/webservices/nf-webservices-wsflushwriter) is called. This may not be **NULL**.

### `writeCallbackState`

User-defined data that will be passed to the writeCallback.
