# WS_BYTE_ARRAY_DESCRIPTION structure

## Description

Specifies constraints on the set of values
which can be deserialized.This type description is used with [WS_BYTE_ARRAY_TYPE](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_type) and is optional.

## Members

### `minByteCount`

Specifies the minimum number of bytes.

### `maxByteCount`

Specifies the maximum number of bytes.
