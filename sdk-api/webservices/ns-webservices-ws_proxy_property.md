# WS_PROXY_PROPERTY structure

## Description

Specifies a proxy property.

## Members

### `id`

Identifies the [WS_PROXY_PROPERTY_ID](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_proxy_property_id).

### `value`

The buffer for the property value.

### `valueSize`

The size of buffer, in bytes.
