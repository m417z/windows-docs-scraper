# WS_SECURITY_ALGORITHM_SUITE_NAME enumeration

## Description

A suite of security algorithms used for tasks such as signing and encrypting.
The values in this enumeration correspond to the suites defined in
WS-SecurityPolicy 1.1section 7.1.

## Constants

### `WS_SECURITY_ALGORITHM_SUITE_NAME_BASIC256:1`

Identifies the Basic256 algorithm suite. This suite uses the following algorithms:

* [WS_SECURITY_ALGORITHM_CANONICALIZATION_EXCLUSIVE](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_security_algorithm_id)
* [WS_SECURITY_ALGORITHM_DIGEST_SHA1](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_security_algorithm_id)
* [WS_SECURITY_ALGORITHM_SYMMETRIC_SIGNATURE_HMAC_SHA1](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_security_algorithm_id)
* [WS_SECURITY_ALGORITHM_ASYMMETRIC_SIGNATURE_RSA_SHA1](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_security_algorithm_id)
* [WS_SECURITY_ALGORITHM_KEY_DERIVATION_P_SHA1](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_security_algorithm_id)
* [WS_SECURITY_ALGORITHM_ASYMMETRIC_KEYWRAP_RSA_OAEP](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_security_algorithm_id)

The minimum symmetric key length is 256; the maximum symmetric key length is 256.
The minimum asymmetric key length is 1024; the maximum asymmetric key length is 4096.
