# WS_CERT_CREDENTIAL_TYPE enumeration

## Description

The type of the certificate credential, used as a selector for
subtypes of [WS_CERT_CREDENTIAL](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_cert_credential).

## Constants

### `WS_SUBJECT_NAME_CERT_CREDENTIAL_TYPE:1`

Type id for the certificate credential [WS_SUBJECT_NAME_CERT_CREDENTIAL](https://learn.microsoft.com/windows/win32/api/webservices/ns-webservices-ws_subject_name_cert_credential).

### `WS_THUMBPRINT_CERT_CREDENTIAL_TYPE:2`

Type id for the certificate credential [WS_THUMBPRINT_CERT_CREDENTIAL](https://learn.microsoft.com/windows/win32/api/webservices/ns-webservices-ws_thumbprint_cert_credential).

### `WS_CUSTOM_CERT_CREDENTIAL_TYPE:3`

Type id for the certificate credential [WS_CUSTOM_CERT_CREDENTIAL](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_custom_cert_credential).
