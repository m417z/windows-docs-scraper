# WS_XML_COMMENT_NODE structure

## Description

Represents a comment. (For example, \<!--The message follows-->)

## Members

### `node`

The base type for all types that derive from [WS_XML_NODE](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_node).

### `value`

The textual content of the comment. In the example, it is the string "The message follows".
