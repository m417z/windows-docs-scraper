# WS_SERVICE_PROPERTY_CLOSE_CALLBACK structure

## Description

Specifies the callback which is called when a channel is about to be closed.
See, [WS_SERVICE_CLOSE_CHANNEL_CALLBACK](https://learn.microsoft.com/windows/desktop/api/webservices/nc-webservices-ws_service_close_channel_callback) for details.

## Members

### `callback`

The close channel callback function reference.
