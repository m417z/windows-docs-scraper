# WS_XML_DECIMAL_TEXT structure

## Description

Represents a 12 byte fixed point value. (e.g. The value 1.23 represents the text "1.23")

## Members

### `text`

The base type for all types that derive from [WS_XML_TEXT](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_text).

### `value`

The value.
