# WS_SECURITY_BINDING_PROPERTY structure

## Description

Specifies a security binding specific setting.

## Members

### `id`

Identifies the [WS_SECURITY_BINDING_PROPERTY_ID](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_security_binding_property_id).

### `value`

A pointer to the value to set.
The pointer must have an alignment compatible with the type
of the property.

### `valueSize`

The size, in bytes, of the memory pointed to by value.
