# WS_XML_DATETIME_TEXT structure

## Description

Represents a datetime formatted as an
xsd:dateTime.

Negative datetime values are not supported.

## Members

### `text`

The base type for all types that derive from [WS_XML_TEXT](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_text).
