# WS_URL_SCHEME_TYPE enumeration

## Description

The set of schemes used with [WsDecodeUrl](https://learn.microsoft.com/windows/desktop/api/webservices/nf-webservices-wsdecodeurl), [WsEncodeUrl](https://learn.microsoft.com/windows/desktop/api/webservices/nf-webservices-wsencodeurl),
and [WsCombineUrl](https://learn.microsoft.com/windows/desktop/api/webservices/nf-webservices-wscombineurl).

## Constants

### `WS_URL_HTTP_SCHEME_TYPE:0`

Denotes the "http" scheme: [WS_HTTP_URL](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_http_url)

### `WS_URL_HTTPS_SCHEME_TYPE:1`

Denotes the "https" scheme: [WS_HTTPS_URL](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_https_url)

### `WS_URL_NETTCP_SCHEME_TYPE:2`

Denotes the "net.tcp" scheme: [WS_NETTCP_URL](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_nettcp_url)

### `WS_URL_SOAPUDP_SCHEME_TYPE:3`

Denotes the "soap.udp" scheme: [WS_SOAPUDP_URL](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_soapudp_url)

### `WS_URL_NETPIPE_SCHEME_TYPE:4`

Windows 8 or later: Denotes the "net.pipe" scheme: [WS_NETPIPE_URL](https://learn.microsoft.com/windows/win32/api/webservices/ns-webservices-ws_netpipe_url)
