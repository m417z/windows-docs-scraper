# WS_POLICY_EXTENSION_TYPE enumeration

## Description

The values in this enumeration are used to identify the sub-types of [WS_POLICY_EXTENSION](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_policy_extension).

## Constants

### `WS_ENDPOINT_POLICY_EXTENSION_TYPE:1`

This value is used in the type field of [WS_POLICY_EXTENSION](https://learn.microsoft.com/windows/win32/api/webservices/ns-webservices-ws_endpoint_policy_extension) to identify a [WS_ENDPOINT_POLICY_EXTENSION](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_endpoint_policy_extension) structure.
