# WS_UNION_FIELD_DESCRIPTION structure

## Description

Represents serialization information about a field within a union.
See [WS_UNION_DESCRIPTION](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_union_description).

## Members

### `value`

The enumeration value that field of the union corresponds to.
See [WS_FIELD_DESCRIPTION](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_field_description) and [WS_UNION_DESCRIPTION](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_union_description).

### `field`

Information about the field. See [WS_FIELD_DESCRIPTION](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_field_description) and [WS_UNION_DESCRIPTION](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_union_description).
