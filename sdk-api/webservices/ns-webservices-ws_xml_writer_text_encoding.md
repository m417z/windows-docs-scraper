# WS_XML_WRITER_TEXT_ENCODING structure

## Description

Indicates that the reader should emit bytes as textual xml.

## Members

### `encoding`

The base type for all types that derive from [WS_XML_WRITER_ENCODING](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_writer_encoding).

### `charSet`

Indicates the text encoding of the bytes. This may not be [WS_CHARSET_AUTO](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_charset).
