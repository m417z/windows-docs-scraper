# WS_XML_BOOL_TEXT structure

## Description

A Boolean value that represents the text "true" or "false".

## Members

### `text`

The base type for all types that derive from [WS_XML_TEXT](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_text).

### `value`

The Boolean value. 0 represents the text "false", while 1 represents the text "true".
