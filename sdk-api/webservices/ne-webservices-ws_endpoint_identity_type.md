# WS_ENDPOINT_IDENTITY_TYPE enumeration

## Description

The type of the endpoint IDentity, used as a selector for subtypes of
[WS_ENDPOINT_IDENTITY](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_endpoint_identity).

## Constants

### `WS_DNS_ENDPOINT_IDENTITY_TYPE:1`

Type ID for [WS_DNS_ENDPOINT_IDENTITY](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_dns_endpoint_identity).

### `WS_UPN_ENDPOINT_IDENTITY_TYPE:2`

Type ID for [WS_UPN_ENDPOINT_IDENTITY](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_upn_endpoint_identity).

### `WS_SPN_ENDPOINT_IDENTITY_TYPE:3`

Type ID for [WS_SPN_ENDPOINT_IDENTITY](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_spn_endpoint_identity).

### `WS_RSA_ENDPOINT_IDENTITY_TYPE:4`

### `WS_CERT_ENDPOINT_IDENTITY_TYPE:5`

Type ID for [WS_UNKNOWN_ENDPOINT_IDENTITY](https://learn.microsoft.com/windows/win32/api/webservices/ns-webservices-ws_unknown_endpoint_identity).
