# WS_URL structure

## Description

The abstract base type for all URL schemes used with [WsDecodeUrl](https://learn.microsoft.com/windows/desktop/api/webservices/nf-webservices-wsdecodeurl) and [WsEncodeUrl](https://learn.microsoft.com/windows/desktop/api/webservices/nf-webservices-wsencodeurl) APIs.

## Members

### `scheme`

A [WS_URL_SCHEME_TYPE](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_url_scheme_type) structure that describes the type of URL scheme.
