# WS_SECURITY_ALGORITHM_PROPERTY structure

## Description

Specifies a cryptographic algorithm setting.

## Members

### `id`

Identifies the [WS_SECURITY_ALGORITHM_PROPERTY_ID](https://learn.microsoft.com/windows/win32/api/webservices/ne-webservices-ws_move_to).

### `value`

A pointer to the value to set.
The pointer must have an alignment compatible with the type
of the property.

### `valueSize`

The size in bytes of the memory pointed to by value.
