# WS_SAML_AUTHENTICATOR_TYPE enumeration

## Description

The type IDs of the SAML token authenticators used on the server side
(For example, relying party) to validate incoming SAML tokens.

## Constants

### `WS_CERT_SIGNED_SAML_AUTHENTICATOR_TYPE:1`

Type ID for the SAML authenticator [WS_CERT_SIGNED_SAML_AUTHENTICATOR](https://learn.microsoft.com/windows/win32/api/webservices/ns-webservices-ws_cert_signed_saml_authenticator).
