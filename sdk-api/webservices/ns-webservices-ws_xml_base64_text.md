# WS_XML_BASE64_TEXT structure

## Description

Represents base64 encoded data. (for example, the three bytes { 0, 0, 0 } represent the text "AAAA".)

## Members

### `text`

The base type for all types that derive from [WS_XML_TEXT](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_xml_text).

### `bytes`

The bytes of data.

### `length`

The length of the bytes of data.
