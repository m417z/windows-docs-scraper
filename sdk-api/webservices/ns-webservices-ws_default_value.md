# WS_DEFAULT_VALUE structure

## Description

Defines a default value for a field. This is used
in a [WS_FIELD_DESCRIPTION](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_field_description).

## Members

### `value`

A pointer to the default value.

### `valueSize`

The size of the default value, in bytes.
