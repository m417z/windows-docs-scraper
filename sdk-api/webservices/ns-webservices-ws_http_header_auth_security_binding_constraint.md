# WS_HTTP_HEADER_AUTH_SECURITY_BINDING_CONSTRAINT structure

## Description

A security binding constraint that corresponds to the
[WS_HTTP_HEADER_AUTH_SECURITY_BINDING](https://learn.microsoft.com/windows/desktop/api/webservices/ns-webservices-ws_http_header_auth_security_binding).

## Members

### `bindingConstraint`

The base binding constraint that this binding constraint derives from.
