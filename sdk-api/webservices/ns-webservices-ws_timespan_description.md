# WS_TIMESPAN_DESCRIPTION structure

## Description

This type description is used with [WS_TIMESPAN_TYPE](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_type) and is optional.
It is used to specify constraints on the set of values
which can be deserialized.

## Members

### `minValue`

The minimum value.

### `maxValue`

The maximum value.
