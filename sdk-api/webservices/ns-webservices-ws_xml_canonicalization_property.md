# WS_XML_CANONICALIZATION_PROPERTY structure

## Description

Specifies a setting that controls how XML canonicalization is done.

## Members

### `id`

Identifies the [WS_XML_CANONICALIZATION_PROPERTY_ID](https://learn.microsoft.com/windows/desktop/api/webservices/ne-webservices-ws_xml_canonicalization_property_id).

### `value`

A pointer to the value to set.
The pointer must have an alignment compatible with the type
of the property.

### `valueSize`

The size in bytes of the memory pointed to by value.
