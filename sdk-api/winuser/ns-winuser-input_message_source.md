# INPUT_MESSAGE_SOURCE structure

## Description

Contains information about the source of the input message.

## Members

### `deviceType`

The device type ([INPUT_MESSAGE_DEVICE_TYPE](https://learn.microsoft.com/windows/desktop/api/winuser/ne-winuser-input_message_device_type)) of the source of the input message.

### `originId`

The ID ([INPUT_MESSAGE_ORIGIN_ID](https://learn.microsoft.com/windows/desktop/api/winuser/ne-winuser-input_message_origin_id)) of the source of the input message.
