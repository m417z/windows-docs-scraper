# SetDisplayAutoRotationPreferences function

## Description

Sets the screen auto-rotation preferences for the current process.

## Parameters

### `orientation` [in]

Pointer to a location in memory with the screen orientation preferences to set for the calling process.

## Return value

TRUE if the method succeeds, otherwise FALSE.

See [GetDisplayAutoRotationPreferences](https://learn.microsoft.com/windows/desktop/api/winuser/nf-winuser-getdisplayautorotationpreferences) for an example of using this function.
