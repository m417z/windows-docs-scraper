# POINTER_DEVICE_CURSOR_INFO structure

## Description

Contains cursor ID mappings for pointer devices.

## Members

### `cursorId`

The assigned cursor ID.

### `cursor`

The [POINTER_DEVICE_CURSOR_TYPE](https://learn.microsoft.com/windows/desktop/api/winuser/ne-winuser-pointer_device_cursor_type) that the ID is mapped to.
