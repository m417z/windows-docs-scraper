# GID_ROTATE_ANGLE_FROM_ARGUMENT macro

## Description

The **GID_ROTATE_ANGLE_FROM_ARGUMENT** macro is used to interpret the
**GID_ROTATE** *ullArgument* value when receiving
the value in the [WM_GESTURE](https://learn.microsoft.com/windows/desktop/wintouch/wm-gesture) structure.

## Parameters

### `_arg_`

A value from a [WM_GESTURE](https://learn.microsoft.com/windows/desktop/wintouch/wm-gesture) message.
