# POINTTOPOINTS macro

## Syntax

```cpp
POINTS POINTTOPOINTS(
    POINT pt
);
```

## Return value

Type: **POINTS**

The return value is a [POINTS](https://learn.microsoft.com/windows/win32/api/windef/ns-windef-points) structure.

## Description

The **POINTTOPOINTS** macro converts a [POINT](https://learn.microsoft.com/windows/win32/api/windef/ns-windef-point) structure to a [POINTS](https://learn.microsoft.com/windows/win32/api/windef/ns-windef-points) structure.

## Parameters

### `pt`

The [POINT](https://learn.microsoft.com/windows/win32/api/windef/ns-windef-point) structure to convert.
