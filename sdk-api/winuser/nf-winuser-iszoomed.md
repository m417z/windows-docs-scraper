# IsZoomed function

## Description

Determines whether a window is maximized.

## Parameters

### `hWnd` [in]

Type: **HWND**

A handle to the window to be tested.

## Return value

Type: **BOOL**

If the window is zoomed, the return value is nonzero.

If the window is not zoomed, the return value is zero.
