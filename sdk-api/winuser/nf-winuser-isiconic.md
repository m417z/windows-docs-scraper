# IsIconic function

## Description

Determines whether the specified window is minimized (iconic).

## Parameters

### `hWnd` [in]

Type: **HWND**

A handle to the window to be tested.

## Return value

Type: **BOOL**

If the window is iconic, the return value is nonzero.

If the window is not iconic, the return value is zero.
