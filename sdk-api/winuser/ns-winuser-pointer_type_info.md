# POINTER_TYPE_INFO structure

## Description

Contains information about the pointer input type.

## Members

### `type`

The pointer input device.

### `DUMMYUNIONNAME`

### `DUMMYUNIONNAME.touchInfo`

A [POINTER_TOUCH_INFO](https://learn.microsoft.com/windows/win32/api/winuser/ns-winuser-pointer_touch_info) structure representing basic touch information common to all pointer types.

### `DUMMYUNIONNAME.penInfo`

A [POINTER_PEN_INFO](https://learn.microsoft.com/windows/win32/api/winuser/ns-winuser-pointer_pen_info) structure representing basic pen information common to all pointer types.
