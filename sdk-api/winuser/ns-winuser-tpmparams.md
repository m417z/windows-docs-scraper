# TPMPARAMS structure

## Description

Contains extended parameters for the [TrackPopupMenuEx](https://learn.microsoft.com/windows/desktop/api/winuser/nf-winuser-trackpopupmenuex) function.

## Members

### `cbSize`

Type: **UINT**

The size of structure, in bytes.

### `rcExclude`

Type: **[RECT](https://learn.microsoft.com/windows/desktop/api/windef/ns-windef-rect)**

The rectangle to be excluded when positioning the window, in screen coordinates.
