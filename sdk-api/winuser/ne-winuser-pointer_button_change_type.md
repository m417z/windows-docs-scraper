# POINTER_BUTTON_CHANGE_TYPE enumeration

## Description

Identifies a change in the state of a button associated with a pointer.

## Constants

### `POINTER_CHANGE_NONE`

No change in button state.

### `POINTER_CHANGE_FIRSTBUTTON_DOWN`

The first button (see [POINTER_FLAG_FIRSTBUTTON](https://learn.microsoft.com/windows/win32/inputmsg/pointer-flags-contants)) transitioned to a pressed state.

### `POINTER_CHANGE_FIRSTBUTTON_UP`

The first button (see [POINTER_FLAG_FIRSTBUTTON](https://learn.microsoft.com/windows/win32/inputmsg/pointer-flags-contants)) transitioned to a released state.

### `POINTER_CHANGE_SECONDBUTTON_DOWN`

The second button (see [POINTER_FLAG_SECONDBUTTON](https://learn.microsoft.com/windows/win32/inputmsg/pointer-flags-contants)) transitioned to a pressed state.

### `POINTER_CHANGE_SECONDBUTTON_UP`

The second button (see [POINTER_FLAG_SECONDBUTTON](https://learn.microsoft.com/windows/win32/inputmsg/pointer-flags-contants)) transitioned to a released state.

### `POINTER_CHANGE_THIRDBUTTON_DOWN`

The third button (see [POINTER_FLAG_THIRDBUTTON](https://learn.microsoft.com/windows/win32/inputmsg/pointer-flags-contants)) transitioned to a pressed state.

### `POINTER_CHANGE_THIRDBUTTON_UP`

The third button (see [POINTER_FLAG_THIRDBUTTON](https://learn.microsoft.com/windows/win32/inputmsg/pointer-flags-contants)) transitioned to a released state.

### `POINTER_CHANGE_FOURTHBUTTON_DOWN`

The fourth button (see [POINTER_FLAG_FOURTHBUTTON](https://learn.microsoft.com/windows/win32/inputmsg/pointer-flags-contants)) transitioned to a pressed state.

### `POINTER_CHANGE_FOURTHBUTTON_UP`

The fourth button (see [POINTER_FLAG_FOURTHBUTTON](https://learn.microsoft.com/windows/win32/inputmsg/pointer-flags-contants)) transitioned to a released state.

### `POINTER_CHANGE_FIFTHBUTTON_DOWN`

The fifth button (see [POINTER_FLAG_FIFTHBUTTON](https://learn.microsoft.com/windows/win32/inputmsg/pointer-flags-contants)) transitioned to a pressed state.

### `POINTER_CHANGE_FIFTHBUTTON_UP`

The fifth button (see [POINTER_FLAG_FIFTHBUTTON](https://learn.microsoft.com/windows/win32/inputmsg/pointer-flags-contants)) transitioned to a released state.
