## Description

Gets the [WindowId](https://learn.microsoft.com/uwp/api/windows.ui.windowid) that corresponds to the specified window handle ([HWND](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)), if the hwnd argument is valid.

## Parameters

### `hwnd`

The handle of the window for which to get the **WindowId**.

### `id`

The identifier that corresponds to *hwnd*, if *hwnd* is valid. Otherwise, null.
