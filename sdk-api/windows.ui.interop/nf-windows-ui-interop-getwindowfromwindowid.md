## Description

Gets the window handle ([HWND](https://learn.microsoft.com/windows/win32/winprog/windows-data-types)) that corresponds to the specified [WindowId](https://learn.microsoft.com/uwp/api/windows.ui.windowid), if the *windowId* argument is valid and the system has an **HWND** that represents the window.

## Parameters

### `id`

The identifier for the window.

### `hwnd`

The window handle that corresponds to *windowId*, if *windowId* is valid and the system has an **HWND** that represents the window. Otherwise, null.
