# IMallocSpy::PreHeapMinimize

## Description

Performs operations required before calling [IMalloc::HeapMinimize](https://learn.microsoft.com/windows/desktop/api/objidl/nf-objidl-imalloc-heapminimize).

## Remarks

This method is included for completeness; it is not anticipated that developers will implement significant functionality in this method.
