# PENDINGTYPE enumeration

## Description

Indicates the level of nesting in the [IMessageFilter::MessagePending](https://learn.microsoft.com/windows/desktop/api/objidl/nf-objidl-imessagefilter-messagepending) method.

## Constants

### `PENDINGTYPE_TOPLEVEL:1`

Top-level call.

### `PENDINGTYPE_NESTED:2`

Nested call.
