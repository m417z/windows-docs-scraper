# ISurrogateService::CatalogRefresh

## Description

Refreshes the catalog.

## Parameters

### `ulReserved` [in]

This parameter is reserved.

## Return value

If the method succeeds, the return value is S_OK. Otherwise, it is E_UNEXPECTED.
