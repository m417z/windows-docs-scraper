# IEnumSTATSTG::Reset

## Description

The **Reset** method resets the enumeration sequence to the beginning of the [STATSTG](https://learn.microsoft.com/windows/desktop/api/objidl/ns-objidl-statstg) structure array.

## Return value

This method supports the S_OK return value.

| Return code | Description |
| --- | --- |
| **S_OK** | The enumeration sequence was successfully reset to the beginning of the enumeration. |
