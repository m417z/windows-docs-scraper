# DXGI_OVERLAY_COLOR_SPACE_SUPPORT_FLAG enumeration

## Description

Specifies support for overlay color space.

## Constants

### `DXGI_OVERLAY_COLOR_SPACE_SUPPORT_FLAG_PRESENT:0x1`

Overlay color space support is present.
