# IFsrmPropertyDefinition2::get_PropertyDefinitionFlags

## Description

[This property is supported for compatibility but it's recommended to use the
[FSRM WMI Classes](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/fsrm-wmi-classes) to manage FSRM. Please see the
[MSFT_FSRMClassificationPropertyDefinition](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/msft-fsrmclassificationpropertydefinition)
class.]

This property contains the flags for the property definition as enumerated by the
[FsrmPropertyDefinitionFlags](https://learn.microsoft.com/windows/desktop/api/fsrmenums/ne-fsrmenums-fsrmpropertydefinitionflags) enumeration.

This property is read-only.
