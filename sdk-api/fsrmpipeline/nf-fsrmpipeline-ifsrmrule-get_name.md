# IFsrmRule::get_Name

## Description

The name of the rule.

This property is read/write.

## Parameters

## Remarks

The name must be unique for its type (see the [IFsrmRule.RuleType](https://learn.microsoft.com/previous-versions/windows/desktop/api/fsrmpipeline/nf-fsrmpipeline-ifsrmrule-get_ruletype) property).
