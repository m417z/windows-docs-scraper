# IFsrmRule::get_RuleType

## Description

The type of the rule.

This property is read-only.

## Parameters

## Remarks

The rule's type is specified when you call the [IFsrmClassificationManager::CreateRule](https://learn.microsoft.com/previous-versions/windows/desktop/api/fsrmpipeline/nf-fsrmpipeline-ifsrmclassificationmanager-createrule) method to create the rule.

The name and rule type properties define a unique rule.
