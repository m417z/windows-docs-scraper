## Description

Frees a **WEBAUTHN_CREDENTIAL_ATTESTATION** previously allocated by calling [WebAuthNAuthenticatorMakeCredential](https://learn.microsoft.com/windows/win32/api/webauthn/nf-webauthn-webauthnauthenticatormakecredential).

## Parameters

### `pWebAuthNCredentialAttestation`

The credential attestation to be freed.
