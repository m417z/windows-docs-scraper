## Description

Contains the SALT values for the Hmac-Secret.

## Members

### `cbFirst`

The size of pbFirst.

### `pbFirst`

The first SALT value.

### `cbSecond`

THe size of pbSecond.

### `pbSecond`

The second SALT value.

## Remarks

SALT values, by default, are converted into RAW Hmac-Secret values as per PRF extension.
