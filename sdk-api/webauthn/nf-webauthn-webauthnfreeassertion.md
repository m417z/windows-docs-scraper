## Description

Frees an assertion previously allocated by calling [WebAuthNAuthenticatorGetAssertion](https://learn.microsoft.com/windows/win32/api/webauthn/nf-webauthn-webauthnauthenticatorgetassertion).

## Parameters

### `pWebAuthNAssertion`

The **WEBAUTHN_ASSERTION** to be freed.
