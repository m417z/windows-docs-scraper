# IAccessible::put_accName

## Description

[[IAccessible::put_accName](https://learn.microsoft.com/windows/desktop/api/oleacc/nf-oleacc-iaccessible-put_accname) is no longer supported. ]

The [IAccessible::put_accName](https://learn.microsoft.com/windows/desktop/api/oleacc/nf-oleacc-iaccessible-put_accname) method is no longer supported. Client applications should use a control-specific workaround, such as the [SetWindowText](https://learn.microsoft.com/windows/desktop/api/winuser/nf-winuser-setwindowtexta) function. Servers should return E_NOTIMPL.

## Parameters

### `varChild` [in, optional]

Not supported.

### `szName` [in]

Not supported.

## Return value

Not supported.
