# EOC_ChangeType enumeration

## Description

Indicates the nature of the change to an event object.

## Constants

### `EOC_NewObject:0`

A new object was added.

### `EOC_ModifiedObject`

An existing object was modified.

### `EOC_DeletedObject`

An existing object was deleted.
