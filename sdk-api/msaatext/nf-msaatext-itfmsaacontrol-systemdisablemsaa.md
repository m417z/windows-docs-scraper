# ITfMSAAControl::SystemDisableMSAA

## Description

Used by MSAA to halt TSF support of an MSAA client.

## Return value

This method can return one of these values.

| Value | Description |
| --- | --- |
| **S_OK** | The method was successful. |
| **E_FAIL** | An unspecified error occurred. |
