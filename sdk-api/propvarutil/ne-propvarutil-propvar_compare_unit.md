# PROPVAR_COMPARE_UNIT enumeration

## Description

These flags are associated with certain [PROPVARIANT](https://learn.microsoft.com/windows/desktop/api/propidl/ns-propidl-propvariant) structure comparisons.

## Constants

### `PVCU_DEFAULT:0`

The default unit.

### `PVCU_SECOND:1`

The second comparison unit.

### `PVCU_MINUTE:2`

The minute comparison unit.

### `PVCU_HOUR:3`

The hour comparison unit.

### `PVCU_DAY:4`

The day comparison unit.

### `PVCU_MONTH:5`

The month comparison unit.

### `PVCU_YEAR:6`

The year comparison unit.
