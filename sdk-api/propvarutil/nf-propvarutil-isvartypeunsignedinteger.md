# IsVarTypeUnsignedInteger function

## Description

Returns whether a [VARTYPE](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/ms221127(v=vs.85)) is an unsigned integer.

## Parameters

### `vt` [in]

Type: **[VARTYPE](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/ms221127(v=vs.85))**

Specifies the [VARTYPE](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/ms221127(v=vs.85)) being queried.

## Return value

Type: **BOOL**

Returns **TRUE** if [VARTYPE](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/ms221127(v=vs.85)) is an unsigned integer; otherwise, **FALSE**.

## Remarks

This is an inline function, with its source code provided in the header. It is not included in any .dll or .lib file.
