# NcIsValidConnectionName function

## Description

[Internet Connection Firewall may be altered or unavailable in subsequent versions. Instead, use the [Windows Firewall API](https://learn.microsoft.com/previous-versions/windows/desktop/ics/windows-firewall-start-page).]

The
**NcIsValidConnectionName** function verifies if the passed in connection name is valid.

## Parameters

### `pszwName` [in]

Connection name to check.

## Return value

**TRUE** if connection name is valid.
