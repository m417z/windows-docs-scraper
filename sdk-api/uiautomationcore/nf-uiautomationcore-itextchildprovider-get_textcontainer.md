# ITextChildProvider::get_TextContainer

## Description

Retrieves this element's nearest ancestor provider that supports the [Text](https://learn.microsoft.com/windows/desktop/WinAuto/uiauto-implementingtextandtextrange) control pattern.

This property is read-only.
