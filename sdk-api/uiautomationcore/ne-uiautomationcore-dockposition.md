# DockPosition enumeration

## Description

Contains values that specify the location of a docking window represented by the [Dock](https://learn.microsoft.com/windows/desktop/WinAuto/uiauto-implementingdock) *control pattern*.

## Constants

### `DockPosition_Top:0`

The window is docked at the top.

### `DockPosition_Left:1`

The window is docked at the left.

### `DockPosition_Bottom:2`

The window is docked at the bottom.

### `DockPosition_Right:3`

The window is docked at the right.

### `DockPosition_Fill:4`

The window is docked on all four sides.

### `DockPosition_None:5`

The window is not docked.
