# TextUnit enumeration

## Description

Contains values that specify units of text for the purposes of navigation.

## Constants

### `TextUnit_Character:0`

Character.

### `TextUnit_Format:1`

Format.

### `TextUnit_Word:2`

Word.

### `TextUnit_Line:3`

Line.

### `TextUnit_Paragraph:4`

Paragraph.

### `TextUnit_Page:5`

Page.

### `TextUnit_Document:6`

Document.
