# IGridItemProvider::get_ContainingGrid

## Description

Specifies the UI Automation provider that implements [IGridProvider](https://learn.microsoft.com/windows/desktop/api/uiautomationcore/nn-uiautomationcore-igridprovider)
and represents the container of this cell or item.

This property is read-only.
