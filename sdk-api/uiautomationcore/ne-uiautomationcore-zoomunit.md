# ZoomUnit enumeration

## Description

Contains possible values for the [IUIAutomationTransformPattern2::ZoomByUnit](https://learn.microsoft.com/windows/desktop/api/uiautomationclient/nf-uiautomationclient-iuiautomationtransformpattern2-zoombyunit) method, which zooms the viewport of a control by the specified unit.

## Constants

### `ZoomUnit_NoAmount:0`

No increase or decrease in zoom.

### `ZoomUnit_LargeDecrement:1`

Decrease zoom by a large decrement.

### `ZoomUnit_SmallDecrement:2`

Decrease zoom by a small decrement.

### `ZoomUnit_LargeIncrement:3`

Increase zoom by a large increment.

### `ZoomUnit_SmallIncrement:4`

Increase zoom by a small increment.
