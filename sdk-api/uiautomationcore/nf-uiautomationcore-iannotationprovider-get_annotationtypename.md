# IAnnotationProvider::get_AnnotationTypeName

## Description

The name of this annotation type.

This property is read-only.

## Parameters

## Remarks

The name of the annotation type can correspond to one of the annotation type identifiers (for example, “Comment” for [AnnotationType_Comment](https://learn.microsoft.com/windows/desktop/WinAuto/uiauto-annotation-type-identifiers)), but it is not required to.
