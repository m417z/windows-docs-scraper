# CapStyle enumeration

## Description

Contains values that specify the value of the CapStyle text attribute.

## Constants

### `CapStyle_None:0`

None.

### `CapStyle_SmallCap:1`

Small capitals.

### `CapStyle_AllCap:2`

All capitals.

### `CapStyle_AllPetiteCaps:3`

All petite capitals.

### `CapStyle_PetiteCaps:4`

Petite capitals.

### `CapStyle_Unicase:5`

Single case.

### `CapStyle_Titling:6`

Title case.

### `CapStyle_Other:-1`

#### - Other

Other.
