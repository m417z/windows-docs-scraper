# ISelectionItemProvider::get_SelectionContainer

## Description

Specifies the provider that implements [ISelectionProvider](https://learn.microsoft.com/windows/desktop/api/uiautomationcore/nn-uiautomationcore-iselectionprovider)
and acts as the container for the calling object.

This property is read-only.
