# ActiveEnd enumeration

## Description

Contains possible values for the [SelectionActiveEnd](https://learn.microsoft.com/windows/desktop/WinAuto/uiauto-textattribute-ids) text attribute, which indicates the location of the caret relative to a text range that represents the currently selected text.

## Constants

### `ActiveEnd_None:0`

The caret is not at either end of the text range.

### `ActiveEnd_Start:1`

The caret is at the beginning of the text range.

### `ActiveEnd_End:2`

The caret is at the end of the text range.
