# IRangeValueProvider::get_Minimum

## Description

Specifies the minimum range value supported by the control.

This property is read-only.

## Parameters

## Remarks

This value should be less than [Maximum](https://learn.microsoft.com/windows/win32/api/uiautomationcore/nf-uiautomationcore-irangevalueprovider-get_maximum).
