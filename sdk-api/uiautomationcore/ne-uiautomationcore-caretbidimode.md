# CaretBidiMode enumeration

## Description

Contains possible values for the [CaretBidiMode](https://learn.microsoft.com/windows/desktop/WinAuto/uiauto-textattribute-ids) text attribute, which indicates whether the caret is in text that flows from left to right, or from right to left.

## Constants

### `CaretBidiMode_LTR:0`

The caret is in text that flows from left to right.

### `CaretBidiMode_RTL:1`

The caret is in text that flows from right to left.
