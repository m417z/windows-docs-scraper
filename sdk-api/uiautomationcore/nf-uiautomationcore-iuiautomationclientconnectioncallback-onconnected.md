# OnConnected function

## Description

Synchonously called when a UI Automation client connects to the provider.

## Parameters

### `clientInfo` [in]

[IUIAutomationClientInfo](https://learn.microsoft.com/windows/win32/api/uiautomationcore/nn-uiautomationcore-iuiautomationclientinfo) object that provides information about the connecting client (valid only during the method execution).

## Return value

Ignored.
