# OnDisconnected function

## Description

Asynchonously called when a UI Automation client disconnects from the provider.

## Parameters

### `clientInfo` [in]

[IUIAutomationClientInfo](https://learn.microsoft.com/windows/win32/api/uiautomationcore/nn-uiautomationcore-iuiautomationclientinfo) object that provides information about the disconnecting client (valid only during the method execution).

## Return value

Ignored.
