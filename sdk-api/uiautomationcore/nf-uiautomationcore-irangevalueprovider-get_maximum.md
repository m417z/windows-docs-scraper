# IRangeValueProvider::get_Maximum

## Description

Specifies the maximum range value supported by the control.

This property is read-only.

## Parameters

## Remarks

This value should be greater than [Minimum](https://learn.microsoft.com/windows/win32/api/uiautomationcore/nf-uiautomationcore-irangevalueprovider-get_minimum).
