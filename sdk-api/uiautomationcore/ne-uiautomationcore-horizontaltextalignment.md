## Description

Defines horizontal text alignment values.

## Constants

### `HorizontalTextAlignment_Left`

Text is left-aligned.

### `HorizontalTextAlignment_Centered`

Text is center-aligned.

### `HorizontalTextAlignment_Right`

Text is right-aligned.

### `HorizontalTextAlignment_Justified`

Text is justified.

The last line of a paragraph is typically left or right-aligned, depending on the language direction.
