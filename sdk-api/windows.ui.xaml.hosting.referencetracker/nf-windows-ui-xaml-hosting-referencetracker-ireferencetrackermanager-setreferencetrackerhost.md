# IReferenceTrackerManager::SetReferenceTrackerHost (windows.ui.xaml.hosting.referencetracker.h)

Registers an [IReferenceTrackerHost](https://learn.microsoft.com/windows/desktop/api/windows.ui.xaml.hosting.referencetracker/nn-windows-ui-xaml-hosting-referencetracker-ireferencetrackerhost) interface with XAML.

### `value` [in]

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
