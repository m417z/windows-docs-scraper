# HTTP_REQUEST_PROPERTY_STREAM_ERROR structure

## Description

The **HTTP\_REQUEST\_PROPERTY\_STREAM\_ERROR** structure represents an HTTP/2 or HTTP/3 stream error code.

## Members

### `ErrorCode`

The protocol stream error code.
