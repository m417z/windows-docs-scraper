# IWSDMessageParameters::SetLocalAddress

## Description

Sets a generic address object representing the source address that should send the message.

## Parameters

### `pAddress` [in]

An [IWSDAddress](https://learn.microsoft.com/windows/desktop/api/wsdbase/nn-wsdbase-iwsdaddress) interface that represents the source address that should send the message.

## Return value

This method can return one of these values.

Possible return values include, but are not limited to, the following.

| Return code | Description |
| --- | --- |
| **S_OK** | Method completed successfully. |
