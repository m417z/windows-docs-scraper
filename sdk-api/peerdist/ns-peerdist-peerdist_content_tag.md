# PEERDIST_CONTENT_TAG structure

## Description

The **PEERDIST_CONTENT_TAG** structure contains a client supplied content tag as an input to the [PeerDistClientOpenContent](https://learn.microsoft.com/windows/desktop/api/peerdist/nf-peerdist-peerdistclientopencontent) API.

## Members

### `Data`

A 16 byte tag associated with the open content.
