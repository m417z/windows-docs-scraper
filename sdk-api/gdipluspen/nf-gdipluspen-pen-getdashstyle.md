# Pen::GetDashStyle

## Description

The **Pen::GetDashStyle** method gets the dash style currently set for this
[Pen](https://learn.microsoft.com/windows/desktop/api/gdipluspen/nl-gdipluspen-pen) object.

## Return value

Type: **[DashStyle](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-dashstyle)**

This method returns an element of the [DashStyle](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-dashstyle) enumeration that indicates the dash style for this
[Pen](https://learn.microsoft.com/windows/desktop/api/gdipluspen/nl-gdipluspen-pen) object.
