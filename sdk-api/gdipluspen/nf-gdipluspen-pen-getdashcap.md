# Pen::GetDashCap

## Description

The **Pen::GetDashCap** method gets the dash cap style currently set for this
[Pen](https://learn.microsoft.com/windows/desktop/api/gdipluspen/nl-gdipluspen-pen) object.

## Return value

Type: **[DashCap](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-dashcap)**

This method returns an element of the [DashCap](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-dashcap) enumeration that indicates the dash cap style currently set for this
[Pen](https://learn.microsoft.com/windows/desktop/api/gdipluspen/nl-gdipluspen-pen) object.
