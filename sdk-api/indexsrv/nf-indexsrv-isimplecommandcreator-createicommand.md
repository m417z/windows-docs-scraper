# ISimpleCommandCreator::CreateICommand

## Description

Creates an ICommand.

## Parameters

### `ppIUnknown`

Returns the IUnknown for the command.

### `pOuterUnk`

Optional outer unknown pointer.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
