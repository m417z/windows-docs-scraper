# MINIDUMP_MODULE_LIST structure

## Description

Contains a list of modules.

## Members

### `NumberOfModules`

The number of structures in the **Modules** array.

### `Modules`

An array of
[MINIDUMP_MODULE](https://learn.microsoft.com/windows/desktop/api/minidumpapiset/ns-minidumpapiset-minidump_module) structures.
