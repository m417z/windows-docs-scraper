# IsEqualGUID macro

## Description

Determines whether two GUIDs are equal.

## Parameters

### `rguid1` [in]

The first GUID.

### `rguid2` [in]

The second GUID.

## Remarks

**IsEqualGUID** is used by the [IsEqualCLSID](https://learn.microsoft.com/windows/desktop/api/guiddef/nf-guiddef-isequalclsid) and [IsEqualIID](https://learn.microsoft.com/windows/desktop/api/guiddef/nf-guiddef-isequaliid) functions.
