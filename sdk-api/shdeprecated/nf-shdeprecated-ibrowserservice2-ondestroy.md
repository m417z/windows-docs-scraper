# IBrowserService2::OnDestroy

## Description

Deprecated. Calls the derived class from the base class on receipt of a [WM_DESTROY](https://learn.microsoft.com/windows/desktop/winmsg/wm-destroy) message. The derived class handles the message.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
