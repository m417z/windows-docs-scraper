# IBrowserService2::OnSetFocus

## Description

Deprecated. Calls the derived class from the base class on receipt of a [WM_SETFOCUS](https://learn.microsoft.com/windows/desktop/inputdev/wm-setfocus) message. The derived class handles the message.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
