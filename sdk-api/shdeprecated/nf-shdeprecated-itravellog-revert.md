# ITravelLog::Revert

## Description

Deprecated. Reverts to the current entry, dropping the result of [ITravelLog::AddEntry](https://learn.microsoft.com/windows/desktop/api/shdeprecated/nf-shdeprecated-itravellog-addentry) in the case of a failed navigation.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
