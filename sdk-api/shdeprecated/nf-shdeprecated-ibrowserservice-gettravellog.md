# IBrowserService::GetTravelLog

## Description

Deprecated. Retrieves the browser's [ITravelLog](https://learn.microsoft.com/windows/desktop/api/shdeprecated/nn-shdeprecated-itravellog).

## Parameters

### `pptl` [out]

Type: **[ITravelLog](https://learn.microsoft.com/windows/desktop/api/shdeprecated/nn-shdeprecated-itravellog)****

The address that receives a pointer to the browser's [ITravelLog](https://learn.microsoft.com/windows/desktop/api/shdeprecated/nn-shdeprecated-itravellog).

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
