# IExpDispSupportXP::OnTranslateAccelerator

## Description

Not implemented.

## Parameters

### `pMsg` [in]

Type: **[MSG](https://learn.microsoft.com/windows/desktop/api/winuser/ns-winuser-msg)***

Specifies a pointer to the MSG structure describing the keystroke to be processed.

### `grfModifiers`

Type: **DWORD**

Specifies the flags describing the state of the Control, Alt, and Shift keys. The value of the flag can be any valid [KEYMODIFIERS](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/ms683763(v=vs.85)) enumeration values.

## Return value

Type: **HRESULT**

Returns **E_NOTIMPL**.
