# IBrowserService2::OnSize

## Description

Deprecated. Calls the derived class from the base class on receipt of a [WM_SIZE](https://learn.microsoft.com/windows/desktop/winmsg/wm-size) message. The derived class handles the message.

## Parameters

### `wParam` [in]

Type: **WPARAM**

Additional information provided by the [WM_SIZE](https://learn.microsoft.com/windows/desktop/winmsg/wm-size) message.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
