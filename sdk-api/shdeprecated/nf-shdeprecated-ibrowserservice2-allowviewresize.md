# IBrowserService2::AllowViewResize

## Description

Deprecated. Informs the base class whether to allow view resizing.

## Parameters

### `f` [in]

Type: **BOOL**

A value of type **BOOL** that indicates whether to allow view resizing.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
