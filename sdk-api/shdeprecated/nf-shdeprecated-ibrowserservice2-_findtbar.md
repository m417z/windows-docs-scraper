# IBrowserService2::_FindTBar

## Description

Deprecated. Returns the index of a browser toolbar item based on Component Object Model (COM) identity rules.

## Parameters

### `punkSrc` [in]

Type: **[IUnknown](https://learn.microsoft.com/windows/desktop/api/unknwn/nn-unknwn-iunknown)***

A pointer to the [IUnknown](https://learn.microsoft.com/windows/desktop/api/unknwn/nn-unknwn-iunknown) of the browser toolbar item.

## Return value

Type: **UINT**

The index of the browser toolbar item.
