# IBrowserService2::_GetToolbarItem

## Description

Deprecated. Gets a specific item from a toolbar.

## Parameters

### `itb` [in]

Type: **int**

The index of the toolbar item to retrieve.

## Return value

Type: **LPTOOLBARITEM**

A pointer to a [TOOLBARITEM](https://learn.microsoft.com/windows/win32/api/shdeprecated/ns-shdeprecated-toolbaritem) structure that represents a toolbar item.

## Remarks

This method is implemented by the derived class.
