# ImmGetCompositionWindow function

## Description

Retrieves information about the composition window.

## Parameters

### `HIMC` [in]

Handle to the input context.

### `lpCompForm` [out]

Pointer to a [COMPOSITIONFORM](https://learn.microsoft.com/windows/desktop/api/imm/ns-imm-compositionform) structure in which the function retrieves information about the composition window.

## Return value

Returns a nonzero value if successful, or 0 otherwise.
