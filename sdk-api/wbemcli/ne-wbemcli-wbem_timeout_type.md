# WBEM_TIMEOUT_TYPE enumeration

## Description

Contains values used to specify the timeout for the [IEnumWbemClassObject::Next](https://learn.microsoft.com/windows/desktop/api/wbemcli/nf-wbemcli-ienumwbemclassobject-next) method.

## Constants

### `WBEM_NO_WAIT:0`

### `WBEM_INFINITE:0xffffffff`
