# WBEM_TEXT_FLAG_TYPE enumeration

## Description

Contains flags to control the execution of the [IWbemClassObject::GetObjectText](https://learn.microsoft.com/windows/desktop/api/wbemcli/nf-wbemcli-iwbemclassobject-getobjecttext) method.

## Constants

### `WBEM_FLAG_NO_FLAVORS:0x1`

Present qualifiers without propagation or flavor information.
