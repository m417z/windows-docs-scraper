# WBEM_CONNECT_OPTIONS enumeration

## Description

Contains flags for the [IWbemLocator::ConnectServer](https://learn.microsoft.com/windows/desktop/api/wbemcli/nf-wbemcli-iwbemlocator-connectserver) method.

## Constants

### `WBEM_FLAG_CONNECT_REPOSITORY_ONLY:0x40`

Reserved for internal use. Do not use.

### `WBEM_FLAG_CONNECT_USE_MAX_WAIT:0x80`

The call returns in 2 minutes or less whether successful or not.
