# CIMTYPE_ENUMERATION enumeration

## Description

The **CIMTYPE_ENUMERATION** enumeration defines values that specify different CIM data types. All the values in the enumeration are types defined by the Distributed Management Task Force (DMTF). For more information about the DMTF and the CIM data types, see [http://www.dmtf.org/standards/cim/cim_spec_v22](https://www.dmtf.org/standards/cim).

## Constants

### `CIM_ILLEGAL:0xfff`

An illegal value.

### `CIM_EMPTY:0`

An empty (null) value.

### `CIM_SINT8:16`

An 8-bit signed integer.

### `CIM_UINT8:17`

An 8-bit unsigned integer.

### `CIM_SINT16:2`

A 16-bit signed integer.

### `CIM_UINT16:18`

A 16-bit unsigned integer.

### `CIM_SINT32:3`

A 32-bit signed integer.

### `CIM_UINT32:19`

A 32-bit unsigned integer.

### `CIM_SINT64:20`

A 64-bit signed integer.

### `CIM_UINT64:21`

A 64-bit unsigned integer.

### `CIM_REAL32:4`

A 32-bit real number.

### `CIM_REAL64:5`

A 64-bit real number.

### `CIM_BOOLEAN:11`

A Boolean value.

### `CIM_STRING:8`

A string value.

### `CIM_DATETIME:101`

A DateTime value.

### `CIM_REFERENCE:102`

Reference (__Path) of another Object.

### `CIM_CHAR16:103`

A 16-bit character value.

### `CIM_OBJECT:13`

An Object value.

### `CIM_FLAG_ARRAY:0x2000`

An array value.
