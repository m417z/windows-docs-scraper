# IRDPSRAPIAttendee::get_Flags

## Description

Retrieves the attendee specific flags that are defined in the [RDPENCOMAPI_ATTENDEE_FLAGS](https://learn.microsoft.com/windows/win32/api/rdpencomapi/ne-rdpencomapi-rdpencomapi_attendee_flags) enumeration type.

This property is read-only.
