# IRDPSRAPISharingSession::Pause

## Description

Pauses the graphics stream that is sent to all viewers from the sharer until [IRDPSRAPISharingSession::Resume](https://learn.microsoft.com/windows/desktop/api/rdpencomapi/nf-rdpencomapi-irdpsrapisharingsession-resume) is called.

## Return value

If the method succeeds, the return value is **S_OK**. Otherwise, the return value is an error code.
