# IRDPSRAPISharingSession::Resume

## Description

Causes the graphics stream that is sent to all viewers from the sharer to resume until either [IRDPSRAPISharingSession::Pause](https://learn.microsoft.com/windows/desktop/api/rdpencomapi/nf-rdpencomapi-irdpsrapisharingsession-pause) or [IRDPSRAPISharingSession::Close](https://learn.microsoft.com/windows/desktop/api/rdpencomapi/nf-rdpencomapi-irdpsrapisharingsession-close) is called.

## Return value

If the method succeeds, the return value is **S_OK**. Otherwise, the return value is an error code.
