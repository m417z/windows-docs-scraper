# IRDPSRAPISharingSession::get_Attendees

## Description

Returns an object implementing the [IRDPSRAPIAttendeeManager](https://learn.microsoft.com/windows/desktop/api/rdpencomapi/nn-rdpencomapi-irdpsrapiattendeemanager) interface.
