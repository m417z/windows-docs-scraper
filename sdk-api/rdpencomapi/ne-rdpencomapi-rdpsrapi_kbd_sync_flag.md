# RDPSRAPI_KBD_SYNC_FLAG enumeration

## Description

Defines values for the type of keys that control the state of a keyboard, such as the Caps Lock
key.

## Constants

### `RDPSRAPI_KBD_SYNC_FLAG_SCROLL_LOCK:1`

### `RDPSRAPI_KBD_SYNC_FLAG_NUM_LOCK:2`

### `RDPSRAPI_KBD_SYNC_FLAG_CAPS_LOCK:4`

### `RDPSRAPI_KBD_SYNC_FLAG_KANA_LOCK:8`
