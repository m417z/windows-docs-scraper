# IRDPSRAPIAudioStream::FreeBuffer

## Description

Releases the hold on the buffer after the [GetBuffer](https://learn.microsoft.com/windows/desktop/api/rdpencomapi/nf-rdpencomapi-irdpsrapiaudiostream-getbuffer) method is called.

## Return value

If the method succeeds, the return value is **S_OK**. Otherwise, the return value is an error code.
