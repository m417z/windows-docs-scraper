# IRDPSRAPIWindow::put_Shared

## Description

Gets or sets the sharing property for a window.

Whether a window is shared or not also depends on the state of the parent application object and on the enabled state of the sharing filter. For more information about the enabled state of the sharing filter, see [Enabled Property of IRDPSRAPIApplicationFilter](https://learn.microsoft.com/windows/desktop/api/rdpencomapi/nf-rdpencomapi-irdpsrapiapplicationfilter-get_enabled).

This property is read/write.
