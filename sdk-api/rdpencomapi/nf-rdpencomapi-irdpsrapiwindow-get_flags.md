# IRDPSRAPIWindow::get_Flags

## Description

Returns the flags on the current window. Flags are one of the values of the enumeration type [RDPSRAPI_WND_FLAGS](https://learn.microsoft.com/windows/win32/api/rdpencomapi/ne-rdpencomapi-rdpsrapi_wnd_flags).

This property is read-only.
