# IRDPSRAPISharingSession::get_VirtualChannelManager

## Description

Retrieves an object implementing the [IRDPSRAPIVirtualChannelManager](https://learn.microsoft.com/windows/desktop/api/rdpencomapi/nn-rdpencomapi-irdpsrapivirtualchannelmanager) interface.

This property is read-only.
