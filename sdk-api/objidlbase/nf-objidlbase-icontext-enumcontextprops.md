# IContext::EnumContextProps

## Description

Returns an [IEnumContextProps](https://learn.microsoft.com/windows/desktop/api/objidl/nn-objidl-ienumcontextprops) interface pointer that can be used to enumerate the context properties in this context.

## Parameters

### `ppEnumContextProps` [out]

The address of the variable that receives the new [IEnumContextProps](https://learn.microsoft.com/windows/desktop/api/objidl/nn-objidl-ienumcontextprops) interface pointer.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
