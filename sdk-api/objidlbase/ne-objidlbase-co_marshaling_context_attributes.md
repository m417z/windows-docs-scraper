# CO_MARSHALING_CONTEXT_ATTRIBUTES enumeration

## Description

Identifies one of the marshaling context attributes that you can query by using the [GetMarshalingContextAttribute](https://learn.microsoft.com/windows/desktop/api/objidl/nf-objidl-imarshalingstream-getmarshalingcontextattribute) method.

## Constants

### `CO_MARSHALING_SOURCE_IS_APP_CONTAINER:0`

The source of the marshaling data is an app container process.
