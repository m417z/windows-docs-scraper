# IRpcStubBuffer::DebugServerRelease

## Description

Releases an interface pointer that was previously returned by [DebugServerQueryInterface](https://learn.microsoft.com/windows/desktop/api/objidl/nf-objidl-irpcstubbuffer-debugserverqueryinterface).

## Parameters

### `pv` [in]

A pointer to the interface that the caller no longer needs.
