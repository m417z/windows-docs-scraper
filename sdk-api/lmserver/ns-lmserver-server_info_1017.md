# SERVER_INFO_1017 structure

## Description

The
**SERVER_INFO_1017** structure contains the network announce rate associated with the specified server.

## Members

### `sv1017_announce`

Specifies the network announce rate, in seconds. This rate determines how often the server is announced to other computers on the network.

For more information about how much the announce rate can vary from the period of time specified by this member, see
[SERVER_INFO_1018](https://learn.microsoft.com/windows/desktop/api/lmserver/ns-lmserver-server_info_1018).
