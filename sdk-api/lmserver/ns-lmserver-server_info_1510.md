# SERVER_INFO_1510 structure

## Description

The
**SERVER_INFO_1510** structure specifies the maximum number of users that can be logged on to the specified server using a single virtual circuit.

## Members

### `sv1510_sessusers`

Specifies the maximum number of users that can be logged on to a server using a single virtual circuit.
