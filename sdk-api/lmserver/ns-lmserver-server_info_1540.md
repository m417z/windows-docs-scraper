# SERVER_INFO_1540 structure

## Description

The
**SERVER_INFO_1540** structure specifies whether the server allows redirected server drives to be shared.

## Members

### `sv1540_enablesharednetdrives`

Specifies whether the server allows redirected server drives to be shared.
