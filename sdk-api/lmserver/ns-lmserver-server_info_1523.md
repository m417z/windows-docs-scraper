# SERVER_INFO_1523 structure

## Description

The
**SERVER_INFO_1523** structure specifies the length of time the server retains information about incomplete search operations.

## Members

### `sv1523_maxkeepsearch`

Specifies the length of time the server retains information about incomplete search operations.
