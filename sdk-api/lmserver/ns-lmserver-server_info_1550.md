# SERVER_INFO_1550 structure

## Description

The
**SERVER_INFO_1550** structure specifies the percentage of free disk space remaining before an alert message is sent.

## Members

### `sv1550_diskspacethreshold`

Specifies the percentage of free disk space remaining before an alert message is sent.
