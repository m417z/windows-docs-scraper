# SERVER_INFO_1502 structure

## Description

The
**SERVER_INFO_1502** structure specifies the maximum number of virtual circuits per client for the specified server.

## Members

### `sv1502_sessvcs`

Specifies the maximum number of virtual circuits permitted per client.
