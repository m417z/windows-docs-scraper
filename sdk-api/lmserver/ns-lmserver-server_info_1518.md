# SERVER_INFO_1518 structure

## Description

The
**SERVER_INFO_1518** structure specifies whether the server is visible to LAN Manager 2.*x* clients.

## Members

### `sv1518_lmannounce`

Specifies whether the server is visible to LAN Manager 2.*x* clients.
