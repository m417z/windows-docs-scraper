# TfLBIClick enumeration

## Description

Elements of the **TfLBIClick** enumeration specify which mouse button was used to click a toolbar item.

## Constants

### `TF_LBI_CLK_RIGHT:1`

The user right-clicked the button.

### `TF_LBI_CLK_LEFT:2`

The user left-clicked the button.
