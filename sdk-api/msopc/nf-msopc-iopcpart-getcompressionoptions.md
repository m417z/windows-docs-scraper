# IOpcPart::GetCompressionOptions

## Description

Gets a value that describes the way part content is compressed.

## Parameters

### `compressionOptions` [out, retval]

A value that describes the way part content is compressed.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
| **E_POINTER** | The *compressionOptions* parameter is **NULL**. |

## Remarks

For more information about parts, see the [Open Packaging Conventions Fundamentals](https://learn.microsoft.com/previous-versions/windows/desktop/opc/open-packaging-conventions-overview) and the *ECMA-376 OpenXML, 1st Edition, Part 2: Open Packaging Conventions (OPC)*.
