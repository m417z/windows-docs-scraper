# IOpcSigningOptions::SetDefaultDigestMethod

## Description

Sets the default digest method that will be used to compute digest values for objects to be signed.

## Parameters

### `digestMethod` [in]

The default digest method.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
| **E_POINTER** | The *digestMethod* parameter is **NULL**. |

## Remarks

To access the default digest method before the signature is generated, call the [IOpcSigningOptions::GetDefaultDigestMethod](https://learn.microsoft.com/previous-versions/windows/desktop/api/msopc/nf-msopc-iopcsigningoptions-getdefaultdigestmethod).

**Important** The default digest method for entities to be signed must be set before the signature is generated by calling the [IOpcDigitalSignatureManager::Sign](https://learn.microsoft.com/previous-versions/windows/desktop/api/msopc/nf-msopc-iopcdigitalsignaturemanager-sign) method.

#### Thread Safety

Packaging objects are not thread-safe.

For more information, see the [Getting Started with the Packaging API](https://learn.microsoft.com/previous-versions/windows/desktop/opc/packaging-api-overview).
