# OPC_RELATIONSHIP_SELECTOR enumeration

## Description

Describes how to interpret the *selectionCriterion* parameter of the [IOpcRelationshipSelector::GetSelectionCriterion](https://learn.microsoft.com/previous-versions/windows/desktop/api/msopc/nf-msopc-iopcrelationshipselector-getselectioncriterion) method.

## Constants

### `OPC_RELATIONSHIP_SELECT_BY_ID:0`

The *selectionCriterion* parameter is a relationship identifier.

### `OPC_RELATIONSHIP_SELECT_BY_TYPE:1`

The *selectionCriterion* parameter is a relationship type.
