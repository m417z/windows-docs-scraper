# IPenInputPanel::put_AttachedEditWindow

## Description

Deprecated. The [PenInputPanel](https://learn.microsoft.com/windows/desktop/tablet/peninputpanel-class) has been replaced by the [Text Input Panel (TIP)](https://learn.microsoft.com/windows/desktop/tablet/text-input-panel-reference).

Sets or gets the window handle of the object that the [PenInputPanel](https://learn.microsoft.com/windows/desktop/tablet/peninputpanel-class) object is attached to.

This property is read/write.

## Parameters

## Remarks

The window handle of an object may change.
