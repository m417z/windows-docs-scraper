# RESV_STYLE structure

## Description

The
**RESV_STYLE** structure contains information about RSVP RESV style.

## Members

### `style_header`

RSVP object header, in the form of an [RsvpObjHdr](https://learn.microsoft.com/previous-versions/windows/desktop/api/lpmapi/ns-lpmapi-rsvpobjhdr) structure.

### `style_word`

RSVP RESV style. Must be one of the following values:

#### STYLE_FF

#### STYLE_SE

#### STYLE_WF
