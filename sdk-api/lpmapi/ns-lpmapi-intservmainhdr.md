# IntServMainHdr structure

## Description

The
**IntServMainHdr** structure is a header for Integrated Services RSVP objects.

## Members

### `ismh_version`

Version of this header.

### `ismh_unused`

Reserved. Do not use.

### `ismh_len32b`

Number of 32-bit WORDs in the object, excluding this header object.
