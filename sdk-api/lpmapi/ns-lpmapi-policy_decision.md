# POLICY_DECISION structure

## Description

The
**POLICY_DECISION** structure contains RSVP policy decision information.

## Members

### `lpvResult`

Local policy value.

### `wPolicyErrCode`

RSVP-defined error code.

### `wPolicyErrValue`

RSVP-defined error value.
