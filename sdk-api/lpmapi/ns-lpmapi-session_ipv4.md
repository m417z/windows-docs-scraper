# Session_IPv4 structure

## Description

The
**Session_IPv4** structure stores information about an IPv4 RSVP SESSION object.

## Members

### `sess_destaddr`

Destination IP address for the session.

### `sess_protid`

Protocol ID for the session.

### `sess_flags`

Session flags.

### `sess_destport`

Destination IP address for the session.
