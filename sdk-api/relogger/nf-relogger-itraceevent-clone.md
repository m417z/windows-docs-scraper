# ITraceEvent::Clone

## Description

The **Clone** method creates a duplicate copy of an event.

## Parameters

### `NewEvent` [out, retval]

Type: **IEvent****

The new event.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
