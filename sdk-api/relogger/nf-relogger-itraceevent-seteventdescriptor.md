# ITraceEvent::SetEventDescriptor

## Description

The **SetEventDescriptor** method sets the event descriptor for an event.

## Parameters

### `EventDescriptor` [in]

Type: **[PCEVENT_DESCRIPTOR](https://learn.microsoft.com/windows/desktop/api/evntprov/ns-evntprov-event_descriptor)**

The event descriptor data.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
