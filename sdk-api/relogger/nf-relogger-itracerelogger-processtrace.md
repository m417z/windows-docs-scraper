# ITraceRelogger::ProcessTrace

## Description

The **ProcessTrace** method delivers events from the associated trace streams to the consumer.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
