# SERVICE_PREFERRED_NODE_INFO structure

## Description

Represents the preferred node on which to run a service.

## Members

### `usPreferredNode`

The node number of the preferred node.

### `fDelete`

If this member is TRUE, the preferred node setting is deleted.
