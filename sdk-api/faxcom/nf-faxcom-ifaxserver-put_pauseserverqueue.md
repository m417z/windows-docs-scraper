# IFaxServer::put_PauseServerQueue

## Description

Sets or retrieves the **PauseServerQueue** property for a [FaxServer](https://learn.microsoft.com/previous-versions/windows/desktop/fax/-mfax-faxserver-client) object. The **PauseServerQueue** property is a Boolean value that indicates whether the fax server has paused the fax job queue.

This property is read/write.

## Parameters

## Remarks

An administrator might pause the fax job queue for various reasons. For example, the administrator may need to perform queue management, or free a fax device for an inbound reception or for another application's use.
