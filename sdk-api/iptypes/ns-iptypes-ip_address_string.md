# IP_ADDRESS_STRING structure

## Description

The **IP_ADDRESS_STRING** structure stores an IPv4 address in dotted decimal notation. The **IP_ADDRESS_STRING** structure definition is also the type definition for the **IP_MASK_STRING** structure.

## Members

### `String`

A character string that represents an IPv4 address or an IPv4 subnet mask in dotted decimal notation.

## Remarks

The **IP_ADDRESS_STRING** structure is used as a parameter in the [IP_ADDR_STRING](https://learn.microsoft.com/windows/desktop/api/iptypes/ns-iptypes-ip_addr_string) structure.
