# eAVDecDDStereoDownMixMode enumeration

## Description

Specifies the stereo downmix mode for a Dolby Digital audio decoder. This enumeration is used with the [CODECAPI_AVDecDDStereoDownMixMode](https://learn.microsoft.com/windows/desktop/medfound/codecapi-avdecddstereodownmixmode) property.

## Constants

### `eAVDecDDStereoDownMixMode_Auto:0`

The decoder selects the mode automatically.

### `eAVDecDDStereoDownMixMode_LtRt:1`

Left total/right total (Lt/Rt) downmix. (Surround compatible.)

### `eAVDecDDStereoDownMixMode_LoRo:2`

Left only/right only (Lo/Ro) downmix. (Stereo.)
