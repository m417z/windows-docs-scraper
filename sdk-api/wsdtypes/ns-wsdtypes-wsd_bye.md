# WSD_BYE structure

## Description

Represents a [Bye](https://learn.microsoft.com/windows/desktop/WsdApi/bye-message) message.

## Members

### `EndpointReference`

Reference to a [WSD_ENDPOINT_REFERENCE](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_endpoint_reference) structure that specifies either the sending or receiving endpoint of the Bye message.

### `Any`

Reference to a [WSDXML_ELEMENT](https://learn.microsoft.com/windows/desktop/api/wsdxmldom/ns-wsdxmldom-wsdxml_element) structure that specifies content allowed by the XML **ANY** keyword.
