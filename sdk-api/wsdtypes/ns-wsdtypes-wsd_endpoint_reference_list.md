# WSD_ENDPOINT_REFERENCE_LIST structure

## Description

Represents a node in a single-linked list of [WSD_ENDPOINT_REFERENCE](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_endpoint_reference) structures.

## Members

### `Next`

Reference to the next node in the linked list of **WSD_ENDPOINT_REFERENCE_LIST** structures.

### `Element`

Reference to a [WSD_ENDPOINT_REFERENCE](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_endpoint_reference) structure that contains the endpoint referenced by this node.
