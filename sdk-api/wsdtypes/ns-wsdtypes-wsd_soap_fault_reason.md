# WSD_SOAP_FAULT_REASON structure

## Description

A collection of reason codes associated with a [WSD_SOAP_FAULT](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_soap_fault). A reason code is a human readable explanation of the fault.

## Members

### `Text`

A [WSD_LOCALIZED_STRING_LIST](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_localized_string_list) structure that contains a collection of localized reason codes.
