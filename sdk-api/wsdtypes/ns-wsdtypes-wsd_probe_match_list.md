# WSD_PROBE_MATCH_LIST structure

## Description

Represents a node in a single-linked list of ProbeMatch message structures.

## Members

### `Next`

Reference to the next node in the linked list of **WSD_PROBE_MATCH_LIST** structures.

### `Element`

Reference to a [WSD_PROBE_MATCH](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_probe_match) structure that contains the ProbeMatch message referenced by this node.
