# WSD_EVENTING_FILTER_ACTION structure

## Description

Represents a boolean expression used for filtering events

## Members

### `Actions`

Reference to a [WSD_URI_LIST](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_uri_list) structure that specifies the URIs used for filtering notifications.

## Remarks

For more information about the evaluation of action filters, see Section 6.1.1, Filtering, in the [Device Profile for Web Services](https://specs.xmlsoap.org/ws/2006/02/devprof/devicesprofile.pdf) specification.
