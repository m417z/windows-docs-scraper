# WSD_RESOLVE_MATCHES structure

## Description

Represents a [ResolveMatches](https://learn.microsoft.com/windows/desktop/WsdApi/resolve-message) message.

## Members

### `ResolveMatch`

Reference to a [WSD_RESOLVE_MATCH](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_resolve_match) structure that contains a child ResolveMatch message.

### `Any`

Reference to a [WSDXML_ELEMENT](https://learn.microsoft.com/windows/desktop/api/wsdxmldom/ns-wsdxmldom-wsdxml_element) structure that specifies extension content allowed by the XML **ANY** keyword.
