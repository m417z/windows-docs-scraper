# WSD_EVENTING_DELIVERY_MODE_PUSH structure

## Description

Represents the endpoint reference used for push delivery of events in a WS-Eventing Subscribe message.

## Members

### `NotifyTo`

Reference to a [WSD_ENDPOINT_REFERENCE](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_endpoint_reference) structure that specifies the endpoint reference to which notifications should be sent.
