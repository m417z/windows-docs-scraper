# RESPONSEBODY_GetMetadata structure

## Description

Represents a WS-MetadataExchange GetMetadata response message.

## Members

### `Metadata`

Reference to a [WSD_METADATA_SECTION_LIST](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_metadata_section_list) structure that contains a node in a single-linked list of metadata sections.
