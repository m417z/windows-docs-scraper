# WSD_HEADER_RELATESTO structure

## Description

Represents a RelatesTo SOAP envelope header block, as specified by the WS-Addressing specification.

## Members

### `RelationshipType`

Reference to a [WSDXML_NAME](https://learn.microsoft.com/windows/desktop/api/wsdxmldom/ns-wsdxmldom-wsdxml_name) structure that contains the relationship type as a qualified name.

### `MessageID`

The identifier of the related message.
