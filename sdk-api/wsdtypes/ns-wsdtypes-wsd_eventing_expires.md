# WSD_EVENTING_EXPIRES structure

## Description

Represents the expiration time of a WS-Eventing message.

## Members

### `Duration`

Reference to a [WSD_DURATION](https://learn.microsoft.com/windows/desktop/api/wsdxml/ns-wsdxml-wsd_duration) structure that specifies the length of time a request or response is valid.

### `DateTime`

Reference to a [WSD_DATETIME](https://learn.microsoft.com/windows/desktop/api/wsdxml/ns-wsdxml-wsd_datetime) structure that specifies the time that the request or response expires.
