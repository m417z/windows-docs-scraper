# WSD_SCOPES structure

## Description

A collection of scopes used in WS-Discovery messaging.

## Members

### `MatchBy`

A matching rule used for scopes.

### `Scopes`

Reference to a [WSD_URI_LIST](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_uri_list) structure that contains a list of scopes.
