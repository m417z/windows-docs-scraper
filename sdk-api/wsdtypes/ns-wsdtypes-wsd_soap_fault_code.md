# WSD_SOAP_FAULT_CODE structure

## Description

Represents a generated SOAP fault code.

## Members

### `Value`

A [WSDXML_NAME](https://learn.microsoft.com/windows/desktop/api/wsdxmldom/ns-wsdxmldom-wsdxml_name) structure that contains the qualified name of the SOAP fault code.

### `Subcode`

A [WSD_SOAP_FAULT_SUBCODE](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_soap_fault_subcode) structure that contains the fault subcode.
