# WSD_SERVICE_METADATA_LIST structure

## Description

Represents a node in a single-linked list of service metadata structures.

## Members

### `Next`

Reference to the next node in the linked list of **WSD_SERVICE_METADATA_LIST** structures.

### `Element`

Reference to a [WSD_SERVICE_METADATA](https://learn.microsoft.com/windows/desktop/api/wsdtypes/ns-wsdtypes-wsd_service_metadata) structure that represents the service metadata referenced by this node.
