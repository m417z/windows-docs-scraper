# RegisterHandwritingInputRoutingCallback function

## Description

Registers an [IHandwritingInputRoutingCallback](https://learn.microsoft.com/windows/win32/api/shellhandwriting/nn-shellhandwriting-ihandwritinginputroutingcallback).

## Parameters

### `callback` [in]

The [IHandwritingInputRoutingCallback](https://learn.microsoft.com/windows/win32/api/shellhandwriting/nn-shellhandwriting-ihandwritinginputroutingcallback) being registered.

## Return value

If the method succeeds, it returns **S_OK**. If it fails, it returns an **HRESULT** error code.
