# ID3D10ShaderReflection1::GetGSInputPrimitive

## Description

Gets the geometry-shader input-primitive description.

## Parameters

### `pPrim` [in]

Type: **[D3D10_PRIMITIVE](https://learn.microsoft.com/windows/desktop/api/d3dcommon/ne-d3dcommon-d3d_primitive)***

A pointer to the input-primitive type (see [D3D10_PRIMITIVE](https://learn.microsoft.com/windows/desktop/api/d3dcommon/ne-d3dcommon-d3d_primitive)).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the following [Direct3D 10 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues).

## Remarks

This method requires Windows Vista Service Pack 1.
