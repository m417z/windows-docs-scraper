# IGPMBackup::Delete

## Description

Removes the Group Policy object (GPO) backup from the backup directory and from the file system.

## Return value

### JScript

Returns **S_OK** if successful. Returns a failure code if an error occurs.

### VB

Returns **S_OK** if successful. Returns a failure code if an error occurs.
