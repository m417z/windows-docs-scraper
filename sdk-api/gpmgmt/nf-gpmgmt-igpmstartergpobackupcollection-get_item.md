# IGPMStarterGPOBackupCollection::get_Item

## Description

[GPMStarterGPOBackup](https://learn.microsoft.com/previous-versions/windows/desktop/api/gpmgmt/nn-gpmgmt-igpmstartergpobackup) object from the collection.
