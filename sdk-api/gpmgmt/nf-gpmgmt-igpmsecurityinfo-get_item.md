# IGPMSecurityInfo::get_Item

## Description

Given an index, returns a
[GPMPermission](https://learn.microsoft.com/previous-versions/windows/desktop/api/gpmgmt/nn-gpmgmt-igpmpermission) object from the collection.

This property is read-only.
