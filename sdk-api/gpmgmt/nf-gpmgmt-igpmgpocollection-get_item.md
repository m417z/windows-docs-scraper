# IGPMGPOCollection::get_Item

## Description

Given an index, returns a pointer to an [GPMGPO](https://learn.microsoft.com/previous-versions/windows/desktop/api/gpmgmt/nn-gpmgmt-igpmgpo) object from the collection.

This property is read-only.
