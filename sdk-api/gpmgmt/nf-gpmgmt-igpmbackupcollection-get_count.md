# IGPMBackupCollection::get_Count

## Description

Returns the number of
[GPMBackup](https://learn.microsoft.com/previous-versions/windows/desktop/api/gpmgmt/nn-gpmgmt-igpmbackup) objects in the collection.

This property is read-only.
