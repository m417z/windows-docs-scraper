# AdjustableArrowCap::SetWidth

## Description

The **AdjustableArrowCap::SetWidth** method sets the width of the arrow cap. The width is the distance between the endpoints of the base of the arrow.

## Parameters

### `width` [in]

Type: **REAL**

Real number that specifies the distance, in units, between the two corners of the base of the arrow.

## Return value

Type: **[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status)**

If the method succeeds, it returns Ok, which is an element of the
**Status** enumeration.

If the method fails, it returns one of the other elements of the
**Status** enumeration.
