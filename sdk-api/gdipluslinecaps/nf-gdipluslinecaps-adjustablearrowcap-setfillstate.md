# AdjustableArrowCap::SetFillState

## Description

The **AdjustableArrowCap::SetFillState** method sets the fill state of the arrow cap. If the arrow cap is not filled, only the outline is drawn.

## Parameters

### `isFilled` [in]

Type: **BOOL**

Boolean value that specifies whether the arrow cap is filled.

## Return value

Type: **[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status)**

If the method succeeds, it returns Ok, which is an element of the
**Status** enumeration.

If the method fails, it returns one of the other elements of the
**Status** enumeration.
