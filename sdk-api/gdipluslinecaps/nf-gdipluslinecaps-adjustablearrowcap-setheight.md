# AdjustableArrowCap::SetHeight

## Description

The **AdjustableArrowCap::SetHeight** method sets the height of the arrow cap. This is the distance from the base of the arrow to its vertex.

## Parameters

### `height` [in]

Type: **REAL**

Real number that specifies the height, in units, for the arrow cap.

## Return value

Type: **[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status)**

If the method succeeds, it returns Ok, which is an element of the
**Status** enumeration.

If the method fails, it returns one of the other elements of the
**Status** enumeration.
