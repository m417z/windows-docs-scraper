# __unnamed_enum_2 enumeration

## Description

This enumeration is used by the [WdsCliLog](https://learn.microsoft.com/windows/desktop/api/wdsclientapi/nf-wdsclientapi-wdsclilog) function.

## Constants

### `WDS_LOG_LEVEL_DISABLED:0`

The event severity level is disabled.

### `WDS_LOG_LEVEL_ERROR:1`

The event severity is error level.

### `WDS_LOG_LEVEL_WARNING:2`

The event severity is warning level.

### `WDS_LOG_LEVEL_INFO:3`

The event severity is information level.
