# IExeServerRegistration::get_IdentityType

## Description

Gets the activation type for the out-of-process server.

## Parameters

### `identityType` [out, retval]

The activation type.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
