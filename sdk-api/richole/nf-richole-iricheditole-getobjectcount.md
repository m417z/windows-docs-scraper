# IRichEditOle::GetObjectCount

## Description

Returns the number of objects currently contained in a rich edit control.

## Return value

Type: **[LONG](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

This method returns the number of objects.
