# PWM_PIN_SET_POLARITY_INPUT structure

## Description

Contains a desired value for polarity of a pin or channel.

## Members

### `Polarity`

The desired polarity for the pin or channel as a [PWM_POLARITY](https://learn.microsoft.com/windows/desktop/api/pwm/ne-pwm-pwm_polarity) value. The polarity is either Active High or Active Low.
