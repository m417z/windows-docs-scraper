# DrawDibStop function

## Description

The **DrawDibStop** function frees the resources used by a DrawDib DC for streaming playback.

## Parameters

### `hdd`

Handle to a DrawDib DC.

## Return value

Returns **TRUE** if successful or **FALSE** otherwise.
