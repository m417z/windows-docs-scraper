# AVIFileAddRef function

## Description

The **AVIFileAddRef** function increments the reference count of an AVI file.

## Parameters

### `pfile`

Handle to an open AVI file.

## Return value

Returns the updated reference count for the file interface.

## Remarks

The argument *pfile* is a pointer to an [IAVIFile](https://learn.microsoft.com/windows/desktop/api/vfw/nn-vfw-iavifile) interface.
