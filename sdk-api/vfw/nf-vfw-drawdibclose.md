# DrawDibClose function

## Description

The **DrawDibClose** function closes a DrawDib DC and frees the resources DrawDib allocated for it.

## Parameters

### `hdd`

Handle to a DrawDib DC.

## Return value

Returns **TRUE** if successful or **FALSE** otherwise.
