# AVIStreamStart function

## Description

The **AVIStreamStart** function returns the starting sample number for the stream.

## Parameters

### `pavi`

Handle to an open stream.

## Return value

Returns the number if successful or -1 otherwise.

## Remarks

The argument *pavi* is a pointer to an [IAVIStream](https://learn.microsoft.com/windows/desktop/api/vfw/nn-vfw-iavistream) interface.
