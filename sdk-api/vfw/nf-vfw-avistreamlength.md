# AVIStreamLength function

## Description

The **AVIStreamLength** function returns the length of the stream.

## Parameters

### `pavi`

Handle to an open stream.

## Return value

Returns the stream's length, in samples, if successful or -1 otherwise.

The argument *pavi* is a pointer to an [IAVIStream](https://learn.microsoft.com/windows/desktop/api/vfw/nn-vfw-iavistream) interface.
