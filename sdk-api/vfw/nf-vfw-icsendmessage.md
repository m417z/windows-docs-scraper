# ICSendMessage function

## Description

The **ICSendMessage** function sends a message to a compressor.

## Parameters

### `hic`

Handle to the compressor to receive the message.

### `msg`

Message to send.

### `dw1`

Additional message-specific information.

### `dw2`

Additional message-specific information.

## Return value

Returns a message-specific result.
