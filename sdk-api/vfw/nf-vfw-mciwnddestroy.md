# MCIWndDestroy macro

## Description

The **MCIWndDestroy** macro closes an MCI device or file associated with an MCIWnd window and destroys the window. You can use this macro or explicitly send the [WM_CLOSE](https://learn.microsoft.com/windows/win32/winmsg/wm-close) message.

## Parameters

### `hwnd`

Handle of the MCIWnd window.
