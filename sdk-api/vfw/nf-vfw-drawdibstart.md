# DrawDibStart function

## Description

The **DrawDibStart** function prepares a DrawDib DC for streaming playback.

## Parameters

### `hdd`

Handle to a DrawDib DC.

### `rate`

Playback rate, in microseconds per frame.

## Return value

Returns **TRUE** if successful or **FALSE** otherwise.
