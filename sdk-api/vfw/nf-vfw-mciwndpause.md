# MCIWndPause macro

## Syntax

```cpp
LONG MCIWndPause(
     hwnd
);
```

## Return value

Type: **LONG**

Returns zero if successful or an error otherwise.

## Description

The **MCIWndPause** macro sends a command to an MCI device to pause playing or recording.

## Parameters

### `hwnd`

Handle of the MCIWnd window.
