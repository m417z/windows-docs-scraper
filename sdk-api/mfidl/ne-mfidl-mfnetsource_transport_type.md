# MFNETSOURCE_TRANSPORT_TYPE enumeration

## Description

Describes the type of transport used in streaming or downloading data (TCP or UDP).

## Constants

### `MFNETSOURCE_UDP:0`

The data transport type is UDP.

### `MFNETSOURCE_TCP`

The data transport type is TCP.
