# MF_BYTE_STREAM_CACHE_RANGE structure

## Description

Specifies a range of bytes.

## Members

### `qwStartOffset`

The offset, in bytes, of the start of the range.

### `qwEndOffset`

The offset, in bytes, of the end of the range.
