# IMFQualityAdvise::GetDropMode

## Description

Retrieves the current drop mode.

## Parameters

### `peDropMode` [out]

Receives the drop mode, specified as a member of the [MF_QUALITY_DROP_MODE](https://learn.microsoft.com/windows/desktop/api/mfidl/ne-mfidl-mf_quality_drop_mode) enumeration.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
