# IMFSensorProfile::AddBlockedControl

## Description

 Adds the specified blocked control .

## Parameters

### `wzBlockedControl` [in]

The blocked control to add.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
