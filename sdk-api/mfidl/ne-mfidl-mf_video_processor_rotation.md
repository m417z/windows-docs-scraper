# MF_VIDEO_PROCESSOR_ROTATION enumeration

## Description

Specifies how to rotate a video image.

## Constants

### `ROTATION_NONE:0`

Do not rotate the image.

### `ROTATION_NORMAL:1`

Rotate the image to the correct viewing orientation.
