# MF_VIDEO_PROCESSOR_MIRROR enumeration

## Description

Specifies how to flip a video image.

## Constants

### `MIRROR_NONE:0`

Do not flip the image.

### `MIRROR_HORIZONTAL:1`

Flip the image horizontally.

### `MIRROR_VERTICAL:2`

Flip the image vertically.
