# IMFClock::GetProperties

## Description

Retrieves the properties of the clock.

## Parameters

### `pClockProperties` [out]

Pointer to an [MFCLOCK_PROPERTIES](https://learn.microsoft.com/windows/desktop/api/mfidl/ns-mfidl-mfclock_properties) structure that receives the properties.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
