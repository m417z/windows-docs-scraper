# IMFTopologyNode::GetNodeType

## Description

Retrieves the node type.

## Parameters

### `pType` [out]

Receives the node type, specified as a member of the [MF_TOPOLOGY_TYPE](https://learn.microsoft.com/windows/desktop/api/mfidl/ne-mfidl-mf_topology_type) enumeration.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
