# IMFOutputTrustAuthority::GetAction

## Description

Retrieves the action that is performed by this output trust authority (OTA).

## Parameters

### `pAction` [out]

Receives a member of the [MFPOLICYMANAGER_ACTION](https://learn.microsoft.com/windows/desktop/api/mfidl/ne-mfidl-mfpolicymanager_action) enumeration.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
