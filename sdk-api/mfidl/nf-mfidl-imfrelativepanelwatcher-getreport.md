## Description

Gets an [IMFRelativePanelReport](https://learn.microsoft.com/windows/win32/api/mfidl/nn-mfidl-imfrelativepanelreport) interface that represents the relative panel location.

## Parameters

### `ppRelativePanelReport`

A pointer to he [IMFRelativePanelReport](https://learn.microsoft.com/windows/win32/api/mfidl/nn-mfidl-imfrelativepanelreport) interface that represents the relative panel location.

## Return value

The function returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
|--------------|------------------------|
|S_OK | The function succeeded.|
