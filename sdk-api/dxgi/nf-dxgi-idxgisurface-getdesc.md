# IDXGISurface::GetDesc

## Description

Get a description of the surface.

## Parameters

### `pDesc` [out]

Type: **[DXGI_SURFACE_DESC](https://learn.microsoft.com/windows/desktop/api/dxgi/ns-dxgi-dxgi_surface_desc)***

A pointer to the surface description (see [DXGI_SURFACE_DESC](https://learn.microsoft.com/windows/desktop/api/dxgi/ns-dxgi-dxgi_surface_desc)).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns S_OK if successful; otherwise, returns one of the error codes that are described in the [DXGI_ERROR](https://learn.microsoft.com/windows/desktop/direct3ddxgi/dxgi-error) topic.
