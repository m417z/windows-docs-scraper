# IDXGIObject::GetParent

## Description

Gets the parent of the object.

## Parameters

### `riid` [in]

Type: **REFIID**

The ID of the requested interface.

### `ppParent` [out]

Type: **void****

The address of a pointer to the parent object.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the [DXGI_ERROR](https://learn.microsoft.com/windows/desktop/direct3ddxgi/dxgi-error) values.
