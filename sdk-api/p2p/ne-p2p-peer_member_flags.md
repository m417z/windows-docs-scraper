# PEER_MEMBER_FLAGS enumeration

## Description

The **PEER_MEMBER_FLAGS** flag allows an application to specify whether all members or only present ones should be enumerated when the [PeerGroupEnumMembers](https://learn.microsoft.com/windows/desktop/api/p2p/nf-p2p-peergroupenummembers) function is called, or to indicate whether or not a member is present within the peer group.

## Constants

### `PEER_MEMBER_PRESENT:0x0001`

The member is present in the peer group.
