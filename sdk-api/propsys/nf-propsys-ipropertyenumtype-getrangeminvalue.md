# IPropertyEnumType::GetRangeMinValue

## Description

Gets a minimum value from an enumeration information structure.

## Parameters

### `ppropvarMin` [out]

Type: **[PROPVARIANT](https://learn.microsoft.com/windows/desktop/api/propidl/ns-propidl-propvariant)***

When this method returns, contains a pointer to the minimum value.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.

## Remarks

For additional information, see [enumeratedList](https://learn.microsoft.com/windows/desktop/properties/propdesc-schema-enumeratedlist).
