# FWP_IP_VERSION enumeration

## Description

The **FWP_IP_VERSION** enumerated type specifies the IP version.

## Constants

### `FWP_IP_VERSION_V4:0`

Specifies IPv4.

### `FWP_IP_VERSION_V6`

Specifies IPv6.

### `FWP_IP_VERSION_NONE`

Reserved.

### `FWP_IP_VERSION_MAX`

Maximum value for testing purposes.
