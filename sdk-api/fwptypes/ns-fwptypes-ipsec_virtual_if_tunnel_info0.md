# IPSEC_VIRTUAL_IF_TUNNEL_INFO0 structure

## Description

The **IPSEC_VIRTUAL_IF_TUNNEL_INFO0** structure is used to store information specific to virtual interface tunneling.

## Members

### `virtualIfTunnelId`

ID of the virtual interface tunnel state.

### `trafficSelectorId`

ID of the virtual interface tunneling traffic selector(s).

## Remarks

 The **IPSEC_VIRTUAL_IF_TUNNEL_INFO0** structure is applicable only to Internet Key Exchange version 2 (IKEv2).

**IPSEC_VIRTUAL_IF_TUNNEL_INFO0** is a specific implementation of IPSEC_VIRTUAL_IF_TUNNEL_INFO. See [WFP Version-Independent Names and Targeting Specific Versions of Windows](https://learn.microsoft.com/windows/desktop/FWP/wfp-version-independent-names-and-targeting-specific-versions-of-windows) for more information.
