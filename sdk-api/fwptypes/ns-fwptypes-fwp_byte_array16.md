# FWP_BYTE_ARRAY16 structure

## Description

The **FWP_BYTE_ARRAY16** structure stores an array of exactly 16 bytes.

## Members

### `byteArray16`

Array of 16 bytes.

## Remarks

This data type is useful for IPv6 addresses.
