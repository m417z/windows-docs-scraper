# FWP_BYTE_ARRAY6 structure

## Description

The **FWP_BYTE_ARRAY6** structure stores an array of exactly 6 bytes. Reserved.

## Members

### `byteArray6`

Array of 6 bytes.

## Remarks

This structure is reserved.
