# FWP_V4_ADDR_AND_MASK structure

## Description

The **FWP_V4_ADDR_AND_MASK** structure specifies IPv4 address and mask in host order..

## Members

### `addr`

Specifies an IPv4 address.

### `mask`

Specifies an IPv4 mask.
