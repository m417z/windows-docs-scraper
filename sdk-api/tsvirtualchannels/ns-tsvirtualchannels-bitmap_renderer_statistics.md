# BITMAP_RENDERER_STATISTICS structure

## Description

Contains statistics for the RemoteFX media redirection bitmap renderer.

## Members

### `dwFramesDelivered`

The number of frames delivered.

### `dwFramesDropped`

The number of frames dropped.
