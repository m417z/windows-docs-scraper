# IKEEXT_INTEGRITY_ALGORITHM0 structure

## Description

The **IKEEXT_INTEGRITY_ALGORITHM0** structure stores the IKE/AuthIP hash algorithm.

## Members

### `algoIdentifier`

The type of hash algorithm.

See [IKEEXT_INTEGRITY_TYPE](https://learn.microsoft.com/windows/desktop/api/iketypes/ne-iketypes-ikeext_integrity_type) for more information.

## Remarks

**IKEEXT_INTEGRITY_ALGORITHM0** is a specific implementation of IKEEXT_INTEGRITY_ALGORITHM. See [WFP Version-Independent Names and Targeting Specific Versions of Windows](https://learn.microsoft.com/windows/desktop/FWP/wfp-version-independent-names-and-targeting-specific-versions-of-windows) for more information.
