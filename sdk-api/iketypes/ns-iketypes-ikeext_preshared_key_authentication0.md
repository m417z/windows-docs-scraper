# IKEEXT_PRESHARED_KEY_AUTHENTICATION0 structure

## Description

The **IKEEXT_PRESHARED_KEY_AUTHENTICATION0** structure stores information needed for pre-shared key authentication.
[IKEEXT_PRESHARED_KEY_AUTHENTICATION1](https://learn.microsoft.com/windows/win32/api/iketypes/ns-iketypes-ikeext_certificate_authentication1) is available.

## Members

### `presharedKey`

The pre-shared key specified by [FWP_BYTE_BLOB](https://learn.microsoft.com/windows/desktop/api/fwptypes/ns-fwptypes-fwp_byte_blob).
