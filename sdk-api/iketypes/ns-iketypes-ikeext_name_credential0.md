# IKEEXT_NAME_CREDENTIAL0 structure

## Description

The **IKEEXT_NAME_CREDENTIAL0** structure is used to store credential name information.

## Members

### `principalName`

Name of the principal.

## Remarks

**IKEEXT_NAME_CREDENTIAL0** is a specific implementation of IKEEXT_NAME_CREDENTIAL. See [WFP Version-Independent Names and Targeting Specific Versions of Windows](https://learn.microsoft.com/windows/desktop/FWP/wfp-version-independent-names-and-targeting-specific-versions-of-windows) for more information.
