# NormalizeState enumeration

## Description

Contains values that specify the behavior of [UiaGetUpdatedCache](https://learn.microsoft.com/windows/desktop/api/uiautomationcoreapi/nf-uiautomationcoreapi-uiagetupdatedcache).

## Constants

### `NormalizeState_None`

No normalization.

### `NormalizeState_View`

Normalize against the condition in the cache request specified by pRequest.

### `NormalizeState_Custom`

Normalize against the condition specified in pNormalizeCondition.
