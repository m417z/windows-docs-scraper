# IUIAutomationElement2::get_CurrentFlowsFrom

## Description

Retrieves an array of elements that indicates the reading order before the current element.

This property is read/write.

## Parameters

## Remarks

This property maps to the Microsoft Accessible Rich Internet Applications (ARIA) [x-ms-aria-flowfrom](https://learn.microsoft.com/previous-versions/hh969192(v=vs.85)) property.
