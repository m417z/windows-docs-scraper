# IUIAutomation6::get_ConnectionRecoveryBehavior

## Description

Indicates whether an accessible technology client adjusts provider request timeouts when the provider is non-responsive.

This property is read/write.

## Parameters

### `connectionRecoveryBehaviorOptions` [in]

Type: [**ConnectionRecoveryBehaviorOptions**](https://learn.microsoft.com/windows/win32/api/uiautomationclient/ne-uiautomationclient-connectionrecoverybehavioroptions)

Value indicating whether provider request timeouts are adjusted. The default is [ConnectionRecoveryBehaviorOptions_Disabled](https://learn.microsoft.com/windows/win32/api/uiautomationclient/ne-uiautomationclient-connectionrecoverybehavioroptions).
