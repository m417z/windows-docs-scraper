# ConnectionRecoveryBehaviorOptions enumeration

## Description

Contains possible values for the [ConnectionRecoveryBehavior](https://learn.microsoft.com/windows/desktop/api/uiautomationclient/nf-uiautomationclient-iuiautomation6-get_connectionrecoverybehavior) property, which indicates whether an accessible technology client adjusts provider request timeouts when the provider is non-responsive.

## Constants

### `ConnectionRecoveryBehaviorOptions_Disabled:0`

Connection recovery is disabled.

### `ConnectionRecoveryBehaviorOptions_Enabled:0x1`

Connection recovery is enabled.
