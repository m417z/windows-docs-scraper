# IUIAutomationCacheRequest::AddProperty

## Description

Adds a property to the cache request.

## Parameters

### `propertyId` [in]

Type: **PROPERTYID**

A property identifier. For a list of property IDs, see [Property Identifiers](https://learn.microsoft.com/windows/desktop/WinAuto/uiauto-entry-propids).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
