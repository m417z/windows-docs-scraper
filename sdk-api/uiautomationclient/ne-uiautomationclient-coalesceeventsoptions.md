# CoalesceEventsOptions enumeration

## Description

Contains possible values for the [CoalesceEvents](https://learn.microsoft.com/windows/desktop/api/uiautomationclient/nf-uiautomationclient-iuiautomation6-get_coalesceevents) property, which indicates whether an accessible technology client receives all events, or a subset where duplicate events are detected and filtered.

## Constants

### `CoalesceEventsOptions_Disabled:0`

Event coalescing is disabled.

### `CoalesceEventsOptions_Enabled:0x1`

Event coalescing is enabled.
