# IUIAutomation::get_ContentViewCondition

## Description

Retrieves a predefined [IUIAutomationCondition](https://learn.microsoft.com/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationcondition) interface that selects content elements.

This property is read-only.
