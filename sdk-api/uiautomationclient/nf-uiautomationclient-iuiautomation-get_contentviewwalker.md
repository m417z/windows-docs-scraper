# IUIAutomation::get_ContentViewWalker

## Description

Retrieves an [IUIAutomationTreeWalker](https://learn.microsoft.com/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationtreewalker) interface used to discover content elements.

This property is read-only.
