# IUIAutomationDockPattern::SetDockPosition

## Description

Sets the dock position of this element.

## Parameters

### `dockPos` [in]

Type: **[DockPosition](https://learn.microsoft.com/windows/desktop/api/uiautomationcore/ne-uiautomationcore-dockposition)**

The new dock position.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
