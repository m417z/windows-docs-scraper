# IUIAutomation::get_RawViewCondition

## Description

Retrieves a predefined [IUIAutomationCondition](https://learn.microsoft.com/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationcondition) interface that selects all UI elements in an unfiltered view.

This property is read-only.

## Parameters

## Remarks

Used by itself, the condition is functionally identical to the one retrieved by [IUIAutomation::CreateTrueCondition](https://learn.microsoft.com/windows/desktop/api/uiautomationclient/nf-uiautomationclient-iuiautomation-createtruecondition).
