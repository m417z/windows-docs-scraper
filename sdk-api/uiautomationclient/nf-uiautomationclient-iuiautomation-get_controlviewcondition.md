# IUIAutomation::get_ControlViewCondition

## Description

Retrieves a predefined [IUIAutomationCondition](https://learn.microsoft.com/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationcondition) interface that selects control elements.

This property is read-only.
