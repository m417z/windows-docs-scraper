# IUIAutomationElement::get_CachedAccessKey

## Description

Retrieves the cached access key character for the element.

This property is read-only.

## Parameters

## Remarks

An access key is a character in the text of a menu, menu item, or label of a control such as a button that activates the attached menu function. For example, the letter "O" is often used to invoke the Open file common dialog box from a File menu. Microsoft UI Automation elements that have the access key property set always implement the [Invoke](https://learn.microsoft.com/windows/desktop/WinAuto/uiauto-implementinginvoke) control pattern.
