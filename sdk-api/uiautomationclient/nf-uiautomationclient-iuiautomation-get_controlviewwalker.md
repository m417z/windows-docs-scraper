# IUIAutomation::get_ControlViewWalker

## Description

Retrieves an [IUIAutomationTreeWalker](https://learn.microsoft.com/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationtreewalker) interface used to discover control elements.

This property is read-only.
