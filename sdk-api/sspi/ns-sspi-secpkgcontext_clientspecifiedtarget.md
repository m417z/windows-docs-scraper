# SecPkgContext_ClientSpecifiedTarget structure

## Description

The **SecPkgContext_ClientSpecifiedTarget** structure specifies the [service principal name](https://learn.microsoft.com/windows/desktop/SecGloss/s-gly) (SPN) of the initial target when calling the [QueryContextAttributes (Digest)](https://learn.microsoft.com/windows/desktop/api/rrascfg/nn-rrascfg-ieapproviderconfig) function.

## Members

### `sTargetName`

The SPN of the initial target.
