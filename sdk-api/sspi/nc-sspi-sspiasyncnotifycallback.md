# SspiAsyncNotifyCallback function

## Description

Callback used for notifying completion of an async SSPI call.

## Parameters

### `Handle`

The async context handle.

### `CallbackData`

Receives the callback data passed by the [SspiSetAsyncNotifyCallback](https://learn.microsoft.com/windows/win32/api/sspi/nf-sspi-sspisetasyncnotifycallback) function as "PVOID CallbackData".
