# SecPkgContext_SubjectAttributes structure

## Description

The **SecPkgContext_SubjectAttributes** structure returns the security attribute information.

## Members

### `AttributeInfo`

Pointer to a **void** that receives the attribute information stored in a [AUTHZ_SECURITY_ATTRIBUTES_INFORMATION](https://learn.microsoft.com/windows/desktop/api/authz/ns-authz-authz_security_attributes_information) structure.
