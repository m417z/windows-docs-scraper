# SEC_WINNT_AUTH_SHORT_VECTOR structure

## Description

Specifies the offset and number of characters in an array of **USHORT** values.

## Members

### `ShortArrayOffset`

The number of characters before the beginning of the data in a [SEC_WINNT_AUTH_PACKED_CREDENTIALS_EX](https://learn.microsoft.com/windows/desktop/api/sspi/ns-sspi-sec_winnt_auth_packed_credentials_ex) structure.

### `ShortArrayCount`

The number of characters in the array.
