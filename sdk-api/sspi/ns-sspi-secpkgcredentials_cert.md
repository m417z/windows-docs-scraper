# SecPkgCredentials_Cert structure

## Description

Specifies the certificate credentials. The [QueryCredentialsAttributes](https://learn.microsoft.com/windows/desktop/api/sspi/nf-sspi-querycredentialsattributesa) function uses this structure.

## Members

### `EncodedCertSize`

Size of the encoded certificate.

### `EncodedCert`

The encoded certificate.
