# WICMetadataHeader structure

## Description

Represents metadata header.

## Members

### `Position`

Type: **ULARGE_INTEGER**

The position of the header.

### `Length`

Type: **ULONG**

The length of the header.

### `Header`

Type: **BYTE***

Pointer to the header.

### `DataOffset`

Type: **ULARGE_INTEGER**

The offset of the header.
