# IWICMetadataReaderInfo::CreateInstance

## Description

Creates an instance of an [IWICMetadataReader](https://learn.microsoft.com/windows/desktop/api/wincodecsdk/nn-wincodecsdk-iwicmetadatareader).

## Parameters

### `ppIReader` [out]

Type: **[IWICMetadataReader](https://learn.microsoft.com/windows/desktop/api/wincodecsdk/nn-wincodecsdk-iwicmetadatareader)****

Pointer that receives a pointer to a metadata reader.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
