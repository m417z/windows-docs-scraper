# IWICMetadataReader::GetEnumerator

## Description

Gets an enumerator of all the metadata items.

## Parameters

### `ppIEnumMetadata` [out]

Type: **[IWICEnumMetadataItem](https://learn.microsoft.com/windows/desktop/api/wincodec/nn-wincodec-iwicenummetadataitem)****

Pointer that receives a pointer to the metadata enumerator.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
