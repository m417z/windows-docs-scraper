# StringFormat::GetAlignment

## Description

The **StringFormat::GetAlignment** method gets an element of the [StringAlignment](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-stringalignment) enumeration that indicates the character alignment of this
[StringFormat](https://learn.microsoft.com/windows/desktop/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) object in relation to the origin of the layout rectangle. A layout rectangle is used to position the displayed string.

## Return value

Type: **[StringAlignment](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-stringalignment)**

This method returns an element of the [StringAlignment](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-stringalignment) enumeration.
