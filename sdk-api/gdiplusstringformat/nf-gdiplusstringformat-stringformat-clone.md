# StringFormat::Clone

## Description

The **StringFormat::Clone** method creates a new
[StringFormat](https://learn.microsoft.com/windows/desktop/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) object and initializes it with the contents of this
**StringFormat** object.

## Return value

Type: **[StringFormat](https://learn.microsoft.com/windows/desktop/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat)***

This method returns a pointer to the new
[StringFormat](https://learn.microsoft.com/windows/desktop/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) object.
