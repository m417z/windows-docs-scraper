# StringFormat::GetTrimming

## Description

The **StringFormat::GetTrimming** method gets an element of the
[StringTrimming](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-stringtrimming) enumeration that indicates the trimming style of this
[StringFormat](https://learn.microsoft.com/windows/desktop/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) object. The trimming style determines how to trim characters from a string that is too large to fit in the layout rectangle.

## Return value

Type: **[StringTrimming](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-stringtrimming)**

This method returns an element of the
[StringTrimming](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-stringtrimming) enumeration.
