# VarR8Pow function

## Description

Performs the power function for variants of type double.

## Parameters

### `dblLeft` [in]

The first variant.

### `dblRight` [in]

The second variant.

### `pdblResult` [out]

The result.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
