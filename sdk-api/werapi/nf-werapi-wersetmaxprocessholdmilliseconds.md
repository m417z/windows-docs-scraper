# WerSetMaxProcessHoldMilliseconds function

## Description

Sets the maximum process hold time for [Windows Error Reporting](https://learn.microsoft.com/windows/win32/api/_wer/) (WER), in milliseconds.

## Parameters

### `dwMilliseconds`

The process hold time, in milliseconds.

## Return value

This function returns **S_OK** on success or an error code on failure.
