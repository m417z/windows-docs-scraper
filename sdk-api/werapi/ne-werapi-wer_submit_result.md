# WER_SUBMIT_RESULT enumeration

## Description

Defines the [Windows Error Reporting](https://learn.microsoft.com/windows/win32/api/_wer/) (WER) submission result options.

## Constants

### `WerReportQueued`

Queued.

### `WerReportUploaded`

Uploaded.

### `WerReportDebug`

Debug.

### `WerReportFailed`

Failed.

### `WerDisabled`

Disabled.

### `WerReportCancelled`

Cancelled.

### `WerDisabledQueue`

Disabled queue.

### `WerReportAsync`

Async.

### `WerCustomAction`

Custom action.

### `WerThrottled`

Throttled.

### `WerReportUploadedCab`

Uploaded [Cabinet](https://learn.microsoft.com/windows/win32/msi/cabinet-files) (CAB) file.

### `WerStorageLocationNotFound`

Storage location not found.

### `WerSubmitResultMax`

Submit the maximum data load.
