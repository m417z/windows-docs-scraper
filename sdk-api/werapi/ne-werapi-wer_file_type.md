# WER_FILE_TYPE enumeration

## Description

Defines the possible [Windows Error Reporting](https://learn.microsoft.com/windows/win32/api/_wer/) (WER) file types for the minidump report.

## Constants

### `WerFileTypeMicrodump`

Micro dump.

### `WerFileTypeMinidump`

Mini dump.

### `WerFileTypeHeapdump`

Heap dump.

### `WerFileTypeUserDocument`

User document.

### `WerFileTypeOther`

Other.

### `WerFileTypeTriagedump`

Triage dump.

### `WerFileTypeCustomDump`

Custom dump.

### `WerFileTypeAuxiliaryDump`

Auxiliary dump.

### `WerFileTypeEtlTrace`

Event Trace Logging (ETL) trace.

### `WerFileTypeAuxiliaryHeapDump`

Auxiliary heap dump.

### `WerFileTypeMax`

Maximum data dump.
