# WER_REGISTER_FILE_TYPE enumeration

## Description

Defines the [Windows Error Reporting](https://learn.microsoft.com/windows/win32/api/_wer/) (WER) file types that can be registered.

## Constants

### `WerRegFileTypeUserDocument`

User document.

### `WerRegFileTypeOther`

Other.

### `WerRegFileTypeMax`

Maximum data dump.
