# WER_REPORT_TYPE enumeration

## Description

Defines the [Windows Error Reporting](https://learn.microsoft.com/windows/win32/api/_wer/) (WER) report types.

## Constants

### `WerReportNonCritical`

Non-critical.

### `WerReportCritical`

Critical.

### `WerReportApplicationCrash`

Application crash.

### `WerReportApplicationHang`

Application hang.

### `WerReportKernel`

Kernel.

### `WerReportInvalid`

Invalid.
