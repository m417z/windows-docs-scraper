# WER_REPORT_SIGNATURE structure

## Description

Contains information about an error report signature generated by [Windows Error Reporting](https://learn.microsoft.com/windows/win32/api/_wer/) (WER).

## Members

### `EventName[WER_MAX_EVENT_NAME_LENGTH` + 1]

The event name.

### `Parameters[WER_MAX_PARAM_COUNT]`

The collection of parameters in the signature.
