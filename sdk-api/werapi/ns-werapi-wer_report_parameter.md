# WER_REPORT_PARAMETER structure

## Description

Contains information about an error report parameter generated by [Windows Error Reporting](https://learn.microsoft.com/windows/win32/api/_wer/) (WER).

## Members

### `Name[WER_MAX_SIGNATURE_NAME_LENGTH` + 1]

The parameter name.

### `Value[WER_MAX_PARAM_LENGTH]`

The parameter value.
