# WER_REPORT_UI enumeration

## Description

Defines the indexes of the [Windows Error Reporting](https://learn.microsoft.com/windows/win32/api/_wer/) (WER) report's customizable UI strings.

## Constants

### `WerUIAdditionalDataDlgHeader`

Additional data dialog header.

### `WerUIIconFilePath`

Icon file path.

### `WerUIConsentDlgHeader`

Consent dialog header.

### `WerUIConsentDlgBody`

Consent dialog body.

### `WerUIOnlineSolutionCheckText`

Online solution check text.

### `WerUIOfflineSolutionCheckText`

Offline solution check text.

### `WerUICloseText`

Close text.

### `WerUICloseDlgHeader`

Close dialog header.

### `WerUICloseDlgBody`

Close dialog body.

### `WerUICloseDlgButtonText`

Close dialog button text.

### `WerUIMax`

Maximum text load.
