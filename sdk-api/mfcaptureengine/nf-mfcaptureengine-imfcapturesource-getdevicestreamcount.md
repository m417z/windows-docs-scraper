# IMFCaptureSource::GetDeviceStreamCount

## Description

Gets the number of device streams.

## Parameters

### `pdwStreamCount` [out]

Receives the number of device streams.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
