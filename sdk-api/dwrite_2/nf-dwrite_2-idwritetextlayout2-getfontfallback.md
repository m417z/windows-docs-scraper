# IDWriteTextLayout2::GetFontFallback

## Description

Get the current font fallback object.

## Parameters

### `fontFallback` [out]

The current font fallback object.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
