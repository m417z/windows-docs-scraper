# IDWriteTextFormat1::SetVerticalGlyphOrientation

## Description

Sets the orientation of a text format.

## Parameters

### `glyphOrientation`

Type: **[DWRITE_VERTICAL_GLYPH_ORIENTATION](https://learn.microsoft.com/windows/win32/api/dwrite_1/ne-dwrite_1-dwrite_vertical_glyph_orientation)**

The orientation to apply to the text format.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
