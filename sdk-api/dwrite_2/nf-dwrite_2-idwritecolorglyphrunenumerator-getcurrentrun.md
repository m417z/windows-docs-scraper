# IDWriteColorGlyphRunEnumerator::GetCurrentRun

## Description

Returns the current glyph run of the enumerator.

## Parameters

### `colorGlyphRun` [out]

Type: **const [DWRITE_COLOR_GLYPH_RUN](https://learn.microsoft.com/windows/win32/api/dwrite_2/ns-dwrite_2-dwrite_color_glyph_run)****

A pointer to the current glyph run.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
