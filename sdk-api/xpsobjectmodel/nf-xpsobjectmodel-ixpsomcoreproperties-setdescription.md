# IXpsOMCoreProperties::SetDescription

## Description

Sets the **description** property.

## Parameters

### `description` [in]

The string to be written to the **description** property. A **NULL** pointer clears this property.

## Return value

If the method succeeds, it returns S_OK; otherwise, it returns an **HRESULT** error code.

## Remarks

The **description** property explains the content.
