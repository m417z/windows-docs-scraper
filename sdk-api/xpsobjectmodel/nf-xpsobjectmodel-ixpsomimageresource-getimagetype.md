# IXpsOMImageResource::GetImageType

## Description

Gets the type of image resource.

## Parameters

### `imageType` [out, retval]

The [XPS_IMAGE_TYPE](https://learn.microsoft.com/windows/win32/api/xpsobjectmodel/ne-xpsobjectmodel-xps_image_type) value that describes the image type in the stream.

## Return value

If the method succeeds, it returns S_OK; otherwise, it returns an **HRESULT** error code.
