# IXpsOMCoreProperties::SetModified

## Description

Sets the **modified** property.

## Parameters

### `modified` [in]

The date and time the package was last changed. A **NULL** pointer clears the **modified** property.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the table that follows. For information about XPS document API return values that are not listed in this table, see [XPS Document Errors](https://learn.microsoft.com/previous-versions/windows/desktop/dd372955(v=vs.85)).

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
| **E_POINTER** | *modified* is **NULL**. |

## Remarks

The **modified** property contains the date and time the package was last changed.
