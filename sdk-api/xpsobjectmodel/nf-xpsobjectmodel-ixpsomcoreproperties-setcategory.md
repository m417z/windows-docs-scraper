# IXpsOMCoreProperties::SetCategory

## Description

Sets the **category** property.

## Parameters

### `category` [in]

The string to be written to the **category** property. A **NULL** pointer clears the **category** property.

## Return value

If the method succeeds, it returns S_OK; otherwise, it returns an **HRESULT** error code.

## Remarks

The **category** property contains a categorization of the content.
