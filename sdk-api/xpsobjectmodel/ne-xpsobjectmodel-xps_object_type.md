# XPS_OBJECT_TYPE enumeration

## Description

Describes the type of an object that is derived from [IXpsOMShareable](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomshareable).

## Constants

### `XPS_OBJECT_TYPE_CANVAS:1`

The object is an [IXpsOMCanvas](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomcanvas) interface.

### `XPS_OBJECT_TYPE_GLYPHS`

The object is an [IXpsOMGlyphs](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomglyphs) interface.

### `XPS_OBJECT_TYPE_PATH`

The object is an [IXpsOMPath](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsompath) interface.

### `XPS_OBJECT_TYPE_MATRIX_TRANSFORM`

The object is an [IXpsOMMatrixTransform](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsommatrixtransform) interface.

### `XPS_OBJECT_TYPE_GEOMETRY`

The object is an [IXpsOMGeometry](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomgeometry) interface.

### `XPS_OBJECT_TYPE_SOLID_COLOR_BRUSH`

The object is an [IXpsOMSolidColorBrush](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomsolidcolorbrush) interface.

### `XPS_OBJECT_TYPE_IMAGE_BRUSH`

The object is an [IXpsOMImageBrush](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomimagebrush) interface.

### `XPS_OBJECT_TYPE_LINEAR_GRADIENT_BRUSH`

The object is an [IXpsOMLinearGradientBrush](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomlineargradientbrush) interface.

### `XPS_OBJECT_TYPE_RADIAL_GRADIENT_BRUSH`

The object is an [IXpsOMRadialGradientBrush](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomradialgradientbrush) interface.

### `XPS_OBJECT_TYPE_VISUAL_BRUSH`

The object is an [IXpsOMVisualBrush](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomvisualbrush) interface.
