# IXpsOMCoreProperties::SetIdentifier

## Description

Sets the **identifier** property.

## Parameters

### `identifier` [in]

The string to be written to the **identifier** property. A **NULL** pointer clears the **identifier** property.

## Return value

If the method succeeds, it returns S_OK; otherwise, it returns an **HRESULT** error code.

## Remarks

The **identifier** property is an unambiguous reference to the resource within a user-defined or application-specific
context.
