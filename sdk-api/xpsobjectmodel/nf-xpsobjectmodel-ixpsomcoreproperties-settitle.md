# IXpsOMCoreProperties::SetTitle

## Description

Sets the **title** property.

## Parameters

### `title` [in]

The string to be written to the **title** property. A **NULL** pointer clears the **title** property.

## Return value

If the method succeeds, it returns S_OK; otherwise, it returns an **HRESULT** error code.

## Remarks

The **title** property contains the name given to the resource.
