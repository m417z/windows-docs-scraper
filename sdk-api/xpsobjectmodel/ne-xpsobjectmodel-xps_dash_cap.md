# XPS_DASH_CAP enumeration

## Description

Specifies the style of a dash cap on a dashed stroke.

## Constants

### `XPS_DASH_CAP_FLAT:1`

Flat-line cap.

### `XPS_DASH_CAP_ROUND`

Round-line cap.

### `XPS_DASH_CAP_SQUARE`

Square-line cap.

### `XPS_DASH_CAP_TRIANGLE`

Triangular-line cap.
