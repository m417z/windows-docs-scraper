# IXpsOMCoreProperties::GetSubject

## Description

Gets the **subject** property.

## Parameters

### `subject` [out, retval]

The string that is read from the **subject** property.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the table that follows. For information about XPS document API return values that are not listed in this table, see [XPS Document Errors](https://learn.microsoft.com/previous-versions/windows/desktop/dd372955(v=vs.85)).

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
| **E_POINTER** | *subject* is **NULL**. |

## Remarks

The **subject** property contains the topic of the resource's content.

This method allocates the memory used by the string that is returned in *subject*. If *subject* is not **NULL**, use the [CoTaskMemFree](https://learn.microsoft.com/windows/desktop/api/combaseapi/nf-combaseapi-cotaskmemfree) function to free the memory.
