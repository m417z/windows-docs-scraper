# IXpsOMCoreProperties::SetCreator

## Description

Sets the **creator** property.

## Parameters

### `creator` [in]

The string to be written to the **creator** property. A **NULL** pointer clears the **creator** property.

## Return value

If the method succeeds, it returns S_OK; otherwise, it returns an **HRESULT** error code.

## Remarks

The **creator** property describes the entity that is primarily responsible for making the content of
the resource.
