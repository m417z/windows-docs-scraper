# IXpsOMObjectFactory::CreateRemoteDictionaryResource

## Description

Creates an [IXpsOMRemoteDictionaryResource](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomremotedictionaryresource) interface that enables the sharing of property resources.

## Parameters

### `dictionary` [in]

The [IXpsOMDictionary](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomdictionary) interface pointer of the dictionary to be associated with this resource.

### `partUri` [in]

The [IOpcPartUri](https://learn.microsoft.com/previous-versions/windows/desktop/api/msopc/nn-msopc-iopcparturi) interface that contains the part name to be assigned to this resource.

### `remoteDictionaryResource` [out, retval]

A pointer to the new [IXpsOMRemoteDictionaryResource](https://learn.microsoft.com/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomremotedictionaryresource) interface.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the table that follows. For information about XPS document API return values that are not listed in this table, see [XPS Document Errors](https://learn.microsoft.com/previous-versions/windows/desktop/dd372955(v=vs.85)).

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
| **E_POINTER** | *dictionary*, *partUri*, or *remoteDictionaryResource* is **NULL**. |
| **XPS_E_NO_CUSTOM_OBJECTS** | *dictionary* does not point to a recognized interface implementation. Custom implementation of XPS Document API interfaces is not supported. |
