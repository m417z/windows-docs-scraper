# IXpsOMCoreProperties::SetSubject

## Description

Sets the **subject** property.

## Parameters

### `subject` [in]

The string to be written to the **subject** property. A **NULL** pointer clears the **subject** property.

## Return value

If the method succeeds, it returns S_OK; otherwise, it returns an **HRESULT** error code.

## Remarks

The **subject** property contains the topic of the resource content.
