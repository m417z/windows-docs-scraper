# IXpsOMCoreProperties::SetLastModifiedBy

## Description

Sets the **lastModifiedBy** property.

## Parameters

### `lastModifiedBy` [in]

The string that contains the value to be written to the **lastModifiedBy** property. A **NULL** pointer clears the **lastModifiedBy** property.

## Return value

If the method succeeds, it returns S_OK; otherwise, it returns an **HRESULT** error code.

## Remarks

The **lastModifiedBy** property describes the user who performs the last modification.
