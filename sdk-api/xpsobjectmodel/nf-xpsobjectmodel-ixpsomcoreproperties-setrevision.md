# IXpsOMCoreProperties::SetRevision

## Description

Sets the **revision** property.

## Parameters

### `revision` [in]

The string to be written to the **revision** property. A **NULL** pointer clears the **revision** property.

## Return value

If the method succeeds, it returns S_OK; otherwise, it returns an **HRESULT** error code.

## Remarks

The **revision** property contains the revision number of the resource.
