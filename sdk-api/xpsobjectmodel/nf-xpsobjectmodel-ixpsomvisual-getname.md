# IXpsOMVisual::GetName

## Description

Gets the **Name** property of the visual.

## Parameters

### `name` [out, retval]

The **Name** property string. If the **Name** property has not been set, a **NULL** pointer is returned.

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the table that follows. For information about XPS document API return values that are not listed in this table, see [XPS Document Errors](https://learn.microsoft.com/previous-versions/windows/desktop/dd372955(v=vs.85)).

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
| **E_POINTER** | *name* is **NULL**. |

## Remarks

This method allocates the memory that is used by the string returned in *name*. If *name* is not **NULL**, use the [CoTaskMemFree](https://learn.microsoft.com/windows/desktop/api/combaseapi/nf-combaseapi-cotaskmemfree) function to free the memory.
