# IXpsOMDictionary::GetCount

## Description

Gets the number of entries in the dictionary.

## Parameters

### `count` [out, retval]

The number of entries in the dictionary.

## Return value

If the method succeeds, it returns S_OK; otherwise, it returns an **HRESULT** error code.
