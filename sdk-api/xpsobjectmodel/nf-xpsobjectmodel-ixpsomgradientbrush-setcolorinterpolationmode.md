# IXpsOMGradientBrush::SetColorInterpolationMode

## Description

Sets the [XPS_COLOR_INTERPOLATION](https://learn.microsoft.com/windows/win32/api/xpsobjectmodel/ne-xpsobjectmodel-xps_color_interpolation) value, which describes the gamma function to be used for color interpolation.

## Parameters

### `colorInterpolationMode` [in]

The [XPS_COLOR_INTERPOLATION](https://learn.microsoft.com/windows/win32/api/xpsobjectmodel/ne-xpsobjectmodel-xps_color_interpolation) value, which describes the gamma function to be used for color interpolation.

## Return value

If the method succeeds, it returns S_OK; otherwise, it returns an **HRESULT** error code.
