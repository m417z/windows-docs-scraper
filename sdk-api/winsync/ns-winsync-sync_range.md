# SYNC_RANGE structure

## Description

Represents a range of item IDs.

## Members

### `pbClosedLowerBound`

The closed lower bound of item IDs that are contained in the range.

### `pbClosedUpperBound`

The closed upper bound of item IDs that are contained in the range.
