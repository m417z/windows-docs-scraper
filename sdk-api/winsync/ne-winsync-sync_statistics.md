# SYNC_STATISTICS enumeration

## Description

Represents types of statistics that convey information about a component.

## Constants

### `SYNC_STATISTICS_RANGE_COUNT:0`

Indicates that the statistic represents the number of ranges that are contained in an [ISyncKnowledge](https://learn.microsoft.com/previous-versions/windows/desktop/api/winsync/nn-winsync-isyncknowledge) object.
