# CERT_LOGOTYPE_AUDIO structure

## Description

The **CERT_LOGOTYPE_AUDIO** structure contains information about an audio logotype.

## Members

### `LogotypeDetails`

A [CERT_LOGOTYPE_DETAILS](https://learn.microsoft.com/windows/desktop/api/wincrypt/ns-wincrypt-cert_logotype_details) structure that contains additional information about the audio data.

### `pLogotypeAudioInfo`

The address of a [CERT_LOGOTYPE_AUDIO_INFO](https://learn.microsoft.com/windows/desktop/api/wincrypt/ns-wincrypt-cert_logotype_audio_info) structure that contains the audio information.
