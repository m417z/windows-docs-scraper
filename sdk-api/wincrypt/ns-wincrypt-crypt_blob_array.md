# CRYPT_BLOB_ARRAY structure

## Description

The **CRYPT_BLOB_ARRAY** structure contains an array of [CRYPT_DATA_BLOB](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/aa381414(v=vs.85)) structures.

## Members

### `cBlob`

The number of elements in the **rgBlob** array.

### `rgBlob`

An array of [CRYPT_DATA_BLOB](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/aa381414(v=vs.85)) structures that contains the data blobs.
