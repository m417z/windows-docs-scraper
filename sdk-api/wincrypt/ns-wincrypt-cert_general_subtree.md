# CERT_GENERAL_SUBTREE structure

## Description

The **CERT_GENERAL_SUBTREE** structure is used in
[CERT_NAME_CONSTRAINTS_INFO](https://learn.microsoft.com/windows/desktop/api/wincrypt/ns-wincrypt-cert_name_constraints_info) structure. This structure provides the identity of a certificate that can be included or excluded.

## Members

### `Base`

[CERT_ALT_NAME_ENTRY](https://learn.microsoft.com/windows/desktop/api/wincrypt/ns-wincrypt-cert_alt_name_entry) containing a base name identifier of a certificate.

### `dwMinimum`

Currently not used.

### `fMaximum`

Currently not used.

### `dwMaximum`

Currently not used.
