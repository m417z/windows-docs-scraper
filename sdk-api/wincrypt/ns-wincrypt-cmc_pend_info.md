# CMC_PEND_INFO structure

## Description

The **CMC_PEND_INFO** structure is a possible member of a
[CMC_STATUS_INFO](https://learn.microsoft.com/windows/desktop/api/wincrypt/ns-wincrypt-cmc_status_info) structure.

## Members

### `PendToken`

A [BLOB](https://learn.microsoft.com/windows/desktop/SecGloss/b-gly) that contains the pending request information.

### `PendTime`

**FILETIME** containing the request.
