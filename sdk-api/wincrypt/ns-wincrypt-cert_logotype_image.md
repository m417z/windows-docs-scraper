# CERT_LOGOTYPE_IMAGE structure

## Description

The **CERT_LOGOTYPE_IMAGE** structure contains information about an image logotype.

## Members

### `LogotypeDetails`

A [CERT_LOGOTYPE_DETAILS](https://learn.microsoft.com/windows/desktop/api/wincrypt/ns-wincrypt-cert_logotype_details) structure that contains additional information about the image.

### `pLogotypeImageInfo`

The address of a [CERT_LOGOTYPE_IMAGE_INFO](https://learn.microsoft.com/windows/desktop/api/wincrypt/ns-wincrypt-cert_logotype_image_info) structure that contains the image information.
