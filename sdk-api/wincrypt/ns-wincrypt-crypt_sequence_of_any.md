# CRYPT_SEQUENCE_OF_ANY structure

## Description

The **CRYPT_SEQUENCE_OF_ANY** structure contains an arbitrary list of encoded BLOBs.

## Members

### `cValue`

Number of elements in the **rgValue** array.

### `rgValue`

An array of
[CRYPT_DER_BLOB](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/aa381414(v=vs.85)) structures.
