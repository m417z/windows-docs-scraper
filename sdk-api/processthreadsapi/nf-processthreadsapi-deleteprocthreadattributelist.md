# DeleteProcThreadAttributeList function

## Description

Deletes the specified list of attributes for process and thread creation.

## Parameters

### `lpAttributeList` [in, out]

The attribute list. This list is created by the [InitializeProcThreadAttributeList](https://learn.microsoft.com/windows/desktop/api/processthreadsapi/nf-processthreadsapi-initializeprocthreadattributelist) function.
