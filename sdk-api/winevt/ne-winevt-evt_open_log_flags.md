# EVT_OPEN_LOG_FLAGS enumeration

## Description

Defines the values that specify whether to open a channel or exported log file.

## Constants

### `EvtOpenChannelPath:0x1`

Open a channel.

### `EvtOpenFilePath:0x2`

Open an exported log file.
