# EVT_CHANNEL_TYPE enumeration

## Description

Defines the type of a channel.

## Constants

### `EvtChannelTypeAdmin:0`

The channel's type is Admin.

### `EvtChannelTypeOperational`

The channel's type is Operational.

### `EvtChannelTypeAnalytic`

The channel's type is Analytic.

### `EvtChannelTypeDebug`

The channel's type is Debug.

## Remarks

For a description of each channel type, see the **type** attribute of the [ChannelType](https://learn.microsoft.com/windows/desktop/WES/eventmanifestschema-channeltype-complextype) complex type.
