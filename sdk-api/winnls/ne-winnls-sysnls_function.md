# SYSNLS_FUNCTION enumeration

## Description

Specifies NLS function capabilities.

## Constants

### `COMPARE_STRING:0x0001`

Value indicating comparison of two strings in the manner of the [CompareString](https://learn.microsoft.com/windows/desktop/api/stringapiset/nf-stringapiset-comparestringw) function or [LCMapString](https://learn.microsoft.com/windows/desktop/api/winnls/nf-winnls-lcmapstringa) with the LCMAP_SORTKEY flag specified.
