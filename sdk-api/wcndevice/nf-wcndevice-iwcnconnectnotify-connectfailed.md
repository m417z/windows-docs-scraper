# IWCNConnectNotify::ConnectFailed

## Description

The **IWCNConnectNotify::ConnectFailed** callback method indicates a [IWCNDevice::Connect](https://learn.microsoft.com/windows/desktop/api/wcndevice/nf-wcndevice-iwcndevice-connect) failure.

## Parameters

### `hrFailure` [in]

An **HRESULT** that specifies the reason for the connection failure.

## Return value

This method does not return a value.
