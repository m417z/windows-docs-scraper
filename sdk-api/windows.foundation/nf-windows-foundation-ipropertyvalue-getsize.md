# IPropertyValue::GetSize (windows.foundation.h)

## Description

Gets the [Size](https://learn.microsoft.com/windows/desktop/api/windows.foundation/ns-windows-foundation-size) value that is stored in the current [IPropertyValue](https://learn.microsoft.com/windows/desktop/api/windows.foundation/nn-windows-foundation-ipropertyvalue) object.

## Parameters

### `value` [out, retval]

Type: **[Size](https://learn.microsoft.com/windows/desktop/api/windows.foundation/ns-windows-foundation-size)***

The value.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
