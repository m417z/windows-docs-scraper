# IPropertyValue::GetPoint (windows.foundation.h)

## Description

Gets the [Point](https://learn.microsoft.com/windows/desktop/api/windows.foundation/ns-windows-foundation-point) value that is stored in the current [IPropertyValue](https://learn.microsoft.com/windows/desktop/api/windows.foundation/nn-windows-foundation-ipropertyvalue) object.

## Parameters

### `value` [out, retval]

Type: **[Point](https://learn.microsoft.com/windows/desktop/api/windows.foundation/ns-windows-foundation-point)***

The value.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
