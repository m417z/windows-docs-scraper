# IPropertyValue::get_Type (windows.foundation.h)

## Description

Gets the data type of the value that is stored in the current [IPropertyValue](https://learn.microsoft.com/windows/desktop/api/windows.foundation/nn-windows-foundation-ipropertyvalue) object.

This property is read-only.
