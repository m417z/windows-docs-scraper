# HatchBrush::GetHatchStyle

## Description

The **HatchBrush::GetHatchStyle** method gets the hatch style of this hatch brush.

## Return value

Type: **[HatchStyle](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-hatchstyle)**

This method returns the hatch style, which is one of the elements of the [HatchStyle](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-hatchstyle) enumeration.
