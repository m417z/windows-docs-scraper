# TextureBrush::GetWrapMode

## Description

The **TextureBrush::GetWrapMode** method gets the wrap mode currently set for this texture brush.

## Return value

Type: **[WrapMode](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-wrapmode)**

This method returns the wrap mode currently set for this texture brush. The value returned is one of the elements of the [WrapMode](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-wrapmode) enumeration.
