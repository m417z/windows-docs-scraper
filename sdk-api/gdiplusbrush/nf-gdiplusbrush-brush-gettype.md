# Brush::GetType

## Description

The **Brush::GetType** method gets the type of this brush.

## Return value

Type: **[BrushType](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-brushtype)**

This method returns the type of this brush. The value returned is one of the elements of the
[BrushType](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-brushtype) enumeration.
