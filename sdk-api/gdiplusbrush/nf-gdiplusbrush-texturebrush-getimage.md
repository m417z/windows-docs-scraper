# TextureBrush::GetImage

## Description

The **TextureBrush::GetImage** method gets a pointer to the [Image](https://learn.microsoft.com/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-image) object that is defined by this texture brush.

## Return value

Type: **[Image](https://learn.microsoft.com/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-image)***

This method returns a pointer to the [Image](https://learn.microsoft.com/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-image) object that is defined by this texture brush.
