# IEnumTfInputProcessorProfiles::Reset

## Description

The [IEnumTfInputProcessorProfiles::Reset](https://learn.microsoft.com/windows/desktop/api/msctf/nf-msctf-ienumtfinputprocessorprofiles-next) method resets the enumerator object by moving the current position to the beginning of the enumeration sequence.

## Return value

This method can return one of these values.

| Value | Description |
| --- | --- |
| **S_OK** | The method was successful. |
