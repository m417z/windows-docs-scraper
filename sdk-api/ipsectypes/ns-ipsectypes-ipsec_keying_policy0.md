# IPSEC_KEYING_POLICY0 structure

## Description

The [IPSEC_KEYING_POLICY1](https://learn.microsoft.com/windows/desktop/api/ipsectypes/ns-ipsectypes-ipsec_keying_policy1) is available.

## Members

### `numKeyMods`

Number of keying modules in the array.

### `keyModKeys`

Array of distinct keying modules.

## Remarks

**IPSEC_KEYING_POLICY0** is a specific implementation of IPSEC_KEYING_POLICY. See [WFP Version-Independent Names and Targeting Specific Versions of Windows](https://learn.microsoft.com/windows/desktop/FWP/wfp-version-independent-names-and-targeting-specific-versions-of-windows) for more information.
