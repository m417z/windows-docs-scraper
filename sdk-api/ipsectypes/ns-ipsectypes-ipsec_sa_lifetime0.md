# IPSEC_SA_LIFETIME0 structure

## Description

The **IPSEC_SA_LIFETIME0** structure stores the lifetime in seconds/kilobytes/packets for an IPsec security association (SA).

## Members

### `lifetimeSeconds`

SA lifetime in seconds.

### `lifetimeKilobytes`

SA lifetime in kilobytes.

### `lifetimePackets`

SA lifetime in packets.

## Remarks

**IPSEC_SA_LIFETIME0** is a specific implementation of IPSEC_SA_LIFETIME. See [WFP Version-Independent Names and Targeting Specific Versions of Windows](https://learn.microsoft.com/windows/desktop/FWP/wfp-version-independent-names-and-targeting-specific-versions-of-windows) for more information.
