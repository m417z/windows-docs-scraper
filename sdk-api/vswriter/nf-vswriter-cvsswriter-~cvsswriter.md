# CVssWriter::~CVssWriter

## Description

**~CVssWriter** is the destructor of the
[CVssWriter](https://learn.microsoft.com/windows/desktop/api/vswriter/nl-vswriter-cvsswriter) class object.

This destructor is virtual and may be overridden by derived objects.
