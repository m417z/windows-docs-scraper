# DHCPV6CAPI_CLASSID structure

## Description

The **DHCPV6CAPI_CLASSID** structure defines an IPv6 client class ID.

## Members

### `Flags`

Reserved for future use. Must be set to 0.

### `Data.size_is`

### `Data.size_is.nBytesData`

### `Data`

Class ID binary data.

### `nBytesData`

Size of **Data**, in bytes.
