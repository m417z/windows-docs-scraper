# DHCPV6Prefix structure

## Description

The **DHCPV6Prefix** contains an IPv6 prefix.

## Members

### `prefix`

128 bit prefix.

### `prefixLength`

Length of the prefix.

### `preferredLifeTime`

Preferred lifetime of the prefix, in seconds.

### `validLifeTime`

The valid lifetime of the prefix in seconds.

### `status`

The status code returned.
