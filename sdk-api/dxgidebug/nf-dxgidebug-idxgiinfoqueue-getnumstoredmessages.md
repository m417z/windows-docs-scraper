# IDXGIInfoQueue::GetNumStoredMessages

## Description

Gets the number of messages currently stored in the message queue.

## Parameters

### `Producer` [in]

 A [DXGI_DEBUG_ID](https://learn.microsoft.com/windows/desktop/direct3ddxgi/dxgi-debug-id) value that identifies the entity that gets the number.

## Return value

Returns the number of messages currently stored in the message queue.

## Remarks

**Note** This API requires the Windows Software Development Kit (SDK) for Windows 8.
