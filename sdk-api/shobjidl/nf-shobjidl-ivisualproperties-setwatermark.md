# IVisualProperties::SetWatermark

## Description

Provides a bitmap to use as a watermark.

## Parameters

### `hbmp` [in]

Type: **HBITMAP**

A handle to the bitmap.

### `vpwf` [in]

Type: **[VPWATERMARKFLAGS](https://learn.microsoft.com/windows/desktop/api/shobjidl/ne-shobjidl-vpwatermarkflags)**

[VPWATERMARKFLAGS](https://learn.microsoft.com/windows/desktop/api/shobjidl/ne-shobjidl-vpwatermarkflags) flags that customize the watermark.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
