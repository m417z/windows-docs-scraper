# VPCOLORFLAGS enumeration

## Description

Specifies the use of a color. Used by [IVisualProperties](https://learn.microsoft.com/windows/desktop/api/shobjidl/nn-shobjidl-ivisualproperties) methods.

## Constants

### `VPCF_TEXT:1`

A text color.

### `VPCF_BACKGROUND:2`

A background color.

### `VPCF_SORTCOLUMN:3`

A sort-column color.

### `VPCF_SUBTEXT:4`

A subtext color.

### `VPCF_TEXTBACKGROUND:5`

**Windows 7 and later**. A text background color.
