# VPWATERMARKFLAGS enumeration

## Description

Specifies watermark flags. Used by [IVisualProperties::SetWatermark](https://learn.microsoft.com/windows/desktop/api/shobjidl/nf-shobjidl-ivisualproperties-setwatermark).

## Constants

### `VPWF_DEFAULT:0`

Default Windows XP behavior.

### `VPWF_ALPHABLEND:0x1`

Alpha blend the respective bitmap; assumed 24-bit color + 8-bit alpha.
