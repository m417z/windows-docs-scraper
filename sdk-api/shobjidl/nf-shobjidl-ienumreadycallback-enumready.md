# IEnumReadyCallback::EnumReady

## Description

Notifies the implementer that the view's item enumeration has completed. This callback interface is provided to the view via [SetEnumReadyCallback](https://learn.microsoft.com/windows/desktop/api/shobjidl/nf-shobjidl-ienumerableview-setenumreadycallback)

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
