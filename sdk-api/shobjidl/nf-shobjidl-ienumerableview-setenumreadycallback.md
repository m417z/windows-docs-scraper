# IEnumerableView::SetEnumReadyCallback

## Description

Sets a callback on the view that is notified when the initial view enumeration is complete.

## Parameters

### `percb` [in]

Type: **[IEnumReadyCallback](https://learn.microsoft.com/windows/desktop/api/shobjidl/nn-shobjidl-ienumreadycallback)***

A pointer to the [IEnumReadyCallback](https://learn.microsoft.com/windows/desktop/api/shobjidl/nn-shobjidl-ienumreadycallback) interface.

## Return value

Type: **HRESULT**

Returns a success value if successful, or an error value otherwise.
