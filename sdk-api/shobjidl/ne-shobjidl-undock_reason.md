# UNDOCK_REASON enumeration

## Description

Values that indicate the reason that a docked accessibility app window has been undocked. Used by [IAccessibilityDockingServiceCallback::Undocked](https://learn.microsoft.com/windows/desktop/api/shobjidl/nf-shobjidl-iaccessibilitydockingservicecallback-undocked).

## Constants

### `UR_RESOLUTION_CHANGE:0`

The accessibility window was undocked because the screen resolution has changed.

### `UR_MONITOR_DISCONNECT:1`

The monitor on which the accessibility window was docked has been disconnected.
