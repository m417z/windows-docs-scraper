# ITrayDeskBand::IsDeskBandShown

## Description

Indicates whether a deskband is shown.

## Parameters

### `clsid` [in]

Type: **REFCLSID**

A reference to a deskband CLSID.

## Return value

Type: **HRESULT**

Returns S_OK if the deskband is shown, S_FALSE if the deskband is not shown, or an error value otherwise.
