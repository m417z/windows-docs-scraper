# IFolderViewOptions::GetFolderViewOptions

## Description

Retrieves the current set of options for the view.

## Parameters

### `pfvoFlags` [out]

Type: **[FOLDERVIEWOPTIONS](https://learn.microsoft.com/windows/desktop/api/shobjidl/ne-shobjidl-folderviewoptions)***

A bitmask that, when this method returns successfully, receives the [FOLDERVIEWOPTIONS](https://learn.microsoft.com/windows/desktop/api/shobjidl/ne-shobjidl-folderviewoptions) values that are currently set.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
