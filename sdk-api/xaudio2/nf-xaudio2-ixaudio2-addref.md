# IXAudio2::AddRef

## Description

Adds a reference to the XAudio2 object.

## Return value

Always 0.

## Remarks

### Platform Requirements

Windows 10 (XAudio2.9); Windows 8, Windows Phone 8 (XAudio 2.8); DirectX SDK (XAudio 2.7)
