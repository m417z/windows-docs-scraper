# IXAudio2Voice::GetVoiceDetails

## Description

Returns information about the creation flags, input channels, and sample rate of a voice.

## Parameters

### `pVoiceDetails` [in, out]

[XAUDIO2_VOICE_DETAILS](https://learn.microsoft.com/windows/desktop/api/xaudio2/ns-xaudio2-xaudio2_voice_details) structure containing information about the voice.

## Return value

This method does not return a value.

## Remarks

### Platform Requirements

Windows 10 (XAudio2.9); Windows 8, Windows Phone 8 (XAudio 2.8); DirectX SDK (XAudio 2.7)
