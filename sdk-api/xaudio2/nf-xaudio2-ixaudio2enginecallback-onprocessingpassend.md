# IXAudio2EngineCallback::OnProcessingPassEnd

## Description

Called by XAudio2 just after an audio processing pass ends.

## Remarks

For information about the [IXAudio2EngineCallback](https://learn.microsoft.com/windows/desktop/api/xaudio2/nn-xaudio2-ixaudio2enginecallback) interface methods, see the XAudio2 [Callbacks](https://learn.microsoft.com/windows/desktop/xaudio2/callbacks) section.

### Platform Requirements

Windows 10 (XAudio2.9); Windows 8, Windows Phone 8 (XAudio 2.8); DirectX SDK (XAudio 2.7)
