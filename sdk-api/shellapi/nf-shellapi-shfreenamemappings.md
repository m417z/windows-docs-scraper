# SHFreeNameMappings function

## Description

Frees a file name mapping object that was retrieved by the [SHFileOperation](https://learn.microsoft.com/windows/desktop/api/shellapi/nf-shellapi-shfileoperationa) function.

## Parameters

### `hNameMappings` [in, optional]

Type: **HANDLE**

A handle to the file name mapping object to be freed.
