# IWbemPath::IsSameClassName

## Description

The
**IWbemPath::IsSameClassName** method tests whether the class name passed in matches the one in the path. The method can return **TRUE** only if the path actually has a class name.

## Parameters

### `wszClass` [in]

Class name to test.

## Return value

This method returns a BOOL indicating whether the class name matches the one in the path.
