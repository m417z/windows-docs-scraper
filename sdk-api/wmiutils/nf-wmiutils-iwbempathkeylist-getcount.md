# IWbemPathKeyList::GetCount

## Description

The
**IWbemPathKeyList::GetCount** method retrieves the number of keys in the path.

## Parameters

### `puKeyCount` [in, out]

Number of keys.

## Return value

This method returns an **HRESULT** indicating the status of the method call.
