# IWbemQuery::FreeMemory

## Description

The **IWbemQuery::FreeMemory** method frees the memory that the parser returns to a caller in a previous call to
[GetAnalysis](https://learn.microsoft.com/windows/desktop/api/wmiutils/nf-wmiutils-iwbemquery-getanalysis).

## Parameters

### `pMem` [in]

Pointer to the memory to be freed.

## Return value

This method returns an **HRESULT** indicating the status of the method call.
