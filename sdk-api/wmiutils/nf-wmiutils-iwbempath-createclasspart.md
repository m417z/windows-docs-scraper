# IWbemPath::CreateClassPart

## Description

The
**IWbemPath::CreateClassPart** method initializes the class or key portion of the path. This method is used when creating a path from scratch.

## Parameters

### `lFlags` [in]

Reserved. Must be 0 (zero).

### `Name` [in]

Initial class name.

## Return value

This method returns an **HRESULT** indicating the status of the method call.
