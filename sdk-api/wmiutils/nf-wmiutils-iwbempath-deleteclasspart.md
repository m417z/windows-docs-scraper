# IWbemPath::DeleteClassPart

## Description

The
**IWbemPath::DeleteClassPart** method deletes the class portion of the path.

## Parameters

### `lFlags` [in]

Reserved. Must be 0 (zero).

## Return value

This method returns an **HRESULT** indicating the status of the method call.
