# IWbemPath::SetServer

## Description

The
**IWbemPath::SetServer** method sets the server portion of the path.

## Parameters

### `Name` [in]

New server name. **NULL** is an acceptable value.

## Return value

This method returns an **HRESULT** indicating the status of the method call.
