# IWbemPathKeyList::RemoveAllKeys

## Description

The
**IWbemPathKeyList::RemoveAllKeys** method removes all keys.

## Parameters

### `uFlags` [in]

Reserved. Must be 0 (zero).

## Return value

This method returns an **HRESULT** indicating the status of the method call.
