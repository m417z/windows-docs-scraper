# IWbemPathKeyList::MakeSingleton

## Description

The
**IWbemPathKeyList::MakeSingleton** method governs whether or not a key is singleton.

## Parameters

### `bSet` [in]

If **TRUE**, the key becomes singleton. If **FALSE**, the key is no longer singleton.

## Return value

This method returns an **HRESULT** indicating the status of the method call.
