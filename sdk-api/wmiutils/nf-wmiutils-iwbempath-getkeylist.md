# IWbemPath::GetKeyList

## Description

The
**IWbemPath::GetKeyList** method retrieves an
[IWbemPathKeyList](https://learn.microsoft.com/windows/desktop/api/wmiutils/nn-wmiutils-iwbempathkeylist) pointer so that the individual key may be accessed.

## Parameters

### `pOut` [out]

Pointer to an
[IWbemPathKeyList](https://learn.microsoft.com/windows/desktop/api/wmiutils/nn-wmiutils-iwbempathkeylist) object.

## Return value

This method returns an **HRESULT** indicating the status of the method call.
