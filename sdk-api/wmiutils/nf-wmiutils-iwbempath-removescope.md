# IWbemPath::RemoveScope

## Description

The **IWbemPath::RemoveScope** method removes a scope
based on the index.

## Parameters

### `uIndex` [in]

Index of the scope to be removed.

## Return value

This method returns an **HRESULT** with one of the following values.
