# IDirect3DDevice9::GetVertexDeclaration

## Description

Gets a vertex shader declaration.

## Parameters

### `ppDecl` [out]

Type: **[IDirect3DVertexDeclaration9](https://learn.microsoft.com/windows/desktop/api/d3d9helper/nn-d3d9helper-idirect3dvertexdeclaration9)****

Pointer to an [IDirect3DVertexDeclaration9](https://learn.microsoft.com/windows/desktop/api/d3d9helper/nn-d3d9helper-idirect3dvertexdeclaration9) object that is returned.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If the method succeeds, the return value is D3D_OK.
The return value can be D3DERR_INVALIDCALL.
