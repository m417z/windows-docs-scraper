# IDirect3DQuery9::GetDataSize

## Description

Gets the number of bytes in the query data.

## Return value

Type: **[DWORD](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

Returns the number of bytes of query data.
