# IDirect3DDevice9Ex::TestCooperativeLevel

## Description

[**TestCooperativeLevel** is no longer available for use. Instead, use [CheckDeviceState](https://learn.microsoft.com/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-checkdevicestate).]

Reports the current cooperative-level status of the Direct3D device for a windowed or full-screen application.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Always returns D3D_OK.
