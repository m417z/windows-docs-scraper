# Graphics::SetCompositingQuality

## Description

The **Graphics::SetCompositingQuality** method sets the compositing quality of this [Graphics](https://learn.microsoft.com/windows/desktop/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) object.

## Parameters

### `compositingQuality` [in]

Type: **[CompositingQuality](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-compositingquality)**

Element of the [CompositingQuality](https://learn.microsoft.com/windows/desktop/api/gdiplusenums/ne-gdiplusenums-compositingquality) enumeration that specifies the compositing quality.

## Return value

Type: **[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status)**

If the method succeeds, it returns Ok, which is an element of the [Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status) enumeration.

If the method fails, it returns one of the other elements of the [Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status) enumeration.
