# Graphics::GetDpiX

## Description

The **Graphics::GetDpiX** method gets the horizontal resolution, in dots per inch, of the display device associated with this
[Graphics](https://learn.microsoft.com/windows/desktop/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) object.

## Return value

Type: **REAL**

This method returns the horizontal resolution, in dots per inch, of the display device associated with this
[Graphics](https://learn.microsoft.com/windows/desktop/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) object.
