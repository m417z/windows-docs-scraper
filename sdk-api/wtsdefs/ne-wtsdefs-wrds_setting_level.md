# WRDS_SETTING_LEVEL enumeration

## Description

Specifies the type of structure contained in the **WRdsSetting** member of the [WRDS_SETTINGS](https://learn.microsoft.com/windows/desktop/api/wtsdefs/ns-wtsdefs-wrds_settings) structure.

## Constants

### `WRDS_SETTING_LEVEL_INVALID`

The type of structure is not defined.

### `WRDS_SETTING_LEVEL_1`

The structure is a [WRDS_SETTINGS_1](https://learn.microsoft.com/windows/desktop/api/wtsdefs/ns-wtsdefs-wrds_settings_1) structure.
