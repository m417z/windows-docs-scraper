# WTS_RCM_SERVICE_STATE enumeration

## Description

Contains information about the state of the Remote Desktop Services service.

## Constants

### `WTS_SERVICE_NONE`

There has been no change in the state of the service.

### `WTS_SERVICE_START`

The RCM service is starting.

### `WTS_SERVICE_STOP`

The RCM service is stopping.

## Remarks

This enumeration type is used by the [WTS_SERVICE_STATE](https://learn.microsoft.com/windows/desktop/api/wtsdefs/ns-wtsdefs-wts_service_state) structure.
