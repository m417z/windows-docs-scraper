# WRDS_LISTENER_SETTINGS structure

## Description

Contains listener setting information for a remote session.

## Members

### `WRdsListenerSettingLevel`

A value of the [WRDS_LISTENER_SETTING_LEVEL](https://learn.microsoft.com/windows/desktop/api/wtsdefs/ne-wtsdefs-wrds_listener_setting_level) enumeration that specifies the type of structure that is contained in the **WRdsListenerSetting** member.

#### WRDS_LISTENER_SETTING_LEVEL_1

The structure is a [WRDS_LISTENER_SETTINGS_1](https://learn.microsoft.com/windows/desktop/api/wtsdefs/ns-wtsdefs-wrds_listener_settings_1) structure.

### `WRdsListenerSetting`

A [WRDS_LISTENER_SETTING](https://learn.microsoft.com/windows/desktop/api/wtsdefs/ns-wtsdefs-wrds_listener_setting) structure that specifies the listener settings.
