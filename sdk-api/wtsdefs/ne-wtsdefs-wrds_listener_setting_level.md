# WRDS_LISTENER_SETTING_LEVEL enumeration

## Description

Used to specify the type of structure that is contained in the **WRdsListenerSetting** member of the [WRDS_LISTENER_SETTINGS](https://learn.microsoft.com/windows/desktop/api/wtsdefs/ns-wtsdefs-wrds_listener_settings) structure.

## Constants

### `WRDS_LISTENER_SETTING_LEVEL_INVALID`

The type of structure is not defined.

### `WRDS_LISTENER_SETTING_LEVEL_1`

The structure is a [WRDS_LISTENER_SETTINGS_1](https://learn.microsoft.com/windows/desktop/api/wtsdefs/ns-wtsdefs-wrds_listener_settings_1) structure.
