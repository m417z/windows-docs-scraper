# WTS_CERT_TYPE enumeration

## Description

Contains values that specify the type of certificate used to obtain a license.

## Constants

### `WTS_CERT_TYPE_INVALID:0`

The certificate is not valid.

### `WTS_CERT_TYPE_PROPRIETORY:1`

The certificate is a custom type.

### `WTS_CERT_TYPE_X509:2`

The certificate adheres to the X.509 standard.

## Remarks

This enumeration is used by the [WTS_LICENSE_CAPABILITIES](https://learn.microsoft.com/windows/desktop/api/wtsdefs/ns-wtsdefs-wts_license_capabilities) structure.
