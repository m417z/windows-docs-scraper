# USB_CLOSE_RAW_DEVICE_PARAMETERS structure

## Description

This structure is not supported.

The **USB_CLOSE_RAW_DEVICE_PARAMETERS** structure is used with the [IOCTL_USB_USER_REQUEST](https://learn.microsoft.com/windows/desktop/api/usbuser/ni-usbuser-ioctl_usb_user_request) I/O control request to close raw access to devices on the bus.

## Members

### `xxx`

Reserved.

## Remarks

The USB_CLOSE_RAW_DEVICE_PARAMETERS structure is used with the USBUSER_OP_CLOSE_RAW_DEVICE user-mode request. For a description of this request, see [IOCTL_USB_USER_REQUEST](https://learn.microsoft.com/windows/desktop/api/usbuser/ni-usbuser-ioctl_usb_user_request).
