# Matrix::Equals

## Description

The **Matrix::Equals** method determines whether the elements of this matrix are equal to the elements of another matrix.

## Parameters

### `matrix` [in]

Type: **const Matrix***

Pointer to a
[Matrix](https://learn.microsoft.com/windows/desktop/api/gdiplusmatrix/nl-gdiplusmatrix-matrix) object that is compared with this
**Matrix** object.

## Return value

Type: **BOOL**

If the elements of the two matrices are the same, this method returns **TRUE**; otherwise, it returns **FALSE**.
