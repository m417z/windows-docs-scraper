# Matrix::IsIdentity

## Description

The **Matrix::IsIdentity** method determines whether this matrix is the identity matrix.

## Return value

Type: **BOOL**

If this matrix is the identity matrix, this method returns **TRUE**; otherwise, it returns **FALSE**.
