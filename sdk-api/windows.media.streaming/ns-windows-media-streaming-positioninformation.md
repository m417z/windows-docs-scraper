# PositionInformation structure

## Description

Contains the current values of media playback position information obtained from the DMR.

## Members

### `streaming.PositionInformation.trackInformation`

### `streaming.PositionInformation.relativeTime`

### `relativeTime`

The current playback position of the DMR.

### `trackInformation`

A [TrackInformation](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/hh829004(v=vs.85)) structure that contains the current track number and duration.
