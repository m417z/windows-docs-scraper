# ACL_REVISION_INFORMATION structure

## Description

The **ACL_REVISION_INFORMATION** structure contains revision information about an [ACL](https://learn.microsoft.com/windows/desktop/api/winnt/ns-winnt-acl) structure.

## Members

### `AclRevision`

Specifies a revision number. The current revision number is ACL_REVISION.
