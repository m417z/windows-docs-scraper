# NtCurrentTeb function

## Description

The **NtCurrentTeb** routine returns a pointer to the Thread Environment Block ([TEB](https://learn.microsoft.com/windows/desktop/api/winternl/ns-winternl-teb)) of the current thread.

## Return value

A pointer to the thread environment block of the current thread.
