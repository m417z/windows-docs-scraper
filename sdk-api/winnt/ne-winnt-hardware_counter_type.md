# HARDWARE_COUNTER_TYPE enumeration

## Description

Defines the types of hardware counters being profiled.

## Constants

### `PMCCounter`

Hardware performance monitoring counters.

### `MaxHardwareCounterType`

The maximum enumeration value for this enumeration.
