# RtlDeleteGrowableFunctionTable function

## Description

Informs the system that a previously reported dynamic function table is no longer in use.

## Parameters

### `DynamicTable` [in]

An opaque reference returned by [RtlAddGrowableFunctionTable.](https://learn.microsoft.com/windows/desktop/api/winnt/nf-winnt-rtladdgrowablefunctiontable)

## Return value

This function does not return a value.
