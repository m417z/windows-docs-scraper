# RtlCaptureContext function

## Description

Retrieves a context record in the context of the caller.

## Parameters

### `ContextRecord` [out]

A pointer to a [CONTEXT](https://learn.microsoft.com/windows/win32/api/winnt/ns-winnt-arm64_nt_context) structure.

## Return value

This function does not return a value.
