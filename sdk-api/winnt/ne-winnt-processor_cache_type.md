# PROCESSOR_CACHE_TYPE enumeration

## Description

Represents the type of processor cache identified in the corresponding [CACHE_DESCRIPTOR](https://learn.microsoft.com/windows/desktop/api/winnt/ns-winnt-cache_descriptor) structure.

## Constants

### `CacheUnified`

The cache is unified.

### `CacheInstruction`

The cache is for processor instructions.

### `CacheData`

The cache is for data.

### `CacheTrace`

The cache is for traces.
