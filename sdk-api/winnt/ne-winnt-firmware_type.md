# FIRMWARE_TYPE enumeration

## Description

Specifies a firmware type.

## Constants

### `FirmwareTypeUnknown`

The firmware type is unknown.

### `FirmwareTypeBios`

The computer booted in legacy BIOS mode.

### `FirmwareTypeUefi`

The computer booted in UEFI mode.

### `FirmwareTypeMax`

Not implemented.
