# ACL_INFORMATION_CLASS enumeration

## Description

The **ACL_INFORMATION_CLASS** enumeration contains values that specify the type of information being assigned to or retrieved from an [access control list](https://learn.microsoft.com/windows/desktop/SecGloss/a-gly) (ACL).

## Constants

### `AclRevisionInformation:1`

Indicates ACL revision information.

### `AclSizeInformation`

Indicates ACL size information.
