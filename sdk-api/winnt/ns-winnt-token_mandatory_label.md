# TOKEN_MANDATORY_LABEL structure

## Description

The **TOKEN_MANDATORY_LABEL** structure specifies the mandatory integrity level for a token.

## Members

### `Label`

A [SID_AND_ATTRIBUTES](https://learn.microsoft.com/windows/desktop/api/winnt/ns-winnt-sid_and_attributes) structure that specifies the mandatory integrity level of the token.
