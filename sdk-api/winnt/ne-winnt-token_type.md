# TOKEN_TYPE enumeration

## Description

The **TOKEN_TYPE** enumeration contains values that differentiate between a [primary token](https://learn.microsoft.com/windows/desktop/SecGloss/p-gly) and an [impersonation token](https://learn.microsoft.com/windows/desktop/SecGloss/i-gly).

## Constants

### `TokenPrimary:1`

Indicates a primary token.

### `TokenImpersonation`

Indicates an impersonation token.
