# ACDGROUP_EVENT enumeration

## Description

The
**ACDGROUP_EVENT** enum describes ACD group events. The
[ITACDGroupEvent::get_Event](https://learn.microsoft.com/windows/desktop/api/tapi3/nf-tapi3-itacdgroupevent-get_event) method returns a member of this enum to indicate the type of ACD group event that occurred.

## Constants

### `ACDGE_NEW_GROUP:0`

A new ACD group has been added.

### `ACDGE_GROUP_REMOVED`

An ACD group has been removed.
