# SHChangeNotifyRegisterThread function

## Description

Enables asynchronous register and deregister of a thread.

## Parameters

### `status`

Type: **[SCNRT_STATUS](https://learn.microsoft.com/windows/desktop/api/shlobj_core/ne-shlobj_core-scnrt_status)**

Indicates whether the function is being used to register or deregister the thread. One of the values of [SCNRT_STATUS](https://learn.microsoft.com/windows/desktop/api/shlobj_core/ne-shlobj_core-scnrt_status).
