# IADesktopP2::UpdateAllDesktopSubscriptions

## Description

Calls the [UpdateAllDesktopSubscriptions](https://learn.microsoft.com/windows/win32/api/shlobj/nf-shlobj-iadesktopp2-updatealldesktopsubscriptions) function to update desktop subscriptions.

## Return value

Type: **HRESULT**

Returns S_OK if successful, or an error code otherwise.
