# IShellFolderBand::GetBandInfoSFB

## Description

Gets information concerning an [IShellFolderBand](https://learn.microsoft.com/windows/desktop/api/shlobj/nn-shlobj-ishellfolderband) object and places it in a [BANDINFOSFB](https://learn.microsoft.com/windows/desktop/api/shlobj/ns-shlobj-bandinfosfb) structure.

## Parameters

### `pbi`

Type: **PBANDINFOSFB**

A pointer to a [BANDINFOSFB](https://learn.microsoft.com/windows/desktop/api/shlobj/ns-shlobj-bandinfosfb) structure.

## Return value

Type: **HRESULT**

Returns S_OK if successful, or an error code otherwise.
