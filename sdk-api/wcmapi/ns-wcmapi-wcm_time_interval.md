# WCM_TIME_INTERVAL structure

## Description

The **WCM_TIME_INTERVAL** structure defines a time interval.

## Members

### `wYear`

Type: **WORD**

Years.

### `wMonth`

Type: **WORD**

Months.

### `wDay`

Type: **WORD**

Days.

### `wHour`

Type: **WORD**

Hours.

### `wMinute`

Type: **WORD**

Minutes.

### `wSecond`

Type: **WORD**

Seconds.

### `wMilliseconds`

Type: **WORD**

Milliseconds.
