# WCM_MEDIA_TYPE enumeration

## Description

The **WCM_MEDIA_TYPE** enumerated type specifies the type of media for a connection.

## Constants

### `wcm_media_unknown`

Unknown media.

### `wcm_media_ethernet`

Ethernet.

### `wcm_media_wlan`

WLAN.

### `wcm_media_mbn`

Mobile broadband.

### `wcm_media_invalid`

Invalid type.

### `wcm_media_max`

Maximum value for testing purposes.
