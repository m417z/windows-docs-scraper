# MetafileHeader::IsDisplay

## Description

The **MetafileHeader::IsDisplay** method determines whether the associated metafile was recorded against a video display device context.

## Return value

Type: **BOOL**

If the associated metafile is in the EMF+ format and was recorded against a video display device context, this method returns **TRUE**; otherwise, it returns **FALSE**.
