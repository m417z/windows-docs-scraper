# MetafileHeader::IsEmfPlusOnly

## Description

The **MetafileHeader::IsEmfPlusOnly** method determines whether the associated metafile is in the EMF+ Only format.

## Return value

Type: **BOOL**

If the associated metafile is in the EMF+ Only format, this method returns **TRUE**; otherwise, it returns **FALSE**. In particular, this method returns **FALSE** if the associated metafile is in the WMF, EMF, or EMF+ Dual format.
