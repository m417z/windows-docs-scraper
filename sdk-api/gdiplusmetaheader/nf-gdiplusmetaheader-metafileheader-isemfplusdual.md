# MetafileHeader::IsEmfPlusDual

## Description

The **MetafileHeader::IsEmfPlusDual** method determines whether the associated metafile is in the EMF+ Dual format.

## Return value

Type: **BOOL**

If the associated metafile is in the EMF+ Dual format, this method returns **TRUE**; otherwise, it returns **FALSE**. In particular, this method returns **FALSE** if the associated metafile is in the WMF, EMF, or EMF+ Only format.
