# MetafileHeader::GetMetafileSize

## Description

The **MetafileHeader::GetMetafileSize** method gets the size, in bytes, of the metafile.

## Return value

Type: **UINT**

This method returns a **UINT** that specifies the size, in bytes, of the metafile.
