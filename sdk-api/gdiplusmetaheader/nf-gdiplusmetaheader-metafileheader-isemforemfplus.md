# MetafileHeader::IsEmfOrEmfPlus

## Description

The **MetafileHeader::IsEmfOrEmfPlus** method determines whether the associated metafile is in either the EMF or EMF+ format.

## Return value

Type: **BOOL**

If the associated metafile is in the EMF or EMF+ format, this method returns **TRUE**; otherwise, it returns **FALSE**. In particular, this method returns **FALSE** if the associated metafile is in the WMF format.
