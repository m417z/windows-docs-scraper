# ITextRow::Insert

## Description

Inserts a row, or rows, at the location identified by the associated [ITextRange2](https://learn.microsoft.com/windows/desktop/api/tom/nn-tom-itextrange2) object.

## Parameters

### `cRow` [in]

Type: **long**

The number of rows to insert.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If the method succeeds, it returns **NOERROR**. Otherwise, it returns an **HRESULT** error code.
