# ITextFont2::SetCharRep

## Description

Sets the character repertoire (writing system).

## Parameters

### `Value` [in]

Type: **long**

The new character repertoire. For a list of possible values, see [ITextFont2::GetCharRep](https://learn.microsoft.com/windows/desktop/api/tom/nf-tom-itextfont2-getcharrep).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If the method succeeds, it returns **NOERROR**. Otherwise, it returns an **HRESULT** error code.
