# ITextFont2::SetProperty

## Description

Sets the value of the specified property.

## Parameters

### `Type` [in]

Type: **long**

The ID of the property value to set. Types are defined by TOM. For a list of types, see [ITextFont2::GetProperty](https://learn.microsoft.com/windows/desktop/api/tom/nf-tom-itextfont2-getproperty).

### `Value` [in]

Type: **long**

The new property value.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If the method succeeds, it returns **NOERROR**. Otherwise, it returns an **HRESULT** error code.
