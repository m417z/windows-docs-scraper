# ITextRange2::GetFormattedText2

## Description

Gets an [ITextRange2](https://learn.microsoft.com/windows/desktop/api/tom/nn-tom-itextrange2) object with the current range's formatted text.

## Parameters

### `ppRange` [out, retval]

Type: **[ITextRange2](https://learn.microsoft.com/windows/desktop/api/tom/nn-tom-itextrange2)****

The formatted text.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If the method succeeds, it returns **NOERROR**. Otherwise, it returns an **HRESULT** error code.
