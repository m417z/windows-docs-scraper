# ITextRange2::GetFont2

## Description

Gets an [ITextFont2](https://learn.microsoft.com/windows/desktop/api/tom/nn-tom-itextfont2) object with the character attributes of the current range.

## Parameters

### `ppFont` [out, retval]

Type: **ITextFont2****

The [ITextFont2](https://learn.microsoft.com/windows/desktop/api/tom/nn-tom-itextfont2) object.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If the method succeeds, it returns **NOERROR**. Otherwise, it returns an **HRESULT** error code.
