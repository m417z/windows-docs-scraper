# ITextStory::SetType

## Description

Sets the story type.

## Parameters

### `Value` [in]

Type: **long**

The story type. The type values are defined in [ITextStory::GetType](https://learn.microsoft.com/windows/desktop/api/tom/nf-tom-itextstory-gettype).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If the method succeeds, it returns **NOERROR**. Otherwise, it returns an **HRESULT** error code.
