# ITextRow::SetCellColorBack

## Description

Sets the background color of the active cell.

## Parameters

### `Value` [in]

Type: **long**

The background color.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If the method succeeds, it returns **NOERROR**. Otherwise, it returns an **HRESULT** error code.

## Remarks

 See [ITextRow::GetCellShading](https://learn.microsoft.com/windows/desktop/api/tom/nf-tom-itextrow-getcellshading) to see how the background color is used together with the foreground color.
