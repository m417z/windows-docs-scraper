# ITextPara2::SetProperty

## Description

Sets the property value.

## Parameters

### `Type` [in]

Type: **long**

The property ID of the property value to set. See [ITextPara2::GetProperty](https://learn.microsoft.com/windows/desktop/api/tom/nf-tom-itextpara2-getproperty) for a list of defined properties.

### `Value` [in]

Type: **long**

The property value to set.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If the method succeeds, it returns **NOERROR**. Otherwise, it returns an **HRESULT** error code.
