# ITextDocument2::SetProperty

## Description

Specifies a new value for a property.

## Parameters

### `Type` [in]

Type: **long**

The identifier of the property. For a list of possible property identifiers, see  [GetProperty](https://learn.microsoft.com/windows/desktop/api/tom/nf-tom-itextdocument2-getproperty).

### `Value` [in]

Type: **long**

The new property value.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If the method succeeds, it returns **NOERROR**. Otherwise, it returns an **HRESULT** error code.
