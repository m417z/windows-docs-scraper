# ITextStory::GetIndex

## Description

Gets the index of a story.

## Parameters

### `pValue` [out, retval]

Type: **long***

The index.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If the method succeeds, it returns **NOERROR**. Otherwise, it returns an **HRESULT** error code.

## Remarks

The index is used with the [ITextDocument2:: GetStory](https://learn.microsoft.com/windows/desktop/api/tom/nf-tom-itextdocument2-getstory) method.
