# ITextRange2::GetPara2

## Description

Gets an [ITextPara2](https://learn.microsoft.com/windows/desktop/api/tom/nn-tom-itextpara2) object with the paragraph attributes of a range.

## Parameters

### `ppPara` [out, retval]

Type: **ITextPara2****

The [ITextPara2](https://learn.microsoft.com/windows/desktop/api/tom/nn-tom-itextpara2) object.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If the method succeeds, it returns **NOERROR**. Otherwise, it returns an **HRESULT** error code.
