# IFaxServer::get_OutboundRouting

## Description

The **IFaxServer::get_OutboundRouting** property creates a [IFaxOutboundRouting](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxoutboundrouting) configuration interface. The interface permits users to configure outbound routing groups and rules.

This property is read-only.

## Parameters

## Remarks

This property is not supported in Windows XP, and will return the error: [FAX_E_NOT_SUPPORTED_ON_THIS_SKU](https://learn.microsoft.com/previous-versions/windows/desktop/fax/-mfax-fax-error-codes).
