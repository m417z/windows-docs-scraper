# IFaxEventLogging::get_GeneralEventsLevel

## Description

The **IFaxEventLogging::get_GeneralEventsLevel** property indicates the level of detail at which the fax service logs general events in the application log. General events include those that are not related to initialization and termination or to inbound and outbound transmissions.

This property is read/write.

## Parameters

## Remarks

To read or to write to this property, a user must have the [farQUERY_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) access right.
