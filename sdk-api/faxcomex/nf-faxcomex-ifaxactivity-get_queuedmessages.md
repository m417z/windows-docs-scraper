# IFaxActivity::get_QueuedMessages

## Description

The **IFaxActivity::get_QueuedMessages** property is a number that represents the total number of fax jobs in the fax job queue that are pending processing. This does not include jobs for which the number of retries has been exceeded.

This property is read-only.

## Parameters

## Remarks

To read this property, a user must have the [farQUERY_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) access right.
