# IFaxOutgoingJob2::get_HasCoverPage

## Description

Specifies if the fax has a cover page.

This property is read-only.

## Parameters

## Remarks

The value is VARIANT_TRUE if there is a cover page; otherwise VARIANT_FALSE.
