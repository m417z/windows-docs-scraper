# IFaxServer2::get_Security2

## Description

The **IFaxServer2::Security2** property returns a [IFaxSecurity2](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxsecurity2) object used to configure security on the fax server.

This property is read-only.
