# IFaxReceiptOptions::put_SMTPPassword

## Description

The **IFaxReceiptOptions::get_SMTPPassword** property is a null-terminated string that contains the Simple Mail Transport Protocol (SMTP) password used for authenticated connections.

This property is read/write.

## Parameters

## Remarks

To read or to write to this property, a user must have the [farQUERY_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) access right.
