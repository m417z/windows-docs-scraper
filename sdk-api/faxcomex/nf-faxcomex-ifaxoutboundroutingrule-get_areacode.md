# IFaxOutboundRoutingRule::get_AreaCode

## Description

The **IFaxOutboundRoutingRule::get_AreaCode** property specifies the area code to which the outbound routing rule applies.

This property is read-only.

## Parameters

## Remarks

If this property is equal to [frrcANY_CODE](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_routing_rule_code_enum), the outbound routing rule applies to all area codes.

To read this property, a user must have the [farQUERY_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) access right.
