# IFaxServer::get_Folders

## Description

The **IFaxServer::get_Folders** property accesses a [IFaxFolders](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxfolders) configuration interface. You can use the interface to access the folders, jobs, and messages on a connected fax server.

This property is read-only.
