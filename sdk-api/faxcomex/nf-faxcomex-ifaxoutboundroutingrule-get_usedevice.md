# IFaxOutboundRoutingRule::get_UseDevice

## Description

The **IFaxOutboundRoutingRule::get_UseDevice** property is a Boolean value that indicates whether the outbound routing rule points to a single fax device.

This property is read/write.

## Parameters

## Remarks

To read or to write to this property, a user must have the [farQUERY_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) access right.
