# IFaxOutgoingQueue::put_DiscountRateEnd

## Description

The **IFaxOutgoingQueue::get_DiscountRateEnd** property is a value that indicates the time at which the discount period for transmitting faxes ends. The discount period applies to outgoing faxes.

This property is read/write.

## Parameters

## Remarks

To read or to write to this property, a user must have the [farQUERY_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) access right.
