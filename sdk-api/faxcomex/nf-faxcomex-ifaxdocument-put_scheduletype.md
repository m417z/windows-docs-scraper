# IFaxDocument::put_ScheduleType

## Description

The **IFaxDocument::get_ScheduleType** property indicates when to schedule the fax job; for example, you can specify that the fax service send the fax immediately, at a specified time, or during a predefined discount period.

This property is read/write.

## Parameters

## Remarks

By default, **IFaxDocument::get_ScheduleType** is set to [fstNOW](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_schedule_type_enum), indicating that the fax will be sent as soon as the device is available.
