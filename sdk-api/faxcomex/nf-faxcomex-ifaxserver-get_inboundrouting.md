# IFaxServer::get_InboundRouting

## Description

The **IFaxServer::get_InboundRouting** property creates a [IFaxInboundRouting](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxinboundrouting) configuration interface. The interface permits access to an inbound fax routing extension and its methods.

This property is read-only.
