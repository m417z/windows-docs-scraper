# IFaxOutgoingMessage2::put_Read

## Description

Indicates if the fax has been read.

**Note** This property is supported only on Windows Vista and later.

This property is read/write.

## Parameters

## Remarks

Possible values are VARIANT_TRUE and VARIANT_FALSE.

A change to this value is not committed to the server until [IFaxOutgoingMessage2::Save](https://learn.microsoft.com/previous-versions/windows/desktop/fax/-mfax-faxoutgoingmessage-save-vb) is called.
