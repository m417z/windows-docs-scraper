# IFaxActivity::get_OutgoingMessages

## Description

The **IFaxActivity::get_OutgoingMessages** property is a number that represents the total number of outgoing fax jobs that the fax service is in the process of sending.

This property is read-only.

## Parameters

## Remarks

To read this property, a user must have the [farQUERY_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) access right.
