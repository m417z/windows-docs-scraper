# IFaxServer::get_LoggingOptions

## Description

The **IFaxServer::get_LoggingOptions** property creates a [IFaxLoggingOptions](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxloggingoptions) configuration interface. The interface permits configuration of both the activity logging options and the event logging categories that the fax service uses.

This property is read-only.
