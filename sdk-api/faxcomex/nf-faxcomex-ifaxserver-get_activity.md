# IFaxServer::get_Activity

## Description

The **IFaxServer::get_Activity** property creates a [IFaxActivity](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxactivity) interface object. The interface permits a fax client application to access information about the activity on a connected fax server, and the fax server status.

This property is read-only.
