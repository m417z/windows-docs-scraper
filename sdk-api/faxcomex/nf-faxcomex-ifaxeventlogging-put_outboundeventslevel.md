# IFaxEventLogging::put_OutboundEventsLevel

## Description

The **IFaxEventLogging::get_OutboundEventsLevel** property indicates the level of detail at which the fax service logs events about outbound fax transmissions in the application log.

This property is read/write.

## Parameters

## Remarks

To read or to write to this property, a user must have the [farQUERY_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) access right.
