# IFaxOutgoingQueue::get_Retries

## Description

The **IFaxOutgoingQueue::get_Retries** property is a value that indicates the number of times that the fax service attempts to retransmit an outgoing fax when the initial transmission fails.

This property is read/write.

## Parameters

## Remarks

To read or to write to this property, a user must have the [farQUERY_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) access right.
