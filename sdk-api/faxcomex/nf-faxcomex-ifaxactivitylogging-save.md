# IFaxActivityLogging::Save

## Description

The **IFaxActivityLogging::Save** method saves the [FaxActivityLogging](https://learn.microsoft.com/previous-versions/windows/desktop/fax/-mfax-faxactivitylogging) object's data.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.

## Remarks

To use this method, a user must have the [farQUERY_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) and [farMANAGE_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) access rights.
