# IFaxInboundRoutingMethod::get_FunctionName

## Description

The **IFaxInboundRoutingMethod::get_FunctionName** property is a null-terminated string that contains the name of the function that executes a specific fax routing procedure.

This property is read-only.

## Parameters

## Remarks

The fax routing extension DLL identified by the [IFaxInboundRoutingMethod::get_ExtensionImageName](https://learn.microsoft.com/previous-versions/windows/desktop/fax/-mfax-faxinboundroutingmethod-extensionimagename-vb) property defines and exports the function.
