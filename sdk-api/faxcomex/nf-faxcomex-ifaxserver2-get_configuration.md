# IFaxServer2::get_Configuration

## Description

The **IFaxServer2::Configuration** property holds a [IFaxConfiguration](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/nn-faxcomex-ifaxconfiguration) object. The object permits a fax client application to access information about the configuration of the connected fax server.

This property is read-only.
