# IFaxOutboundRoutingRule::get_Status

## Description

The **IFaxOutboundRoutingRule::get_Status** property indicates the current status of the outbound routing rule; for example, whether the rule is valid and whether it can apply to fax jobs.

This property is read-only.

## Parameters

## Remarks

To read this property, a user must have the [farQUERY_CONFIG](https://learn.microsoft.com/previous-versions/windows/desktop/api/faxcomex/ne-faxcomex-fax_access_rights_enum) access right.
