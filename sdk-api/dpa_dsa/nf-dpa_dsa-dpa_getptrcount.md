# DPA_GetPtrCount macro

## Syntax

```cpp
int DPA_GetPtrCount(
  [in] HDPA hdpa
);
```

## Return value

Type: **int**

Returns the number of pointers (elements) the DPA contains.

## Description

Gets the number of pointers in a dynamic pointer array (DPA).

## Parameters

### `hdpa` [in]

A handle to an existing DPA.
