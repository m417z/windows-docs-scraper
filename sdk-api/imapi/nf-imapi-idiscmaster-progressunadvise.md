# IDiscMaster::ProgressUnadvise

## Description

Cancels progress notifications for an application.

## Parameters

### `vCookie` [in]

Value returned by a previous call to the
[ProgressAdvise](https://learn.microsoft.com/windows/desktop/api/imapi/nf-imapi-idiscmaster-progressadvise) method.
