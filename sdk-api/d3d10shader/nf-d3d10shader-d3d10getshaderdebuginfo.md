# D3D10GetShaderDebugInfo function

## Description

Get shader debug info. Debug info is generated by [D3D10CompileShader](https://learn.microsoft.com/windows/desktop/api/d3d10shader/nf-d3d10shader-d3d10compileshader) and is embedded in the body of the shader.

## Parameters

### `pShaderBytecode`

Type: **const void***

A pointer to the compiled shader. To get this pointer see [Getting a Pointer to a Compiled Shader](https://learn.microsoft.com/windows/desktop/direct3dhlsl/dx-graphics-hlsl-using-shaders-10).

### `BytecodeLength`

Type: **[SIZE_T](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

Length of the shader bytecode buffer.

### `ppDebugInfo`

Type: **[ID3D10Blob](https://learn.microsoft.com/windows/desktop/api/d3dcommon/nn-d3dcommon-id3d10blob)****

Pointer to an [ID3D10Blob Interface](https://learn.microsoft.com/windows/win32/api/d3d10_1shader/ns-d3d10_1shader-d3d10_shader_debug_info) used to return debug info. For information about the layout of this buffer, see [D3D10_SHADER_DEBUG_INFO](https://learn.microsoft.com/windows/desktop/api/d3d10_1shader/ns-d3d10_1shader-d3d10_shader_debug_info).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the following [Direct3D 10 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues).
