# ID3D10ShaderReflection::GetDesc

## Description

Get a shader description.

## Parameters

### `pDesc` [in]

Type: **[D3D10_SHADER_DESC](https://learn.microsoft.com/windows/desktop/api/d3d10shader/ns-d3d10shader-d3d10_shader_desc)***

A pointer to a shader description. See [D3D10_SHADER_DESC](https://learn.microsoft.com/windows/desktop/api/d3d10shader/ns-d3d10shader-d3d10_shader_desc).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the following [Direct3D 10 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues).
