# ID3D10ShaderReflectionType::GetDesc

## Description

Get the description of a shader-reflection-variable type.

## Parameters

### `pDesc` [in]

Type: **[D3D10_SHADER_TYPE_DESC](https://learn.microsoft.com/windows/desktop/api/d3d10shader/ns-d3d10shader-d3d10_shader_type_desc)***

A pointer to a shader-type description (see [D3D10_SHADER_TYPE_DESC](https://learn.microsoft.com/windows/desktop/api/d3d10shader/ns-d3d10shader-d3d10_shader_type_desc)).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the following [Direct3D 10 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues).
