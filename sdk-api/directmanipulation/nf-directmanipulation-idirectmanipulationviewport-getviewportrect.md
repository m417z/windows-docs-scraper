# IDirectManipulationViewport::GetViewportRect

## Description

Retrieves the rectangle for the viewport relative to the origin of the viewport coordinate system specified by [SetViewportRect](https://learn.microsoft.com/previous-versions/windows/desktop/api/directmanipulation/nf-directmanipulation-idirectmanipulationviewport-setviewportrect).

## Parameters

### `viewport` [out, retval]

The bounding rectangle relative to the viewport coordinate system.

## Return value

If the method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
