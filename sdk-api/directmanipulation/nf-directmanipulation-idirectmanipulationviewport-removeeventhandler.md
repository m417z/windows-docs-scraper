# IDirectManipulationViewport::RemoveEventHandler

## Description

Removes an existing event handler from the viewport.

## Parameters

### `cookie` [in]

A value that was returned by a previous call to [AddEventHandler](https://learn.microsoft.com/previous-versions/windows/desktop/api/directmanipulation/nf-directmanipulation-idirectmanipulationviewport-addeventhandler).

## Return value

If the method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
