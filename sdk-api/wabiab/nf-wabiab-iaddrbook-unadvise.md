## Description

Unregisters the caller from the Windows Address Book (WAB) for notifications.

## Parameters

### `ulConnection`

Value of type ULONG that specifies the connection number returned by the matching call to [IAddrBook::Advise](https://learn.microsoft.com/windows/win32/api/wabiab/nf-wabiab-iaddrbook-advise).

## Return value

HRESULT
