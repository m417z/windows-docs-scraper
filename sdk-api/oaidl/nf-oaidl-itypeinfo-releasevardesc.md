# ITypeInfo::ReleaseVarDesc

## Description

Releases a VARDESC previously returned by [ITypeInfo::GetVarDesc](https://learn.microsoft.com/previous-versions/windows/desktop/api/oaidl/nf-oaidl-itypeinfo-getvardesc).

## Parameters

### `pVarDesc` [in]

The VARDESC to be freed.
