# IRecordInfo::IsMatchingType

## Description

Determines whether the record that is passed in matches that of the current record information.

## Parameters

### `pRecordInfo` [in]

The information of the record.

## Return value

| Return code | Description |
| --- | --- |
| **TRUE** | The record that is passed in matches the current record information. |
| **FALSE** | The record that is passed in does not match the current record information. |
