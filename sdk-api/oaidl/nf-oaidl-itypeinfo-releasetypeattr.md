# ITypeInfo::ReleaseTypeAttr

## Description

Releases a TYPEATTR previously returned by [ITypeInfo::GetTypeAttr](https://learn.microsoft.com/previous-versions/windows/desktop/api/oaidl/nf-oaidl-itypeinfo-gettypeattr).

## Parameters

### `pTypeAttr` [in]

The TYPEATTR to be freed.
