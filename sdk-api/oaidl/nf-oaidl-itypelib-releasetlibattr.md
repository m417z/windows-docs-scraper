# ITypeLib::ReleaseTLibAttr

## Description

Releases the TLIBATTR originally obtained from [GetLibAttr](https://learn.microsoft.com/previous-versions/windows/desktop/api/oaidl/nf-oaidl-itypelib-getlibattr).

## Parameters

### `pTLibAttr` [in]

The TLIBATTR to be freed.
