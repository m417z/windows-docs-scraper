# BINDPTR structure

## Description

Describes a pointer.

## Members

### `lpfuncdesc`

Pointer to a function.

### `lpvardesc`

Pointer to a variable, constant, or data member.

### `lptcomp`

The [ITypeComp](https://learn.microsoft.com/previous-versions/windows/desktop/api/oaidl/nn-oaidl-itypecomp) that binds the pointer.
