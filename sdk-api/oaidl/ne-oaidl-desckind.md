# DESCKIND enumeration

## Description

Identifies the type description being bound to.

## Constants

### `DESCKIND_NONE:0`

No match was found.

### `DESCKIND_FUNCDESC`

A [FUNCDESC](https://learn.microsoft.com/windows/desktop/api/oaidl/ns-oaidl-funcdesc) was returned.

### `DESCKIND_VARDESC`

A [VARDESC](https://learn.microsoft.com/windows/desktop/api/oaidl/ns-oaidl-vardesc) was returned.

### `DESCKIND_TYPECOMP`

A TYPECOMP was returned.

### `DESCKIND_IMPLICITAPPOBJ`

An IMPLICITAPPOBJ was returned.

### `DESCKIND_MAX`

The end of the enum.
