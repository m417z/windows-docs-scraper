# CALLCONV enumeration

## Description

Identifies the calling convention used by a member function described in the METHODDATA structure.

## Constants

### `CC_FASTCALL:0`

### `CC_CDECL:1`

### `CC_MSCPASCAL`

### `CC_PASCAL`

### `CC_MACPASCAL`

### `CC_STDCALL`

### `CC_FPFASTCALL`

### `CC_SYSCALL`

### `CC_MPWCDECL`

### `CC_MPWPASCAL`

### `CC_MAX`
