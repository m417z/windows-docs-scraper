# ITypeInfo::ReleaseFuncDesc

## Description

Releases a FUNCDESC previously returned by [ITypeInfo::GetFuncDesc](https://learn.microsoft.com/previous-versions/windows/desktop/api/oaidl/nf-oaidl-itypeinfo-getfuncdesc).

## Parameters

### `pFuncDesc` [in]

The FUNCDESC to be freed.
