# IWiaVideo::put_PreviewVisible

## Description

The **IWiaVideo::PreviewVisible** property specifies whether the video playback is visible in its parent window. This does not affect the [WIAVIDEO_STATE](https://learn.microsoft.com/windows/win32/api/wiavideo/ne-wiavideo-wiavideo_state) of the video.

This property is read/write.
