## Description

Evaluates to a Boolean value that indicates whether two IIDs are equal.

## Syntax

```cpp
BOOL IsEqualIID(
  riid1,
  riid2
);
```

## Parameters

### `riid1`

The first IID.

### `riid2`

The second IID.

## Return value

`TRUE` if the two IIDs are equal; otherwise, `FALSE`.
