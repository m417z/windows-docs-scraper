# IShellFolderViewDual3::put_SortColumns

## Description

Sets the names of the columns to be sorted.

## Parameters

### `bstrSortColumns` [in]

Type: **[BSTR](https://learn.microsoft.com/previous-versions/windows/desktop/automat/bstr)**

The **BSTR** that contains the names of the columns to be sorted for the current folder.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
