# IShellFolderViewDual3::get_SortColumns

## Description

Gets the names of the columns used to sort the current folder.

## Parameters

### `pbstrSortColumns` [out]

Type: **[BSTR](https://learn.microsoft.com/previous-versions/windows/desktop/automat/bstr)***

A **BSTR** that contains the column names.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
