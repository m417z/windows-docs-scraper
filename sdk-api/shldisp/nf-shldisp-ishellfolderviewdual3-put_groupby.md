# IShellFolderViewDual3::put_GroupBy

## Description

Sets the column used in grouping the folder view.

## Parameters

### `bstrGroupBy` [in]

Type: **[BSTR](https://learn.microsoft.com/previous-versions/windows/desktop/automat/bstr)**

A **BSTR** that contains the column name.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
