# IShellFolderViewDual::get_Script

## Description

Gets the scripting object for the view.

## Parameters

### `ppDisp` [out]

Type: **[IDispatch](https://learn.microsoft.com/previous-versions/windows/desktop/api/oaidl/nn-oaidl-idispatch)****

The scripting object for the view. This represents the scripting automation model.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
