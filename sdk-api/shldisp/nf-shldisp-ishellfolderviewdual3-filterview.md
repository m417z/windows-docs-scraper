# IShellFolderViewDual3::FilterView

## Description

Sets the filter on the contents of the current view.

## Parameters

### `bstrFilterText` [in]

Type: **[BSTR](https://learn.microsoft.com/previous-versions/windows/desktop/automat/bstr)**

The BSTR that names the filter view for the current folder.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
