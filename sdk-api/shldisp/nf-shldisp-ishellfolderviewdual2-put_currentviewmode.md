# IShellFolderViewDual2::put_CurrentViewMode

## Description

Sets the current view mode of the current folder.

## Parameters

### `ViewMode` [in]

Type: **uint**

Sets the current view mode. For a list of possible values see [FOLDERVIEWMODE](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/ne-shobjidl_core-folderviewmode).

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
