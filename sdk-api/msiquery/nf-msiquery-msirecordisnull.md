# MsiRecordIsNull function

## Description

The
**MsiRecordIsNull** function reports a null record field.

## Parameters

### `hRecord` [in]

Handle to a record.

### `iField` [in]

Specifies the field to check.

## Return value

This function returns BOOL.

## Remarks

The *iField* parameter is based on 1 (one).
