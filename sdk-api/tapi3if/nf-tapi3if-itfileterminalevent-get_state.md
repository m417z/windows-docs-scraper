# ITFileTerminalEvent::get_State

## Description

The
**get_State** method gets information on the new file terminal state.

## Parameters

### `pState` [out]

[TERMINAL_MEDIA_STATE](https://learn.microsoft.com/windows/desktop/api/tapi3if/ne-tapi3if-terminal_media_state) descriptor of the new terminal state.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
