# ITLegacyWaveSupport::IsFullDuplex

## Description

The
**IsFullDuplex** method gets an indicator of whether the address supports wave devices.

## Parameters

### `pSupport`

Pointer to
[FULLDUPLEX_SUPPORT](https://learn.microsoft.com/windows/desktop/api/tapi3if/ne-tapi3if-fullduplex_support) enumerator member, such as FDS_SUPPORTED.

## Return value

This method can return one of these values.

| Return code | Description |
| --- | --- |
| **S_OK** | Method succeeded. |
