# ITPhone::put_ButtonFunction

## Description

The
**put_ButtonFunction** method sets the button function.

## Parameters

### `lButtonID` [in]

Button identifier.

### `ButtonFunction` [in]

The
[PHONE_BUTTON_FUNCTION](https://learn.microsoft.com/windows/desktop/api/tapi3if/ne-tapi3if-phone_button_function) descriptor for the button's function.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
