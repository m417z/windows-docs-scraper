# IEnumAddress::Reset

## Description

The
**Reset** method resets to the beginning of the enumeration sequence. This method is hidden from Visual Basic and scripting languages.

## Return value

This method can return one of these values.

| Return code | Description |
| --- | --- |
| **S_OK** | Method succeeded. |
| **E_OUTOFMEMORY** | Insufficient memory exists to perform the operation. |
