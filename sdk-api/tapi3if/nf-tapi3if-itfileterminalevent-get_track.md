# ITFileTerminalEvent::get_Track

## Description

The
**get_Track** method returns the track terminal that generated this event.

## Parameters

### `ppTrackTerminal` [out]

Pointer to
[ITFileTrack](https://learn.microsoft.com/windows/desktop/api/tapi3if/nn-tapi3if-itfiletrack) interface.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
