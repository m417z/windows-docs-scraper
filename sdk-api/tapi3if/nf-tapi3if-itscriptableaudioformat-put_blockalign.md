# ITScriptableAudioFormat::put_BlockAlign

## Description

The
**put_BlockAlign** method sets the **nBlockAlign** member in the
[WAVEFORMATEX](https://learn.microsoft.com/previous-versions/dd757713(v=vs.85)) structure.

## Parameters

### `nNewVal` [in]

New value for the **nBlockAlign** member in the
[WAVEFORMATEX](https://learn.microsoft.com/previous-versions/dd757713(v=vs.85)) structure.

## Return value

This method can return one of these values.

| Return code | Description |
| --- | --- |
| **S_OK** | Method succeeded. |
