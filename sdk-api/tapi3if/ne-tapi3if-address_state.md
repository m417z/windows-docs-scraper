# ADDRESS_STATE enumeration

## Description

The
**ADDRESS_STATE** enum is used by the
[ITAddress::get_State](https://learn.microsoft.com/windows/desktop/api/tapi3if/nf-tapi3if-itaddress-get_state) method to check the address state.

## Constants

### `AS_INSERVICE:0`

Normal state; the address can be used.

### `AS_OUTOFSERVICE`

The address is temporarily out of service, but may go back into service at some time.
