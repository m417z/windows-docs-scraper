# PHONE_BUTTON_MODE enumeration

## Description

The
**PHONE_BUTTON_MODE** enum describes the mode of a phone button.

## Constants

### `PBM_DUMMY:0`

Dummy button.

### `PBM_CALL`

Call button.

### `PBM_FEATURE`

Feature button.

### `PBM_KEYPAD`

Keypad button.

### `PBM_LOCAL`

Local function button, such as mute or volume control.

### `PBM_DISPLAY`

Display button.
