# ITScriptableAudioFormat::put_Channels

## Description

The
**put_Channels** method sets the **nChannels** member in the
[WAVEFORMATEX](https://learn.microsoft.com/previous-versions/dd757713(v=vs.85)) structure.

## Parameters

### `nNewVal` [in]

New value for the **nChannels** member in the
[WAVEFORMATEX](https://learn.microsoft.com/previous-versions/dd757713(v=vs.85)) structure.

## Return value

This method can return one of these values.

| Return code | Description |
| --- | --- |
| **S_OK** | Method succeeded. |
