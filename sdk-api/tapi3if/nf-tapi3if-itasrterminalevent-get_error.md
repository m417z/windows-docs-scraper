# ITASRTerminalEvent::get_Error

## Description

The
**get_Error** method returns an HRESULT cast of the error associated with the terminal event.

## Parameters

### `phrErrorCode` [out]

Pointer to error code.

## Return value

If the method succeeds, it returns S_OK. Otherwise, it returns an error value.
