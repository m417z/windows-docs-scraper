# ITSubStream::StopSubStream

## Description

The
**StopSubStream** method stops the substream. For additional information, see
[ITStream::StopStream](https://learn.microsoft.com/windows/desktop/api/tapi3if/nf-tapi3if-itstream-stopstream).

## Return value

This method can return one of these values.

| Value | Meaning |
| --- | --- |
| **S_OK** | Method succeeded. |
| **TAPI_E_NOTSUPPORTED** | The provider does not support this operation. |
