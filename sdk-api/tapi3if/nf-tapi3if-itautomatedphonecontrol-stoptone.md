# ITAutomatedPhoneControl::StopTone

## Description

The
**StopTone** method ceases any tone that is currently being played. If no tone is being played, the method performs no action and returns success.

## Return value

If the method succeeds, it returns S_OK. Otherwise, it returns an error value.
