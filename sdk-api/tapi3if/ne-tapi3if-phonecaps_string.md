# PHONECAPS_STRING enumeration

## Description

The
**PHONECAPS_STRING** enum is used by methods that set or get phone capabilities described by a string.

## Constants

### `PCS_PHONENAME:0`

Name of the phone.

### `PCS_PHONEINFO`

Phone information string.

### `PCS_PROVIDERINFO`

Phone provider string.
