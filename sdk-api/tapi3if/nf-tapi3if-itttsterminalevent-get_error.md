# ITTTSTerminalEvent::get_Error

## Description

The
**get_Error** method gets an **HRESULT** cast of the error code involved in the terminal event.

## Parameters

### `phrErrorCode` [out]

Pointer to the **HRESULT** cast of the error code.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
