# ITLocationInfo::get_CancelCallWaitingCode

## Description

The
**get_CancelCallWaitingCode** method gets the dial digits and modifier characters that must be prefixed to a dialable string to cancel call waiting.

## Parameters

### `ppCode` [out]

Pointer to **BSTR** representation of dial digits and modifier characters required to cancel call waiting.

## Return value

This method can return one of these values.

| Return code | Description |
| --- | --- |
| **S_OK** | Method succeeded. |
| **E_POINTER** | The *ppCode* parameter is not a valid pointer. |
| **E_OUTOFMEMORY** | Insufficient memory exists to perform the operation. |

## Remarks

The application must use
[SysFreeString](https://learn.microsoft.com/previous-versions/windows/desktop/api/oleauto/nf-oleauto-sysfreestring) to free the memory allocated for the *ppCode* parameter.

The value that this method returns corresponds to the **dwCancelCallWaitingSize** and **dwCancelCallWaitingOffset** members of TAPI 2's
[LINELOCATIONENTRY](https://learn.microsoft.com/windows/desktop/api/tapi/ns-tapi-linelocationentry) structure.
