# PHONECAPS_BUFFER enumeration

## Description

The
**PHONECAPS_BUFFER** enum is used by methods that set or get phone capabilities described by a buffer.

## Constants

### `PCB_DEVSPECIFICBUFFER:0`

Device-specific phone capabilities.
