# ITFileTerminalEvent::get_Cause

## Description

The
**get_Cause** method gets the cause associated with this event.

## Parameters

### `pCause` [out]

[FT_STATE_EVENT_CAUSE](https://learn.microsoft.com/windows/desktop/api/tapi3if/ne-tapi3if-ft_state_event_cause) descriptor of the cause of this event.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
