# PHONE_HOOK_SWITCH_DEVICE enumeration

## Description

The
**PHONE_HOOK_SWITCH_DEVICE** enum is used to indicate the types of switch hooks on a phone device.

## Constants

### `PHSD_HANDSET:0x1`

The handset's hookswitch.

### `PHSD_SPEAKERPHONE:0x2`

The speakerphone's hookswitch.

### `PHSD_HEADSET:0x4`

The headset's hookswitch.
