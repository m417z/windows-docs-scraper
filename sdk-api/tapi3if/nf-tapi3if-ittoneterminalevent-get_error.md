# ITToneTerminalEvent::get_Error

## Description

The
**get_Error** method returns an **HRESULT** cast of the error code involved in the event.

## Parameters

### `phrErrorCode` [out]

**HRESULT** cast of the error code.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
