# ITPhone::put_ButtonText

## Description

The
**put_ButtonText** method sets the button text.

## Parameters

### `lButtonID` [in]

Button identifier.

### `bstrButtonText` [in]

The **BSTR** representation of the button text.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
