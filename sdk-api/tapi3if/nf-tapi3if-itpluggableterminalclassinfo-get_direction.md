# ITPluggableTerminalClassInfo::get_Direction

## Description

The
**get_Direction** method gets the direction supported by the terminal.

## Parameters

### `pDirection` [out]

The
[TERMINAL_DIRECTION](https://learn.microsoft.com/windows/desktop/api/tapi3if/ne-tapi3if-terminal_direction) descriptor for the direction supported by the terminal.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
