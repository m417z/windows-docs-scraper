# ITPhoneEvent::get_Event

## Description

The
**get_Event** method returns a
[PHONE_EVENT](https://learn.microsoft.com/windows/desktop/api/tapi3if/ne-tapi3if-phone_event) value specifying the type of phone event that occurred.

## Parameters

### `pEvent` [out]

The
[PHONE_EVENT](https://learn.microsoft.com/windows/desktop/api/tapi3if/ne-tapi3if-phone_event) descriptor of the event.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
