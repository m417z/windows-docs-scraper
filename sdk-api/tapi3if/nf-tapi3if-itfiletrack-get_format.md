# ITFileTrack::get_Format

## Description

The
**get_Format** method gets the file terminal's format.

## Parameters

### `ppmt` [out]

Pointer to an
**AM_MEDIA_TYPE** descriptor of the terminal format. For more information on **AM_MEDIA_TYPE**, see the DirectX documentation.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
