# ITAutomatedPhoneControl::StopRinger

## Description

The
**StopRinger** method ceases any incoming ring that is currently being produced. If no ring is being produced, the method performs no action and returns success.

## Return value

If the method succeeds, it returns S_OK. Otherwise, it returns an error value.
