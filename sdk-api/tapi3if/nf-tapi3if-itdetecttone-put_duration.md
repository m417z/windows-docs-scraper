# ITDetectTone::put_Duration

## Description

The
**put_Duration** method sets the length of time during which a tone should be present before the TAPI Server generates a tone event.

## Parameters

### `lDuration` [in]

Specifies the tone duration, in milliseconds, during which the specified tone should be present.

## Return value

This method can return one of these values.

| Return code | Description |
| --- | --- |
| **S_OK** | Method succeeded. |
