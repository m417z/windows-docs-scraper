# TERMINAL_STATE enumeration

## Description

The
**TERMINAL_STATE** enum describes the current state of a terminal device. This enum is returned by the
[ITTerminal::get_State](https://learn.microsoft.com/windows/desktop/api/tapi3if/nf-tapi3if-itterminal-get_state) method.

## Constants

### `TS_INUSE:0`

The terminal is currently in use.

### `TS_NOTINUSE`

The terminal is not currently in use.
