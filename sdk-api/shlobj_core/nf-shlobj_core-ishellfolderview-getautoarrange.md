# IShellFolderView::GetAutoArrange

## Description

[**GetAutoArrange** is available for use in the operating systems specified in the Requirements section. It may be altered or unavailable in subsequent versions. Instead, use [IFolderView::GetAutoArrange](https://learn.microsoft.com/windows/desktop/api/shobjidl_core/nf-shobjidl_core-ifolderview-getautoarrange).]

Gets the current state of the folder's Auto Arrange mode.

## Return value

Type: **HRESULT**

Returns S_OK if the folder is in Auto Arrange mode.
