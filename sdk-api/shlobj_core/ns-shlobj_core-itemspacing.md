# ITEMSPACING structure

## Description

Stores the dimensions of the two possible sizes of icon spacing that are available for display: small and large. Used by [IShellFolderView::GetItemSpacing](https://learn.microsoft.com/windows/desktop/api/shlobj_core/nf-shlobj_core-ishellfolderview-getitemspacing).

## Members

### `cxSmall`

Type: **int**

The width of the spacing between icons in small icon mode, in pixels.

### `cySmall`

Type: **int**

The height of the spacing between icons in small icon mode, in pixels.

### `cxLarge`

Type: **int**

The width of the spacing between icons in large icon mode, in pixels.

### `cyLarge`

Type: **int**

The height of the spacing between icons in large icon mode, in pixels.
