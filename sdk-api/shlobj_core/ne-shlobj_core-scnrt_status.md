# SCNRT_STATUS enumeration

## Description

Indicates whether to enable or disable Async Register and Deregister for [SHChangeNotifyRegisterThread](https://learn.microsoft.com/windows/desktop/api/shlobj/nf-shlobj-shchangenotifyregisterthread).

## Constants

### `SCNRT_ENABLE:0`

Enable Async Register and Deregister for [SHChangeNotifyRegisterThread](https://learn.microsoft.com/windows/desktop/api/shlobj/nf-shlobj-shchangenotifyregisterthread).

### `SCNRT_DISABLE:1`

Disable Async Register and Deregister for [SHChangeNotifyRegisterThread](https://learn.microsoft.com/windows/desktop/api/shlobj/nf-shlobj-shchangenotifyregisterthread).
