# ISearchContext::GetSearchText

## Description

Retrieves the text that is in the browser's Address bar.

## Parameters

### `pbstrSearchText` [in]

Type: **[BSTR](https://learn.microsoft.com/previous-versions/windows/desktop/automat/bstr)**

The **BSTR** that receives the text in the Address bar.

## Return value

Type: **HRESULT**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
