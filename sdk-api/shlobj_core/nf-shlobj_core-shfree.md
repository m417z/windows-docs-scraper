# SHFree function

## Description

[This function is available through Windows XP Service Pack 2 (SP2) and Windows Server 2003. It might be altered or unavailable in subsequent versions of Windows. Use [CoTaskMemFree](https://learn.microsoft.com/windows/desktop/api/combaseapi/nf-combaseapi-cotaskmemfree) instead.]

Frees the memory allocated by [SHAlloc](https://learn.microsoft.com/windows/desktop/api/shlobj_core/nf-shlobj_core-shalloc).

## Parameters

### `pv` [in]

Type: **void***

A pointer to the memory allocated by [SHAlloc](https://learn.microsoft.com/windows/desktop/api/shlobj_core/nf-shlobj_core-shalloc).
