# IObjMgr::Append

## Description

Appends an object to the collection of managed objects.

## Parameters

### `punk` [in]

Type: **[IUnknown](https://learn.microsoft.com/windows/desktop/api/unknwn/nn-unknwn-iunknown)***

The address of the [IUnknown](https://learn.microsoft.com/windows/desktop/api/unknwn/nn-unknwn-iunknown) interface of the object to be added to the list.

## Return value

Type: **HRESULT**

Returns S_OK if successful, or a COM error code otherwise.
