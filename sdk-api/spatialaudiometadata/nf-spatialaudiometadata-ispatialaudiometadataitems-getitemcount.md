# ISpatialAudioMetadataItems::GetItemCount

## Description

The current number of items stored by the [ISpatialAudioMetadataItems](https://learn.microsoft.com/windows/desktop/api/spatialaudiometadata/nn-spatialaudiometadata-ispatialaudiometadataitems).

## Parameters

### `itemCount` [out]

The current number of stored items.

## Return value

If the method succeeds, it returns S_OK.
