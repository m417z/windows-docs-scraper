# ISpatialAudioMetadataItems::GetMaxValueBufferLength

## Description

The size of the largest command value defined by the metadata format for the [ISpatialAudioMetadataItems](https://learn.microsoft.com/windows/desktop/api/spatialaudiometadata/nn-spatialaudiometadata-ispatialaudiometadataitems).

## Parameters

### `maxValueBufferLength` [out]

The size of the largest command value defined by the metadata format.

## Return value

If the method succeeds, it returns S_OK.
