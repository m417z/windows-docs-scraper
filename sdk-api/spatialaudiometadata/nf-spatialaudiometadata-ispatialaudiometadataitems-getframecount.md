# ISpatialAudioMetadataItems::GetFrameCount

## Description

Gets the total frame count of the [ISpatialAudioMetadataItems](https://learn.microsoft.com/windows/desktop/api/spatialaudiometadata/nn-spatialaudiometadata-ispatialaudiometadataitems), which defines valid item offsets.

## Parameters

### `frameCount` [out]

The total frame count.

## Return value

If the method succeeds, it returns S_OK.
