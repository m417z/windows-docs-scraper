# ISpatialAudioMetadataItems::GetInfo

## Description

Gets the total frame count for the [ISpatialAudioMetadataItems](https://learn.microsoft.com/windows/desktop/api/spatialaudiometadata/nn-spatialaudiometadata-ispatialaudiometadataitems), which defines valid item offsets.

## Parameters

### `info` [out]

The total frame count, which defines valid item offsets.

## Return value

If the method succeeds, it returns S_OK.
