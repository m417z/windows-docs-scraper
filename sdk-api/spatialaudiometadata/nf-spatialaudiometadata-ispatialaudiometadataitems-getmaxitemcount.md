# ISpatialAudioMetadataItems::GetMaxItemCount

## Description

The maximum number of items allowed by the [ISpatialAudioMetadataItems](https://learn.microsoft.com/windows/desktop/api/spatialaudiometadata/nn-spatialaudiometadata-ispatialaudiometadataitems), defined when the object is created.

## Parameters

### `maxItemCount` [out]

The maximum number of items allowed.

## Return value

If the method succeeds, it returns S_OK.
