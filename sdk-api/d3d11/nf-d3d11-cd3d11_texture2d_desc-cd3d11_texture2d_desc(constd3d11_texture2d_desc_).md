# CD3D11_TEXTURE2D_DESC::CD3D11_TEXTURE2D_DESC(const D3D11_TEXTURE2D_DESC &)

## Description

Instantiates a new instance of a [CD3D11_TEXTURE2D_DESC](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/jj151700(v=vs.85)) structure that is initialized with a [D3D11_TEXTURE2D_DESC](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_texture2d_desc) structure.

## Parameters

### `o` [ref]

Type: **const [D3D11_TEXTURE2D_DESC](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_texture2d_desc)**

Address of the [D3D11_TEXTURE2D_DESC](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_texture2d_desc) structure that initializes the [CD3D11_TEXTURE2D_DESC](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/jj151700(v=vs.85)) structure.
