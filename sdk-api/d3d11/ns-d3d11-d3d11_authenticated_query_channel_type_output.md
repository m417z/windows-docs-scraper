# D3D11_AUTHENTICATED_QUERY_CHANNEL_TYPE_OUTPUT structure

## Description

Contains the response to a **D3D11_AUTHENTICATED_QUERY_CHANNEL_TYPE** query.

## Members

### `Output`

A [D3D11_AUTHENTICATED_QUERY_OUTPUT](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_authenticated_query_output) structure that contains a Message Authentication Code (MAC) and other data.

### `ChannelType`

A [D3D11_AUTHENTICATED_CHANNEL_TYPE](https://learn.microsoft.com/windows/desktop/api/d3d11/ne-d3d11-d3d11_authenticated_channel_type) value that specifies the channel type.
