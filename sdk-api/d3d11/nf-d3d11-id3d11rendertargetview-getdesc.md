# ID3D11RenderTargetView::GetDesc

## Description

Get the properties of a render target view.

## Parameters

### `pDesc` [out]

Type: **[D3D11_RENDER_TARGET_VIEW_DESC](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_render_target_view_desc)***

Pointer to the description of a render target view (see [D3D11_RENDER_TARGET_VIEW_DESC](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_render_target_view_desc)).
