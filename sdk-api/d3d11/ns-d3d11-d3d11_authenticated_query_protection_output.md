# D3D11_AUTHENTICATED_QUERY_PROTECTION_OUTPUT structure

## Description

Contains the response to a **D3D11_AUTHENTICATED_QUERY_PROTECTION** query.

## Members

### `Output`

A [D3D11_AUTHENTICATED_QUERY_OUTPUT](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_authenticated_query_output) structure that contains a Message Authentication Code (MAC) and other data.

### `ProtectionFlags`

A [D3D11_AUTHENTICATED_PROTECTION_FLAGS](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_authenticated_protection_flags) union that specifies the protection level.
