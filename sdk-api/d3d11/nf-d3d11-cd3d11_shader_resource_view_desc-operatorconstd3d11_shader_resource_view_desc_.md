# CD3D11_SHADER_RESOURCE_VIEW_DESC::operator const D3D11_SHADER_RESOURCE_VIEW_DESC&

## Description

This operator returns the address of a [D3D11_SHADER_RESOURCE_VIEW_DESC](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_shader_resource_view_desc) structure that contains the data from the [CD3D11_SHADER_RESOURCE_VIEW_DESC](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/jj151684(v=vs.85)) instance.
