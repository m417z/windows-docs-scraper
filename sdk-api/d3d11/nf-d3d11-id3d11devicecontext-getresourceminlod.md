# ID3D11DeviceContext::GetResourceMinLOD

## Description

Gets the minimum level-of-detail (LOD).

## Parameters

### `pResource` [in]

Type: **[ID3D11Resource](https://learn.microsoft.com/windows/desktop/api/d3d11/nn-d3d11-id3d11resource)***

A pointer to an [ID3D11Resource](https://learn.microsoft.com/windows/desktop/api/d3d11/nn-d3d11-id3d11resource) which represents the resource.

## Return value

Type: **FLOAT**

Returns the minimum LOD.
