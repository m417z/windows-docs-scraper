# ID3D11DepthStencilView::GetDesc

## Description

Get the depth-stencil view.

## Parameters

### `pDesc` [out]

Type: **[D3D11_DEPTH_STENCIL_VIEW_DESC](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_depth_stencil_view_desc)***

Pointer to a depth-stencil-view description (see [D3D11_DEPTH_STENCIL_VIEW_DESC](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_depth_stencil_view_desc)).
