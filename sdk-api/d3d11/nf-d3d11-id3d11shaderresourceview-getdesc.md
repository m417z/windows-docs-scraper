# ID3D11ShaderResourceView::GetDesc

## Description

Get the shader resource view's description.

## Parameters

### `pDesc` [out]

Type: **[D3D11_SHADER_RESOURCE_VIEW_DESC](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_shader_resource_view_desc)***

A pointer to a [D3D11_SHADER_RESOURCE_VIEW_DESC](https://learn.microsoft.com/windows/desktop/api/d3d11/ns-d3d11-d3d11_shader_resource_view_desc) structure to be filled with data about the shader resource view.
