# IADsADSystemInfo::GetAnyDCName

## Description

The **IADsADSystemInfo::GetAnyDCName** method retrieves the DNS name of a domain controller in the local computer's domain.

## Parameters

### `pszDCName` [out]

Name of a domain controller, such as "ADServer1.domain1.Fabrikam.com".

## Return value

This method supports the standard **HRESULT** return values. For more information, see [ADSI Error Codes](https://learn.microsoft.com/windows/desktop/ADSI/adsi-error-codes).
