# IADsADSystemInfo::GetTrees

## Description

The **IADsADSystemInfo::GetTrees** method retrieves the DNS names of all the directory trees in the local computer's forest.

## Parameters

### `pvTrees` [out]

A Variant array of strings that contains the names of the directory trees within the forest.

## Return value

This method supports the standard **HRESULT** return values. For more information, see [ADSI Error Codes](https://learn.microsoft.com/windows/desktop/ADSI/adsi-error-codes).
