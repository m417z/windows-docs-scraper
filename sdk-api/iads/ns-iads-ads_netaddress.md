# ADS_NETADDRESS structure

## Description

The **ADS_NETADDRESS** structure is an ADSI representation of the **Net Address** attribute syntax.

## Members

### `AddressType`

Types of communication protocols.

### `AddressLength`

Address length in bytes.

### `Address`

A network address.
