# IADsPropertyList::Skip

## Description

The **IADsPropertyList::Skip** method skips a specified number of items, counting from the current cursor position, in the property list.

## Parameters

### `cElements` [in]

Number of elements to be skipped.

## Return value

This method supports the standard HRESULT return values, including S_OK. For more information and other return values, see [ADSI Error Codes](https://learn.microsoft.com/windows/desktop/ADSI/adsi-error-codes).
