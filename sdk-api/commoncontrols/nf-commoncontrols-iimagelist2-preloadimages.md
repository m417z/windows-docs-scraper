# IImageList2::PreloadImages

## Description

Preloads images, as specified.

## Parameters

### `pimldp` [in]

Type: **[IMAGELISTDRAWPARAMS](https://learn.microsoft.com/windows/desktop/api/commoncontrols/ns-commoncontrols-imagelistdrawparams)***

A pointer to an [IMAGELISTDRAWPARAMS](https://learn.microsoft.com/windows/desktop/api/commoncontrols/ns-commoncontrols-imagelistdrawparams) structure containing information about an image list draw operation.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
