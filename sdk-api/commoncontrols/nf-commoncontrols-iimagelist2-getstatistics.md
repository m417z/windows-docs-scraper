# IImageList2::GetStatistics

## Description

Gets an image list statistics structure.

## Parameters

### `pils` [in, out]

Type: **[IMAGELISTSTATS](https://learn.microsoft.com/windows/desktop/api/commoncontrols/ns-commoncontrols-imageliststats)***

A pointer to the [IMAGELISTSTATS](https://learn.microsoft.com/windows/desktop/api/commoncontrols/ns-commoncontrols-imageliststats) structure.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
