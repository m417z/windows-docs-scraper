# POWER_POLICY structure

## Description

Contains power policy settings that are unique to each power scheme.

## Members

### `user`

A
[USER_POWER_POLICY](https://learn.microsoft.com/windows/desktop/api/powrprof/ns-powrprof-user_power_policy) structure that defines user power policy settings.

### `mach`

A
[MACHINE_POWER_POLICY](https://learn.microsoft.com/windows/desktop/api/powrprof/ns-powrprof-machine_power_policy) structure that defines computer power policy settings.
