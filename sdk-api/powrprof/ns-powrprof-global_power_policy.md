# GLOBAL_POWER_POLICY structure

## Description

Contains global power policy settings that apply to all power schemes.

## Members

### `user`

A
[GLOBAL_USER_POWER_POLICY](https://learn.microsoft.com/windows/desktop/api/powrprof/ns-powrprof-global_user_power_policy) structure that defines the global user power policy settings.

### `mach`

A
[GLOBAL_MACHINE_POWER_POLICY](https://learn.microsoft.com/windows/desktop/api/powrprof/ns-powrprof-global_machine_power_policy) structure that defines the global computer power policy settings.
