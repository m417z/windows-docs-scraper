# WINML_BINDING_DESC structure

## Description

**These APIs have been deprecated and should no longer be used:** Please use [Windows.AI.MachineLearning](https://learn.microsoft.com/uwp/api/windows.ai.machinelearning) instead.

Contains a description of the WinML binding.

## Members

### `Name`

The name of the WinML binding.

### `BindType`

A [WINML_BINDING_TYPE](https://learn.microsoft.com/windows/desktop/api/winml/ne-winml-winml_binding_type) containing the type of the WinML binding.

### `Tensor`

A [WINML_TENSOR_BINDING_DESC](https://learn.microsoft.com/windows/desktop/api/winml/ns-winml-winml_tensor_binding_desc) containing the description of the tensor binding.

### `Sequence`

A [WINML_SEQUENCE_BINDING_DESC](https://learn.microsoft.com/windows/desktop/api/winml/ns-winml-winml_sequence_binding_desc) containing the description of the sequence binding.

### `Map`

A [WINML_MAP_BINDING_DESC](https://learn.microsoft.com/windows/desktop/api/winml/ns-winml-winml_map_binding_desc) containing the description of the map binding.

### `Image`

A [WINML_IMAGE_BINDING_DESC](https://learn.microsoft.com/windows/desktop/api/winml/ns-winml-winml_image_binding_desc) containing the description of the image binding.

### `Resource`

A [WINML_RESOURCE_BINDING_DESC](https://learn.microsoft.com/windows/desktop/api/winml/ns-winml-winml_resource_binding_desc) containing the description of the resource binding.
