# IWinMLModel::GetDescription

## Description

**These APIs have been deprecated and should no longer be used:** Please use [Windows.AI.MachineLearning](https://learn.microsoft.com/uwp/api/windows.ai.machinelearning) instead.

Retrieves the WinML model description.

## Parameters

### `ppDescription` [out]

A pointer to a [WINML_MODEL_DESC](https://learn.microsoft.com/windows/desktop/api/winml/ns-winml-winml_model_desc) containing the model description.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
