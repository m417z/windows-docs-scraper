# IWinMLRuntime::EvaluateModel

## Description

**These APIs have been deprecated and should no longer be used:** Please use [Windows.AI.MachineLearning](https://learn.microsoft.com/uwp/api/windows.ai.machinelearning) instead.

Evaluates a WinML model.

## Parameters

### `pContext` [in]

A pointer to the [WinMLEvaluationContext](https://learn.microsoft.com/windows/desktop/api/winml/nn-winml-iwinmlevaluationcontext) to evaluate.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
