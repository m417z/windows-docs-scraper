# WINML_IMAGE_BINDING_DESC structure

## Description

**These APIs have been deprecated and should no longer be used:** Please use [Windows.AI.MachineLearning](https://learn.microsoft.com/uwp/api/windows.ai.machinelearning) instead.

Contains properties for the binding of the type image.

## Members

### `ElementType`

A [WINML_TENSOR_DATA_TYPE](https://learn.microsoft.com/windows/desktop/api/winml/ne-winml-winml_tensor_data_type) describing the element tensor data type.

### `NumDimensions`

The image tensor dimension count.

### `pShape`

Pointer to the shape of the image.

### `DataSize`

Size of image data in bytes.

### `pData`

Pointer to the image data.
