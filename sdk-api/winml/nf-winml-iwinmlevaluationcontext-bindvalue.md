# IWinMLEvaluationContext::BindValue

## Description

**These APIs have been deprecated and should no longer be used:** Please use [Windows.AI.MachineLearning](https://learn.microsoft.com/uwp/api/windows.ai.machinelearning) instead.

Binds the input/output to the given model.

## Parameters

### `pDescriptor` [in]

A pointer to a [WINML_BINDING_DESC](https://learn.microsoft.com/windows/desktop/api/winml/ns-winml-winml_binding_desc) containing the input/output binding descriptor.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
