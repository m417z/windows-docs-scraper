# WINML_BINDING_TYPE enumeration

## Description

**These APIs have been deprecated and should no longer be used:** Please use [Windows.AI.MachineLearning](https://learn.microsoft.com/uwp/api/windows.ai.machinelearning) instead.

Specifies the different types of WinML bindings.

## Constants

### `WINML_BINDING_UNDEFINED:0`

Binding type undefined.

### `WINML_BINDING_TENSOR`

Binding of type tensor.

### `WINML_BINDING_SEQUENCE`

Binding of type sequence.

### `WINML_BINDING_MAP`

Binding of type map.

### `WINML_BINDING_IMAGE`

Binding of type image.

### `WINML_BINDING_RESOURCE`

Binding of type resource.
