# WINML_FEATURE_TYPE enumeration

## Description

**These APIs have been deprecated and should no longer be used:** Please use [Windows.AI.MachineLearning](https://learn.microsoft.com/uwp/api/windows.ai.machinelearning) instead.

Specifies the different types of WinML features.

## Constants

### `WINML_FEATURE_UNDEFINED:0`

### `WINML_FEATURE_TENSOR`

### `WINML_FEATURE_SEQUENCE`

### `WINML_FEATURE_MAP`

### `WINML_FEATURE_IMAGE`

#### - WINML_TENSOR_IMAGE

Feature of type image.

#### - WINML_TENSOR_MAP

Feature of type map.

#### - WINML_TENSOR_SEQUENCE

Feature of type sequence.

#### - WINML_TENSOR_TENSOR

Feature of type tensor.

#### - WINML_TENSOR_UNDEFINED

Feature type undefined.
