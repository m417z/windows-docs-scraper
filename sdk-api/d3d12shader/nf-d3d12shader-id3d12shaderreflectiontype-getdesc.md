# ID3D12ShaderReflectionType::GetDesc

## Description

Gets the description of a shader-reflection-variable type.

## Parameters

### `pDesc` [out]

Type: **[D3D12_SHADER_TYPE_DESC](https://learn.microsoft.com/windows/desktop/api/d3d12shader/ns-d3d12shader-d3d12_shader_type_desc)***

A pointer to a shader-type description (see [D3D12_SHADER_TYPE_DESC](https://learn.microsoft.com/windows/desktop/api/d3d12shader/ns-d3d12shader-d3d12_shader_type_desc)).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the [Direct3D 12 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d12/d3d12-graphics-reference-returnvalues).

## Remarks

This method's interface is hosted in the out-of-box DLL D3DCompiler_xx.dll.
