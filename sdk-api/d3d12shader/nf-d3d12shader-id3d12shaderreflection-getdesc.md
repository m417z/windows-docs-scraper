# ID3D12ShaderReflection::GetDesc

## Description

Gets a shader description.

## Parameters

### `pDesc` [out]

Type: **[D3D12_SHADER_DESC](https://learn.microsoft.com/windows/desktop/api/d3d12shader/ns-d3d12shader-d3d12_shader_desc)***

A shader description, as a pointer to a [D3D12_SHADER_DESC](https://learn.microsoft.com/windows/desktop/api/d3d12shader/ns-d3d12shader-d3d12_shader_desc) structure.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the following [Direct3D 12 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d12/d3d12-graphics-reference-returnvalues).

## Remarks

This method's interface is hosted in the out-of-box DLL D3DCompiler_xx.dll.
