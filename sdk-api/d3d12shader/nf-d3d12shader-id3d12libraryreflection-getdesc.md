# ID3D12LibraryReflection::GetDesc

## Description

Fills the library descriptor structure for the library reflection.

## Parameters

### `pDesc` [out]

Type: **[D3D12_LIBRARY_DESC](https://learn.microsoft.com/windows/desktop/api/d3d12shader/ns-d3d12shader-d3d12_library_desc)***

A pointer to a [D3D12_LIBRARY_DESC](https://learn.microsoft.com/windows/desktop/api/d3d12shader/ns-d3d12shader-d3d12_library_desc) structure that receives a description of the library reflection.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the [Direct3D 12 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d12/d3d12-graphics-reference-returnvalues).
