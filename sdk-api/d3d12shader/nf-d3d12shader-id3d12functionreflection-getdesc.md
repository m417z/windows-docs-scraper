# ID3D12FunctionReflection::GetDesc

## Description

Fills the function descriptor structure for the function.

## Parameters

### `pDesc` [out]

Type: **[D3D12_FUNCTION_DESC](https://learn.microsoft.com/windows/desktop/api/d3d12shader/ns-d3d12shader-d3d12_function_desc)***

A pointer to a [D3D12_FUNCTION_DESC](https://learn.microsoft.com/windows/desktop/api/d3d12shader/ns-d3d12shader-d3d12_function_desc) structure that receives a description of the function.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the [Direct3D 12 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d12/d3d12-graphics-reference-returnvalues).
