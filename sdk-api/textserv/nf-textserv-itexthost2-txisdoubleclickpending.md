# ITextHost2::TxIsDoubleClickPending

## Description

Discovers whether the message queue contains a [WM_LBUTTONDBLCLK](https://learn.microsoft.com/windows/desktop/inputdev/wm-lbuttondblclk) message that is pending for the text host window.

## Return value

Type: **[BOOL](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

Returns **TRUE** if a [WM_LBUTTONDBLCLK](https://learn.microsoft.com/windows/desktop/inputdev/wm-lbuttondblclk) message is pending, or **FALSE** if not.
