# RedEyeCorrection::SetParameters

## Description

The **RedEyeCorrection::SetParameters** method sets the parameters of this [RedEyeCorrection](https://learn.microsoft.com/windows/desktop/api/gdipluseffects/nl-gdipluseffects-redeyecorrection) object.

## Parameters

### `parameters` [in]

Type: **const [RedEyeCorrectionParams](https://learn.microsoft.com/windows/desktop/api/gdipluseffects/ns-gdipluseffects-redeyecorrectionparams)***

Pointer to a [RedEyeCorrectionParams](https://learn.microsoft.com/windows/desktop/api/gdipluseffects/ns-gdipluseffects-redeyecorrectionparams) structure that specifies the parameters.

## Return value

Type: **[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status)**

If the method succeeds, it returns **Ok**, which is an element of the
[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status) enumeration.

If the method fails, it returns one of the other elements of the
[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status) enumeration.
