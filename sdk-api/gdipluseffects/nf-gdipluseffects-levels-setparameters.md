# Levels::SetParameters

## Description

The **Levels::SetParameters** method sets the parameters of this [Levels](https://learn.microsoft.com/windows/desktop/api/gdipluseffects/nl-gdipluseffects-levels) object.

## Parameters

### `parameters` [in]

Type: **const [LevelsParams](https://learn.microsoft.com/windows/desktop/api/gdipluseffects/ns-gdipluseffects-levelsparams)***

Pointer to a [LevelsParams](https://learn.microsoft.com/windows/desktop/api/gdipluseffects/ns-gdipluseffects-levelsparams) structure that specifies the parameters.

## Return value

Type: **[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status)**

If the method succeeds, it returns **Ok**, which is an element of the
[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status) enumeration.

If the method fails, it returns one of the other elements of the
[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status) enumeration.
