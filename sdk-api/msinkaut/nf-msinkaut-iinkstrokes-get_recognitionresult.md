# IInkStrokes::get_RecognitionResult

## Description

Gets the [IInkRecognitionResult](https://learn.microsoft.com/windows/desktop/api/msinkaut/nn-msinkaut-iinkrecognitionresult) object of the [InkStrokes](https://learn.microsoft.com/previous-versions/windows/desktop/legacy/ms703293(v=vs.85)) collection.

This property is read-only.

## Parameters

## Remarks

The value of this property is **NULL** if recognition has not been done or if the results were not persisted.
