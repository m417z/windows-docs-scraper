# IInkRecognitionAlternate::get_LineNumber

## Description

Gets the line number of the ink that makes up the alternate.

This property is read-only.

## Parameters

## Remarks

Line numbers begin with 1.

**Note** The [Recognizer](https://learn.microsoft.com/windows/desktop/api/msinkaut/nn-msinkaut-iinkrecognizer) object automatically determines the metrics for how lines are spaced.
