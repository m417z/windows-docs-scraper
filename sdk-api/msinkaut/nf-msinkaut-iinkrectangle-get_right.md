# IInkRectangle::get_Right

## Description

Gets or sets the right position of the [InkRectangle](https://learn.microsoft.com/windows/desktop/tablet/inkrectangle-class) object.

This property is read/write.

## Parameters

## Remarks

**Note** A point is within an [InkRectangle](https://learn.microsoft.com/windows/desktop/tablet/inkrectangle-class) if it lies on the [Left](https://learn.microsoft.com/windows/desktop/api/msinkaut/nf-msinkaut-iinkrectangle-get_left) or [Top](https://learn.microsoft.com/windows/desktop/api/msinkaut/nf-msinkaut-iinkrectangle-get_top) side or is within all four sides. A point on the **Right** or [Bottom](https://learn.microsoft.com/windows/desktop/api/msinkaut/nf-msinkaut-iinkrectangle-get_bottom) side is considered outside the rectangle.

The default value of this property is 0.
