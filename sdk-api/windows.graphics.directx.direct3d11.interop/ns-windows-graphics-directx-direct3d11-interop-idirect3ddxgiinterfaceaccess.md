# IDirect3DDxgiInterfaceAccess interface

## Description

IDirect3DDxgiInterfaceAccess is a COM interface, which must be implemented by anything that implements [IDirect3DDevice](https://learn.microsoft.com/uwp/api/windows.graphics.directx.direct3d11.idirect3ddevice) or [IDirect3DSurface](https://learn.microsoft.com/uwp/api/windows.graphics.directx.direct3d11.idirect3dsurface). **ID3D12GraphicsCommandList5** inherits from the [IUnknown interface](https://learn.microsoft.com/windows/desktop/api/unknwn/nn-unknwn-iunknown).
