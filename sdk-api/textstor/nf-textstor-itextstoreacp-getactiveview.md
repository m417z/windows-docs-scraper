# ITextStoreACP::GetActiveView

## Description

The **ITextStoreACP::GetActiveView** method returns a [TsViewCookie](https://learn.microsoft.com/windows/desktop/TSF/tsviewcookie) data type that specifies the current active view.

## Parameters

### `pvcView` [out]

Receives the **TsViewCookie** data type that specifies the current active view.

## Return value

This method has no return values.
