# IAnchor::SetGravity

## Description

Sets the gravity of the anchor.

## Parameters

### `gravity` [in]

Contains a value from the [TsGravity](https://learn.microsoft.com/windows/win32/api/textstor/ne-textstor-tsgravity) enumeration that specifies a new forward or backward gravity for the anchor.

## Return value

This method can return one of these values.

| Value | Description |
| --- | --- |
| **S_OK** | The method was successful. |
