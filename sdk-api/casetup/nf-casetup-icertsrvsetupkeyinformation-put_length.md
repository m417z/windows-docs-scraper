# ICertSrvSetupKeyInformation::put_Length

## Description

The **Length** property gets or sets the strength of the key to one of the values supported by the [cryptographic service provider](https://learn.microsoft.com/windows/desktop/SecGloss/c-gly) (CSP).

This property is read/write.
