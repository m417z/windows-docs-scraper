# IOfflineFilesEvents3::PrefetchFileBegin

## Description

Reports that a file prefetch operation has begun.

## Parameters

### `pszPath` [in]

The UNC path of the file.

## Return value

Returns **S_OK** if successful, or an error value otherwise.
