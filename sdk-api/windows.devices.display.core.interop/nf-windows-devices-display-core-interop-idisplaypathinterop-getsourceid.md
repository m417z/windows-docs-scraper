## Description

Gets the unique identifier of the [DisplaySource](https://learn.microsoft.com/uwp/api/windows.devices.display.core.displaysource) object.

## Parameters

### `pSourceId` [out]

The unique identifier.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

This method returns **S_OK** if it succeeded, otherwise a failure code indicating why it failed.
