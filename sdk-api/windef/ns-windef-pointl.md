# POINTL structure

## Description

The POINTL structure defines the x- and y-coordinates of a point.

## Members

### `x`

Specifies the *x*-coordinate of the point.

### `y`

Specifies the *y*-coordinate of the point.

## Remarks

The POINTL structure is identical to the [POINT](https://learn.microsoft.com/windows/desktop/api/windef/ns-windef-point) structure.
