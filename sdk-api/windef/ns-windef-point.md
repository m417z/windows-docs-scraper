# POINT structure

## Description

The POINT structure defines the x- and y-coordinates of a point.

## Members

### `x`

Specifies the *x*-coordinate of the point.

### `y`

Specifies the *y*-coordinate of the point.

## Remarks

The POINT structure is identical to the [POINTL](https://learn.microsoft.com/windows/desktop/api/windef/ns-windef-pointl) structure.
