# ID2D1SvgElement::SetAttributeValue(PCWSTR,const D2D1_SVG_LENGTH &)

## Description

Sets an attribute of this element as a length value.

## Parameters

### `name` [in]

Type: **PCWSTR**

Name of the attribute to set.

### `value` [ref]

Type: **const [D2D1_SVG_LENGTH](https://learn.microsoft.com/windows/desktop/api/d2d1svg/ns-d2d1svg-d2d1_svg_length)**

The new value of the attribute.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

This method returns an HRESULT success or error code.
