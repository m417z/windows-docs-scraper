# ID2D1SvgElement::SetAttributeValue(PCWSTR,D2D1_SVG_LINE_CAP)

## Description

Sets an attribute of this element as a line cap value. This method can be used to set the value of the stroke-linecap property.

## Parameters

### `name` [in]

Type: **PCWSTR**

Name of the attribute to set.

### `value`

Type: **[D2D1_SVG_LINE_CAP](https://learn.microsoft.com/windows/desktop/api/d2d1svg/ne-d2d1svg-d2d1_svg_line_cap)**

The new value of the attribute.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

This method returns an HRESULT success or error code.
