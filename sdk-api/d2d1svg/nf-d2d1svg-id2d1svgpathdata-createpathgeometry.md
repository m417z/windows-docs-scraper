# ID2D1SvgPathData::CreatePathGeometry

## Description

Creates a path geometry object representing the path data.

## Parameters

### `fillMode`

Type: **[D2D1_FILL_MODE](https://learn.microsoft.com/windows/desktop/api/d2d1/ne-d2d1-d2d1_fill_mode)**

Fill mode for the path geometry object.

### `pathGeometry` [out]

Type: **ID2D1PathGeometry1****

On completion, pathGeometry will contain a point to the created [ID2D1PathGeometry1](https://learn.microsoft.com/windows/desktop/api/d2d1_1/nn-d2d1_1-id2d1pathgeometry1) object.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

This method returns an HRESULT success or error code.
