# ID2D1SvgElement::GetAttributeValue(PCWSTR,D2D1_SVG_LENGTH)

## Description

Gets an attribute of this element as a length value.

## Parameters

### `name` [in]

Type: **PCWSTR**

The name of the attribute.

### `value` [out]

Type: **[D2D1_SVG_LENGTH](https://learn.microsoft.com/windows/desktop/api/d2d1svg/ns-d2d1svg-d2d1_svg_length)***

The value of the attribute.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

This method returns an HRESULT success or error code.
