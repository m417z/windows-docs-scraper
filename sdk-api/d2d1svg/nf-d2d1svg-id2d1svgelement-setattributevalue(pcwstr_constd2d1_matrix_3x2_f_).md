# ID2D1SvgElement::SetAttributeValue(PCWSTR,const D2D1_MATRIX_3X2_F &)

## Description

Sets an attribute of this element as a matrix value. This method can be used to set the value of a transform or gradientTransform attribute.

## Parameters

### `name` [in]

Type: **PCWSTR**

Name of the attribute to set.

### `value` [ref]

Type: **const D2D1_MATRIX_3X2_F**

The new value of the attribute.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

This method returns an HRESULT success or error code.
