# D2D1_SVG_ASPECT_ALIGN enumeration

## Description

The alignment portion of the SVG preserveAspectRatio attribute.

The alignment is set to SVG's 'none' value.

### `D2D1_SVG_ASPECT_ALIGN_X_MAX_Y_MAX:9`
