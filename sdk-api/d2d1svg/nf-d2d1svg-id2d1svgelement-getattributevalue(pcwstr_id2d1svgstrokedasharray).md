# ID2D1SvgElement::GetAttributeValue(PCWSTR,ID2D1SvgStrokeDashArray)

## Description

Gets an attribute of this element as a stroke dash array. This method can be used to get the value of the stroke-dasharray property.

## Parameters

### `name` [in]

Type: **PCWSTR**

The name of the attribute.

### `value`

Type: **ID2D1SvgStrokeDashArray****

The value of the attribute.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

This method returns an HRESULT success or error code.
