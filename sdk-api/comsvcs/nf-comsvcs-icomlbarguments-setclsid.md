# ICOMLBArguments::SetCLSID

## Description

Sets the object's CLSID.

## Parameters

### `pCLSID` [in]

The object's CLSID.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
