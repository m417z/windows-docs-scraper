# IMessageMover::get_SourcePath

## Description

Retrieves the current path of the source (input) queue.

## Parameters

### `pVal` [out]

The path.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
