# IContextProperties::SetProperty

## Description

Sets a context object property.

## Parameters

### `name` [in]

The name of the context object property to be set. See [GetProperty](https://learn.microsoft.com/windows/desktop/api/comsvcs/nf-comsvcs-icontextproperties-getproperty) for a list of valid property names.

### `property` [in]

The context object property value.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
