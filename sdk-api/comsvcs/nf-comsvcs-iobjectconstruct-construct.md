# IObjectConstruct::Construct

## Description

Constructs an object using the specified parameters.

## Parameters

### `pCtorObj` [in]

A reference to an implementation of the [IObjectConstructString](https://learn.microsoft.com/windows/desktop/api/comsvcs/nn-comsvcs-iobjectconstructstring) interface.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
