# ObjectContext::get_ContextInfo

## Description

Retrieves the context information object of the current object's context.

## Parameters

### `ppContextInfo` [out]

A reference to a [ContextInfo](https://learn.microsoft.com/windows/desktop/api/comsvcs/nn-comsvcs-contextinfo) interface that contains the context information.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
