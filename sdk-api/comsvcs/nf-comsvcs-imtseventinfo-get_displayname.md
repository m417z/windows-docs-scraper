# IMtsEventInfo::get_DisplayName

## Description

Retrieves the display name of the object.

## Parameters

### `sDisplayName` [out]

The display name of the object.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_FAIL, and S_OK.
