# IMtsEvents::PostEvent

## Description

Posts a user-defined event to an event sink.

## Parameters

### `vEvent` [in]

A pointer to the name of the event.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_FAIL, and S_OK.
