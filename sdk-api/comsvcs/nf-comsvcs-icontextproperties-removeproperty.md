# IContextProperties::RemoveProperty

## Description

Removes a context object property.

## Parameters

### `name` [in]

The name of the context object property to be removed. See [GetProperty](https://learn.microsoft.com/windows/desktop/api/comsvcs/nf-comsvcs-icontextproperties-getproperty) for a list of valid property names.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
