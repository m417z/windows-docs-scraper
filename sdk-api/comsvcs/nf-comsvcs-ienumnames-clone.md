# IEnumNames::Clone

## Description

Creates an enumerator that contains the same enumeration state as the current one.

## Parameters

### `ppenum` [out]

Address of a pointer to the [IEnumNames](https://learn.microsoft.com/windows/desktop/api/comsvcs/nn-comsvcs-ienumnames) interface on the enumeration object.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
