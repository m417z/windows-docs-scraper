# IGetContextProperties::EnumNames

## Description

Retrieves a list of the names of the current context properties.

## Parameters

### `ppenum` [out, retval]

An [IEnumNames](https://learn.microsoft.com/windows/desktop/api/comsvcs/nn-comsvcs-ienumnames) interface providing access to a list of the names of the current context properties.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
