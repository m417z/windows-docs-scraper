# IObjPool::PutEndTx

## Description

Destroys the pooled object when the transaction ends.

## Parameters

### `pObj` [in]

A reference to the [IUnknown](https://learn.microsoft.com/windows/desktop/api/unknwn/nn-unknwn-iunknown) of the pooled object.
