# IServicePoolConfig::get_ClassFactory

## Description

Retrieves a class factory for the pooled objects.

## Parameters

### `pFactory` [out]

A pointer to the [IClassFactory](https://learn.microsoft.com/windows/desktop/api/unknwnbase/nn-unknwnbase-iclassfactory) interface pointer.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
