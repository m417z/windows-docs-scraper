# IMessageMover::get_DestPath

## Description

Retrieves the path of the destination (output) queue.

## Parameters

### `pVal` [out]

The path.

## Return value

This method can return the standard return values E_INVALIDARG, E_OUTOFMEMORY, E_UNEXPECTED, E_FAIL, and S_OK.
