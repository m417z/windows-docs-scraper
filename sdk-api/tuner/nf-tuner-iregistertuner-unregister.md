# IRegisterTuner::Unregister

## Description

\[The feature associated with this page, [Microsoft TV Technologies](https://learn.microsoft.com/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal), is a legacy feature. Microsoft strongly recommends that new code does not use this feature.\]

This feature is expected to be available on a future version of the Windows operating system.

The **Unregister** method unregisters an apartment-threaded tuner and a tuner marshaller that were registered by **IRegisterTuner::Register**.

## Return value

When the method succeeds, it returns S_OK. Otherwise it returns an **HRESULT** error code.
