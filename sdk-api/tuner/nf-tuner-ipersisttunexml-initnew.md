# IPersistTuneXml::InitNew

## Description

\[The feature associated with this page, [Microsoft TV Technologies](https://learn.microsoft.com/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal), is a legacy feature. Microsoft strongly recommends that new code does not use this feature.\]

This method is not implemented in the current release. It was created to make the [IPersistTuneXml](https://learn.microsoft.com/previous-versions/windows/desktop/api/tuner/nn-tuner-ipersisttunexml) interface compatible with the **IPersistStreamInit** interface and is reserved for future use.

## Return value

Always returns **S_OK**.
