# IESOpenMmiEvent::GetDialogNumber

## Description

\[The feature associated with this page, [Microsoft TV Technologies](https://learn.microsoft.com/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal), is a legacy feature. Microsoft strongly recommends that new code does not use this feature.\]

The **GetDialogNumber** method returns the dialog number generated by the sending device. The dialog number identifies the dialog to the receiver while the receiver is processing the **OpenMMI**  event.

## Parameters

### `pDialogRequest` [out]

Pointer to the value of the dialog request from the operation that generated the **OpenMMI** request, for example, a call to [IBDA::CheckEntitlementToken](https://learn.microsoft.com/windows/desktop/api/bdaiface/nf-bdaiface-ibda_conditionalaccessex-checkentitlementtoken).

### `pDialogNumber` [out, retval]

Gets the number generated by the device that is opening the dialog, which identifies the dialog.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
