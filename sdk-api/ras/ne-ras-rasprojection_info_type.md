# RASPROJECTION_INFO_TYPE enumeration

## Description

The **RASPROJECTION_INFO_TYPE** enumeration is used in the [RAS_PROJECTION_INFO](https://learn.microsoft.com/windows/desktop/api/ras/ns-ras-ras_projection_info) structure to represent either a [RASPPP_PROJECTION_INFO](https://learn.microsoft.com/windows/desktop/api/ras/ns-ras-rasppp_projection_info) or [RASIKEV2_PROJECTION_INFO](https://learn.microsoft.com/windows/desktop/api/ras/ns-ras-rasikev2_projection_info) structure.

## Constants

### `PROJECTION_INFO_TYPE_PPP:1`

Represents a [RASPPP_PROJECTION_INFO](https://learn.microsoft.com/windows/desktop/api/ras/ns-ras-rasppp_projection_info) structure.

### `PROJECTION_INFO_TYPE_IKEv2`

Represents a [RASIKEV2_PROJECTION_INFO](https://learn.microsoft.com/windows/desktop/api/ras/ns-ras-rasikev2_projection_info) structure.
