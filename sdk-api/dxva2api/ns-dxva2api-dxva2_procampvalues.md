# DXVA2_ProcAmpValues structure

## Description

Contains values for DirectX Video Acceleration (DXVA) video processing operations.

## Members

### `Brightness`

Brightness value.

### `Contrast`

Contrast value.

### `Hue`

Hue value.

### `Saturation`

Saturation value.
