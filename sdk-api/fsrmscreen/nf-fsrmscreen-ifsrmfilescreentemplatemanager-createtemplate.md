# IFsrmFileScreenTemplateManager::CreateTemplate

## Description

Creates a file screen template object.

## Parameters

### `fileScreenTemplate` [out]

An [IFsrmFileScreenTemplate](https://learn.microsoft.com/previous-versions/windows/desktop/api/fsrmscreen/nn-fsrmscreen-ifsrmfilescreentemplate) interface to the
newly create template. To add the template to FSRM, call the
[IFsrmFileScreenTemplate::Commit](https://learn.microsoft.com/previous-versions/windows/desktop/api/fsrm/nf-fsrm-ifsrmobject-commit) method.

## Return value

The method returns the following return values.
