# IFsrmFileScreenException::get_AllowedFileGroups

## Description

[This property is supported for compatibility but it's recommended to use the
[FSRM WMI Classes](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/fsrm-wmi-classes) to manage FSRM. Please see the
[MSFT_FSRMFileScreenException](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/msft-fsrmfilescreenexception) class.]

Retrieves or sets the names of the file groups that contain the file name patterns of the files that
are allowed in the directory.

This property is read/write.
