# IFsrmFileScreenManager::get_ActionVariableDescriptions

## Description

Retrieves the descriptions for the macros contained in the
[IFsrmFileScreenManager::ActionVariables](https://learn.microsoft.com/previous-versions/windows/desktop/api/fsrmscreen/nf-fsrmscreen-ifsrmfilescreenmanager-get_actionvariables)
property.

This property is read-only.
