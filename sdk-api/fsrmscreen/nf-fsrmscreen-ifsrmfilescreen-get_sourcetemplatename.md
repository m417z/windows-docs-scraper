# IFsrmFileScreen::get_SourceTemplateName

## Description

[This property is supported for compatibility but it's recommended to use the
[FSRM WMI Classes](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/fsrm-wmi-classes) to manage FSRM. Please see the
[MSFT_FSRMFileScreen](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/msft-fsrmfilescreen) class.]

Retrieves the name of the template from which this file screen object was derived.

This property is read-only.
