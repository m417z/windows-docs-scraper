# IWSMan::get_Error

## Description

Gets additional error information, in an XML stream, for the preceding call to an [IWSMan](https://learn.microsoft.com/windows/desktop/api/wsmandisp/nn-wsmandisp-iwsman) method if Windows Remote Management service was unable to create an [IWSManSession](https://learn.microsoft.com/windows/desktop/api/wsmandisp/nn-wsmandisp-iwsmansession) object, an [IWSManConnectionOptions](https://learn.microsoft.com/windows/desktop/api/wsmandisp/nn-wsmandisp-iwsmanconnectionoptions) object, or an [IWSManResourceLocator](https://learn.microsoft.com/windows/desktop/api/wsmandisp/nn-wsmandisp-iwsmanresourcelocator) object.

This property is read-only.
