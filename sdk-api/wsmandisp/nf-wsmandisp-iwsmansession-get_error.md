# IWSManSession::get_Error

## Description

Gets additional error information in an XML stream for the preceding call to an [IWSManSession](https://learn.microsoft.com/windows/desktop/api/wsmandisp/nn-wsmandisp-iwsmansession) object method.

This property is read-only.
