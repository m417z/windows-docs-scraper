# IWSManEx::EnumerationFlagHierarchyDeep

## Description

The **IWSManEx::EnumerationFlagHierarchyDeep** method returns the value of the enumeration constant **EnumerationFlagHierarchyDeep** for use in the *flags* parameter of the [IWSManSession::Enumerate](https://learn.microsoft.com/windows/desktop/api/wsmandisp/nf-wsmandisp-iwsmansession-enumerate) method.

**EnumerationFlagHierarchyDeep** is a constant in the **_WSManEnumFlags** enumeration and is described in [Enumeration Constants](https://learn.microsoft.com/windows/desktop/WinRM/enumeration-constants).

## Parameters

### `flags` [out]

The value of the constant.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
