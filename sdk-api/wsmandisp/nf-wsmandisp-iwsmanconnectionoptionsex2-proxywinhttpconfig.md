# IWSManConnectionOptionsEx2::ProxyWinHttpConfig

## Description

Returns the value of the proxy access type flag **WSManProxyWinHttpConfig** for use in the *accessType* parameter of the [IWSManConnectionOptionsEx2::SetProxy](https://learn.microsoft.com/windows/desktop/api/wsmandisp/nf-wsmandisp-iwsmanconnectionoptionsex2-setproxy) method.

## Parameters

### `value` [out, retval]

Specifies the value of the constant.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
