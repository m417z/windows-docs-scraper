# IWSManEnumerator::get_AtEndOfStream

## Description

Indicates that the end of items in the
[IWSManEnumerator](https://learn.microsoft.com/windows/desktop/api/wsmandisp/nn-wsmandisp-iwsmanenumerator) object has been reached by calls to
[IWSManEnumerator::ReadItem](https://learn.microsoft.com/windows/desktop/api/wsmandisp/nf-wsmandisp-iwsmanenumerator-readitem).

This property is read-only.
