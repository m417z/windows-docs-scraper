# CStdAsyncStubBuffer2_Disconnect function

## Description

[CStdAsyncStubBuffer2_Disconnect is not supported and may be altered or unavailable in the future.]

Implements the [IRpcStubBuffer::Disconnect](https://learn.microsoft.com/windows/desktop/api/objidl/nf-objidl-irpcstubbuffer-disconnect) method and disconnects the server object from the stub.

## Parameters

### `pthis` [in]

Pointer to the [IRpcStubBuffer](https://learn.microsoft.com/windows/desktop/api/objidl/nn-objidl-irpcstubbuffer) object.

## Remarks

This function is used internally by proxies that are generated by MIDL.
