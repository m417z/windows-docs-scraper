# RdcNeedType enumeration

## Description

Defines the set of
data chunks used to generate a remote copy.

## Constants

### `RDCNEED_SOURCE:0`

The chunk is a source chunk.

### `RDCNEED_TARGET:1`

This value is reserved for future use.

### `RDCNEED_SEED:2`

The chunk is a seed chunk.

### `RDCNEED_SEED_MAX:255`

This value is reserved for future use.
