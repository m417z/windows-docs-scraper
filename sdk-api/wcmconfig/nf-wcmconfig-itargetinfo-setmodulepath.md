# ITargetInfo::SetModulePath

## Description

Sets the module path for the offline installation location.

## Parameters

### `Module` [in]

The name of the module.

### `Path` [in]

The module path.

## Return value

This method returns an HRESULT value. **S_OK** indicates success.
