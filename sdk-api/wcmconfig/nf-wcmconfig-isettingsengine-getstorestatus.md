# ISettingsEngine::GetStoreStatus

## Description

Gets the status of the schema store.

## Parameters

### `Reserved` [in]

Reserved. Must be **NULL**.

### `Status` [out]

A [WcmUserStatus](https://learn.microsoft.com/windows/win32/api/wcmconfig/ne-wcmconfig-wcmuserstatus) value that indicates the status of the store.

## Return value

This method returns an HRESULT value. **S_OK** indicates success.
