# ISettingsEngine::SetTargetInfo

## Description

Sets the current offline target for the engine.

## Parameters

### `Target` [in]

An [ITargetInfo](https://learn.microsoft.com/previous-versions/windows/desktop/api/wcmconfig/nn-wcmconfig-itargetinfo) value that specifies the target.

## Return value

This method returns an HRESULT value. **S_OK** indicates success.
