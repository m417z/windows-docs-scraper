# ISettingsItem::HasChild

## Description

Determines whether the current item has a child item.

## Parameters

### `ItemHasChild` [out]

**True** if a child item is present, **false** otherwise.

## Return value

This method returns an HRESULT value. **S_OK** indicates success.
