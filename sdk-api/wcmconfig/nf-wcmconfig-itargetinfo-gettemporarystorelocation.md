# ITargetInfo::GetTemporaryStoreLocation

## Description

Gets the current temporary store location.

## Parameters

### `TemporaryStoreLocation` [out]

The current temporary store location.

## Return value

This method returns an HRESULT value. **S_OK** indicates success.
