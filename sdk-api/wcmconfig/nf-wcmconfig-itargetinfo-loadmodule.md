# ITargetInfo::LoadModule

## Description

Loads the module from the offline installation location.

## Parameters

### `Module` [in]

The name of the module.

### `ModuleHandle` [out]

The module handle.

## Return value

This method returns an HRESULT value. **S_OK** indicates success.
