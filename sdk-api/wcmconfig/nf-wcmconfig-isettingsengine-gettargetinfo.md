# ISettingsEngine::GetTargetInfo

## Description

Gets the current offline target for the engine.

## Parameters

### `Target` [out]

A pointer to an [ITargetInfo](https://learn.microsoft.com/previous-versions/windows/desktop/api/wcmconfig/nn-wcmconfig-itargetinfo) object that is the current target for the engine.

## Return value

This method returns an HRESULT value. **S_OK** indicates success.
