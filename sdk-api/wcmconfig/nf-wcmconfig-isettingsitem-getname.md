# ISettingsItem::GetName

## Description

Gets the name of the item.

## Parameters

### `Name` [out]

The name of the item.

## Return value

This method returns an HRESULT value. **S_OK** indicates success. It may return **E_OUTOFMEMORY** if there are insufficient resources to allocate Name.
