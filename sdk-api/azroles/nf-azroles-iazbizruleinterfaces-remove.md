# IAzBizRuleInterfaces::Remove

## Description

The **Remove** method removes the specified interface from the list of interfaces The number of interfaces in the list of interfaces that can be called by BizRule scripts.

## Parameters

### `bstrInterfaceName` [in]

The name of the interface to remove.

## Return value

 If the method succeeds, it returns **S_OK**.

If the method fails, it returns an error code. For a list of common error codes, see [Common HRESULT Values](https://learn.microsoft.com/windows/desktop/SecCrypto/common-hresult-values).
