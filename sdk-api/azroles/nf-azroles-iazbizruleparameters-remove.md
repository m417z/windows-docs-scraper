# IAzBizRuleParameters::Remove

## Description

The **Remove** method removes the specified parameter from the list of parameters available to business rule (BizRule) scripts.

## Parameters

### `varParameterName` [in]

The name of the parameter to remove.

## Return value

 If the method succeeds, it returns **S_OK**.

If the method fails, it returns an error code. For a list of common error codes, see [Common HRESULT Values](https://learn.microsoft.com/windows/desktop/SecCrypto/common-hresult-values).
