# BoundingOrientedBox::Intersects(const BoundingFrustum &)

## Description

Tests the [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) for intersection with a [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).

## Parameters

### `fr` [in, ref]

The [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) to test for intersection.

## Return value

A boolean value indicating whether the [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) intersects the [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
