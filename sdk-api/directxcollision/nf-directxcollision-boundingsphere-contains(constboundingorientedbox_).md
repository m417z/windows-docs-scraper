# BoundingSphere::Contains(const BoundingOrientedBox &)

## Description

Tests whether the [BoundingSphere](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere) contains the specified [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox).

## Parameters

### `box` [in, ref]

The [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) to test against.

## Return value

A [ContainmentType](https://learn.microsoft.com/windows/win32/api/directxcollision/ne-directxcollision-containmenttype) value indicating whether the [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) is contained in the [BoundingSphere](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere).

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
