# BoundingFrustum::Intersects(const BoundingFrustum &)

## Description

Test the [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) for intersection with another **BoundingFrustum**.

## Parameters

### `fr` [in, ref]

The [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) to test for intersection.

## Return value

A bool value indicating whether the [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) intersects the specified **BoundingFrustum**.

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
