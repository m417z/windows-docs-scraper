# BoundingFrustum::Contains(const BoundingBox &)

## Description

Tests whether the [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) contains the specified [BoundingBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingbox).

## Parameters

### `box` [in, ref]

The [BoundingBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingbox) to test against.

## Return value

A [ContainmentType](https://learn.microsoft.com/windows/win32/api/directxcollision/ne-directxcollision-containmenttype) value indicating whether the [BoundingBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingbox) is contained in the [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
