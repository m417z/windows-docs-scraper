# BoundingFrustum::operator-assign(BoundingFrustum &&)

## Description

Copies values from another [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).

## Parameters

### `unnamedParam1` [in, ref]

The [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) to copy values from.

## Return value

The updated [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
