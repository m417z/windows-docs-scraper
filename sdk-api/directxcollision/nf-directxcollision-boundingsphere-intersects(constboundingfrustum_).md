# BoundingSphere::Intersects(const BoundingFrustum &)

## Description

Test the [BoundingSphere](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere) for intersection with a [BoundingFrustum](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).

## Parameters

### `fr` [in, ref]

The [BoundingSphere](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere) to test for intersection.

## Return value

A bool value indicating whether the [BoundingSphere](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere) intersects the specified **BoundingSphere**.

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
