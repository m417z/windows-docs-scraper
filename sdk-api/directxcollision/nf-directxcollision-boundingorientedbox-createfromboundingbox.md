# BoundingOrientedBox::CreateFromBoundingBox

## Description

Creates a [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) from a [BoundingBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingbox).

## Parameters

### `Out` [out, ref]

The new [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox).

### `box` [in, ref]

The [BoundingBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingbox) the [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) should contain.

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
