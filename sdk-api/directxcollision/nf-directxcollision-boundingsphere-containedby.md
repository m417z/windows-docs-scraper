# BoundingSphere::ContainedBy

## Description

Tests whether the [BoundingSphere](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere) is contained by the specified frustum.

## Parameters

### `Plane0` [in]

A plane describing the frustum.

### `Plane1` [in]

A plane describing the frustum.

### `Plane2` [in]

A plane describing the frustum.

### `Plane3` [in]

A plane describing the frustum.

### `Plane4` [in]

A plane describing the frustum.

### `Plane5` [in]

A plane describing the frustum.

## Return value

A [ContainmentType](https://learn.microsoft.com/windows/win32/api/directxcollision/ne-directxcollision-containmenttype) value indicating whether the frustum contains the [BoundingSphere](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere).

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
