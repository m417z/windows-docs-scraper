# BoundingOrientedBox::Intersects(FXMVECTOR)

## Description

Tests the [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) for intersection with a plane.

## Parameters

### `Plane` [in]

A vector describing the plane

## Return value

A [PlaneIntersectionType](https://learn.microsoft.com/windows/win32/api/directxcollision/ne-directxcollision-planeintersectiontype) value indicating the intersection status.

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
