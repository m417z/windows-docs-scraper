# BoundingBox::Intersects(const BoundingOrientedBox &)

## Description

Test the [BoundingBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingbox) for intersection with a [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox).

## Parameters

### `box` [in, ref]

The [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) to test for intersection.

## Return value

A boolean value indicating whether the [BoundingBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingbox) intersects the specified [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox).

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
