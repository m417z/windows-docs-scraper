# BoundingOrientedBox::operator-assign

## Description

Copies values from another [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox).

## Parameters

### `unnamedParam1` [in, ref]

The [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) to copy values from.

## Return value

The updated [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox).

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
