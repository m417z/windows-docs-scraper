# BoundingOrientedBox::Intersects(FXMVECTOR,FXMVECTOR,FXMVECTOR)

## Description

Tests the [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) for intersection with a triangle.

## Parameters

### `V0` [in]

A vector describing the triangle.

### `V1` [in]

A vector describing the triangle.

### `V2` [in]

A vector describing the triangle.

## Return value

A boolean value indicating whether the [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) intersects the triangle.

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
