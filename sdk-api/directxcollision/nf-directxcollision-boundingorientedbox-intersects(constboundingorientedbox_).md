# BoundingOrientedBox::Intersects(const BoundingOrientedBox &)

## Description

Tests the [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) for intersection with a **BoundingOrientedBox**.

## Parameters

### `box` [in, ref]

The [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) to test for intersection.

## Return value

A boolean value indicating whether the [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) intersects the **BoundingOrientedBox**.

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
