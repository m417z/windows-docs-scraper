# BoundingOrientedBox::Contains(const BoundingSphere &)

## Description

Tests whether the [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) contains a [BoundingSphere](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere).

## Parameters

### `sh` [in, ref]

The [BoundingSphere](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere) to test against.

## Return value

A [ContainmentType](https://learn.microsoft.com/windows/win32/api/directxcollision/ne-directxcollision-containmenttype) indicating whether the [BoundingSphere](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere) is contained in the [BoundingOrientedBox](https://learn.microsoft.com/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox).

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
