## Description

Sets the presentation buffer associated with this presentation surface.

## Parameters

### `presentationBuffer`

Type: **[IPresentationBuffer](https://learn.microsoft.com/windows/win32/api/presentation/nn-presentation-ipresentationbuffer)**

The presentation buffer associated with this presentation surface.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/winprog/windows-data-types)**

If the function succeeds, it returns `S_OK`; otherwise, it returns an `HRESULT` value that indicates the error.
