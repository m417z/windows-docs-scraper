## Description

Creates a presentation manager.

## Parameters

### `ppPresentationManager`

Type: **[IPresentationManager](https://learn.microsoft.com/windows/win32/api/presentation/nn-presentation-ipresentationmanager) \*\***

The address of a pointer to the created presentation manager.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/winprog/windows-data-types)**

If the function succeeds, it returns `S_OK`; otherwise, it returns an `HRESULT` value that indicates the error.
