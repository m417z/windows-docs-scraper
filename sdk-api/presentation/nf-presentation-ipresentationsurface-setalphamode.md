## Description

Sets the transparency behavior of the presentation surface.

## Parameters

### `alphaMode`

Type: **[DXGI_ALPHA_MODE](https://learn.microsoft.com/windows/win32/api/dxgi1_2/ne-dxgi1_2-dxgi_alpha_mode)**

The transparency behavior of the presentation surface.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/winprog/windows-data-types)**

If the function succeeds, it returns `S_OK`; otherwise, it returns an `HRESULT` value that indicates the error.
