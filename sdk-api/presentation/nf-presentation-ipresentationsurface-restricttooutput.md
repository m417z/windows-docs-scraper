## Description

Restricts presentation to a specific display adapter output.

## Parameters

### `output`

Type: **[IUnknown](https://learn.microsoft.com/windows/win32/api/unknwn/nn-unknwn-iunknown)**

An object that represents the adapter output. Only an [IDXGIOutput](https://learn.microsoft.com/windows/win32/api/dxgi/nn-dxgi-idxgioutput) is accepted.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/desktop/winprog/windows-data-types)**

If the function succeeds, it returns `S_OK`; otherwise, it returns an `HRESULT` value that indicates the error.
