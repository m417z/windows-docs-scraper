# IDCompositionDevice3::CreateBrightnessEffect

## Description

Creates an instance of [IDCompositionBrightnessEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionbrightnesseffect).

## Parameters

### `brightnessEffect` [out]

Type: **[IDCompositionBrightnessEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionbrightnesseffect)****

Receives the created instance of [IDCompositionBrightnessEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionbrightnesseffect).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
