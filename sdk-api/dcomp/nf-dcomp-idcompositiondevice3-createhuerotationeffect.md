# IDCompositionDevice3::CreateHueRotationEffect

## Description

Creates an instance of [IDCompositionHueRotationEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionhuerotationeffect).

## Parameters

### `hueRotationEffect` [out]

Type: **[IDCompositionHueRotationEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionhuerotationeffect)****

Receives the created instance of [IDCompositionHueRotationEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionhuerotationeffect).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
