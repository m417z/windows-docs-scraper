# IDCompositionAffineTransform2DEffect::SetBorderMode

## Description

Sets the border mode to use with the effect.

## Parameters

### `borderMode` [in]

Type: **[D2D1_BORDER_MODE](https://learn.microsoft.com/windows/desktop/Direct2D/2d-affine-transform)**

Specifies the border mode to use with the effect.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
