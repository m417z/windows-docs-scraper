# IDCompositionDevice3::CreateSaturationEffect

## Description

Creates an instance of [IDCompositionSaturationEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionsaturationeffect).

## Parameters

### `saturationEffect` [out]

Type: **[IDCompositionSaturationEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionsaturationeffect)****

Receives the created instance of [IDCompositionSaturationEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionsaturationeffect).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
