# IDCompositionDevice3::CreateTurbulenceEffect

## Description

Creates an instance of [IDCompositionTurbulenceEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionturbulenceeffect).

## Parameters

### `turbulenceEffect` [out]

Type: **[IDCompositionTurbulenceEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionturbulenceeffect)****

Receives the created instance of [IDCompositionTurbulenceEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionturbulenceeffect).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
