# IDCompositionDevice3::CreateBlendEffect

## Description

Creates an instance of [IDCompositionBlendEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionblendeffect).

## Parameters

### `blendEffect` [out]

Type: **[IDCompositionBlendEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionblendeffect)****

Receives the created instance of [IDCompositionBlendEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionblendeffect).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
