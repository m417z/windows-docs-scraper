# IDCompositionDevice3::CreateCompositeEffect

## Description

Creates an instance of [IDCompositionCompositeEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositioncompositeeffect).

## Parameters

### `compositeEffect` [out]

Type: **[IDCompositionCompositeEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositioncompositeeffect)****

Receives the created instance of [IDCompositionCompositeEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositioncompositeeffect).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
