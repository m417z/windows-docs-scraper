# IDCompositionTurbulenceEffect::SetSize

## Description

Sets the size of the turbulence output.

## Parameters

### `size` [in, ref]

Type: **const [D2D1_VECTOR_2F](https://learn.microsoft.com/windows/desktop/api/dcommon/ns-dcommon-d2d_vector_2f)**

The size of the turbulence output

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
