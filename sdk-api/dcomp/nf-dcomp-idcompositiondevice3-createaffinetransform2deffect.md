# IDCompositionDevice3::CreateAffineTransform2DEffect

## Description

Creates an instance of [IDCompositionAffineTransform2DEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionaffinetransform2deffect).

## Parameters

### `affineTransform2dEffect` [out]

Type: **[IDCompositionAffineTransform2DEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionaffinetransform2deffect)****

Recieves the created instance of [IDCompositionAffineTransform2DEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionaffinetransform2deffect).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
