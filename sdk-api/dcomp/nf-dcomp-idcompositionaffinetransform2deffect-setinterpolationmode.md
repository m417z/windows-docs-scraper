# IDCompositionAffineTransform2DEffect::SetInterpolationMode

## Description

Sets the interpolation mode of the effect.

## Parameters

### `interpolationMode` [in]

Type: **[D2D1_2DAFFINETRANSFORM_INTERPOLATION_MODE](https://learn.microsoft.com/windows/desktop/Direct2D/2d-affine-transform)**

Specifies the interpolation mode of the effect.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
