# IDCompositionAffineTransform2DEffect::SetTransformMatrix

## Description

Sets the transform matrix of the effect.

## Parameters

### `transformMatrix` [in, ref]

Type: **const [D2D1_MATRIX_3X2_F](https://learn.microsoft.com/windows/desktop/Direct2D/d2d1-matrix-3x2-f)**

Specifies the transform matrix for the effect to use.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
