# IDCompositionDevice3::CreateColorMatrixEffect

## Description

Creates an instance of [IDCompositionColorMatrixEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositioncolormatrixeffect).

## Parameters

### `colorMatrixEffect` [out]

Type: **[IDCompositionColorMatrixEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositioncolormatrixeffect)****

Receives the created instance of [IDCompositionColorMatrixEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositioncolormatrixeffect).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
