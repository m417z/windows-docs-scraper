# IDCompositionDevice3::CreateLinearTransferEffect

## Description

Creates an instance of [IDCompositionLinearTransferEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionlineartransfereffect).

## Parameters

### `linearTransferEffect` [out]

Type: **[IDCompositionLinearTransferEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionlineartransfereffect)****

Receives the created instance of [IDCompositionLinearTransferEffect](https://learn.microsoft.com/windows/desktop/api/dcomp/nn-dcomp-idcompositionlineartransfereffect).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
