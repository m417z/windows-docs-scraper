# ID3D10EffectDepthStencilViewVariable::GetDepthStencil

## Description

Get a depth-stencil-view resource.

## Parameters

### `ppResource` [out]

Type: **[ID3D10DepthStencilView](https://learn.microsoft.com/windows/desktop/api/d3d10/nn-d3d10-id3d10depthstencilview)****

The address of a pointer to a depth-stencil-view interface. See [ID3D10DepthStencilView Interface](https://learn.microsoft.com/windows/desktop/api/d3d10/nn-d3d10-id3d10depthstencilview).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the following [Direct3D 10 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues).
