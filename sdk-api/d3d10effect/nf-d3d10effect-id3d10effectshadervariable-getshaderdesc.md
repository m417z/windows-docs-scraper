# ID3D10EffectShaderVariable::GetShaderDesc

## Description

Get a shader description.

## Parameters

### `ShaderIndex` [in]

Type: **[UINT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

A zero-based index.

### `pDesc` [in]

Type: **[D3D10_EFFECT_SHADER_DESC](https://learn.microsoft.com/windows/desktop/api/d3d10effect/ns-d3d10effect-d3d10_effect_shader_desc)***

A pointer to a shader description (see [D3D10_EFFECT_SHADER_DESC](https://learn.microsoft.com/windows/desktop/api/d3d10effect/ns-d3d10effect-d3d10_effect_shader_desc)).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the following [Direct3D 10 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues).
