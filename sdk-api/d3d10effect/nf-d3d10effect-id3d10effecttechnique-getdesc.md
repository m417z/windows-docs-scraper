# ID3D10EffectTechnique::GetDesc

## Description

Get a technique description.

## Parameters

### `pDesc` [in]

Type: **[D3D10_TECHNIQUE_DESC](https://learn.microsoft.com/windows/desktop/api/d3d10effect/ns-d3d10effect-d3d10_technique_desc)***

A pointer to a technique description (see [D3D10_TECHNIQUE_DESC](https://learn.microsoft.com/windows/desktop/api/d3d10effect/ns-d3d10effect-d3d10_technique_desc)).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the following [Direct3D 10 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues).
