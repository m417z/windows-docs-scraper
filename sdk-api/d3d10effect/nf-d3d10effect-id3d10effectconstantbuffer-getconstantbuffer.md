# ID3D10EffectConstantBuffer::GetConstantBuffer

## Description

Get a constant-buffer.

## Parameters

### `ppConstantBuffer` [out]

Type: **[ID3D10Buffer](https://learn.microsoft.com/windows/desktop/api/d3d10/nn-d3d10-id3d10buffer)****

The address of a pointer to a constant-buffer interface. See [ID3D10Buffer Interface](https://learn.microsoft.com/windows/desktop/api/d3d10/nn-d3d10-id3d10buffer).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the following [Direct3D 10 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues).
