# ID3D10EffectPass::IsValid

## Description

Test a pass to see if it contains valid syntax.

## Return value

Type: **[BOOL](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

**TRUE** if the code syntax is valid; otherwise **FALSE**.
