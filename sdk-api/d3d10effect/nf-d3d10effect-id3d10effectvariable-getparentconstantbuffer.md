# ID3D10EffectVariable::GetParentConstantBuffer

## Description

Get a constant buffer.

## Return value

Type: **[ID3D10EffectConstantBuffer](https://learn.microsoft.com/windows/desktop/api/d3d10effect/nn-d3d10effect-id3d10effectconstantbuffer)***

A pointer to a [ID3D10EffectConstantBuffer Interface](https://learn.microsoft.com/windows/desktop/api/d3d10effect/nn-d3d10effect-id3d10effectconstantbuffer).

## Remarks

Effect variables are read-from or written-to a constant buffer.
