# ID3D10EffectType::IsValid

## Description

Tests that the effect type is valid.

## Return value

Type: **[BOOL](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

**TRUE** if it is valid; otherwise **FALSE**.
