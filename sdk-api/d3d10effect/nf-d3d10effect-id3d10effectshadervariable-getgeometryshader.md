# ID3D10EffectShaderVariable::GetGeometryShader

## Description

Get a geometry shader.

## Parameters

### `ShaderIndex` [in]

Type: **[UINT](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)**

A zero-based index.

### `ppGS` [out]

Type: **[ID3D10GeometryShader](https://learn.microsoft.com/windows/desktop/api/d3d10/nn-d3d10-id3d10geometryshader)****

A pointer to a [ID3D10GeometryShader Interface](https://learn.microsoft.com/windows/desktop/api/d3d10/nn-d3d10-id3d10geometryshader).

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

Returns one of the following [Direct3D 10 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d10/d3d10-graphics-reference-returnvalues).
