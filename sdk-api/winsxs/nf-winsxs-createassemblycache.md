# CreateAssemblyCache function

## Description

The **CreateAssemblyCache** function obtains an instance of the [IAssemblyCache](https://learn.microsoft.com/windows/desktop/api/winsxs/nn-winsxs-iassemblycache) interface.

## Parameters

### `ppAsmCache`

Pointer the location that receives the [IAssemblyCache](https://learn.microsoft.com/windows/desktop/api/winsxs/nn-winsxs-iassemblycache) pointer.

### `dwReserved`

Reserved.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
