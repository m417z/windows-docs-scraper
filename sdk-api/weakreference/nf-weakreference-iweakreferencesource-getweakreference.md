## Description

Retrieves a weak reference from an [IWeakReferenceSource](https://learn.microsoft.com/windows/win32/api/weakreference/nn-weakreference-iweakreferencesource).

## Parameters

### `weakReference`

Type: [out, retval] **[IWeakReference](https://learn.microsoft.com/windows/win32/api/weakreference/nn-weakreference-iweakreference)****

The weak reference.

## Return value

Type: **HRESULT**

If this method succeeds, then it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
