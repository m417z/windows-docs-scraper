# Region::Region(IN const RectF &)

## Description

Creates a region that is defined by a rectangle.

## Parameters

### `rect` [in, ref]

Type: **const [RectF](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf)**

Reference to a rectangle.
