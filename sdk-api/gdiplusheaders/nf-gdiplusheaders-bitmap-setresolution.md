# Bitmap::SetResolution

## Description

The **Bitmap::SetResolution** method sets the resolution of this
[Bitmap](https://learn.microsoft.com/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-bitmap) object.

## Parameters

### `xdpi` [in]

Type: **REAL**

Real number that specifies the horizontal resolution in dots per inch.

### `ydpi` [in]

Type: **REAL**

Real number that specifies the vertical resolution in dots per inch.

## Return value

Type: **[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status)**

If the method succeeds, it returns **Ok**, which is an element of the
[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status) enumeration.

If the method fails, it returns one of the other elements of the
[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status) enumeration.
