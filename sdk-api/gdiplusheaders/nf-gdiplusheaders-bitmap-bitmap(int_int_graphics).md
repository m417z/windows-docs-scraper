# Bitmap::Bitmap(IN INT,IN INT,IN Graphics)

## Description

Creates a **Bitmap::Bitmap** object based on a [Graphics](https://learn.microsoft.com/windows/desktop/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) object, a width, and a height.

## Parameters

### `width` [in]

Type: **INT**

Integer that specifies the width, in pixels, of the bitmap.

### `height` [in]

Type: **INT**

Integer that specifies the height, in pixels, of the bitmap.

### `target` [in]

Type: **[Graphics](https://learn.microsoft.com/windows/desktop/api/gdiplusgraphics/nl-gdiplusgraphics-graphics)***

Pointer to a [Graphics](https://learn.microsoft.com/windows/desktop/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) object that contains information used to initialize certain properties (for example, dots per inch) of the new **Bitmap::Bitmap** object.
