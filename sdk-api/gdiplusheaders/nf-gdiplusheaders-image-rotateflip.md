# Image::RotateFlip

## Description

The **Image::RotateFlip** method rotates and flips this image.

## Parameters

### `rotateFlipType` [in]

Type: **[RotateFlipType](https://learn.microsoft.com/windows/desktop/api/gdiplusimaging/ne-gdiplusimaging-rotatefliptype)**

Element of the [RotateFlipType](https://learn.microsoft.com/windows/desktop/api/gdiplusimaging/ne-gdiplusimaging-rotatefliptype) enumeration that specifies the type of rotation and the type of flip.

## Return value

Type: **[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status)**

If the method succeeds, it returns Ok, which is an element of the
[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status) enumeration.

If the method fails, it returns one of the other elements of the
[Status](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/ne-gdiplustypes-status) enumeration.
