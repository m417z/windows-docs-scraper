# Region::Region(IN const Rect &)

## Description

Creates a region that is defined by a rectangle.

## Parameters

### `rect` [in, ref]

Type: **const [Rect](https://learn.microsoft.com/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect)**

Reference to a rectangle.
