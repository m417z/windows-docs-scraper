# Image::GetPaletteSize

## Description

The **Image::GetPaletteSize** method gets the size, in bytes, of the color palette of this
[Image](https://learn.microsoft.com/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-image) object.

## Return value

Type: **INT**

This method returns the size, in bytes, of the color palette of this
[Image](https://learn.microsoft.com/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-image) object.
