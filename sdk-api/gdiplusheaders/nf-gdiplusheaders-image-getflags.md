# Image::GetFlags

## Description

The **Image::GetFlags** method gets a set of flags that indicate certain attributes of this
[Image](https://learn.microsoft.com/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-image) object.

## Return value

Type: **UINT**

This method returns a value that holds a set of single-bit flags. The flags are defined in the [ImageFlags](https://learn.microsoft.com/windows/desktop/api/gdiplusimaging/ne-gdiplusimaging-imageflags) enumeration.
