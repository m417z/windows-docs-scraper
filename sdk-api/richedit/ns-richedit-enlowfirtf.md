# ENLOWFIRTF structure

## Description

Contains information about an unsupported Rich Text Format (RTF) keyword in a Microsoft Rich Edit control.

## Members

### `nmhdr`

Type: **[NMHDR](https://learn.microsoft.com/windows/win32/api/richedit/ns-richedit-nmhdr)**

Specifies an [NMHDR](https://learn.microsoft.com/windows/win32/api/richedit/ns-richedit-nmhdr) structure.

### `szControl`

Type: **[CHAR](https://learn.microsoft.com/windows/desktop/WinProg/windows-data-types)***

The unsupported RTF keyword.

## Remarks

This structure is used with the [EN_LOWFIRTF](https://learn.microsoft.com/windows/win32/controls/en-lowfirtf) message.
