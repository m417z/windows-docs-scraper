# INCREMENTAL_ACCESS_INFO structure

## Description

Contains access information used by an incremental crawl, such as the last access date and modification time.

## Members

### `dwSize`

Type: **DWORD**

Size of the file in bytes.

### `ftLastModifiedTime`

Type: **[FILETIME](https://learn.microsoft.com/windows/desktop/api/minwinbase/ns-minwinbase-filetime)**

Last time the file was modified.
