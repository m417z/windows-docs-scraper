# PROXY_ACCESS enumeration

## Description

Used by [ISearchManager](https://learn.microsoft.com/windows/desktop/api/searchapi/nn-searchapi-isearchmanager) to state proxy use.

## Constants

### `PROXY_ACCESS_PRECONFIG:0`

Use proxy as set by Internet settings.

### `PROXY_ACCESS_DIRECT`

Do not use a proxy.

### `PROXY_ACCESS_PROXY`

Use the specified proxy.
