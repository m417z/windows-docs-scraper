# TIMEOUT_INFO structure

## Description

Stores time-out values for connections and data.

## Members

### `dwSize`

Type: **DWORD**

The size of the structure, in bytes.

### `dwConnectTimeout`

Type: **DWORD**

The time-out value for a connection, in seconds.

### `dwDataTimeout`

Type: **DWORD**

The time-out value for data, in seconds.
