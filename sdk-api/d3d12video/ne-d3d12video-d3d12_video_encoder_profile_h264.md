## Description

Specifies the encoder profiles for H.264 encoding.

## Constants

### `D3D12_VIDEO_ENCODER_PROFILE_H264_MAIN`

Main profile.

### `D3D12_VIDEO_ENCODER_PROFILE_H264_HIGH`

High profile.

### `D3D12_VIDEO_ENCODER_PROFILE_H264_HIGH_10`

High 10 profile.

## Remarks

Use this enumeration to specify the encoder profile in a [D3D12_VIDEO_ENCODER_PROFILE_DESC](https://learn.microsoft.com/windows/win32/api/d3d12video/ns-d3d12video-d3d12_video_encoder_profile_desc) structure.
