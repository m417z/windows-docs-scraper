## Description

Gets the [ID3D12ProtectedResourceSession](https://learn.microsoft.com/windows/win32/api/d3d12/nn-d3d12-id3d12protectedresourcesession) that was passed into [ID3D12VideoDevice2::CreateVideoExtensionCommand](https://learn.microsoft.com/windows/win32/api/d3d12video/nf-d3d12video-id3d12videodevice2-createvideoextensioncommand) when the [ID3D12VideoExtensionCommand](https://learn.microsoft.com/windows/win32/api/d3d12video/nn-d3d12video-id3d12videoextensioncommand) was created.

## Parameters

### `riid`

The globally unique identifier (GUID) for the **ID3D12ProtectedResourceSession** interface.

### `ppProtectedSession`

Receives a void pointer representing the **ID3D12ProtectedResourceSession** interface.

## Return value

This method returns HRESULT.
