## Description

Specifies support flags for input motion vector unit precision modes.

## Constants

### `D3D12_VIDEO_ENCODER_FRAME_INPUT_MOTION_UNIT_PRECISION_SUPPORT_FLAG_NONE` : 0x0

No precision modes are supported.

### `D3D12_VIDEO_ENCODER_FRAME_INPUT_MOTION_UNIT_PRECISION_SUPPORT_FLAG_FULL_PIXEL` : 0x1

Full pixel precision is supported.

### `D3D12_VIDEO_ENCODER_FRAME_INPUT_MOTION_UNIT_PRECISION_SUPPORT_FLAG_HALF_PIXEL` : 0x2

Half pixel precision is supported.

### `D3D12_VIDEO_ENCODER_FRAME_INPUT_MOTION_UNIT_PRECISION_SUPPORT_FLAG_QUARTER_PIXEL` : 0x4

Quarter pixel precision is supported.
