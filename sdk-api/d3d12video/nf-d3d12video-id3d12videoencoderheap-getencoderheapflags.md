## Description

Gets the encoder heap flags with which the video encoder heap was initialized.

## Return value

The bitwise OR combination of values from the [D3D12_VIDEO_ENCODER_HEAP_FLAGS](https://learn.microsoft.com/windows/win32/api/d3d12video/ne-d3d12video-d3d12_video_encoder_heap_flags) enumeration specified in the [D3D12_VIDEO_ENCODER_HEAP_DESC](https://learn.microsoft.com/windows/win32/api/d3d12video/ns-d3d12video-d3d12_video_encoder_heap_desc) passed into [ID3D12VideoDevice3::CreateVideoEncoderHeap](https://learn.microsoft.com/windows/win32/api/d3d12video/nf-d3d12video-id3d12videodevice3-createvideoencoderheap).
