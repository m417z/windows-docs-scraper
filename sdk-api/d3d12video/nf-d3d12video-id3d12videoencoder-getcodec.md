## Description

Gets the codec associated with the video encoder.

## Return value

The value from the [D3D12_VIDEO_ENCODER_CODEC](https://learn.microsoft.com/windows/win32/api/d3d12video/ne-d3d12video-d3d12_video_encoder_codec) enumeration specified in the [D3D12_VIDEO_ENCODER_DESC](https://learn.microsoft.com/windows/win32/api/d3d12video/ns-d3d12video-d3d12_video_encoder_desc) passed into [ID3D12VideoDevice3::CreateVideoEncoder](https://learn.microsoft.com/windows/win32/api/d3d12video/nf-d3d12video-id3d12videodevice3-createvideoencoder).
