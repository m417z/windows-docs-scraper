## Description

Specifies the input source for video encoder input maps.

## Constants

### `D3D12_VIDEO_ENCODER_INPUT_MAP_SOURCE_CPU_BUFFER` : 0

Indicates the input is a CPU structure/buffer.

### `D3D12_VIDEO_ENCODER_INPUT_MAP_SOURCE_GPU_TEXTURE` : 1

Indicates the input is an [ID3D12Resource](https://learn.microsoft.com/windows/win32/api/d3d12/nn-d3d12-id3d12resource) texture.
