# PRJ_VIRTUALIZATION_INSTANCE_INFO structure

## Description

Information about a virtualization instance.

## Members

### `InstanceID`

An ID corresponding to a specific virtualization instance.

### `WriteAlignment`

The value used for the byteOffset and length parameters of [PrjWriteFileData](https://learn.microsoft.com/windows/desktop/api/projectedfslib/nf-projectedfslib-prjwritefiledata).
