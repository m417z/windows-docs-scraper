# PRJ_EXT_INFO_TYPE enumeration

## Description

Specifies extended information types for [PRJ_EXTENDED_INFO](https://learn.microsoft.com/windows/desktop/api/projectedfslib/ns-projectedfslib-prj_extended_info).

## Constants

### `PRJ_EXT_INFO_TYPE_SYMLINK:1`

This [PRJ_EXTENDED_INFO](https://learn.microsoft.com/windows/desktop/api/projectedfslib/ns-projectedfslib-prj_extended_info) specifies the target of a symbolic link.
