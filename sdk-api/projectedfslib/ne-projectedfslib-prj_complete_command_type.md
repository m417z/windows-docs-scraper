# PRJ_COMPLETE_COMMAND_TYPE enumeration

## Description

Specifies command types.

## Constants

### `PRJ_COMPLETE_COMMAND_TYPE_NOTIFICATION:1`

The provider is completing a call to its [PRJ_NOTIFICATION_CB](https://learn.microsoft.com/windows/desktop/api/projectedfslib/nc-projectedfslib-prj_notification_cb) callback.

### `PRJ_COMPLETE_COMMAND_TYPE_ENUMERATION:2`

The provider is completing a call to its [PRJ_GET_DIRECTORY_ENUMERATION_CB](https://learn.microsoft.com/windows/desktop/api/projectedfslib/nc-projectedfslib-prj_get_directory_enumeration_cb) callback.
