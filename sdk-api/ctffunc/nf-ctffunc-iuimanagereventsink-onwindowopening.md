# IUIManagerEventSink::OnWindowOpening

## Description

Called by the TSF before opening an IME UI.

## Parameters

### `prcBounds` [in]

Pointer to a **RECT** structure defining the affected area (in screen coordinates).

## Return value

Ignored.
