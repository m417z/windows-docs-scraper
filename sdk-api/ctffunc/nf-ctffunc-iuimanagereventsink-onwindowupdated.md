# IUIManagerEventSink::OnWindowUpdated

## Description

Called by the TSF after resizing and/or relocating the opened IME UI.

## Parameters

### `prcUpdatedBounds` [in]

Pointer to a **RECT** structure defining the affected area (in screen coordinates).

## Return value

Ignored.
