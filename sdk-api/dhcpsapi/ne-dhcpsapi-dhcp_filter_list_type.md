# DHCP_FILTER_LIST_TYPE enumeration

## Description

The **DHCP_FILTER_LIST_TYPE** enumeration specifies the types of filter lists available on the DHCP server.

## Constants

### `Deny`

The filter list is a deny list.

### `Allow`

The filter list is an allow list.
