# acmStreamMessage function

## Description

The **acmStreamMessage** function sends a driver-specific message to an ACM driver.

## Parameters

### `has`

Handle to an open conversion stream.

### `uMsg`

Message to send.

### `lParam1`

Message parameter.

### `lParam2`

Message parameter.

## Return value

Returns the value returned by the ACM device driver.
