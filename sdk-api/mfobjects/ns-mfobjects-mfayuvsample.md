# MFAYUVSample structure

## Description

Describes a 4:4:4:4 Y'Cb'Cr' sample.

## Members

### `bCrValue`

Cr (chroma difference) value.

### `bCbValue`

Cb (chroma difference) value.

### `bYValue`

Y (luma) value.

### `bSampleAlpha8`

Alpha value.
