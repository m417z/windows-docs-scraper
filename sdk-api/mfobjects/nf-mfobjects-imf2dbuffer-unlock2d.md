# IMF2DBuffer::Unlock2D

## Description

Unlocks a buffer that was previously locked. Call this method once for each call to [IMF2DBuffer::Lock2D](https://learn.microsoft.com/windows/desktop/api/mfobjects/nf-mfobjects-imf2dbuffer-lock2d).

## Return value

The method returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return code | Description |
| --- | --- |
| **S_OK** | The method succeeded. |
