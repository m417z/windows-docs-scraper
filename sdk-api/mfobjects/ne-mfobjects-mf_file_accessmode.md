# MF_FILE_ACCESSMODE enumeration

## Description

Specifies the requested access mode for opening a file.

## Constants

### `MF_ACCESSMODE_READ:1`

Read mode.

### `MF_ACCESSMODE_WRITE:2`

Write mode.

### `MF_ACCESSMODE_READWRITE:3`

Read and write mode.
