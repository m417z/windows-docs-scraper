# IMetaDataTables::GetNumTables

## Description

Gets the number of tables in the scope of the current [IMetaDataTables](https://learn.microsoft.com/windows/desktop/api/rometadataapi/nn-rometadataapi-imetadatatables) instance.

## Parameters

### `pcTables` [out]

A pointer to the number of tables in the current instance scope.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
