# IFsrmReportManager::GetReportJob

## Description

Retrieves the specified report job.

## Parameters

### `taskName` [in]

The task name that identifies the report job to retrieve. The string is limited to 230 characters.

### `reportJob` [out]

An [IFsrmReportJob](https://learn.microsoft.com/previous-versions/windows/desktop/api/fsrmreports/nn-fsrmreports-ifsrmreportjob) interface to the retrieved job.

## Return value

The method returns the following return values.
