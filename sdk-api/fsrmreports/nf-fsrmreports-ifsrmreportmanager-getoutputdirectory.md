# IFsrmReportManager::GetOutputDirectory

## Description

Retrieves the local directory path where the reports with the specified context are stored.

## Parameters

### `context` [in]

The report context (for example, if the report is scheduled or run on demand). For possible values, see the [FsrmReportGenerationContext](https://learn.microsoft.com/windows/desktop/api/fsrmenums/ne-fsrmenums-fsrmreportgenerationcontext) enumeration.

### `path` [out]

The local directory path where the reports are stored.

## Return value

The method returns the following return values.
