# IFsrmFileConditionProperty::get_Operator

## Description

[This property is supported for compatibility but it's recommended to use the
[FSRM WMI Classes](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/fsrm-wmi-classes) to manage FSRM. Please see the
[MSFT_FSRMFMJCondition](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/msft-fsrmfmjcondition) class.]

Specifies the comparison operator, as enumerated by the
[FsrmPropertyConditionType](https://learn.microsoft.com/windows/desktop/api/fsrmenums/ne-fsrmenums-fsrmpropertyconditiontype)
enumeration.

This property is read/write.
