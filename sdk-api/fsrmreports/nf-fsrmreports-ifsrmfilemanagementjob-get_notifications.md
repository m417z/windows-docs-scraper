# IFsrmFileManagementJob::get_Notifications

## Description

[This method is supported for compatibility but it's recommended to use the
[FSRM WMI Classes](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/fsrm-wmi-classes) to manage FSRM. Please see the
[MSFT_FSRMFMJNotification](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/msft-fsrmfmjnotification) class.]

The list of notification periods defined for the job.

This property is read-only.
