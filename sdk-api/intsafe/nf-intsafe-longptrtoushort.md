# LongPtrToUShort function

## Description

Converts a value of type **LONG_PTR** to a value of type **USHORT**.

## Parameters

### `lOperand` [in]

The value to convert.

### `pusResult` [out]

The converted value.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
