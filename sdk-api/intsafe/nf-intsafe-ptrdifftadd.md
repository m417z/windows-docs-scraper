# PtrdiffTAdd function

## Description

Adds two values of type **ptrdiff_t**.

## Parameters

### `Augend` [in]

The first value.

### `Addend` [in]

The second value.

### `pResult` [out]

The result.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
