# Int8ToUInt8 function

## Description

Converts a value of type **INT8** to a value of type **UINT8**.

## Parameters

### `i8Operand` [in]

The value to convert.

### `pu8Result` [out]

The converted value.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
