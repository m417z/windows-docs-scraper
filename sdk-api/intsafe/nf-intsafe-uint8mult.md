# UInt8Mult function

## Description

Multiplies two values of type **UINT8**.

## Parameters

### `u8Multiplicand` [in]

The first value.

### `u8Multiplier` [in]

The second value.

### `pu8Result` [out]

The result.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
