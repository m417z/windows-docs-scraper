# UInt8Sub function

## Description

Subtracts one value of type **UINT8** from another.

## Parameters

### `u8Minuend` [in]

The first value.

### `u8Subtrahend` [in]

The value to subtract.

### `pu8Result` [out]

The result.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
