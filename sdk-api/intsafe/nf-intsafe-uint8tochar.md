# UInt8ToChar function

## Description

Converts a value of type **UINT8** to a value of type **CHAR**.

## Parameters

### `u8Operand` [in]

The value to convert.

### `pch` [out]

The converted value.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
