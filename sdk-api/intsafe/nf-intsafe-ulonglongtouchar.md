# ULongLongToUChar function

## Description

Converts a value of type **ULONGLONG** to a value of type **UCHAR**.

## Parameters

### `ullOperand` [in]

The value to convert.

### `pch` [out]

The converted value.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
