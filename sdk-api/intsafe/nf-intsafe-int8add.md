# Int8Add function

## Description

Adds two values of type **INT8**.

## Parameters

### `i8Augend` [in]

The first value.

### `i8Addend` [in]

The second value.

### `pi8Result` [out]

The result.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
