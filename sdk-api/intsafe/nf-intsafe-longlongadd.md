# LongLongAdd function

## Description

Adds two values of type **LONGLONG**.

## Parameters

### `llAugend` [in]

The first value.

### `llAddend` [in]

The second value.

### `pllResult` [out]

The result.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
