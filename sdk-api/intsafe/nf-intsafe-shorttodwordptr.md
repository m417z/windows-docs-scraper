# ShortToDWordPtr function

## Description

Converts a value of type **SHORT** to a value of type **DWORD_PTR**.

## Parameters

### `sOperand` [in]

The value to convert.

### `pdwResult` [out]

The converted value.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
