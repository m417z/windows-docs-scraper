# LongPtrToInt8 function

## Description

Converts a value of type **LONG_PTR** to a value of type **INT8**.

## Parameters

### `lOperand` [in]

The value to convert.

### `pi8Result` [out]

The converted value.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
