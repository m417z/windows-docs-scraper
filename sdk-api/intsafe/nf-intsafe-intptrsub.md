# IntPtrSub function

## Description

Subtracts one value of type **INT_PTR** from another.

## Parameters

### `iMinuend` [in]

The first value.

### `iSubtrahend` [in]

The value to subtract.

### `piResult` [out]

The result.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
