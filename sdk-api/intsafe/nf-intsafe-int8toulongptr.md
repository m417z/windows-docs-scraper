# Int8ToULongPtr function

## Description

Converts a value of type **INT8** to a value of type **ULONG_PTR**.

## Parameters

### `i8Operand` [in]

The value to convert.

### `pulResult` [out]

The converted value.

## Return value

If this function succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
