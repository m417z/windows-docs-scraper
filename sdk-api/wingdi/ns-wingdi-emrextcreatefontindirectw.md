# EMREXTCREATEFONTINDIRECTW structure

## Description

The **EMREXTCREATEFONTINDIRECTW** structure contains members for the [CreateFontIndirect](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-createfontindirecta) enhanced metafile record.

## Members

### `emr`

The base structure for all record types.

### `ihFont`

Index to the font in handle table.

### `elfw`

Logical font.
