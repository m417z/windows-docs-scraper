# EMRSETMAPPERFLAGS structure

## Description

The **EMRSETMAPPERFLAGS** structure contains members for the **SetMapperFlags** enhanced metafile record.

## Members

### `emr`

The base structure for all record types.

### `dwFlags`

Font mapper flag.
