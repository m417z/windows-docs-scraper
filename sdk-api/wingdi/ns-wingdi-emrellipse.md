# EMRELLIPSE structure

## Description

The **EMRELLIPSE** and **EMRRECTANGLE** structures contain members for the [Ellipse](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-ellipse) and [Rectangle](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-rectangle) enhanced metafile records.

## Members

### `emr`

Base structure for all record types.

### `rclBox`

Bounding rectangle in logical units.
