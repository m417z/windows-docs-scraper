# EMRFILLPATH structure

## Description

The **EMRFILLPATH**, **EMRSTROKEANDFILLPATH**, and **EMRSTROKEPATH** structures contain members for the [FillPath](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-fillpath), [StrokeAndFillPath](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-strokeandfillpath), and [StrokePath](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-strokepath) enhanced metafile records.

## Members

### `emr`

Base structure for all record types.

### `rclBounds`

Bounding rectangle, in device units.
