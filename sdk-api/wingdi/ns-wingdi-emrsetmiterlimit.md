# EMRSETMITERLIMIT structure

## Description

The **EMRSETMITERLIMIT** structure contains members for the **SetMiterLimit** enhanced metafile record.

## Members

### `emr`

The base structure for all record types.

### `eMiterLimit`

New miter limit.
