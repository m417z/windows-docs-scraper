# EMRCREATEPEN structure

## Description

The **EMRCREATEPEN** structure contains members for the **CreatePen** enhanced metafile record.

## Members

### `emr`

The base structure for all record types.

### `ihPen`

Index to pen in handle table.

### `lopn`

Logical pen.
