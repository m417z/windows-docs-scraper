# EMRANGLEARC structure

## Description

The **EMRANGLEARC** structure contains members for the [AngleArc](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-anglearc) enhanced metafile record.

## Members

### `emr`

The base structure for all record types.

### `ptlCenter`

Logical coordinates of a circle's center.

### `nRadius`

A circle's radius, in logical units.

### `eStartAngle`

An arc's start angle, in degrees.

### `eSweepAngle`

An arc's sweep angle, in degrees.
