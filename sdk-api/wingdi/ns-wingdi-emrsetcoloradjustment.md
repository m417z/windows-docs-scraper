# EMRSETCOLORADJUSTMENT structure

## Description

The **EMRSETCOLORADJUSTMENT** structure contains members for the **SetColorAdjustment** enhanced metafile record.

## Members

### `emr`

The base structure for all record types.

### `ColorAdjustment`

A [COLORADJUSTMENT](https://learn.microsoft.com/windows/desktop/api/wingdi/ns-wingdi-coloradjustment) structure.
