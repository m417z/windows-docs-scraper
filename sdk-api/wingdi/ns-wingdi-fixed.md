# FIXED structure

## Description

The **FIXED** structure contains the integral and fractional parts of a fixed-point real number.

## Members

### `value`

The integer part of the number.

### `fract`

The fractional part of the number.

## Remarks

The **FIXED** structure is used to describe the elements of the [MAT2](https://learn.microsoft.com/windows/desktop/api/wingdi/ns-wingdi-mat2) structure.
