# POINTFX structure

## Description

The **POINTFX** structure contains the coordinates of points that describe the outline of a character in a TrueType font.

## Members

### `x`

The x-component of a point on the outline of a TrueType character.

### `y`

The y-component of a point on the outline of a TrueType character.

## Remarks

The **POINTFX** structure is a member of the [TTPOLYCURVE](https://learn.microsoft.com/windows/desktop/api/wingdi/ns-wingdi-ttpolycurve) and [TTPOLYGONHEADER](https://learn.microsoft.com/windows/desktop/api/wingdi/ns-wingdi-ttpolygonheader) structures. Values in the **POINTFX** structure are specified in device units.
