# EMRSELECTCLIPPATH structure

## Description

Contains parameters for the [SelectClipPath](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-selectclippath), [SetBkMode](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-setbkmode), [SetMapMode](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-setmapmode), [SetPolyFillMode](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-setpolyfillmode), [SetROP2](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-setrop2), [SetStretchBltMode](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-setstretchbltmode), [SetTextAlign](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-settextalign), [SetICMMode](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-seticmmode) , and [SetLayout](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-setlayout) enhanced metafile records.

## Members

### `emr`

The base structure for all record types.

### `iMode`

A value and meaning that varies depending on the function contained in the enhanced metafile record. For a description of this member, see the documentation of the functions contained in this record.
