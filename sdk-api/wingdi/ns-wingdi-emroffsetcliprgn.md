# EMROFFSETCLIPRGN structure

## Description

The **EMROFFSETCLIPRGN** structure contains members for the **OffsetClipRgn** enhanced metafile record.

## Members

### `emr`

The base structure for all record types.

### `ptlOffset`

The logical coordinates of offset.
