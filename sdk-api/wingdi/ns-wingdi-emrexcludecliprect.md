# EMREXCLUDECLIPRECT structure

## Description

The **EMREXCLUDECLIPRECT** and **EMRINTERSECTCLIPRECT** structures contain members for the [ExcludeClipRect](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-excludecliprect) and [IntersectClipRect](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-intersectcliprect) enhanced metafile records.

## Members

### `emr`

Base structure for all record types.

### `rclClip`

Clipping rectangle in logical units.
