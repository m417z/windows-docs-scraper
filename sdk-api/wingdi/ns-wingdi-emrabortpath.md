# EMRABORTPATH structure

## Description

Contains data for the [AbortPath](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-abortpath), [BeginPath](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-beginpath), [EndPath](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-endpath), [CloseFigure](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-closefigure), [FlattenPath](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-flattenpath), [WidenPath](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-widenpath), [SetMetaRgn](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-setmetargn), [SaveDC](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-savedc), and [RealizePalette](https://learn.microsoft.com/windows/desktop/api/wingdi/nf-wingdi-realizepalette) enhanced metafile records.

## Members

### `emr`

The base structure for all record types.
