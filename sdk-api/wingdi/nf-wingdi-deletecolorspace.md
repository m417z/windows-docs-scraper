# DeleteColorSpace function

## Description

The **DeleteColorSpace** function removes and destroys a specified [color space](https://learn.microsoft.com/windows/win32/wcs/c#color-space).

## Parameters

### `hcs`

Specifies the handle to a color space to delete.

## Return value

If this function succeeds, the return value is **TRUE**.

If this function fails, the return value is **FALSE**.
