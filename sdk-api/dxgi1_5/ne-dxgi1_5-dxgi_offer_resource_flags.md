# DXGI_OFFER_RESOURCE_FLAGS enumeration

## Description

Specifies flags for the [OfferResources1](https://learn.microsoft.com/windows/desktop/api/dxgi1_5/nf-dxgi1_5-idxgidevice4-offerresources1) method.

## Constants

### `DXGI_OFFER_RESOURCE_FLAG_ALLOW_DECOMMIT:0x1`

Indicates the ability to allow memory de-commit by the DirectX Graphics Kernel.
