# D3DX11_SCAN_DIRECTION enumeration

## Description

Direction to perform scan in.

## Constants

### `D3DX11_SCAN_DIRECTION_FORWARD:1`

Scan forward.

### `D3DX11_SCAN_DIRECTION_BACKWARD`

Scan backward.
