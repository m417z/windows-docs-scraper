# SetProviderStatusInfoFreeFunc callback function

## Description

Frees a
[SetProviderStatusInfo](https://learn.microsoft.com/windows/desktop/api/timeprov/ns-timeprov-setproviderstatusinfo) structure.

## Parameters

### `pspsi` [in]

A pointer to the
[SetProviderStatusInfo](https://learn.microsoft.com/windows/desktop/api/timeprov/ns-timeprov-setproviderstatusinfo) structure to be freed.
