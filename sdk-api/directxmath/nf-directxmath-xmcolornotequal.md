# XMColorNotEqual function

## Description

Tests to see whether two colors are unequal.

## Parameters

### `C1` [in]

**XMVECTOR** describing the first color to compare.

### `C2` [in]

**XMVECTOR** describing the second color to compare.

## Return value

Returns true if any component of *C1* is different from the corresponding component of *C2*. Returns
false otherwise.

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
