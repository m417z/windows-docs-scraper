# XMVectorGetX function

## Description

Retrieve the `x` component of an [XMVECTOR Data Type](https://learn.microsoft.com/windows/desktop/dxmath/xmvector-data-type).

## Parameters

### `V` [in]

A valid 4D vector storing floating-point data

## Return value

The value in the `x` component of the 4D vector storing floating-point data *V*

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
