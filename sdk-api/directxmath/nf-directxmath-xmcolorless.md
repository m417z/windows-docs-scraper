# XMColorLess function

## Description

Tests whether all the components of the first color are less than the corresponding components of the second color.

## Parameters

### `C1` [in]

**XMVECTOR** describing the first color to compare.

### `C2` [in]

**XMVECTOR** describing the second color to compare.

## Return value

Returns true if every component of *C1* is less than the corresponding component in *C2*. Returns
false otherwise.

## Remarks

### Platform Requirements

Microsoft Visual Studio 2010 or Microsoft Visual Studio 2012 with the Windows SDK for Windows 8. Supported for Win32 desktop apps, Windows Store apps, and Windows Phone 8 apps.
