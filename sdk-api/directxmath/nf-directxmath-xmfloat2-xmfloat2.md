# XMFLOAT2::XMFLOAT2

## Description

Default constructor for `XMFLOAT2`.

Default constructor for [XMFLOAT2](https://learn.microsoft.com/windows/desktop/api/directxmath/ns-directxmath-xmfloat2).

**Note** This constructor is only available under C++.

## Remarks

`XMFLOAT2()` constructs a new instance of the `XMFLOAT2` structure without
defining the value of any of its members.
