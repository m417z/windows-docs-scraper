# XMFLOAT2::XMFLOAT2(float,float)

## Description

Initializes a new instance of `XMFLOAT2` from two `float` arguments.

This constructor initializes a new instance of [XMFLOAT2](https://learn.microsoft.com/windows/desktop/api/directxmath/ns-directxmath-xmfloat2) from two
`float` arguments.

**Note** This constructor is only available under C++.

## Parameters

### `_x`

Value of the x-coordinate of the vector, the **x** member of the new
`XMFLOAT2` instance.

### `_y`

Value of the y-coordinate of the vector, the **y** member of the new
`XMFLOAT2` instance.

## Remarks

The following pseudocode demonstrates the operation of this constructor:

```

	XMFLOAT2 instance;

	instance.x = _x;
	instance.y = _y;

```
