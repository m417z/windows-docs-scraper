# CF_OPERATION_ACK_DEHYDRATE_FLAGS enumeration

## Description

Flags to acknowledge the dehydration of a placeholder file or directory.

## Constants

### `CF_OPERATION_ACK_DEHYDRATE_FLAG_NONE:0x00000000`

No acknowledge dehydration flag.
