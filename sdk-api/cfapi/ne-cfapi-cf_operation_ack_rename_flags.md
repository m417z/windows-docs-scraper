# CF_OPERATION_ACK_RENAME_FLAGS enumeration

## Description

Flags for acknowledging placeholder file or directory renaming.

## Constants

### `CF_OPERATION_ACK_RENAME_FLAG_NONE:0x00000000`

No acknowledgment of placeholder renaming.
