# CF_OPERATION_RETRIEVE_DATA_FLAGS enumeration

## Description

Flags to retrieve data for a placeholder file or folder.

## Constants

### `CF_OPERATION_RETRIEVE_DATA_FLAG_NONE:0x00000000`

No retrieve data flag.
