# CF_OPERATION_ACK_DELETE_FLAGS enumeration

## Description

Flags to acknowledge the deletion of a placeholder file or directory.

## Constants

### `CF_OPERATION_ACK_DELETE_FLAG_NONE:0x00000000`

No deletion acknowledgment flags.
