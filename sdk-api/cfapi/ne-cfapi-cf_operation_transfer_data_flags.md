# CF_OPERATION_TRANSFER_DATA_FLAGS enumeration

## Description

Flags to transfer data to hydrate a placeholder file or folder.

## Constants

### `CF_OPERATION_TRANSFER_DATA_FLAG_NONE:0x00000000`

No transfer data flag.
