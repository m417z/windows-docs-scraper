# IBootTrigger::put_Delay

## Description

Gets or sets a value that indicates the amount of time between when the system is booted and when the task is started.

This property is read/write.

## Parameters

## Remarks

When reading or writing your own XML for a task, the boot delay is specified using the [Delay](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-delay-boottriggertype-element) element of the Task Scheduler schema.
