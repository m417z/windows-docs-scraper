# ITaskSettings::put_StopIfGoingOnBatteries

## Description

Gets or sets a Boolean value that indicates that the task will be stopped if the computer is going onto batteries.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, this setting is specified in the [StopIfGoingOnBatteries](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-stopifgoingonbatteries-settingstype-element) element of the Task Scheduler schema.
