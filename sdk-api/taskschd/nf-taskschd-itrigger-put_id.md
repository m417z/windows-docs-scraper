# ITrigger::put_Id

## Description

Gets or sets the identifier for the trigger.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, the trigger identifier is specified in the Id attribute of the individual trigger elements (for example, the Id attribute of the [BootTrigger](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-boottrigger-triggergroup-element) element) of the Task Scheduler schema.
