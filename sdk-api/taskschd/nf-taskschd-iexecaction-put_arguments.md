# IExecAction::put_Arguments

## Description

Gets or sets the arguments associated with the command-line operation.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML, the command-line operation arguments are specified in the [Arguments](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-arguments-exectype-element) element of the Task Scheduler schema.
