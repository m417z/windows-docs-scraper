# IIdleSettings::put_IdleDuration

## Description

 Gets or sets a value that indicates the amount of time that the computer must be in an idle state before the task is run.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, this setting is specified in the [Duration](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-duration-idlesettingstype-element) element of the Task Scheduler schema.
