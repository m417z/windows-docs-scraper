# ITrigger::get_Enabled

## Description

Gets or sets a Boolean value that indicates whether the trigger is enabled.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, the enabled property is specified using the [Enabled](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-enabled-triggerbasetype-element) element of the Task Scheduler schema.
