# ITaskService::get_ConnectedDomain

## Description

Gets the name of the domain to which the [TargetServer](https://learn.microsoft.com/windows/desktop/api/taskschd/nf-taskschd-itaskservice-get_targetserver) computer is connected.

This property is read-only.
