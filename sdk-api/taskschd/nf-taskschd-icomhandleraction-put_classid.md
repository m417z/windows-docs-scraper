# IComHandlerAction::put_ClassId

## Description

Gets or sets the identifier of the handler class.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML, the class of a COM handler is specified in the [ClassId](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-classid-comhandlertype-element) element of the Task Scheduler schema.
