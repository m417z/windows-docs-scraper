# ITaskSettings::get_RestartCount

## Description

Gets or sets the number of times that the Task Scheduler will attempt to restart the task.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, this setting is specified in the [Count](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-count-restarttype-element) element of the Task Scheduler schema.
