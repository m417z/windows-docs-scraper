# ITaskSettings::get_RestartInterval

## Description

Gets or sets a value that specifies how long the Task Scheduler will attempt to restart the task.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, this setting is specified in the [Interval](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-interval-restarttype-element) element of the Task Scheduler schema.
