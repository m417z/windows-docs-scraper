# IRepetitionPattern::get_Interval

## Description

Gets or sets the amount of time between each restart of the task.

This property is read/write.

## Parameters

## Remarks

If you specify a repetition duration for a task, you must also specify the repetition interval.

When reading or writing XML for a task, the repetition interval is specified in the [Interval](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-interval-repetitiontype-element) element of the Task Scheduler schema.
