# ITaskSettings::get_MultipleInstances

## Description

Gets or sets the policy that defines how the Task Scheduler deals with multiple instances of the task.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, this setting is specified in the [MultipleInstancesPolicy](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-multipleinstancespolicy-settingstype-element) element of the Task Scheduler schema.
