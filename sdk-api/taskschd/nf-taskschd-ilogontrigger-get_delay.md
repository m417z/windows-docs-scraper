# ILogonTrigger::get_Delay

## Description

Gets or sets a value that indicates the amount of time between when the user logs on and when the task is started.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, the logon trigger delay is specified using the [Delay](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-delay-logontriggertype-element) element of the Task Scheduler schema.
