# ITaskSettings::get_Enabled

## Description

Gets or sets a Boolean value that indicates that the task is enabled. The task can be performed only when this setting is True.

This property is read-only.

## Parameters

## Remarks

When reading or writing XML for a task, this setting is specified in the [Enabled (settingsType)](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-enabled-settingstype-element) element of the Task Scheduler schema.
