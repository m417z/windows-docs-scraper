# IComHandlerAction::get_Data

## Description

Gets or sets additional data that is associated with the handler.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML, the data of a COM handler is specified in the [Data](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-data-comhandlertype-element) element of the Task Scheduler schema.
