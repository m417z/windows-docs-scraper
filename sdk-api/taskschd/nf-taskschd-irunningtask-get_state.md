# IRunningTask::get_State

## Description

Gets an identifier for the state of the running task.

This property is read-only.

## Parameters

## Remarks

The [IRunningTask::Refresh](https://learn.microsoft.com/windows/desktop/api/taskschd/nf-taskschd-irunningtask-refresh) method is called before the property value is returned.
