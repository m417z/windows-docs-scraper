# IRegistrationInfo::put_Version

## Description

Gets or sets the version number of the task.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, the version number of the task is specified using the [Version](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-version-registrationinfotype-element) element of the Task Scheduler schema.
