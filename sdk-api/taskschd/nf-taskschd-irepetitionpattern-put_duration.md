# IRepetitionPattern::put_Duration

## Description

Gets or sets how long the pattern is repeated.

This property is read/write.

## Parameters

## Remarks

If you specify a repetition duration for a task, you must also specify the repetition interval.

When reading or writing XML for a task, the repetition duration is specified in the [Duration](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-duration-repetitiontype-element) element of the Task Scheduler schema.
