# ITaskSettings2::put_UseUnifiedSchedulingEngine

## Description

Gets or sets a Boolean value that indicates that the Unified Scheduling Engine will be utilized to run this task.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, this setting is specified in the [UseUnifiedSchedulingEngine](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-useunifiedschedulingengine-settingstype-element) element of the Task Scheduler schema.
