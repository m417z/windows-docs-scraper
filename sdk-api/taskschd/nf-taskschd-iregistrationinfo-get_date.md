# IRegistrationInfo::get_Date

## Description

Gets or sets the date and time when the task is registered.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, the registration date is specified using the [Date](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-date-registrationinfotype-element) element of the Task Scheduler schema.
