# IRegistrationInfo::put_URI

## Description

Gets or sets the URI of the task.

This property is read/write.

## Parameters

## Remarks

When reading or writing XML for a task, the task URI is specified using the [URI](https://learn.microsoft.com/windows/desktop/TaskSchd/taskschedulerschema-uri-registrationinfotype-element) element of the Task Scheduler schema.
