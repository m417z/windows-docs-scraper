# IBDA_DigitalDemodulator2::put_Pilot

## Description

\[The feature associated with this page, [Microsoft TV Technologies](https://learn.microsoft.com/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal), is a legacy feature. Microsoft strongly recommends that new code does not use this feature.\]

Sets the current pilot mode for Digital Video Broadcasting-S2 (DVB-S2).

## Parameters

### `pPilot` [in]

Pointer to a variable that contains the pilot mode, specified as a member of the [Pilot](https://learn.microsoft.com/previous-versions/windows/desktop/mstv/pilot) enumeration.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
