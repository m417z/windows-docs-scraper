# IBDA_DigitalDemodulator2::put_RollOff

## Description

\[The feature associated with this page, [Microsoft TV Technologies](https://learn.microsoft.com/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal), is a legacy feature. Microsoft strongly recommends that new code does not use this feature.\]

Sets the demodulator's roll-off factor.

## Parameters

### `pRollOff` [in]

Pointer to a variable that contains the roll-off factor, specified as a member of the [RollOff](https://learn.microsoft.com/previous-versions/windows/desktop/mstv/rolloff) enumeration.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
