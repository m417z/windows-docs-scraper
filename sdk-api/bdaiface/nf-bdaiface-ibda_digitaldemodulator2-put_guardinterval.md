# IBDA_DigitalDemodulator2::put_GuardInterval

## Description

\[The feature associated with this page, [Microsoft TV Technologies](https://learn.microsoft.com/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal), is a legacy feature. Microsoft strongly recommends that new code does not use this feature.\]

Sets the demodulator's guard interval.

## Parameters

### `pGuardInterval` [in]

Pointer to a variable that contains the guard interval, specified as a member of the [GuardInterval](https://learn.microsoft.com/previous-versions/windows/desktop/mstv/guardinterval) enumeration.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
