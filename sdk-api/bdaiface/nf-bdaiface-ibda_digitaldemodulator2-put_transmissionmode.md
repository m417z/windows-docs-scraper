# IBDA_DigitalDemodulator2::put_TransmissionMode

## Description

\[The feature associated with this page, [Microsoft TV Technologies](https://learn.microsoft.com/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal), is a legacy feature. Microsoft strongly recommends that new code does not use this feature.\]

Sets the demodulator's transmission mode.

## Parameters

### `pTransmissionMode` [in]

Pointer to a variable that contains the transmission mode, specified as a member of the [TransmissionMode](https://learn.microsoft.com/previous-versions/windows/desktop/mstv/transmissionmode) enumeration.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
