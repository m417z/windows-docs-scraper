# IWRdsGraphicsChannel::Close

## Description

Called to close the channel. The [IWRdsGraphicsChannelEvents::OnClose](https://learn.microsoft.com/windows/desktop/api/wrdsgraphicschannels/nf-wrdsgraphicschannels-iwrdsgraphicschannelevents-onclose) method will be called when the channel is completely closed.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
