# SR_RESOURCE_TYPE_REPLICATED_DISKS_RESULT structure

## Description

Describes a retrieved set of replicated disks.

## Members

### `Count`

The number of replicated disks in the result set.

### `ReplicatedDisks`

The retrieved array of replicated disks.
