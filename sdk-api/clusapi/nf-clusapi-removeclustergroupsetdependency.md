# RemoveClusterGroupSetDependency function

## Description

Removes a groupset from a groupset's dependency expression.

## Parameters

### `hGroupSet` [in]

The groupset from which to remove the dependency.

### `hDependsOn` [in]

The collection to remove

## Return value

If the operation succeeds, the function returns **ERROR_SUCCESS**.

If the operation fails,
the function returns a [system error code](https://learn.microsoft.com/windows/desktop/Debug/system-error-codes).
