# SetClusterGroupSetDependencyExpression function

## Description

Sets the dependency expression for a cluster groupset.

## Parameters

### `hGroupSet` [in]

The collection to receive the dependency expression

### `lpszDependencyExprssion` [in]

The dependency expression for *hCollection*

## Return value

If the operation succeeds, the function returns **ERROR_SUCCESS**.

If the operation fails,
the function returns a [system error code](https://learn.microsoft.com/windows/desktop/Debug/system-error-codes).
