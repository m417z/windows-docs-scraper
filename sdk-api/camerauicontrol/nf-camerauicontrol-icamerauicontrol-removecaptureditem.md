# ICameraUIControl::RemoveCapturedItem

## Description

Removes the captured item.

## Parameters

### `pszPath` [in]

The path to the item to delete.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
