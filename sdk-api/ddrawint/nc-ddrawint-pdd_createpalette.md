## Description

The **DdCreatePalette** callback function creates a DirectDrawPalette object for the specified DirectDraw object.

## Parameters

### `unnamedParam1`

Points to a [DD_CREATEPALETTEDATA](https://learn.microsoft.com/windows/desktop/api/ddrawint/ns-ddrawint-dd_createpalettedata) structure that contains the information necessary to create the DirectDrawPalette object.
