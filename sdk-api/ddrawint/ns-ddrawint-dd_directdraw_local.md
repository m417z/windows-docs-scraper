# DD_DIRECTDRAW_LOCAL structure

## Description

The DD_DIRECTDRAW_LOCAL structure contains driver information that is relevant to the current DirectDraw process only.

## Members

### `lpGbl`

Points to the [DD_DIRECTDRAW_GLOBAL](https://learn.microsoft.com/windows/desktop/api/ddrawint/ns-ddrawint-dd_directdraw_global) structure that describes the driver's device.
