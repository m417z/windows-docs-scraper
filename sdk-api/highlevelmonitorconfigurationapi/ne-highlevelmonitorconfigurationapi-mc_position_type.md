# MC_POSITION_TYPE enumeration

## Description

Specifies whether to get or set the vertical or horizontal position of a monitor's display area.

## Constants

### `MC_HORIZONTAL_POSITION`

Horizontal position.

### `MC_VERTICAL_POSITION`

Vertical position.
