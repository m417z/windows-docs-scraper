# MC_GAIN_TYPE enumeration

## Description

Specifies whether to get or set a monitor's red, green, or blue gain.

## Constants

### `MC_RED_GAIN`

Red gain.

### `MC_GREEN_GAIN`

Green gain.

### `MC_BLUE_GAIN`

Blue gain.
