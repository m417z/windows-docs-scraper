# USER_MODALS_INFO_1001 structure

## Description

The
**USER_MODALS_INFO_1001** structure contains the minimum length for passwords in the security database, which is the security accounts manager (SAM) database or, in the case of domain controllers, the Active Directory.

## Members

### `usrmod1001_min_passwd_len`

Specifies the minimum allowable password length. Valid values for this element are zero through PWLEN.
