# USER_INFO_1053 structure

## Description

The
**USER_INFO_1053** structure contains user information for network accounts. This information level is valid only when you call the
[NetUserSetInfo](https://learn.microsoft.com/windows/desktop/api/lmaccess/nf-lmaccess-netusersetinfo) function.

## Members

### `usri1053_home_dir_drive`

Specifies the drive letter to assign to the user's home directory for logon purposes. The user is specified in the *username* parameter to the
**NetUserSetInfo** function.
