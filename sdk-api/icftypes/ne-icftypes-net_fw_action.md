# NET_FW_ACTION enumeration

## Description

The **NET_FW_ACTION** enumerated type specifies the action for a rule or default setting.

## Constants

### `NET_FW_ACTION_BLOCK:0`

Block traffic.

### `NET_FW_ACTION_ALLOW`

Allow traffic.

### `NET_FW_ACTION_MAX`

Maximum traffic.
