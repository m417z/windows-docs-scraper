# IBitsPeerCacheRecord::GetFileSize

## Description

Gets the size of the file.

## Parameters

### `pVal` [out]

Size of the file, in bytes.

## Return value

The method returns the following return values.

| Return code | Description |
| --- | --- |
| **S_OK** | Success |
