# HrtfDirectivityType enumeration

## Description

Indicates one of several stock directivity patterns.

## Constants

### `OmniDirectional:0`

The sound emission is in all directions.

### `Cardioid`

The sound emission is a cardioid shape.

### `Cone`

The sound emission is a cone.
