# HrtfEnvironment enumeration

## Description

Indicates one of several stock environment types.

## Constants

### `Small:0`

A small room.

### `Medium`

A medium-sized room.

### `Large`

A large, enclosed space.

### `Outdoors`

An outdoor space.
