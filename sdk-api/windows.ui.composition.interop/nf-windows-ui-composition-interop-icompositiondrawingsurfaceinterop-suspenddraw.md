# ICompositionDrawingSurfaceInterop::SuspendDraw (windows.ui.composition.interop.h)

## Description

Suspends drawing on the surface object.

## Return value

Type: **HRESULT**

If this method succeeds, it returns S_OK. Otherwise, it returns an HRESULT error code.
