# PROGRESS_DIALOG_CHECKBOX_ID enumeration

## Description

The `PROGRESS_DIALOG_CHECKBOX_ID` enumeration type indicates the check box on the [IPhotoProgressDialog](https://learn.microsoft.com/windows/desktop/api/photoacquire/nn-photoacquire-iphotoprogressdialog) object.

## Constants

### `PROGRESS_DIALOG_CHECKBOX_ID_DEFAULT:0`

Specifies PROGRESS_DIALOG_CHECKBOX_ID_DEFAULT .
