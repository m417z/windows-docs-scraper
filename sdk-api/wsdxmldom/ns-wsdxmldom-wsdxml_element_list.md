# WSDXML_ELEMENT_LIST structure

## Description

Represents a node in a linked list of XML elements.

## Members

### `Next`

Reference to the next node in the linked list of **WSDXML_ELEMENT_LIST** structures.

### `Element`

The [WSDXML_ELEMENT](https://learn.microsoft.com/windows/desktop/api/wsdxmldom/ns-wsdxmldom-wsdxml_element) structure referenced by this node.
