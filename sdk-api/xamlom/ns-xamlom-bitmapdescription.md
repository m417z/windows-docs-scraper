# BitmapDescription structure

## Description

Represents information about the bitmap stored in [IBitmapData](https://learn.microsoft.com/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ibitmapdata).

## Members

### `Width`

The width of the bitmap.

### `Height`

The height of the bitmap.

### `Format`

The format of the bitmap.

### `AlphaMode`

The alpha mode of the bitmap.
