# IBitmapData::GetBitmapDescription

## Description

Gets a [BitmapDescription](https://learn.microsoft.com/previous-versions/windows/desktop/api/xamlom/ns-xamlom-bitmapdescription) that describes the bitmap data stored in the [IBitmapData](https://learn.microsoft.com/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ibitmapdata).

## Parameters

### `pBitmapDescription` [out]

Information about the bitmap stored in the [IBitmapData](https://learn.microsoft.com/previous-versions/windows/desktop/api/xamlom/nn-xamlom-ibitmapdata).

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
