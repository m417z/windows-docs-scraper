# IXamlDiagnostics::GetApplication

## Description

Gets an instance of the application.

## Parameters

### `ppApplication` [out, retval]

The application.

## Return value

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
