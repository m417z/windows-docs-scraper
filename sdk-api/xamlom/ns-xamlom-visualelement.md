# VisualElement structure

## Description

Represents a XAML element in the Live Visual Tree in Microsoft Visual Studio.

## Members

### `Handle`

A handle to the object.

### `SrcInfo`

Information about the XAML source document.

### `Type`

The type of the object.

### `Name`

The name of the XAML element, if it has an [x:Name](https://learn.microsoft.com/windows/uwp/xaml-platform/x-name-attribute) defined in markup.

### `NumChildren`

The number of children the XAML element has.
