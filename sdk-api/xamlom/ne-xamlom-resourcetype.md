# ResourceType enumeration

## Description

Defines constants that specify the type of a resource in a resource dictionary.

## Constants

### `ResourceTypeStatic:0`

The resource is a [StaticResource](https://learn.microsoft.com/windows/uwp/xaml-platform/staticresource-markup-extension).

### `ResourceTypeTheme`

The resource is a [ThemeResource](https://learn.microsoft.com/windows/uwp/xaml-platform/themeresource-markup-extension).
