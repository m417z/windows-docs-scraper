# PropertyChainSource structure

## Description

Represents the source object (a [Style](https://learn.microsoft.com/uwp/api/windows.ui.xaml.style)) of a target type.

## Members

### `Handle`

A handle to the style.

### `TargetType`

The target type of the style, defined in markup.

### `Name`

The name of the style, if it has an [x:Name](https://learn.microsoft.com/windows/uwp/xaml-platform/x-name-attribute) defined in markup.

### `Source`

Where the style is defined in the application .

### `SrcInfo`

### `srcInfo`

Source information about where the style is defined.
