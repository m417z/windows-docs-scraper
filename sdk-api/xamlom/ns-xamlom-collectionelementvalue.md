# CollectionElementValue structure

## Description

Represents an element in a collection.

## Members

### `Index`

The collection index where the element was found.

### `ValueType`

The type of the element object.

### `Value`

The value of the element.

### `MetadataBits`

A bit field representing [MetadataBit](https://learn.microsoft.com/previous-versions/windows/desktop/api/xamlom/ne-xamlom-metadatabit)s.
