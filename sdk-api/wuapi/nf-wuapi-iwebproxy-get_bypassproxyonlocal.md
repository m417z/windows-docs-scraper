# IWebProxy::get_BypassProxyOnLocal

## Description

Gets and sets a Boolean value that indicates whether local addresses bypass the proxy server.

This property is read/write.

## Parameters

## Remarks

The value of the **BypassProxyOnLocal** property is ignored if the value of the [AutoDetect](https://learn.microsoft.com/windows/desktop/api/wuapi/nf-wuapi-iwebproxy-get_autodetect) property is set to **VARIANT_TRUE**.
