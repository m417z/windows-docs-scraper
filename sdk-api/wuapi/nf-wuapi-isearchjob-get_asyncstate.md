# ISearchJob::get_AsyncState

## Description

Gets the caller-specific state object that is passed to the [IUpdateSearch.BeginSearch](https://learn.microsoft.com/windows/desktop/api/wuapi/nf-wuapi-iupdatesearcher-beginsearch) method.

This property is read-only.
