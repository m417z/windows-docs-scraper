# IAutomaticUpdatesResults::get_LastInstallationSuccessDate

## Description

Gets the last time and Coordinated Universal Time (UTC) date when Automatic Updates successfully installed any updates, even if some failures occurred.

This property is read-only.

## Parameters

## Remarks

Calls to **LastInstallationSuccessDate** by public users do not update this property. Only the [AutomaticUpdates](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iautomaticupdates) object will update this property.
