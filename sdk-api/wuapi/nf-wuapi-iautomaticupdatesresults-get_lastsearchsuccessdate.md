# IAutomaticUpdatesResults::get_LastSearchSuccessDate

## Description

Gets the last time and Coordinated Universal Time (UTC) date when AutomaticUpdates successfully searched for updates.

This property is read-only.

## Parameters

## Remarks

Calls to **LastSearchSuccessDate** by public users do not update this property. Only the [AutomaticUpdates](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iautomaticupdates) object will update this property.
