# IInstallationBehavior::get_RebootBehavior

## Description

Gets an [InstallationRebootBehavior](https://learn.microsoft.com/windows/desktop/api/wuapi/ne-wuapi-installationrebootbehavior) enumeration that specifies the restart behavior that occurs when you install or uninstall the update.

This property is read-only.
