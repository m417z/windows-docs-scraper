# IAutomaticUpdatesSettings::put_NotificationLevel

## Description

[**Set** is no longer supported. Starting with
Windows 10 calls to **Set** always return
**S_OK**, but do nothing.]

Gets and sets how users are notified about Automatic Update events.

This property is read/write.
