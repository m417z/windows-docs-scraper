# IUpdateDownloadContentCollection::get_Item

## Description

Gets the download content for an update from an [IUpdateDownloadContentCollection](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iupdatedownloadcontentcollection) interface.

This property is read-only.
