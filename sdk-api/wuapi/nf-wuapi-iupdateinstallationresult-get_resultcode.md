# IUpdateInstallationResult::get_ResultCode

## Description

Gets an [OperationResultCode](https://learn.microsoft.com/windows/desktop/api/wuapi/ne-wuapi-operationresultcode) value that specifies the result of an operation on an update.

This property is read-only.
