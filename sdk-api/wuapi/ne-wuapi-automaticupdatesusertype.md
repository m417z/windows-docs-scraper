# AutomaticUpdatesUserType enumeration

## Description

Defines the type of user.

## Constants

### `auutCurrentUser:1`

The context of the current user.

### `auutLocalAdministrator:2`

Any administrator on the local computer.

## Remarks

The AutomaticUpdatesUserType is used in conjunction with the [IAutomaticUpdatesSettings2::CheckPermission](https://learn.microsoft.com/windows/desktop/api/wuapi/nf-wuapi-iautomaticupdatessettings2-checkpermission) method.
