# IUpdateHistoryEntry::get_UpdateIdentity

## Description

Gets the [IUpdateIdentity](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iupdateidentity) interface that contains the identity of the update.

This property is read-only.
