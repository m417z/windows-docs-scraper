# IAutomaticUpdates2::get_Results

## Description

Returns a pointer to an [IAutomaticUpdatesResults](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iautomaticupdatesresults) interface.

## Parameters

### `retval` [out]

A pointer to an [IAutomaticUpdatesResults](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iautomaticupdatesresults) interface.

## Return value

Returns **S_OK** if successful. Otherwise, returns a COM or Windows error code.
