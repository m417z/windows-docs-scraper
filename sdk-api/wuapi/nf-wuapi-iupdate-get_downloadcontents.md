# IUpdate::get_DownloadContents

## Description

Gets file information about the download contents of the update.

This property is read-only.

## Parameters

### `retval`

An [IUpdateDownloadContentCollection](https://learn.microsoft.com/windows/win32/api/wuapi/nn-wuapi-iupdatedownloadcontentcollection) that allows you to enumerate the URLs for download contents associated with an update.
