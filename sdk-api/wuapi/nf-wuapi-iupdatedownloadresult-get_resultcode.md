# IUpdateDownloadResult::get_ResultCode

## Description

Gets an [OperationResultCode](https://learn.microsoft.com/windows/desktop/api/wuapi/ne-wuapi-operationresultcode) enumeration value that specifies the result of an operation on the update.

This property is read-only.
