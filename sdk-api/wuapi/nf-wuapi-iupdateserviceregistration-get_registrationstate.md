# IUpdateServiceRegistration::get_RegistrationState

## Description

Gets an [UpdateServiceRegistrationState](https://learn.microsoft.com/windows/desktop/api/wuapi/ne-wuapi-updateserviceregistrationstate) value that indicates the current state of the service registration.

This property is read-only.
