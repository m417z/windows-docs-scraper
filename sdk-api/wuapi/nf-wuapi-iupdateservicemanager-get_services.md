# IUpdateServiceManager::get_Services

## Description

Gets an [IUpdateServiceCollection](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iupdateservicecollection) of the services that are registered with WUA.

This property is read-only.
