# IDownloadJob::get_IsCompleted

## Description

Gets the setting that indicates whether the call to [IUpdateDownloader.BeginDownload](https://learn.microsoft.com/windows/desktop/api/wuapi/nf-wuapi-iupdatedownloader-begindownload) was processed completely.

This property is read-only.
