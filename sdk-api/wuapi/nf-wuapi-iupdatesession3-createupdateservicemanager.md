# IUpdateSession3::CreateUpdateServiceManager

## Description

Returns a pointer to an [IUpdateServiceManager2](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iupdateservicemanager2) interface for the session.

## Parameters

### `retval` [out]

A pointer to an [IUpdateServiceManager2](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iupdateservicemanager2) interface for the session.

## Return value

Returns **S_OK** if successful. Otherwise, returns a COM or Windows error code.
