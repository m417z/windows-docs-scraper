# IInstallationBehavior::get_Impact

## Description

Gets an [InstallationImpact](https://learn.microsoft.com/windows/desktop/api/wuapi/ne-wuapi-installationimpact) enumeration that indicates how the installation or uninstallation of the update affects the computer.

This property is read-only.
