# IUpdateSearcher::put_CanAutomaticallyUpgradeService

## Description

Gets and sets a Boolean value that indicates whether future calls to the [BeginSearch](https://learn.microsoft.com/windows/desktop/api/wuapi/nf-wuapi-iupdatesearcher-beginsearch) and [Search](https://learn.microsoft.com/windows/desktop/api/wuapi/nf-wuapi-iupdatesearcher-search) methods result in an automatic upgrade to Windows Update Agent (WUA). Currently, this property's valid value corresponds to the option that does not automatically upgrade WUA.

This property is read/write.
