# IUpdate::get_Type

## Description

Gets the type of the update.

This property is read-only.

## Parameters

### `retval`

A member of the [UpdateType](https://learn.microsoft.com/windows/win32/api/wuapi/ne-wuapi-updatetype) enumeration specifying the type of the update.
