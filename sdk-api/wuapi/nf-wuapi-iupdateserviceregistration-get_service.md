# IUpdateServiceRegistration::get_Service

## Description

Gets a pointer to an [IUpdateService2](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iupdateservice2) interface. This property is the default property.

This property is read-only.
