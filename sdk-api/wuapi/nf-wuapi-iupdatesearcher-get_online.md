# IUpdateSearcher::get_Online

## Description

Gets and sets a Boolean value that indicates whether the [UpdateSearcher](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iupdatesearcher) goes online to search for updates.

This property is read/write.
