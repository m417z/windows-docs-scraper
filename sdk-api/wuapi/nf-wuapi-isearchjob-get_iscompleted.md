# ISearchJob::get_IsCompleted

## Description

Gets a Boolean value that indicates whether the call to the [IUpdateSearch.BeginSearch](https://learn.microsoft.com/windows/desktop/api/wuapi/nf-wuapi-iupdatesearcher-beginsearch) method is completely processed.

This property is read-only.
