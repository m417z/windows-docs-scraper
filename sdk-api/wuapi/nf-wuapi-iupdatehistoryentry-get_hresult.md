# IUpdateHistoryEntry::get_HResult

## Description

Gets the **HRESULT** value that is returned from the operation on an update.

This property is read-only.

## Parameters

## Remarks

The returned value is a mapped exception code. To retrieve the actual exception code, use the [UnmappedResultCode](https://learn.microsoft.com/windows/desktop/api/wuapi/nf-wuapi-iupdatehistoryentry-get_unmappedresultcode) property.
