# IWindowsDriverUpdate5::get_AutoSelection

## Description

Gets an [AutoSelectionMode](https://learn.microsoft.com/windows/desktop/api/wuapi/ne-wuapi-autoselectionmode) value indicating the automatic selection mode of an update in the Control Panel of Windows Update.

This property is read-only.

## Parameters

## Remarks

The AutoSelection property indicates whether the update will be automatically selected when the user views the available updates in the Windows Update user interface.
