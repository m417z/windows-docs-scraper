# IWindowsDriverUpdate5::get_AutoDownload

## Description

Gets an [AutoDownloadMode](https://learn.microsoft.com/windows/desktop/api/wuapi/ne-wuapi-autodownloadmode) value that indicates the automatic download mode of update.

This property is read-only.

## Parameters

## Remarks

The AutoDownload property indicates whether the update will be automatically downloaded by Automatic Updates.
