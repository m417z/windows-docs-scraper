# IWebProxy::get_ReadOnly

## Description

Gets a Boolean value that indicates whether the [WebProxy](https://learn.microsoft.com/windows/desktop/api/wuapi/nn-wuapi-iwebproxy) object is read-only.

This property is read-only.
