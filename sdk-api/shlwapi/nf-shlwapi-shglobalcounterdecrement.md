# SHGlobalCounterDecrement function

## Description

Decrements a global counter.

## Parameters

### `id` [in]

Type: **const [SHGLOBALCOUNTER](https://learn.microsoft.com/windows/desktop/api/shlwapi/ne-shlwapi-shglobalcounter)**

The [SHGLOBALCOUNTER](https://learn.microsoft.com/windows/desktop/api/shlwapi/ne-shlwapi-shglobalcounter) to decrement.

## Return value

Type: **long**

The value of the counter after the decrement.
