# tagSFBS_FLAGS enumeration

## Description

Specifies how the [StrFormatByteSizeEx](https://learn.microsoft.com/windows/desktop/api/shlwapi/nf-shlwapi-strformatbytesizeex) function should handle rounding of undisplayed digits.

## Constants

### `SFBS_FLAGS_ROUND_TO_NEAREST_DISPLAYED_DIGIT:0x0001`

Round to the nearest displayed digit.

### `SFBS_FLAGS_TRUNCATE_UNDISPLAYED_DECIMAL_DIGITS:0x0002`

Discard undisplayed digits.
