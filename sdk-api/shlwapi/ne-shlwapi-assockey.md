# ASSOCKEY enumeration

## Description

Specifies the type of key to be returned by [IQueryAssociations::GetKey](https://learn.microsoft.com/windows/desktop/api/shlwapi/nf-shlwapi-iqueryassociations-getkey).

## Constants

### `ASSOCKEY_SHELLEXECCLASS:1`

A key that is passed to [ShellExecuteEx](https://learn.microsoft.com/windows/desktop/api/shellapi/nf-shellapi-shellexecuteexa) through a [SHELLEXECUTEINFO](https://learn.microsoft.com/windows/desktop/api/shellapi/ns-shellapi-shellexecuteinfoa) structure.

### `ASSOCKEY_APP`

An **Application** key for the file type.

### `ASSOCKEY_CLASS`

A ProgID or class key.

### `ASSOCKEY_BASECLASS`

A BaseClass value.

### `ASSOCKEY_MAX`
