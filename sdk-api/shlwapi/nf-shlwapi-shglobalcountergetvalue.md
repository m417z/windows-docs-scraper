# SHGlobalCounterGetValue function

## Description

Gets the current value of a global counter.

## Parameters

### `id` [in]

Type: **const [SHGLOBALCOUNTER](https://learn.microsoft.com/windows/desktop/api/shlwapi/ne-shlwapi-shglobalcounter)**

The [SHGLOBALCOUNTER](https://learn.microsoft.com/windows/desktop/api/shlwapi/ne-shlwapi-shglobalcounter) for which to retrieve the current value.

## Return value

Type: **long**

The current value of the counter.
