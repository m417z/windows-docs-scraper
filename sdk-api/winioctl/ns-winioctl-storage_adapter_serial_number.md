# _STORAGE_ADAPTER_SERIAL_NUMBER structure

## Description

The NULL-terminated Unicode string of the adapter serial number for the StorageAdapterSerialNumberProperty as defined in [STORAGE_PROPERTY_ID](https://learn.microsoft.com/windows/win32/api/winioctl/ne-winioctl-storage_property_id).

## Members

### `Version`

The version of this structure. The Size serves as the version.

### `Size`

The size of this structure.

### `SerialNumber`

The serial number.
