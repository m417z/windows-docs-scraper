# PREVENT_MEDIA_REMOVAL structure

## Description

Provides removable media locking data. It is used by the
[IOCTL_STORAGE_MEDIA_REMOVAL](https://learn.microsoft.com/windows/desktop/api/winioctl/ni-winioctl-ioctl_storage_media_removal) control code.

## Members

### `PreventMediaRemoval`

If this member is **TRUE**, the media is to be locked. Otherwise, it is not.
