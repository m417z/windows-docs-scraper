# GET_LENGTH_INFORMATION structure

## Description

Contains disk, volume, or partition length information used by the
[IOCTL_DISK_GET_LENGTH_INFO](https://learn.microsoft.com/windows/desktop/api/winioctl/ni-winioctl-ioctl_disk_get_length_info) control code.

## Members

### `Length`

The length of the disk, volume, or partition, in bytes.
