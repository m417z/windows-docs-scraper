# RETRIEVAL_POINTER_BASE structure

## Description

Contains the output for the [FSCTL_GET_RETRIEVAL_POINTER_BASE](https://learn.microsoft.com/windows/desktop/api/winioctl/ni-winioctl-fsctl_get_retrieval_pointer_base) control code.

## Members

### `FileAreaOffset`

The volume-relative sector offset to the first allocatable unit on the file system, also referred to as the base of the cluster heap.
