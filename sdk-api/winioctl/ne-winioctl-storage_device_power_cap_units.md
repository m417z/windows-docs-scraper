# STORAGE_DEVICE_POWER_CAP_UNITS enumeration

## Description

The units of the maximum power threshold.

## Constants

### `StorageDevicePowerCapUnitsPercent`

Units in percent.

### `StorageDevicePowerCapUnitsMilliwatts`

Units in milliwatts.
