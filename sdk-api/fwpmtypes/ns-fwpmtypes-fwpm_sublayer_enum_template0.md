# FWPM_SUBLAYER_ENUM_TEMPLATE0 structure

## Description

The **FWPM_SUBLAYER_ENUM_TEMPLATE0** structure is used for enumerating sublayers.

## Members

### `providerKey`

Uniquely identifies the provider associated with this sublayer. If this value is non-NULL, only options with the specifies provider will be returned.

## Remarks

**FWPM_SUBLAYER_ENUM_TEMPLATE0** is a specific implementation of FWPM_SUBLAYER_ENUM_TEMPLATE. See [WFP Version-Independent Names and Targeting Specific Versions of Windows](https://learn.microsoft.com/windows/desktop/FWP/wfp-version-independent-names-and-targeting-specific-versions-of-windows) for more information.
