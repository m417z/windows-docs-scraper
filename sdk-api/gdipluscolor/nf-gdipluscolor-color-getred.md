# Color::GetRed

## Description

The **Color::GetRed** method gets the red component of this [Color](https://learn.microsoft.com/windows/desktop/api/gdipluscolor/nl-gdipluscolor-color) object.

## Return value

Type: **BYTE**

This method returns the red component of this color.
