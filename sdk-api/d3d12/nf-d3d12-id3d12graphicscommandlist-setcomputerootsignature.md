# ID3D12GraphicsCommandList::SetComputeRootSignature

## Description

Sets the layout of the compute root signature.

## Parameters

### `pRootSignature` [in, optional]

Type: **[ID3D12RootSignature](https://learn.microsoft.com/windows/desktop/api/d3d12/nn-d3d12-id3d12rootsignature)***

A pointer to the [ID3D12RootSignature](https://learn.microsoft.com/windows/desktop/api/d3d12/nn-d3d12-id3d12rootsignature) object.
