# D3D12_FEATURE_DATA_ROOT_SIGNATURE structure

## Description

Indicates root signature version support.

## Members

### `HighestVersion`

On input, specifies the highest version [D3D_ROOT_SIGNATURE_VERSION](https://learn.microsoft.com/windows/desktop/api/d3d12/ne-d3d12-d3d_root_signature_version) to check for. On output specifies the highest version, up to the input version specified, actually available.
