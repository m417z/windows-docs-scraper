# D3D_ROOT_SIGNATURE_VERSION enumeration

## Description

Specifies the version of root signature layout.

## Constants

### `D3D_ROOT_SIGNATURE_VERSION_1:0x1`

Version one of root signature layout.

### `D3D_ROOT_SIGNATURE_VERSION_1_0:0x1`

Version one of root signature layout.

### `D3D_ROOT_SIGNATURE_VERSION_1_1:0x2`

Version 1.1 of root signature layout. Refer to [Root Signature Version 1.1](https://learn.microsoft.com/windows/desktop/direct3d12/root-signature-version-1-1).
