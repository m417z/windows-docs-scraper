# ID3D12PipelineState::GetCachedBlob

## Description

Gets the cached blob representing the pipeline state.

## Parameters

### `ppBlob` [out]

Type: **ID3DBlob****

After this method returns, points to the cached blob representing the pipeline state.

## Return value

Type: **[HRESULT](https://learn.microsoft.com/windows/win32/com/structure-of-com-error-codes)**

This method returns one of the [Direct3D 12 Return Codes](https://learn.microsoft.com/windows/desktop/direct3d12/d3d12-graphics-reference-returnvalues).

## Remarks

Refer to the remarks for [D3D12_CACHED_PIPELINE_STATE](https://learn.microsoft.com/windows/desktop/api/d3d12/ns-d3d12-d3d12_cached_pipeline_state).
