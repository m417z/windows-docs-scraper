# IFsrmQuotaObject::ApplyTemplate

## Description

[This method is supported for compatibility but it's recommended to use the
[FSRM WMI Classes](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/fsrm-wmi-classes) to manage FSRM. Please see the
[MSFT_FSRMQuota](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/msft-fsrmquota) class.]

Applies the property values of the specified quota template to this quota object.

## Parameters

### `quotaTemplateName` [in]

The name of the quota template. The string is limited to 4,000 characters.

## Return value

The method returns the following return values.
