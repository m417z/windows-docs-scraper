# IFsrmQuotaTemplate::put_Name

## Description

[This property is supported for compatibility but it's recommended to use the
[FSRM WMI Classes](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/fsrm-wmi-classes) to manage FSRM. Please see the
[MSFT_FSRMQuota](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/msft-fsrmquota) class.]

Retrieves and sets the name of the quota template.

This property is read/write.

## Parameters

## Remarks

If you do not specify a name, FSRM generates a unique name that begins with
"QuotaTemplate".

#### Examples

For an example, see [Using Templates to Define Quotas](https://learn.microsoft.com/previous-versions/windows/desktop/fsrm/using-templates-to-define-quotas).
