# INetFwRule3::get_SecureFlags

## Description

Specifies which firewall verifications of security levels provided by IPsec must be guaranteed to allow the collection. The allowed values must correspond to those of the [NET_FW_AUTHENTICATE_TYPE](https://learn.microsoft.com/windows/win32/api/icftypes/ne-icftypes-net_fw_authenticate_type) enumeration.

This property is read/write.
