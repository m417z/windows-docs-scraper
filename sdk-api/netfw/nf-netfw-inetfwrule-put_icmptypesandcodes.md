# INetFwRule::put_IcmpTypesAndCodes

## Description

Specifies the list of ICMP types and codes for this rule.

This property is read/write.

## Parameters

## Remarks

This property is optional.

Also see the restrictions on changing properties described in the Remarks section of the [INetFwRule](https://learn.microsoft.com/previous-versions/windows/desktop/api/netfw/nn-netfw-inetfwrule) interface page.

The *icmpTypesAndCodes* parameter is a list of ICMP types and codes separated by semicolon. "*" indicates all ICMP types and codes.
